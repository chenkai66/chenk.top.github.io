company-skill.com is the visual console: Knowledge Base browses distilled Skills, Ingest feeds new docs, Workspaces switch products, Routing Playground tests intent decisions, Telemetry runs adversarial recall evals.

Built for any company — distill an entire organization into one Skill that agents call directly. No human writes a single line of Skill content.{{ end }}
