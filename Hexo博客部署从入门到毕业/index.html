域名购买流程

开启自定义域名的HTTPS服务

现在我们可以通过你购买的域名进入你的网站啦！这是历史性的一刻，但还是有一点点缺憾，你有发现在进入你网站时左上角的感叹号吗？这是提醒你现在的博客是不安全的访问链接，也就是你的HTTP还没有经过加密。这会导致很多人实际上打不开你的博客，比如我们从微信直接分享的博客链接是打不开的，那么我们就需要将其做，这里推荐的网站是 https://app.netlify.com/。Netlify 和 GitHub pages 的作用一样，是托管静态代码的服务器。但比起Github，Netlify的速度更快，因为它的服务器更多。一般而言我们直接遵循网站内的提示即可完成所有操作，并没有什么坑点。

抛开链接 Github 等操作不讲，我们第一步是按照提示完成 Domains 部分信息的填写：

在Domains信息填完之后，实际上有一个 Check DNS configuration 的提示，这里你需要讲提示的四个name servers在你购买域名的网站进行修改。以阿里云为例：

  1. 登录域名控制台。

  2. 前往域名列表页签，单击待修改域名操作列下的管理。

  3. 前往 DNS修改页签，单击修改 DNS 服务器。

  4. 在 DNS 修改页面，输入新的 DNS 服务器，然后单击确认。

基本上修改完 DNS 服务器之后另一边的 Netlify 的 Domains 界面那个黄色感叹号就消失了，变成前面的绿色勾勾，过一会可以点击 HTTPS 那个标签下的 update，再等个半小时就完成了 HTTP 到 HTTPS 的转换！
