PINN实验代码说明

本目录包含"物理信息神经网络：从Ritz方法到现代求解器"一文的4个完整实验代码。

环境要求

  • Python 3.7+
  • PyTorch 1.8+
  • NumPy
  • Matplotlib
  • SciPy（实验3需要）

实验列表

实验1：一维热方程 (experiment1_heat_equation.py)

问题：求解一维热方程 \(\frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}\)

运行：

python experiment1_heat_equation.py

输出： - experiment1_results.png: 训练损失、预测解对比、误差分布 - experiment1_3d.png: 3D可视化

实验2：二维泊松方程 (experiment2_poisson.py)

问题：求解L形区域上的Poisson方程 \(-\Delta u = f\)

运行：

python experiment2_poisson.py

输出： - experiment2_results.png: 训练损失、预测解、L形区域示意图 - experiment2_3d.png: 3D可视化

实验3：Burgers方程 (experiment3_burgers.py)

问题：求解Burgers方程 \(\frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = \nu \frac{\partial^2 u}{\partial x^2}\)，包含自适应采样

运行：

python experiment3_burgers.py

输出： - experiment3_results.png: 训练损失、不同时刻的解、激波演化

实验4：激活函数对比 (experiment4_activation.py)

问题：对比Tanh、Sine、Swish、GELU四种激活函数在Poisson方程上的性能

运行：

python experiment4_activation.py

输出： - experiment4_results.png: 四种激活函数的训练曲线和预测对比 - experiment4_table.png: 性能对比表格

注意事项

  1. GPU加速：代码会自动检测并使用GPU（如果可用），可显著加速训练。

  2. 训练时间：每个实验的训练时间取决于硬件配置：

    • CPU: 10-30分钟
    • GPU: 2-10分钟

  3. 随机性：由于随机采样和初始化，每次运行的结果可能略有不同。代码已设置随机种子以确保可复现性。

  4. 内存要求：建议至少4GB内存。如果内存不足，可以减少采样点数（N_r, N_b, N_i）。

参数调整

所有实验的关键参数都可以在代码中调整：

  • 网络架构：layers 参数（如 [2, 50, 50, 50, 1]）
  • 采样点数：N_r（PDE残差点）、N_b（边界点）、N_i（初始条件点）
  • 学习率：optimizer 中的 lr 参数
  • 训练轮数：range(epochs) 中的 epochs
