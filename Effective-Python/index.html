Formatted String

Using the % operator to stuff values into a C-style format string has many problems and is cumbersome to write. Just like:

Actually, there are three main troubles using the C-style format string: 1. If the values in the tuple on the right of % change in type or order, the program may fail due to incompatibility when converting types; 2. Before filling in the template, it is often necessary to do a little processing of the value to be filled in. But in this way, the whole expression may be written very long, which makes people feel more confusing; 3. If you want to fill multiple positions in the format string with the same value, you must repeat the value correspondingly multiple times in the tuple to the right of the % operator.
