Part 6: VRM (Voltage Regulator Module) Deep Dive

What is VRM?

VRM = Voltage Regulator Module

Function: Converts 12V PSU power to CPU-required 1.0-1.4V

Analogy: - VRM = Transformer (converts 220V mains to 5V phone charger) - More phases = More transformers (more stable power)

VRM Phase Count Reference

CPU TDP Recommended VRM Motherboard Tier Price
< 65W (i5-13400) 6+2 phase Entry B760 $120
125W (i5-13600K) 10+2 phase Mid-range B760 $140
150W (i7-13700K) 14+2 phase High-end Z790 $200
250W+ (i9-13900K OC) 18+2 phase+ Flagship Z790 $280+

Dangerous example: > ‚ùå i9-13900K overclocked (300W) + Entry B760 (8 phase) > > Result: VRM overheats ‚Üí throttling ‚Üí possibly burn motherboard!
