  • Prefix Tuning —— Optimizing Continuous Prompts for Generation

    Full fine-tuning is effective, but expensive: each task typically requires storing an entire copy of the model’s updated weights. Prefix-Tuning takes a more minimal approach—freeze the pretrained language model, and learn a small set of continuous “prefix” vectors that steer generation by injecting task-specific context into the Transformer’s attention computation. This note focuses on what is actually being optimized (token-level prefix vs. per-layer key/value prefixes), why reparameterization via an MLP helps training stability, and how Prefix-Tuning compares to adapters and prompt-based methods in both quality and parameter/storage efficiency.

  • Session-based Recommendation with Graph Neural Networks (SR-GNN)

    Session-based recommendation is challenging when you only observe a short click sequence and have little or no long-term user profile. SR-GNN tackles this by turning each session into a directed graph, where repeated items and multi-step transitions form richer structure than a plain sequence. A gated GNN propagates information over this session graph to learn item representations, and the model then aggregates them into a session representation to score next-item candidates. This note explains the session-graph construction, the gated message passing update, and how SR-GNN produces the final ranking—highlighting why this graph view often outperforms purely sequential baselines on standard SBR benchmarks.

  • Graph Contextualized Self-Attention Network for Session-based Recommendation

    GC-SAN is a session-based recommendation model that combines two complementary views of a session: a graph view that captures local item-transition structure, and a self-attention view that captures long-range dependencies and global intent within the same session. The core idea is to first encode the session as a graph and learn item representations with a GNN, then use stacked self-attention layers to model global interactions across clicked items. Finally, GC-SAN fuses local and global signals into a session representation to predict the next click. This note focuses on the main architectural choices, how the fusion is done, and why this hybrid tends to outperform pure RNN/GNN baselines on real-world datasets.

  • HCGR —— Hyperbolic Contrastive Graph Representation Learning for Session-based Recommendation

    HCGR is a session-based recommendation method motivated by a simple mismatch: user interests often have hierarchical structure (coarse intent → fine-grained choices), while standard Euclidean embeddings tend to represent similarity in a “flat” way. The paper moves the representation space to hyperbolic geometry (Lorentz model), where tree-like hierarchies can be represented more naturally, and combines it with graph learning over sessions. On top of that, HCGR introduces a hyperbolic attention mechanism and a contrastive objective to stabilize and enrich the learned representations. This note focuses on the intuition (why hyperbolic), the key building blocks (Lorentz operations, attention, contrastive pairs), and what gains the paper reports on public benchmarks.
