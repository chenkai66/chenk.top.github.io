2. Positive definite kernels and RKHS

A kernel is (symmetric) positive definite if the Gram matrix (K_{ij}=K(x_i,x_j)) is PSD for any finite set ({x_i}). Such kernels correspond to an RKHS where evaluation is continuous and the reproducing property holds.
