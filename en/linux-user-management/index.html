Linux is multi-user by design, so user and group management directly affects security, access control, and day-to-day operations. This note covers the core commands, important system files, and common admin workflows.

1. Key files

  • /etc/passwd: user entries
  • /etc/shadow: password hashes (root-only)
  • /etc/group: group entries
  • /etc/sudoers and /etc/sudoers.d/: sudo policy
