дёҖгҖҒдёәд»Җд№Ҳ SVD й…Қеҫ—дёҠвҖңзҡҮеҶ вҖқдәҢеӯ—

з¬¬ 8 з« зҡ„и°ұе®ҡзҗҶз»ҷеҮәдәҶ $A = Q\Lambda Q^{\!\top}$пјҢз®ҖжҙҒдјҳзҫҺпјҢдҪҶжңүдёӘзЎ¬жҖ§йҷҗеҲ¶пјҡ$A$ еҝ…йЎ»жҳҜеҜ№з§°зҹ©йҳөгҖӮзҺ°е®һдёӯзҡ„зҹ©йҳөеӨ§еӨҡдёҚеҜ№з§°пјҢз”ҡиҮідёҚжҳҜж–№йҳөпјҡ

  • дёҖеј  $1920 \times 1080$ зҡ„еӣҫзүҮпјӣ
  • з”ЁжҲ·вҖ”з”өеҪұиҜ„еҲҶзҹ©йҳөпјҲзҷҫдёҮиЎҢпјҢж•°еҚғеҲ—пјүпјӣ
  • ж–ҮжЎЈвҖ”иҜҚйЎ№зҹ©йҳөпјҲNLP дёӯж–ҮжЎЈж•° Г— иҜҚжұҮйҮҸпјүпјӣ
  • еҹәеӣ иЎЁиҫҫзҹ©йҳөпјҲз”ҹзү©дҝЎжҒҜеӯҰпјүгҖӮ
$$A = U\,\Sigma\,V^{\!\top}.$$

иҝҷжҳҜзәҝжҖ§д»Јж•°йҮҢжңҖејәеӨ§гҖҒжңҖйҖҡз”Ёзҡ„еҲҶи§Јж–№жі•гҖӮ

дёҖдёӘж‘„еҪұзұ»жҜ”

жҠҠз…§зүҮзңӢжҲҗеғҸзҙ зҹ©йҳөпјҢSVD е‘ҠиҜүжҲ‘дёүд»¶дәӢпјҡ

  1. д»»дҪ•з…§зүҮйғҪеҸҜд»ҘжӢҶи§ЈдёәиӢҘе№ІвҖңеҹәзЎҖеұӮвҖқзҡ„еҸ еҠ гҖӮ
  2. иҝҷдәӣеұӮжҢүйҮҚиҰҒжҖ§жҺ’еәҸпјҡз¬¬дёҖеұӮжҠ“дё»з»“жһ„пјҢз¬¬дәҢеұӮжҠ“ж¬ЎиҰҒз»ҶиҠӮпјҢз¬¬дёүеұӮжҠ“жӣҙзІҫз»Ҷзҡ„йғЁеҲҶгҖӮ
  3. еҸӘдҝқз•ҷеүҚеҮ еұӮе°ұиғҪиҝҳеҺҹеӨ§йғЁеҲҶеӣҫеғҸдҝЎжҒҜгҖӮ

иҝҷе°ұеғҸд№җйҳҹеҪ•йҹіз”ұдё»е”ұгҖҒеҗүд»–гҖҒиҙқж–ҜгҖҒйј“еҸ еҠ иҖҢжҲҗпјҡеҺ»жҺүиғҢжҷҜе’ҢеЈ°пјҢжӯҢжӣІиҝҳиғҪеҗ¬пјӣеҺ»жҺүдё»е”ұпјҢж•ҙйҰ–жӯҢе°ұеһ®дәҶгҖӮSVD жҠҠиҝҷз§Қзӣҙи§үйҮҸеҢ–дәҶпјҡеҘҮејӮеҖјзІҫзЎ®иЎЎйҮҸдәҶжҜҸдёҖеұӮзҡ„иҙЎзҢ®гҖӮ

жң¬з« еӯҰд№ зӣ®ж Ү

  • жҺҢжҸЎ SVD зҡ„е®ҡд№үеҸҠе…¶дёүжӯҘеҮ дҪ•ж„Ҹд№үпјҡж—ӢиҪ¬ вҶ’ жӢүдјё вҶ’ ж—ӢиҪ¬гҖӮ
  • еӯҰдјҡйҖҡиҝҮ $A^{\!\top}\!A$ е’Ң $AA^{\!\top}$ и®Ўз®—еҘҮејӮеҖје’ҢеҘҮејӮеҗ‘йҮҸгҖӮ
  • д»Һ $U$ е’Ң $V$ зӣҙжҺҘиҜ»еҮәеӣӣдёӘеҹәжң¬еӯҗз©әй—ҙгҖӮ
  • зҗҶи§ЈдҪҺз§©йҖјиҝ‘дёҺ EckartвҖ“Young е®ҡзҗҶвҖ”вҖ”ж•°жҚ®еҺӢзј©иғҢеҗҺзҡ„жңҖдјҳжҖ§еҺҹзҗҶгҖӮ
  • жҺҢжҸЎдјӘйҖҶпјҡдёҖз§ҚйҖӮз”ЁдәҺдёҚеҸҜйҖҶжҲ–йқһж–№йҳөзҡ„вҖңжңҖдҪійҖҶвҖқгҖӮ
  • еҸ‘зҺ° PCA е…¶е®һе°ұжҳҜ SVD зҡ„еҸҰдёҖз§ҚеҪўејҸгҖӮ
  • еә”з”ЁйўҶеҹҹпјҡеӣҫеғҸеҺӢзј©гҖҒжҺЁиҚҗзі»з»ҹгҖҒжҪңеңЁиҜӯд№үеҲҶжһҗгҖҒдҝЎеҸ·еҺ»еҷӘгҖҒзү№еҫҒи„ёгҖӮ

еүҚзҪ®зҹҘиҜҶ

  • зү№еҫҒеҖјдёҺзү№еҫҒеҗ‘йҮҸпјҲз¬¬ 6 з« пјү
  • жӯЈдәӨзҹ©йҳөдёҺжҠ•еҪұпјҲз¬¬ 7 з« пјү
  • еҜ№з§°зҹ©йҳөдёҺи°ұе®ҡзҗҶпјҲз¬¬ 8 з« пјү

дәҢгҖҒSVD зҡ„е®ҡд№ү

еҹәжң¬е®ҡзҗҶ

$$A = U\,\Sigma\,V^{\!\top}$$

е…¶дёӯпјҡ

  • $U \in \mathbb{R}^{m\times m}$ жҳҜжӯЈдәӨзҹ©йҳөпјҢеҲ—еҗ‘йҮҸжҳҜе·ҰеҘҮејӮеҗ‘йҮҸ $u_1, \ldots, u_m$пјӣ
  • $V \in \mathbb{R}^{n\times n}$ жҳҜжӯЈдәӨзҹ©йҳөпјҢеҲ—еҗ‘йҮҸжҳҜеҸіеҘҮејӮеҗ‘йҮҸ $v_1, \ldots, v_n$пјӣ
  • $\Sigma \in \mathbb{R}^{m\times n}$ жҳҜеҜ№и§’зҹ©йҳөпјҢдё»еҜ№и§’зәҝдёҠжҳҜеҘҮејӮеҖј $\sigma_1 \ge \sigma_2 \ge \cdots \ge 0$пјҢе…¶дҪҷдҪҚзҪ®е…Ёдёәйӣ¶гҖӮ
$$A = U_r\,\Sigma_r\,V_r^{\!\top},\qquad U_r\in\mathbb{R}^{m\times r},\ \Sigma_r\in\mathbb{R}^{r\times r},\ V_r\in\mathbb{R}^{n\times r}.$$

еҘҮејӮеҖјжңүдёүдёӘйҮҚиҰҒзү№жҖ§пјҡ

  • е®ғд»¬ж°ёиҝңжҳҜйқһиҙҹе®һж•°пјҲзү№еҫҒеҖјеҸҜиғҪжҳҜиҙҹж•°жҲ–еӨҚж•°пјүгҖӮ
  • жҢүжғҜдҫӢд»ҺеӨ§еҲ°е°ҸжҺ’еҲ—гҖӮ
  • д»»дҪ•зҹ©йҳөйғҪжңү SVDгҖӮиҝҷжҳҜзү№еҫҒеҲҶи§ЈеҒҡдёҚеҲ°зҡ„пјҢд№ҹжҳҜ SVD зҡ„ж ёеҝғдјҳеҠҝгҖӮ

еҮ дҪ•ж„Ҹд№үпјҡдёүжӯҘе®ҢжҲҗеҸҳжҚў

еҲҶи§Ј $A = U\Sigma V^{\!\top}$ еҸҜд»ҘзңӢдҪңдёҖдёӘдёүжӯҘиҝҮзЁӢпјҢд»ҺеҸіеҫҖе·Ұдҫқж¬ЎдҪңз”Ёпјҡ

  1. ж—ӢиҪ¬ $V^{\!\top}$пјҡжҠҠиҫ“е…ҘеҜ№йҪҗеҲ° $A$ зҡ„иҮӘз„¶ж–№еҗ‘гҖӮ
  2. жӢүдјё $\Sigma$пјҡз¬¬ $i$ дёӘеқҗж Үд№ҳд»Ҙ $\sigma_i$гҖӮ
  3. ж—ӢиҪ¬ $U$пјҡжҠҠз»“жһңиҪ¬еҲ°иҫ“еҮәз©әй—ҙзҡ„жңҖз»ҲдҪҚзҪ®гҖӮ

з”ЁжҸүйқўеӣўжқҘжҜ”е–»пјҡе…ҲжҠҠйқўеӣўиҪ¬еҲ°еҗҲйҖӮзҡ„и§’еәҰпјҲ$V^{\!\top}$пјүпјҢеҶҚз”Ёж“Җйқўжқ–еҺӢжүҒжӢүй•ҝпјҲ$\Sigma$пјүпјҢжңҖеҗҺжҠҠж“ҖеҘҪзҡ„йқўйҘјж‘ҶеҲ°йңҖиҰҒзҡ„ж–№еҗ‘пјҲ$U$пјүгҖӮ

еӣҫдёӯи·ҹиёӘдәҶдёҖдёӘеҚ•дҪҚеңҶзҡ„еҸҳеҢ–иҝҮзЁӢгҖӮдёӨж¬Ўж—ӢиҪ¬дҝқжҢҒдәҶжӯЈдәӨеҹәзҡ„жҖ§иҙЁпјҢеҸӘжңүдёӯй—ҙзҡ„еҜ№и§’йҳө $\Sigma$ ж”№еҸҳдәҶеҪўзҠ¶пјҢжҠҠеңҶеҸҳжҲҗдәҶеҚҠиҪҙй•ҝеәҰдёә $\sigma_1, \sigma_2$ зҡ„жӨӯеңҶгҖӮ

еҚ•дҪҚеңҶеҸҳжӨӯеңҶ

иҝҳеҸҜд»Ҙз”ЁдёҖе№…еӣҫжө“зј©ж•ҙдёӘиҝҮзЁӢпјҡе·Ұиҫ№жҳҜеҚ•дҪҚеңҶпјҢеҸіиҫ№жҳҜ $A$ дҪңз”ЁеҗҺзҡ„жӨӯеңҶгҖӮ

зңӢеӣҫиҰҒзӮ№пјҡ

  • еҸіеҘҮејӮеҗ‘йҮҸ $v_1, v_2$пјҡиҫ“е…Ҙдҫ§зҡ„жӯЈдәӨж–№еҗ‘пјҢиў«жҳ е°„еҲ°жӨӯеңҶзҡ„дё»иҪҙдёҠгҖӮ
  • е·ҰеҘҮејӮеҗ‘йҮҸ $u_1, u_2$пјҡиҫ“еҮәдҫ§дё»иҪҙзҡ„жӯЈдәӨж–№еҗ‘гҖӮ
  • еҘҮејӮеҖј $\sigma_1, \sigma_2$пјҡжӨӯеңҶзҡ„еҚҠиҪҙй•ҝеәҰгҖӮ
$$A\,v_i \;=\; \sigma_i\, u_i.$$

иҝҷе°ұжҳҜ SVD зҡ„ж ёеҝғе…¬ејҸпјҢжүҖжңүз»“и®әйғҪд»ҺиҝҷйҮҢеұ•ејҖгҖӮ

еӨ–з§Ҝеұ•ејҖ

$$A = \sigma_1 u_1 v_1^{\!\top} + \sigma_2 u_2 v_2^{\!\top} + \cdots + \sigma_r u_r v_r^{\!\top}.$$

жҜҸдёӘ $u_i v_i^{\!\top}$ жҳҜз§©дёә 1 зҡ„зҹ©йҳөпјҢеҘҮејӮеҖјжҳҜжқғйҮҚгҖӮиҝҷз§Қи§Ҷи§’жҳҜдҪҺз§©йҖјиҝ‘зҡ„е…ій”®пјҡдҝқз•ҷжқғйҮҚеӨ§зҡ„йЎ№пјҢиҲҚејғжқғйҮҚе°Ҹзҡ„йЎ№гҖӮ

дёүгҖҒSVD зҡ„и®Ўз®—

жЎҘжўҒпјҡ$A^{\!\top}\!A$ е’Ң $AA^{\!\top}$

$$A^{\!\top}\!A = V\,\Sigma^{\!\top}\!\Sigma\,V^{\!\top}, \qquad AA^{\!\top} = U\,\Sigma\Sigma^{\!\top}\,U^{\!\top}.$$

$A^{\!\top}\!A$ е’Ң $AA^{\!\top}$ йғҪжҳҜеҜ№з§°еҚҠжӯЈе®ҡзҹ©йҳөпјҢеӣ жӯӨеҸҜд»Ҙз”Ёи°ұе®ҡзҗҶеҲҶжһҗгҖӮд»Һи°ұеҲҶи§Јзҡ„и§’еәҰзңӢпјҡ

  • $V$ зҡ„еҲ—жҳҜ $A^{\!\top}\!A$ зҡ„жӯЈдәӨзү№еҫҒеҗ‘йҮҸпјҲз§°дёәеҸіеҘҮејӮеҗ‘йҮҸпјүгҖӮ
  • $U$ зҡ„еҲ—жҳҜ $AA^{\!\top}$ зҡ„жӯЈдәӨзү№еҫҒеҗ‘йҮҸпјҲз§°дёәе·ҰеҘҮејӮеҗ‘йҮҸпјүгҖӮ
  • $\sigma_i = \sqrt{\lambda_i}$пјҢе…¶дёӯ $\lambda_i$ жҳҜиҝҷдёӨдёӘзҹ©йҳөе…ұжңүзҡ„йқһиҙҹзү№еҫҒеҖјгҖӮ

дёәд»Җд№Ҳз”Ё $A^{\!\top}\!A$пјҹ еҸҜд»ҘзҗҶи§Јдёә $A$ “дҪңз”ЁдёӨж¬Ў”пјҡе…ҲйҖҡиҝҮ $A$ еүҚиҝӣпјҢеҶҚйҖҡиҝҮ $A^{\!\top}$ еӣһйҖҖгҖӮиҝҷдёҖжқҘеӣһдјҡи®©жҹҗдёӘж–№еҗ‘иў«ж”ҫеӨ§ $\sigma^2$ еҖҚпјҢжүҖд»Ҙ $A^{\!\top}\!A$ зҡ„зү№еҫҒеҖјжӯЈеҘҪжҳҜеҘҮејӮеҖјзҡ„е№іж–№гҖӮ

и®Ўз®—жӯҘйӘӨ

еҒҮи®ҫ $A$ жҳҜ $m \times n$ зҹ©йҳөпјҢдё” $m \ge n$пјҡ

  1. жһ„йҖ  $A^{\!\top}\!A$пјҢи®Ўз®—е…¶зү№еҫҒеҖј $\lambda_1 \ge \cdots \ge \lambda_n \ge 0$ е’ҢеҜ№еә”зҡ„жӯЈдәӨеҚ•дҪҚзү№еҫҒеҗ‘йҮҸ $v_1, \ldots, v_n$гҖӮиҝҷдәӣеҗ‘йҮҸз»„жҲҗ $V$ зҡ„еҲ—гҖӮ
  2. и®ҫ $\sigma_i = \sqrt{\lambda_i}$гҖӮ
  3. еҜ№жҜҸдёӘ $\sigma_i > 0$пјҢе®ҡд№ү $u_i = A v_i / \sigma_i$гҖӮиҝҷдәӣеҗ‘йҮҸжһ„жҲҗ $U$ зҡ„еүҚ $r$ еҲ—гҖӮ
  4. еҰӮжһң $r < m$пјҢз”Ё GramвҖ“Schmidt ж–№жі•е°Ҷ $\{u_1, \ldots, u_r\}$ жү©еұ•дёә $\mathbb{R}^m$ зҡ„жӯЈдәӨеҹәпјҢиЎҘйҪҗ $U$гҖӮ

е®һйҷ…ж•°еҖји®Ўз®—дёӯпјҢжІЎдәәдјҡиҝҷд№Ҳз®— вҖ”вҖ” жҳҫејҸжһ„йҖ  $A^{\!\top}\!A$ дјҡи®©жқЎд»¶ж•°е№іж–№еҢ–пјҢеҜјиҮҙзІҫеәҰжҚҹеӨұгҖӮз”ҹдә§д»Јз ҒйҖҡеёёдҪҝз”ЁеҸҢеҜ№и§’еҢ–з»“еҗҲ QR з®—жі•жҲ–еҲҶжІ»жі•гҖӮдёҠйқўзҡ„жҺЁеҜјеҸӘжҳҜдёәдәҶиҜҙжҳҺеҺҹзҗҶгҖӮ

дёҖдёӘжүӢз®—дҫӢеӯҗ

$$A^{\!\top}\!A = \begin{pmatrix} 1 & 1 \\ 1 & 2 \end{pmatrix}, \qquad \det(A^{\!\top}\!A - \lambda I) = \lambda^2 - 3\lambda + 1.$$$$\sigma_1 = \sqrt{\tfrac{3+\sqrt 5}{2}} \approx 1.618, \qquad \sigma_2 = \sqrt{\tfrac{3-\sqrt 5}{2}} \approx 0.618.$$

жұӮеҮә $A^{\!\top}\!A$ зҡ„зү№еҫҒеҗ‘йҮҸеҫ—еҲ° $V$пјҢеҶҚйҖҡиҝҮ $u_i = A v_i / \sigma_i$ еҫ—еҲ° $U$гҖӮйӘҢиҜҒдёҖдёӢпјҡ$\sigma_1 \sigma_2 = 1 = |\det A|$пјҢз»“жһңиҮӘжҙҪгҖӮ

зү№еҫҒеҖј vs еҘҮејӮеҖјпјҡзӣҙи§ӮеҜ№жҜ”

еҜ№з§°зҹ©йҳөзҡ„зү№еҫҒеҖје’ҢеҘҮејӮеҖјпјҲжңҖеӨҡе·®жӯЈиҙҹеҸ·пјүжҳҜдёҖеӣһдәӢгҖӮдҪҶеҜ№дёҖиҲ¬зҡ„ $A$пјҢдёӨиҖ…е®Ңе…ЁдёҚеҗҢпјҢеҜ№жҜ”иө·жқҘеҫҲжңүеҗҜеҸ‘жҖ§гҖӮ

зү№еҫҒеҗ‘йҮҸеҘҮејӮеҗ‘йҮҸ
е®ҡд№ү$A x = \lambda x$$A v = \sigma u$пјҢдё” $u, v$ еҗ„иҮӘжӯЈдәӨ
жӯЈдәӨжҖ§дёҖиҲ¬дёҚжӯЈдәӨ$V$ е’Ң $U$ йғҪжҳҜжӯЈдәӨзҹ©йҳө
еҸ–еҖј$\lambda_i \in \mathbb{C}$$\sigma_i \in \mathbb{R}_{\ge 0}$
еҮ дҪ•еҗ«д№үдёҚеҸҳж–№еҗ‘пјҢжҢү $\lambda$ зј©ж”ҫиҫ“е…ҘжӯЈдәӨеҹәжҳ е°„еҲ°иҫ“еҮәжӯЈдәӨеҹәпјҢжҢү $\sigma$ зј©ж”ҫ
еӯҳеңЁжҖ§ж–№йҳөпјҢдё”жңӘеҝ…еҸҜеҜ№и§’еҢ–д»»ж„Ҹзҹ©йҳөйғҪеӯҳеңЁ

еӣҫдёӯеҗҢдёҖдёӘйқһеҜ№з§° $A$ зҡ„зү№еҫҒеҗ‘йҮҸе‘Ҳж–ңи§’пјҢиҖҢ SVD еңЁиҫ“е…Ҙе’Ңиҫ“еҮәдёӨдҫ§йғҪз»ҷеҮәдәҶеһӮзӣҙзҡ„ж–№еҗ‘гҖӮ

SVD дёҺеӣӣдёӘеҹәжң¬еӯҗз©әй—ҙ

SVD иғҪжңҖжё…жҷ°ең°жҸӯзӨәзҹ©йҳөзҡ„з»“жһ„гҖӮеҒҮи®ҫ $A \in \mathbb{R}^{m \times n}$пјҢз§©дёә $r$пјҡ

еӯҗз©әй—ҙз»ҙеәҰSVD жҸҗдҫӣзҡ„жӯЈдәӨеҹәжүҖеңЁз©әй—ҙ
иЎҢз©әй—ҙ $\mathcal{C}(A^{\!\top})$$r$$v_1, \ldots, v_r$$\mathbb{R}^n$
йӣ¶з©әй—ҙ $\mathcal{N}(A)$$n - r$$v_{r+1}, \ldots, v_n$$\mathbb{R}^n$
еҲ—з©әй—ҙ $\mathcal{C}(A)$$r$$u_1, \ldots, u_r$$\mathbb{R}^m$
е·Ұйӣ¶з©әй—ҙ $\mathcal{N}(A^{\!\top})$$m - r$$u_{r+1}, \ldots, u_m$$\mathbb{R}^m$

иЎҢз©әй—ҙзҡ„жӯЈдәӨеҹә $\{v_1, \ldots, v_r\}$ жҳ е°„еҲ°еҲ—з©әй—ҙзҡ„жӯЈдәӨеҹә $\{u_1, \ldots, u_r\}$пјҢжҜҸдёӘж–№еҗ‘жҢү $\sigma_i$ жӢүдјёгҖӮйӣ¶з©әй—ҙзҡ„жүҖжңүеҶ…е®№йғҪиў«еҺӢзј©еҲ°йӣ¶зӮ№гҖӮиҝҷе°ұжҳҜзҹ©йҳө $A$ зҡ„е…ЁйғЁдҪңз”ЁгҖӮ

дә”гҖҒдҪҺз§©йҖјиҝ‘пјҡеҺӢзј©иғҢеҗҺзҡ„е®ҡзҗҶ

EckartвҖ“Young е®ҡзҗҶ

$$A_k = \sigma_1 u_1 v_1^{\!\top} + \cdots + \sigma_k u_k v_k^{\!\top}.$$$$\|A - A_k\|_F \;=\; \min_{\operatorname{rank}(B) \le k} \|A - B\|_F \;=\; \sqrt{\sigma_{k+1}^{\,2} + \cdots + \sigma_r^{\,2}}.$$

з”Ёз®—еӯҗпјҲ2-пјүиҢғж•°иЎЎйҮҸж—¶пјҢ$\|A - A_k\|_2 = \sigma_{k+1}$гҖӮ

жүҖд»ҘпјҢ$A_k$ дёҚд»…жҳҜдёҖдёӘдҪҺз§©иҝ‘дјјпјҢиҖҢдё”жҳҜзҗҶи®әдёҠжңҖдјҳзҡ„гҖӮд»»дҪ•е…¶д»–з§©дёә $k$ зҡ„зҹ©йҳөйғҪж— жі•жҜ”е®ғжӣҙжҺҘиҝ‘еҺҹзҹ©йҳөгҖӮ

зұ»жҜ” MP3пјҡMP3 еҺӢзј©дјҡдёўејғдәәиҖійҡҫд»ҘеҜҹи§үзҡ„й«ҳйў‘жҲҗеҲҶгҖӮSVD жҲӘж–ӯеҜ№зҹ©йҳөеҒҡдәҶзұ»дјјзҡ„дәӢпјҡеҺ»жҺүиғҪйҮҸжңҖе°Ҹзҡ„йғЁеҲҶпјҢдҝқз•ҷжңҖйҮҚиҰҒзҡ„дҝЎжҒҜгҖӮ

дёҖеұӮдёҖеұӮеҸ еҠ 

жҠҠжҜҸдёҖеұӮеҚ•зӢ¬з”»еҮәжқҘжӣҙе®№жҳ“зҗҶи§ЈгҖӮжҜҸдёӘ $\sigma_i u_i v_i^{\!\top}$ йғҪжҳҜдёҖеј з§©дёә 1 зҡ„еӣҫеғҸпјҢйҖҗжӯҘеҸ еҠ еҗҺи¶ҠжқҘи¶ҠжҺҘиҝ‘еҺҹеӣҫгҖӮ

дёҠжҺ’жҳҜеүҚдёүеұӮз§© 1 еӣҫеғҸпјҲзәўиүІдёәжӯЈпјҢи“қиүІдёәиҙҹпјүд»ҘеҸҠеҺҹеӣҫпјӣдёӢжҺ’жҳҜеҸ еҠ  1 еұӮгҖҒ2 еұӮгҖҒ3 еұӮеҗҺзҡ„з»“жһңпјҢеҸідҫ§жҳҜеҘҮејӮеҖјжҹұзҠ¶еӣҫгҖӮд»…д»…дёүеұӮе°ұиғҪзңӢеҮәеӣҫеғҸзҡ„дё»иҰҒз»“жһ„гҖӮ

иғҪйҮҸи§Ҷи§’

$$\|A\|_F^2 = \sigma_1^2 + \sigma_2^2 + \cdots + \sigma_r^2.$$$$\text{иғҪйҮҸжҜ”дҫӢ} = \frac{\sigma_1^2 + \cdots + \sigma_k^2}{\sigma_1^2 + \cdots + \sigma_r^2}.$$

еҜ№дәҺеӨ§еӨҡж•°иҮӘз„¶ж•°жҚ®пјҢеҘҮејӮеҖјиЎ°еҮҸеҫ—еҫҲеҝ«гҖӮжҜ”еҰӮдёҖеј  $1000 \times 1000$ зҡ„з…§зүҮпјҢеүҚ 50 дёӘеҘҮејӮеҖјйҖҡеёёиғҪдҝқз•ҷ 95% зҡ„иғҪйҮҸгҖӮ

import numpy as np

def compress(img, k):
    """еҜ№ 2D ж•°з»„еҒҡз§© k SVD иҝ‘дјј"""
    U, s, Vt = np.linalg.svd(img, full_matrices=False)
    return U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]

йҖҶзҹ©йҳөдёҚеӯҳеңЁж—¶жҖҺд№ҲеҠһпјҹ

и§Јж–№зЁӢ $A x = b$ ж—¶пјҢйҖҡеёёеёҢжңӣеҫ—еҲ° $x = A^{-1} b$гҖӮдҪҶ $A^{-1}$ еҸӘжңүеңЁ $A$ жҳҜж–№йҳөдё”ж»Ўз§©ж—¶жүҚеӯҳеңЁгҖӮиҝҷж—¶пјҢMooreвҖ“Penrose дјӘйҖҶ $A^{+}$ е°ұжҲҗдәҶдёҖдёӘйҖҡз”Ёзҡ„вҖңжңҖдҪіжӣҝд»Јж–№жЎҲвҖқгҖӮ

дјӘйҖҶзҡ„дҪңз”Ё

  1. жңҖе°ҸдәҢд№ҳи§Јпјҡи®© $\|A x - b\|_2$ жңҖе°Ҹпјӣ

  • з¬¬ $i$ дёӘдё»жҲҗеҲҶзҡ„ж–№е·®пјҡзӯүдәҺ $\sigma_i^2 / (n - 1)$гҖӮ
  • йҷҚеҲ° $k$ з»ҙпјҡеҸ–еүҚ $k$ дёӘдё»жҲҗеҲҶпјҢе…¬ејҸжҳҜ $X_k = U_k \Sigma_k$гҖӮ

import numpy as np

def pca(X, k):
    """з”Ё SVD е®һзҺ° PCAпјҢиҝ”еӣһ (еҫ—еҲҶ, дё»жҲҗеҲҶж–№еҗ‘, и§ЈйҮҠж–№е·®жҜ”дҫӢ)гҖӮ"""
    Xc = X - X.mean(axis=0)
    U, s, Vt = np.linalg.svd(Xc, full_matrices=False)
    components = Vt[:k]                    # k x p
    scores = Xc @ components.T             # n x k
    var = (s ** 2) / (X.shape[0] - 1)
    return scores, components, (var / var.sum())[:k]

import numpy as np

def svd_via_eigen(A):
    """йҖҡиҝҮ A^T A зҡ„зү№еҫҒеҲҶи§ЈжқҘжҰӮеҝөжҖ§ең°и®Ўз®— SVDгҖӮ
    е®һйҷ…йЎ№зӣ®дёӯиҜ·з”Ё np.linalg.svdпјҢиҝҷйҮҢд»…з”ЁдәҺж•ҷеӯҰгҖӮ"""
    ATA = A.T @ A
    eigvals, V = np.linalg.eigh(ATA)            # зү№еҫҒеҖјжҢүеҚҮеәҸжҺ’еҲ—
    idx = np.argsort(eigvals)[::-1]
    eigvals, V = eigvals[idx], V[:, idx]

    sigma = np.sqrt(np.maximum(eigvals, 0.0))
    r = int((sigma > 1e-10).sum())
    U = (A @ V[:, :r]) / sigma[:r]              # жҢүеҲ—е№ҝж’ӯ
    return U, sigma[:r], V[:, :r].T

A = np.array([[3.0, 2.0], [2.0, 3.0]])
U, s, Vt = svd_via_eigen(A)
print("еҘҮејӮеҖј:", s)
print("йҮҚе»ә:\n", U @ np.diag(s) @ Vt)

import numpy as np
import matplotlib.pyplot as plt

def plot_compression(img, ks):
    U, s, Vt = np.linalg.svd(img, full_matrices=False)
    fig, axes = plt.subplots(1, len(ks) + 1, figsize=(3 * (len(ks) + 1), 3))
    axes[0].imshow(img, cmap="gray"); axes[0].set_title("еҺҹеӣҫ"); axes[0].axis("off")
    for ax, k in zip(axes[1:], ks):
        rec = U[:, :k] @ np.diag(s[:k]) @ Vt[:k]
        energy = (s[:k] ** 2).sum() / (s ** 2).sum() * 100
        ax.imshow(rec, cmap="gray")
        ax.set_title(f"k={k}  ({energy:.0f}%)"); ax.axis("off")
    plt.tight_layout(); plt.show()
