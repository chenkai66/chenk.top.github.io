机器学习数学推导（八）：支持向量机

从最大间隔到核技巧，完整推导 SVM 的理论框架——拉格朗日对偶、KKT 条件、SMO 算法与核函数构造。

引子。 假设有两团点，能将它们分开的直线有无数条，哪一条才是“最佳选择”？SVM 给出了一个非常几何化的答案：位于两类点之间“最宽空白走廊”正中间的那条直线。把这个简单的想法用拉格朗日对偶推导一遍，就能得到三个重要结果：稀疏模型（只有走廊边界上的点起作用）、具有全局最优解的二次规划问题，以及几乎算是附赠的核技巧——它让线性方法在无限维空间中也能刻画出弯曲的决策边界。

你将学到什么

  • “最宽走廊”如何转化为带线性约束的凸二次规划问题
  • 对偶问题为何比原问题更实用，KKT 条件怎样实现稀疏性
  • 软间隔的原理，以及正则化参数 $C$ 到底在控制什么
  • 核技巧：用 $K(x, z)$ 替代 $\phi(x)^\top \phi(z)$，完全不用显式构造 $\phi$
  • Mercer 条件与标准核函数家族（线性、多项式、RBF、sigmoid）
  • SMO 算法：为何每次只优化两个变量，裁剪的机制是什么，算法为何能收敛
  • Hinge 损失——从 SVM 通往其他监督学习方法的桥梁

前置知识

  • 线性代数：内积、超平面、投影
  • 微积分：拉格朗日乘子、偏导数
  • 了解凸对偶性会有帮助，但不是必须的
  • 熟悉 第 7 篇：决策树

1. 硬间隔 SVM

1.1 函数间隔与几何间隔

假设二分类标签 $y_i \in \{-1, +1\}$，线性决策规则为 $\hat{y} = \operatorname{sign}(w^\top x + b)$。对于一个点到边界的距离，有两种定义方式：

$$\hat{\gamma}_i \;=\; y_i\,(w^\top x_i + b) \qquad\text{（函数间隔）}$$$$\gamma_i \;=\; \frac{y_i\,(w^\top x_i + b)}{\lVert w \rVert} \qquad\text{（几何间隔）}$$

函数间隔在分类正确时为正，但它依赖于 $(w, b)$ 的缩放——把 $(w, b)$ 放大两倍，函数间隔也会翻倍。几何间隔则是点 $x_i$ 到超平面的真实欧氏距离，带上标签的符号，并且对 $(w, b)$ 的缩放不敏感。这种尺度不变性让优化问题变得合理，否则“增大间隔”就没有明确意义，因为只要缩小 $\lVert w \rVert$ 就能实现。

公式怎么来的？ 对任意点 $x_0$，它在超平面 $w^\top x + b = 0$ 上的最近点是正交投影，位移向量为 $-(w^\top x_0 + b)/\lVert w \rVert^2 \cdot w$，模长为 $\lvert w^\top x_0 + b\rvert / \lVert w \rVert$。乘上 $y_i$ 后，分类正确时结果为正。

1.2 最大间隔规划

目标是找到让最坏情况下几何间隔最大的超平面：

$$\max_{w, b}\; \min_i\; \frac{y_i\,(w^\top x_i + b)}{\lVert w \rVert}.$$

这里 $(w, b)$ 只在正比例缩放下唯一，因此我固定一个尺度：要求最近点的函数间隔恰好为 $1$。等价形式如下：

$$\boxed{\;\min_{w, b}\; \tfrac{1}{2}\lVert w \rVert^2 \quad \text{s.t.} \quad y_i(w^\top x_i + b) \;\ge\; 1, \quad i = 1, \dots, N.\;}$$

这是一个带线性约束的凸二次规划问题。目标函数严格凸，最优解 $w^\*$ 唯一；只要数据中两个类别都存在，$b^\*$ 也唯一。

满足约束取等号的样本（$y_i(w^\top x_i + b) = 1$）称为支持向量。几何上看，它们位于边界两侧的两条平行间隔线上；代数上看，它们是唯一决定 $w^\*$ 的点。

1.3 拉格朗日对偶推导

为每个约束引入拉格朗日乘子 $\alpha_i \ge 0$：

$$L(w, b, \alpha) \;=\; \tfrac{1}{2}\lVert w \rVert^2 \;-\; \sum_{i=1}^N \alpha_i \bigl[\,y_i(w^\top x_i + b) - 1\,\bigr].$$

第一步——对 $w$ 和 $b$ 求极小值。 令 $\partial_w L = 0$ 和 $\partial_b L = 0$：

$$w^\* \;=\; \sum_{i=1}^N \alpha_i y_i x_i, \qquad \sum_{i=1}^N \alpha_i y_i \;=\; 0.$$

第二步——回代。 把 $w^\* = \sum_i \alpha_i y_i x_i$ 代入 $L$：

$$W(\alpha) \;=\; \sum_{i=1}^N \alpha_i \;-\; \tfrac{1}{2} \sum_{i, j} \alpha_i \alpha_j y_i y_j\, x_i^\top x_j.$$

得到对偶问题：

$$\boxed{\;\max_{\alpha}\; \sum_i \alpha_i - \tfrac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j\, x_i^\top x_j \quad \text{s.t.} \quad \alpha_i \ge 0, \;\; \sum_i \alpha_i y_i = 0.\;}$$

这里有两点关键性质：

  1. 数据只通过内积 $x_i^\top x_j$ 出现。将其替换为核函数 $K(x_i, x_j)$，整个推导依然成立——这就是核技巧的核心思想。
  2. 对偶问题有 $N$ 个标量变量和一条等式约束。当特征维度 $d \gg N$ 时，对偶问题比原问题更高效。

1.4 KKT 条件与稀疏性

原问题是凸优化问题，约束是仿射的，Slater 条件成立，因此强对偶性成立。最优解 $(w^\*, b^\*, \alpha^\*)$ 必须满足以下条件：

  • 原始可行性： $y_i(w^{\*\top} x_i + b^\*) \ge 1$。
  • 对偶可行性： $\alpha_i^\* \ge 0$。
  • 稳定性： $w^\* = \sum_i \alpha_i^\* y_i x_i$，$\sum_i \alpha_i^\* y_i = 0$。
  • 互补松弛： $\alpha_i^\* \cdot \bigl[\, y_i(w^{\*\top} x_i + b^\*) - 1 \,\bigr] = 0$。

最后一项是重点。对每个 $i$，以下两种情况只能选其一：

  • $\alpha_i^\* = 0$——点严格在间隔带之外，对 $w^\*$ 没有贡献。
  • $y_i(w^{\*\top} x_i + b^\*) = 1$——点恰好落在间隔线上，允许 $\alpha_i^\* > 0$。

最终分类器只由第二种点支撑：

$$f(x) \;=\; \sum_{i \in \mathrm{SV}} \alpha_i^\* y_i\, x_i^\top x \;+\; b^\*.$$

预测时可以丢弃所有非支持向量的训练点。这就是 SVM 的稀疏性。

import numpy as np
from sklearn.svm import SVC

# 一个小的线性可分例子
X = np.array([[1, 2], [2, 3], [3, 3],
              [1, 0], [2, 1], [0, 1]], dtype=float)
y = np.array([1, 1, 1, -1, -1, -1])

clf = SVC(kernel="linear", C=1e6).fit(X, y)   # C 极大 ≈ 硬间隔
w, b = clf.coef_[0], clf.intercept_[0]

print(f"w = {w},  b = {b:.4f}")
print(f"#SV = {len(clf.support_)} / {len(X)}")
for i in clf.support_:
    print(f"  SV idx {i}: y(w·x+b) = {y[i] * (X[i] @ w + b):.4f}  "
          f"（应≈1）")

2. 软间隔 SVM

2.1 松弛变量与参数 $C$

真实数据往往有重叠。我给每个点分配一个非负的松弛量 $\xi_i$，允许它“违规”：

$$\min_{w, b, \xi}\; \tfrac{1}{2}\lVert w \rVert^2 \;+\; C \sum_i \xi_i \quad \text{s.t.} \quad y_i(w^\top x_i + b) \ge 1 - \xi_i,\; \xi_i \ge 0.$$

$\xi_i$ 的含义很简单：

  • $\xi_i = 0$：点在间隔带外，分类正确。
  • $0 < \xi_i \le 1$：点在间隔带内，但仍在正确的一侧。
  • $\xi_i > 1$：点被错误分类。

$C > 0$ 是用来权衡两个目标的参数。$C$ 大时，模型对松弛量惩罚更重，间隔变窄，接近硬间隔模型；$C$ 小时，模型对松弛量宽容，间隔变宽，更能容忍噪声。

2.2 对偶问题：盒约束

还是用拉格朗日方法。这次为间隔约束引入乘子 $\alpha_i \ge 0$，为 $\xi_i \ge 0$ 引入乘子 $\mu_i \ge 0$。令 $\partial_\xi L = 0$，得到 $\alpha_i + \mu_i = C$，所以 $0 \le \alpha_i \le C$。对偶问题如下：

$$\boxed{\;\max_{\alpha}\; \sum_i \alpha_i - \tfrac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j\, x_i^\top x_j \quad \text{s.t.} \quad 0 \le \alpha_i \le C, \;\; \sum_i \alpha_i y_i = 0.\;}$$

和硬间隔相比，唯一的区别是多了个上界 $\alpha_i \le C$。KKT 条件把样本分成了三类：

类型条件含义
非活跃点$\alpha_i = 0$，$\xi_i = 0$点安全地位于间隔带外
边界支持向量$0 < \alpha_i < C$，$\xi_i = 0$点恰好落在间隔线上，可用于计算 $b^\*$
上界支持向量$\alpha_i = C$，$\xi_i > 0$点在间隔带内或被错误分类

偏置 $b^\*$ 可以通过任一边界支持向量计算：$b^\* = y_i - \sum_j \alpha_j^\* y_j x_j^\top x_i$。为了数值稳定性，我会对所有边界支持向量取平均。

2.3 Hinge 损失视角

原问题中，最优松弛量可以写成 $\xi_i^\* = \max(0,\, 1 - y_i(w^\top x_i + b))$。把它代回原问题，消掉 $\xi_i$：

$$\min_{w, b}\; \tfrac{1}{2}\lVert w \rVert^2 + C \sum_i \max\bigl(0,\, 1 - y_i(w^\top x_i + b)\bigr).$$

右边的求和项就是 hinge 损失。换句话说，软间隔 SVM 其实就是在做“L2 正则化 + hinge 损失”的经验风险最小化。从这个角度看，SVM 和其他线性分类器没什么本质区别，只是损失函数不同而已。

Hinge 损失在 $m = 1$ 处有个尖点，并且在 $m > 1$ 时恒为零。这正是支持向量稀疏性的来源：那些自信分类正确的点既不会贡献梯度，也不会影响 $\alpha$。

3. 核函数

3.1 核技巧

把输入映射到某个特征空间 $\phi: \mathbb{R}^d \to \mathcal{H}$，然后在 $\mathcal{H}$ 中运行线性 SVM。对偶目标函数变为：

$$W(\alpha) \;=\; \sum_i \alpha_i \;-\; \tfrac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j \;\phi(x_i)^\top \phi(x_j).$$

整个过程中，$\phi$ 只出现在内积里。把这个内积定义为核函数：

$$K(x, z) \;=\; \phi(x)^\top \phi(z).$$

凡是出现 $x_i^\top x_j$ 的地方（无论是对偶问题还是预测），都替换成 $K(x_i, x_j)$。我不需要显式构造 $\phi$，也不用存储特征，甚至不用关心 $\dim \mathcal{H}$ 是否有限。

这张图直观说明了核技巧为什么有效。左图中，内外两类数据在二维平面里无法用直线分开。通过映射 $\phi(x) = (x_1, x_2, x_1^2 + x_2^2)$，外环在第三维被抬高，水平面 $x_1^2 + x_2^2 = c$ 就能轻松分开两类数据。如果想在 $\mathbb{R}^3$ 中计算内积，直接用 $K(x, z) = (1 + x^\top z)^2$ 就行，完全不需要显式写出 $\phi$。

3.2 常用核函数

核公式特征空间
线性$K(x, z) = x^\top z$原始输入空间
多项式$K(x, z) = (\gamma\, x^\top z + r)^p$所有不超过 $p$ 次的单项式
RBF（高斯）$K(x, z) = \exp(-\gamma\,\lVert x - z\rVert^2)$无限维
Sigmoid$K(x, z) = \tanh(\gamma\, x^\top z + r)$仅在某些参数下半正定

RBF 直觉
每个训练点会生成一个半径约为 $1/\sqrt{\gamma}$ 的“鼓包”。$\gamma$ 大时，鼓包窄，决策边界紧贴每个点，容易过拟合；$\gamma$ 小时，鼓包宽，边界更平滑，但可能欠拟合。

3.3 Mercer 定理

定理
对称函数 $K(x, z)$ 能表示为某个希尔伯特空间中的内积 $\phi(x)^\top \phi(z)$，当且仅当对于任意有限点集 $\{x_i\}_{i=1}^N$，核矩阵 $\mathbf{K}_{ij} = K(x_i, x_j)$ 半正定，即：

$$\sum_{i, j} c_i c_j\, K(x_i, x_j) \;\ge\; 0, \quad \forall\, c \in \mathbb{R}^N.$$

这就是核技巧的理论依据：半正定的 $K$ 本质上就是某个内积——再生核希尔伯特空间（RKHS）的构造保证了这一点。

有用的封闭性规则
如果 $K_1$ 和 $K_2$ 是核，那么以下也是核：$K_1 + K_2$、$\lambda K_1$（$\lambda \ge 0$）、$K_1 \cdot K_2$、$f(x) K_1(x, z) f(z)$、$K_1$ 的非负系数多项式、$\exp(K_1)$。RBF 核实际上就是通过这些规则反复应用在线性核上得到的。

# 验证 Gram 矩阵是否半正定
gamma = 5.0
K = np.exp(-gamma * np.sum((X[:, None] - X[None, :]) ** 2, axis=-1))
eig = np.linalg.eigvalsh(K)
print(f"特征值: {eig.round(4)}  全部 ≥ 0: {np.all(eig >= -1e-10)}")

4. SMO 算法

4.1 为什么一次更新两个变量

对偶问题是一个包含 $N$ 个变量的 QP 问题，带有一个等式约束 $\sum_i \alpha_i y_i = 0$。通用 QP 求解器的时间复杂度是 $O(N^3)$，空间复杂度是 $O(N^2)$——对于几万个点的数据集来说，这完全不可接受。

如果尝试用朴素的坐标下降法（固定其他变量，只优化一个 $\alpha_i$），会发现这是行不通的：等式约束要求任何合法的移动都必须至少改变两个变量。因此，在保持可行性的前提下，最少需要同时更新两个变量。这就是 Sequential Minimal Optimization（Platt, 1998）的核心思想：选一对变量，解析求解这个两变量的 QP 问题，然后重复。

4.2 两变量子问题

假设我选择了变量 $1, 2$，并固定其他变量。此时，等式约束可以简化为：

$$\alpha_1 y_1 + \alpha_2 y_2 \;=\; -\sum_{i \ge 3} \alpha_i y_i \;=:\; \zeta \quad (\text{常数}).$$

这样一来，$\alpha_1$ 就由 $\alpha_2$ 决定，而对偶目标函数退化为关于 $\alpha_2$ 的一元二次函数。对它求导并令导数为零，得到无约束更新公式：

$$\alpha_2^{\text{new, unc}} \;=\; \alpha_2^{\text{old}} \;+\; \frac{y_2(E_1 - E_2)}{\eta},$$

其中

$$E_i \;=\; f(x_i) - y_i, \qquad \eta \;=\; K(x_1, x_1) + K(x_2, x_2) - 2K(x_1, x_2) \;\ge\; 0.$$

$\eta$ 是特征空间中 $\lVert \phi(x_1) - \phi(x_2) \rVert^2$ 的平方距离，因此非负。

4.3 裁剪到 $[L, H]$

更新后的变量对必须满足 $0 \le \alpha_1, \alpha_2 \le C$，同时还要符合等式约束。将这两个条件结合起来，$\alpha_2$ 的取值范围就被限制在区间 $[L, H]$ 内，具体端点取决于两个标签的符号关系：

$$\begin{cases} y_1 \neq y_2: & L = \max(0,\, \alpha_2^{\text{old}} - \alpha_1^{\text{old}}), \quad H = \min(C,\, C + \alpha_2^{\text{old}} - \alpha_1^{\text{old}}). \\[2pt] y_1 = y_2: & L = \max(0,\, \alpha_1^{\text{old}} + \alpha_2^{\text{old}} - C), \quad H = \min(C,\, \alpha_1^{\text{old}} + \alpha_2^{\text{old}}). \end{cases}$$

裁剪后，回代计算：

$$\alpha_2^{\text{new}} \;=\; \operatorname{clip}(\alpha_2^{\text{new, unc}},\, L,\, H), \qquad \alpha_1^{\text{new}} \;=\; \alpha_1^{\text{old}} + y_1 y_2\,(\alpha_2^{\text{old}} - \alpha_2^{\text{new}}).$$

4.4 启发式规则与收敛性

  • 外层循环： 在两种模式之间交替：(a) 遍历所有样本；(b) 只遍历非边界支持向量（$0 < \alpha_i < C$）。因为边界上的变量几乎不会移动。
  • 第一变量： 选择违反 KKT 条件超过容差的任意 $\alpha_i$。
  • 第二变量： 选择使 $\lvert E_1 - E_2 \rvert$ 最大的变量，这近似于最大步长启发式。

只要选中的变量对不在最优位置，每一步都会严格改进对偶目标函数。结合目标函数有界和有限步长集合，SMO 必然收敛。实际运行中，SMO 比通用 QP 求解器快几个数量级。为了让每步复杂度真正达到 $O(N)$ 而不是 $O(N^2)$，关键在于增量维护误差缓存 $E_i$。

5. 实践要点

  • 特征一定要标准化。 SVM 基于欧氏几何，如果特征量纲不一致，大尺度特征会主导间隔计算。
  • 同时调整 $C$ 和 $\gamma$。 它们的乘积大致决定了模型的容量。用交叉验证在对数尺度上搜索：$C \in \{10^{-2}, \dots, 10^3\}$，$\gamma \in \{10^{-3}, \dots, 10^1\}$。
  • 样本量 $N \gg 10^5$ 时，优先选择线性 SVM（LIBLINEAR）或随机方法。 核 SVM 的训练时间复杂度大约是 $O(N^2)$。
  • 多分类问题。 标准 SVM 只支持二分类。可以用 One-vs-Rest（$K$ 个模型）或 One-vs-One（$K(K-1)/2$ 个模型加投票）。scikit-learn 的 SVC 默认使用 OvO。
  • 概率输出。 SVM 输出的是距离，不是校准后的概率。如果需要概率，可以用 Platt scaling（对决策值再拟合一个逻辑回归）或者保序回归。

问题。 超平面 $w = (3, 4)^\top$，$b = -1$。点 $x_0 = (1, 1)^\top$，标签 $y = +1$。求几何间隔。

解答。

$$\gamma = \frac{y\,(w^\top x_0 + b)}{\lVert w \rVert} = \frac{1 \cdot (3 + 4 - 1)}{\sqrt{9 + 16}} = \frac{6}{5} = 1.2.$$

练习 2 — 核函数的加法封闭性

问题。 证明：如果 $K_1, K_2$ 是有效核，则 $K = K_1 + K_2$ 也是有效核。

解答。 对任意有限点集 $\{x_i\}$ 和任意 $c \in \mathbb{R}^N$：

$$\sum_{i, j} c_i c_j K(x_i, x_j) = \sum_{i, j} c_i c_j K_1(x_i, x_j) + \sum_{i, j} c_i c_j K_2(x_i, x_j) \ge 0,$$

因为 $K_1$ 和 $K_2$ 都满足 Mercer 条件，两项都非负。所以 $K$ 是半正定的，是有效核。

练习 3 — 从支持向量数量看 $C$

问题。 100 个训练样本：$C = 0.1$ 时有 30 个支持向量；$C = 100$ 时有 15 个支持向量。解释原因。

解答。 $C$ 小的时候，松弛变量代价低，优化器会扩大间隔带，更多点落入间隔带内，变成 $\alpha = C$ 的边界支持向量。$C$ 大的时候，松弛变量惩罚重，间隔带变窄，落进带内的点减少，支持向量数量也减少。一般规律是：正则化越强（$C$ 越小），间隔越宽，支持向量越多。

练习 4 — 调试 $\gamma$

问题。 使用 RBF 核：$\gamma = 0.01$ 时训练准确率 95 %，测试准确率 60 %；$\gamma = 100$ 时训练准确率 100 %，测试准确率 70 %。哪个更好？下一步该怎么做？

解答。 两个都不好。$\gamma = 0.01$ 时，核函数的“鼓包”太宽，模型欠拟合——训练勉强及格，测试表现差。$\gamma = 100$ 时，“鼓包”像针尖一样细，模型记住每个训练点，严重过拟合。接下来应该和 $C$ 一起，在 $\gamma \in \{0.1, 0.3, 1, 3, 10, 30\}$ 上交叉验证，找到最优组合，缩小训练和测试的差距。

练习 5 — SVM 和逻辑回归的对比

问题。 什么时候该用哪个？

解答。

需求更合适的选择理由
输出校准概率逻辑回归SVM 只输出带符号的距离
小数据 + 非线性边界RBF SVM核技巧让模型稀疏且灵活
百万级样本线性逻辑回归 / 线性 SVM（SGD）核 SVM 的复杂度接近 $O(N^2)$
稀疏特征选择逻辑回归 + L1 正则$w$ 自然稀疏，而不是样本稀疏
对远离边界的样本鲁棒SVMhinge 损失忽略已经正确分类的样本

Q&A

为什么要最大化间隔？

从几何上看，间隔越大，模型对噪声的容忍度越高。只要扰动小于间隔，预测结果就不会变。从统计上看，基于间隔的理论界（比如通过 Rademacher 复杂度推导）表明，泛化误差会随着间隔增大而减小。所以，SVM 其实是在隐式地控制模型复杂度。

为什么用对偶形式？

主要有三个原因。第一，数据只以内积的形式出现，这为使用核方法提供了可能。第二，当 $d \gg N$ 时，对偶问题的规模比原问题小得多。第三，KKT 条件揭示了稀疏性：大多数 $\alpha_i = 0$，这让预测变得高效。

SMO 中如果 $\eta = 0$ 怎么办？

$\eta = \lVert \phi(x_1) - \phi(x_2) \rVert^2$，所以 $\eta = 0$ 表示两个点在特征空间中重合。这种情况下，目标函数在可行区间上关于 $\alpha_2$ 是线性的，最优解就是取 $L$ 或 $H$ 中让目标值更大的那个端点。

SVM 能做回归吗？

可以。支持向量回归（SVR） 使用 $\varepsilon$-不敏感损失函数 $\max(0, |y - f(x)| - \varepsilon)$。误差小于 $\varepsilon$ 的样本不计入损失，超出 $\varepsilon$ 的样本成为支持向量。对偶形式的求解框架同样适用，只不过每个样本对应两个乘子（$\alpha_i, \alpha_i^\*$，分别表示上下边界）。

为什么要标准化输入？

$\eta$ 和核矩阵都涉及 $\lVert x \rVert$。如果某个特征的量纲特别大，它会主导其他特征，导致决策边界偏向坐标轴对齐。把每个特征调整为零均值、单位方差，通常只需一行代码，但能显著提升训练速度和模型精度。
