иҝҷзҜҮж–Үз« еҸӘи°Ҳ demo д№ӢеҗҺзҡ„дәӢгҖӮз«ӢеңәжҳҜжҳҺзЎ®зҡ„пјҡдёҖдёӘз”ҹдә§зә§ LLM зі»з»ҹпјҢжң¬иҙЁдёҠжҳҜдёҖдёӘжҷ®йҖҡзҡ„еҲҶеёғејҸзі»з»ҹпјҢеҸӘдёҚиҝҮиў«ејәиЎҢеЎһдәҶдёҖдёӘйқһзЎ®е®ҡжҖ§зҡ„з»„д»¶иҝӣеҺ»вҖ”вҖ”иҖҢе·ҘзЁӢдёҠз»қеӨ§йғЁеҲҶзҡ„зІҫеҠӣпјҢйғҪз”ЁжқҘвҖңеңҲдҪҸвҖқиҝҷз§ҚдёҚзЎ®е®ҡжҖ§гҖӮдёӢйқўдјҡжҢүдёғдёӘз»ҙеәҰеұ•ејҖпјҡеә”з”ЁеҲҶеұӮгҖҒе·ҘдҪңжөҒжЁЎејҸгҖҒ RAG/еҫ®и°ғ/Prompt зҡ„еҸ–иҲҚгҖҒйғЁзҪІжӢ“жү‘гҖҒжҲҗжң¬гҖҒеҸҜи§ӮжөӢжҖ§гҖҒдјҒдёҡйӣҶжҲҗгҖӮжҜҸдёҖиҠӮйғҪеҠӣжұӮз®ҖзҹӯгҖҒе…·дҪ“пјҢеҸӘиҒҡз„ҰзңҹжӯЈиғҪжҳҫи‘—еҪұе“Қж ёеҝғжҢҮж Үзҡ„е…ій”®жқ жқҶгҖӮ

дҪ е°ҶеӯҰеҲ°д»Җд№Ҳ#

  • еҰӮдҪ•жҠҠ LLM еә”з”ЁжӢҶжҲҗдә”еұӮпјҢи®©вҖңйқһзЎ®е®ҡжҖ§вҖқиў«е…іеңЁдёҖеұӮйҮҢ
  • еӣӣз§Қе·ҘдҪңжөҒжЁЎејҸпјҲchainгҖҒ branchгҖҒ loopгҖҒ parallelпјүпјҢд»ҘеҸҠе®ғд»¬еҗ„иҮӘдёҚиҜҘз”ЁеңЁе“ӘйҮҢ
  • дёҖдёӘиғҪжүӣдҪҸзҺ°е®һиҖғйӘҢзҡ„вҖңPrompt vs RAG vs еҫ®и°ғвҖқеҶізӯ–и§„еҲҷ
  • дёҖдёӘеёҰиҜӯд№үзј“еӯҳгҖҒ LLM GatewayгҖҒеӨҡжЎЈжЁЎеһӢжұ зҡ„йғЁзҪІжӢ“жү‘
  • е…ӯж №жҲҗжң¬жқ жқҶпјҢд»ҘеҸҠжӢүеҠЁе®ғд»¬зҡ„жӯЈзЎ®йЎәеәҸ
  • дёҖдёӘжҠҠеӣӣдёӘ LLM дё“еұһдҝЎеҸ·иЎҘеӣһеҺ»зҡ„еҸҜи§ӮжөӢжҖ§ж–№жЎҲ
  • и®©еӨ§еӨҡж•°иҜ•зӮ№дёҠдёҚдәҶзәҝзҡ„е…ӯдёӘдјҒдёҡйӣҶжҲҗвҖңй—Ёж§ӣвҖқ

еүҚзҪ®зҹҘиҜҶ#

дҪ еҶҷиҝҮ REST жҺҘеҸЈпјҢз”ЁиҝҮејӮжӯҘ Python жҲ– NodeпјҢеӨ§иҮҙдәҶи§Ј DockerпјҢе№¶и°ғз”ЁиҝҮиҮіе°‘дёҖе®¶ LLM еҺӮе•Ҷзҡ„ SDKгҖӮж–Үдёӯж¶үеҸҠзҡ„е·ҘзЁӢеҒҡжі•иҮӘеёҰи¶іеӨҹзҡ„дёҠдёӢж–ҮпјҢж— йңҖеңЁз”ҹдә§зҺҜеўғдёӯиҝҗиЎҢиҝҮеҗ‘йҮҸж•°жҚ®еә“гҖӮ

LLM еә”з”Ёзҡ„еҲҶеұӮжһ¶жһ„#

жһ„е»ә LLM дә§е“ҒжңҖжңүз”Ёзҡ„еҝғжҷәжЁЎеһӢжҳҜе°Ҷе…¶и§ҶдёәдёҖдёӘжҷ®йҖҡзҡ„дә”еұӮеә”з”ЁпјҢеҸӘжҳҜе…¶дёӯжҒ°еҘҪжңүдёҖеұӮжҳҜжҰӮзҺҮжҖ§зҡ„гҖӮиҮӘдёҠиҖҢдёӢпјҡ

  1. дҪ“йӘҢеұӮвҖ”вҖ”WebгҖҒз§»еҠЁз«ҜгҖҒ IDE жҸ’д»¶гҖҒдјҒдёҡ IM жңәеҷЁдәәгҖӮжөҒејҸ UX еә”иҜҘдҪҸеңЁиҝҷйҮҢпјҢиҖҢдёҚжҳҜжЁЎеһӢйҮҢгҖӮ
  2. Agent / зј–жҺ’еұӮвҖ”вҖ”и§„еҲ’еҷЁгҖҒе·Ҙе…·и·Ҝз”ұгҖҒи®°еҝҶгҖҒжҠӨж ҸгҖӮдҪ зҡ„д»Јз ҒеӨ§еӨҙйғҪеңЁиҝҷгҖӮ
  3. жЈҖзҙўдёҺдёҠдёӢж–ҮеұӮвҖ”вҖ”RAGгҖҒиҜӯд№үзј“еӯҳгҖҒдјҡиҜқеӯҳеӮЁпјҢд»ҘеҸҠжңҖеҗҺжҠҠж¶ҲжҒҜеҲ—иЎЁз»„иЈ…еҮәжқҘзҡ„ prompt builderгҖӮ
  4. жЁЎеһӢжңҚеҠЎеұӮвҖ”вҖ”дёҖдёӘ LLM Gateway йЎ¶еңЁеӨҡдёӘжЁЎеһӢдҫӣеә”е•ҶеүҚйқўпјҢиҙҹиҙЈи·Ҝз”ұгҖҒйҷҚзә§гҖҒжөҒејҸиҫ“еҮәгҖӮ
  5. е·Ҙе…·дёҺж•°жҚ®йқўвҖ”вҖ”SQLгҖҒжҗңзҙўгҖҒд»Јз Ғжү§иЎҢгҖҒз¬¬дёүж–№ APIгҖҒеҗ‘йҮҸеә“гҖӮиҝҷдёҖеұӮжҳҜзЎ®е®ҡжҖ§зҡ„пјҢеҸҜд»ҘеғҸжҷ®йҖҡжңҚеҠЎдёҖж ·жөӢиҜ•гҖӮ

жЁӘеҲҮиҝҷдә”еұӮзҡ„жҳҜи®ӨиҜҒжҺҲжқғгҖҒе®Ўи®ЎдёҺ PII и„ұж•ҸгҖҒйҷҗжөҒгҖҒеҸҜи§ӮжөӢжҖ§гҖҒжҲҗжң¬ж ёз®—е’ҢиҜ„дј°зӯүеёёи§ҒеҠҹиғҪгҖӮ

иҝҷз§ҚеҲҶеұӮд№ӢжүҖд»ҘйҮҚиҰҒпјҢжҳҜеӣ дёәе®ғжҳҺзЎ®дәҶе·ҘзЁӢжҠ•е…Ҙзҡ„йҮҚзӮ№гҖӮеҰӮжһңдёҖдёӘеӣўйҳҹжҠҠйҮҚиҜ•йҖ»иҫ‘еҶҷеңЁдҪ“йӘҢеұӮгҖҒжҠҠ prompt жЁЎжқҝеЎһиҝӣжЁЎеһӢеұӮгҖҒжҠҠйҷҗжөҒеҲ°еӨ„йғҪжқҘдёҖд»ҪпјҢйӮЈд№ҲиҝҷдёӘйЎ№зӣ®еҫҲеҸҜиғҪж— жі•дёҠзәҝгҖӮе°ҶжҜҸдёӘе…іжіЁзӮ№еҺӢеӣһеҲ°е®ғеә”жңүзҡ„еұӮж¬ЎпјҢжЁЎеһӢеұӮе°ұиғҪдҝқжҢҒиҪ»йҮҸеҢ–гҖӮ

з¬¬дәҢдёӘжҺЁи®әжҳҜпјҢеҸҜйқ жҖ§е·ҘдҪңеӨ§йғЁеҲҶдёҚжҳҜ LLM зҡ„е·ҘдҪңгҖӮеҜ№е·ҘзЁӢе®һзҺ°иҖҢиЁҖпјҢжЁЎеһӢжң¬иҙЁдёҠжҳҜдёҖдёӘ HTTP й»‘зӣ’гҖӮзңҹжӯЈжңүи¶Јзҡ„й—®йўҳвҖ”вҖ”еҰӮжӯЈзЎ®зҡ„е·Ҙе…·жҳҜеҗҰиў«и°ғз”ЁгҖҒеҸӮж•°жҳҜеҗҰжӯЈзЎ®гҖҒжҳҜеҗҰи®ҝй—®дәҶжӯЈзЎ®з§ҹжҲ·зҡ„ж•°жҚ®гҖҒжҳҜеҗҰеңЁйў„з®—д№ӢеҶ…вҖ”вҖ”йғҪеӯҳеңЁдәҺзј–жҺ’е’ҢжЈҖзҙўеұӮпјҢиҝҷдәӣйғҪжҳҜзЎ®е®ҡжҖ§зҡ„ Python жҲ– TypeScript д»Јз ҒпјҢеҸҜд»ҘиҝӣиЎҢеҚ•е…ғжөӢиҜ•гҖӮ

е·ҘдҪңжөҒжЁЎејҸпјҡиғҪз”ЁжңҖз®ҖеҚ•зҡ„е°ұз”ЁжңҖз®ҖеҚ•зҡ„#

Anthropic еңЁ Building Effective Agents вҶ— пјҲ2024пјүйҮҢи®ІиҝҮдёҖдёӘеҖјеҫ—еҸҚеӨҚдҪ“дјҡзҡ„и§ӮзӮ№пјҡеҫҲеӨҡдәәеҳҙйҮҢзҡ„ “agent”пјҢе…¶е®һз”ЁдёҖдёӘеӣәе®ҡзҡ„е°Ҹе·ҘдҪңжөҒе°ұиғҪи§ЈеҶіеҫ—жӣҙеҘҪгҖӮе·ҘдҪңжөҒеҸҜйў„жөӢгҖҒдҫҝе®ңгҖҒеҘҪиҜ„дј°гҖӮ AgentвҖ”вҖ”LLM иҮӘе·ұеҠЁжҖҒеҶіе®ҡдёӢдёҖжӯҘвҖ”вҖ”еҸӘеңЁд»»еҠЎз©әй—ҙзңҹзҡ„ејҖж”ҫж—¶жүҚиҜҘз”ЁгҖӮ

е®һйҷ…з”ҹдә§йҮҢпјҢдҪ еҸӘйңҖиҰҒеӣӣз§ҚжЁЎејҸгҖӮ

Chain пјҲй“ҫејҸпјүгҖӮйЎәеәҸжөҒж°ҙзәҝпјҡжҠҪеҸ– вҶ’ иҪ¬жҚў вҶ’ жҖ»з»“гҖӮеҪ“жӯҘйӘӨзЎ®е®ҡгҖҒйЎәеәҸе·ІзҹҘж—¶з”Ёе®ғгҖӮе®һзҺ°е°ұжҳҜдёҖдёӘеҮҪж•°еҲ—иЎЁпјҢдёҚйңҖиҰҒжЎҶжһ¶гҖӮ

Branch пјҲеҲҶж”Ҝи·Ҝз”ұпјүгҖӮе…Ҳз”ЁдёҖдёӘе°ҸжЁЎеһӢеҲӨдёҖдёӢж„ҸеӣҫпјҢеҶҚеҲҶеҸ‘пјҡ FAQ иө°е°ҸжЁЎеһӢпјҢеҶҷд»Јз Ғиө° tool-using и·Ҝеҫ„пјҢзЎ¬жҺЁзҗҶиө°ж——иҲ°жЁЎеһӢгҖӮиҝҷдёҖз§ҚжЁЎејҸиҙЎзҢ®дәҶдҪ дёҚжҚўжЁЎеһӢдҫӣеә”е•Ҷе°ұиғҪжӢҝеҲ°зҡ„еӨ§йғЁеҲҶжҲҗжң¬иҠӮзәҰвҖ”вҖ”жҲ‘д»¬еңЁиҮӘе·ұзҡ„йғЁзҪІйҮҢи§ӮеҜҹеҲ°пјҢеҠ дёҖдёӘ LLM и·Ҝз”ұ + дёүжЎЈжЁЎеһӢжұ пјҢзӣёжҜ”вҖңе…ЁйғЁиө°жңҖеӨ§жЁЎеһӢвҖқйҖҡеёёиғҪзңҒ 60вҖ“80% зҡ„иҙ№з”ЁгҖӮ

Loop пјҲеҸҚжҖқеҫӘзҺҜпјүгҖӮз”ҹжҲҗ вҶ’ иҜ„еҲӨ вҶ’ ж”№еҶҷгҖӮиҜ„еҲӨж–№еҸҜд»ҘжҳҜ LLM-as-judgeгҖҒжӯЈеҲҷгҖҒи·‘еҚ•жөӢгҖҒжҲ–иҖ…дёҖдёӘ SQL dry-runгҖӮеҫӘзҺҜеҝ…йЎ»и®ҫзҪ®жҳҺзЎ®зҡ„з»ҲжӯўжқЎд»¶пјҡиҝӯд»Јж¬Ўж•°пјҲйҖҡеёёдёҚи¶…иҝҮ 3 ж¬ЎпјүгҖҒжҖ» token йў„з®—пјҢжҲ–дёҖдёӘдёҚдҫқиө–еҗҺз»ӯжЁЎеһӢи°ғз”Ёзҡ„зӢ¬з«ӢеҲӨе®ҡжқЎд»¶гҖӮ

Parallel пјҲе№¶иЎҢ + иҒҡеҗҲпјүгҖӮжүҮеҮәеҲ° N дёӘ workerпјҢеҶҚ reduceгҖӮйҖӮеҗҲеҲҶеқ—ж‘ҳиҰҒгҖҒ self-consistency жҠ•зҘЁгҖҒеӨҡжәҗж•ҙеҗҲгҖӮдёӨжқЎиӯҰе‘Ҡпјҡе№¶иЎҢи°ғз”ЁдјҡжҠҠжҲҗжң¬д№ҳд»Ҙ Nпјӣ reduce йӮЈдёҖжӯҘйҖҡеёёйңҖиҰҒз”ЁжңҖеӨ§зҡ„жЁЎеһӢпјҢеӣ дёәз»јеҗҲжҜ”з”ҹжҲҗжӣҙйҡҫгҖӮ

еҠ д»»дҪ•дёҖз§ҚжЁЎејҸд№ӢеүҚйғҪиҜҘй—®дёҖеҸҘпјҡиҝҷдёҖжӯҘпјҢиғҪдёҚиғҪз”Ёйқһ LLM жңҚеҠЎеҒҡпјҹ еҰӮжһңдёҖдёӘжӯЈеҲҷгҖҒдёҖжқЎ SQLгҖҒдёҖдёӘеҮҪж•°и°ғз”Ёе°ұиғҪжҗһе®ҡпјҢе°ұеҲ«еҸ¬е”ӨжЁЎеһӢгҖӮжҜҸдёҖж¬Ў LLM и°ғз”ЁйғҪдјҡеј•е…ҘйўқеӨ–зҡ„е»¶иҝҹгҖҒжҲҗжң¬е’ҢдёҚзЎ®е®ҡжҖ§гҖӮ

RAG vs еҫ®и°ғ vs Prompt е·ҘзЁӢ#

иҝҷжҳҜд»»дҪ• LLM и®ҫи®ЎиҜ„е®ЎйғҪдјҡиў«й—®еҲ°зҡ„й—®йўҳпјҢзӯ”жЎҲе…¶е®һз®ҖеҚ•пјҡз”ЁжңҖдҫҝе®ңзҡ„гҖҒиғҪжҠҠзјәеҸЈеЎ«дёҠзҡ„йӮЈдёӘе·Ҙе…·гҖӮзјәеҸЈеҲҶдёәдёӨзұ»пјҢйҖҡеёёеҸҜеңЁдёҖеҲҶй’ҹеҶ…е®ҢжҲҗеҲқжӯҘиҜҠж–ӯгҖӮ

еҰӮжһңжЁЎеһӢйЈҺж јжҲ–ж јејҸдёҚеҜ№пјҢдҪҶдәӢе®һеӨ§иҮҙжҳҜеҜ№зҡ„пјҢйӮЈжҳҜиЎҢдёәзјәеҸЈгҖӮе…ҲеҒҡ prompt е·ҘзЁӢпјҡ few-shot зӨәдҫӢгҖҒ CoT и„ҡжүӢжһ¶гҖҒз»“жһ„еҢ–иҫ“еҮәзәҰжқҹгҖӮ Schulhoff зӯүдәәзҡ„ The Prompt Report вҶ— пјҲ2024пјүзі»з»ҹж•ҙзҗҶдәҶйӮЈдәӣи·ЁжЁЎеһӢд»Јйҷ…д»Қз„¶жңүж•Ҳзҡ„еҒҡжі•гҖӮеҪ“ Prompt е·ҘзЁӢе·Іж— жі•иҝӣдёҖжӯҘжҸҗеҚҮж•Ҳжһңж—¶пјҢеҶҚиҖғиҷ‘еҫ®и°ғвҖ”вҖ”е°Ҫз®Ў LoRA е’Ң DPO жҳҫи‘—йҷҚдҪҺдәҶеҫ®и°ғжҲҗжң¬пјҢе…¶иҝӯд»Јж•ҲзҺҮд»ҚиҝңдҪҺдәҺ Prompt и°ғдјҳгҖӮ

еҰӮжһңжЁЎеһӢж №жң¬дёҚзҹҘйҒ“вҖ”вҖ”дҪ зҡ„дә§е“ҒгҖҒдҪ зҡ„е®ўжҲ·гҖҒдҪ дёҠдёӘеӯЈеәҰзҡ„ж•…йҡңвҖ”вҖ”йӮЈжҳҜзҹҘиҜҶзјәеҸЈпјҢй»ҳи®Өзӯ”жЎҲе°ұжҳҜ RAGгҖӮ RAG еңЁдёүж–№йқўжҳҫи‘—дјҳдәҺ Prompt е·ҘзЁӢе’Ңеҫ®и°ғпјҡж”ҜжҢҒжәҜжәҗпјҲзӯ”жЎҲеҸҜе®Ўи®ЎпјүгҖҒжӣҙж–°иҝ…йҖҹпјҲж–ҮжЎЈеҜје…ҘвҶ’йҮҚж–° embeddingвҶ’дёҠзәҝпјҢе…ЁзЁӢж•°еҲҶй’ҹпјүгҖҒиҜӯж–ҷжү©еұ•иғҪеҠӣиҝңи¶…еҫ®и°ғжһҒйҷҗгҖӮ Lewis зӯүдәәйӮЈзҜҮ RAG еҺҹе§Ӣи®әж–Ү вҶ— пјҲNeurIPS 2020пјүиҮід»Ҡд»ҚжҳҜжңҖе№ІеҮҖзҡ„еҪўејҸеҢ–е®ҡд№үпјӣ Karpukhin зӯүдәәзҡ„ Dense Passage Retrieval вҶ— пјҲEMNLP 2020пјүеҲҷз»ҷеҮәдәҶдҪ зңҹжӯЈдјҡйғЁзҪІзҡ„йӮЈз§Қ bi-encoder жЈҖзҙўеҷЁгҖӮ

RAG + еҫ®и°ғ иҝҷз§Қз»„еҗҲжӢіжҳҜзңҹе®һеӯҳеңЁзҡ„пјҢдҪҶз”Ёеҫ—дёҚеӨҡгҖӮеҸӘеңЁд»ҘдёӢеҮ з§Қжғ…еҶөжүҚеҖјеҫ—пјҡиҜӯж–ҷзЁіе®ҡгҖҒйўҶеҹҹзӘ„гҖҒиҙЁйҮҸжҜ”иҝӯд»ЈйҖҹеәҰйҮҚиҰҒгҖӮдҫӢеӯҗпјҡеҚ•дёҖдё“з§‘зҡ„еҢ»еӯҰй—®зӯ”гҖҒеӣәе®ҡ schema зҡ„еҶ…йғЁе·Ҙе…·гҖҒејәзӣ‘з®ЎдёӢзҡ„йҮ‘иһҚж‘ҳиҰҒгҖӮиҝҷз§ҚеңәжҷҜдёӢиҰҒд№Ҳеҫ®и°ғжЈҖзҙўеҷЁгҖҒиҰҒд№Ҳеҫ®и°ғйҳ…иҜ»еҷЁпјҢ RAG д»Қз„¶иҙҹиҙЈвҖңж–°йІңеәҰвҖқпјҢиҝҗз»ҙеӨҚжқӮеәҰиҮӘе·ұжүӣгҖӮ

еҮ жқЎе®һжҲҳз»ҸйӘҢпјҡ

  • дёҖдёӘдёӢеҚҲжҗ“еҮәжқҘзҡ„вҖңжңҙзҙ  RAGвҖқпјҢеңЁйҷӨдәҶжңҖе®№жҳ“зҡ„жҹҘиҜўд№ӢеӨ–зҡ„йўҳзӣ®дёҠеҲҶж•°йғҪдјҡеҫҲйҡҫзңӢгҖӮеўһеҠ еӨҚжқӮеәҰзҡ„еҗҲзҗҶйЎәеәҸеӨ§иҮҙжҳҜпјҡжӣҙеҘҪзҡ„ chunking вҶ’ ж··еҗҲжЈҖзҙўпјҲdense + BM25пјү вҶ’ cross-encoder йҮҚжҺ’ вҶ’ query ж”№еҶҷ / HyDEгҖӮжҜҸеҠ дёҖжӯҘйғҪдјҡжҺЁй«ҳе»¶иҝҹе’ҢжҲҗжң¬пјҢжүҖд»ҘеҸӘжңүжҢҮж Үе‘ҠиҜүдҪ еӨұиҙҘжЁЎејҸзЎ®е®һеҮәзҺ°дәҶпјҢжүҚеҠ гҖӮ
  • й•ҝдёҠдёӢж–ҮжЁЎеһӢе№¶дёҚиғҪеҸ–д»Ј RAGгҖӮ Liu зӯүдәәйӮЈзҜҮ Lost in the Middle вҶ— пјҲTACL 2024пјүе·Із»ҸиҜҙжё…жҘҡпјҡе“ӘжҖ•жҳҜ 128k token зҡ„зӘ—еҸЈпјҢжЁЎеһӢеҜ№ејҖеӨҙе’Ңз»“е°ҫзҡ„жіЁж„ҸеҠӣд№ҹиҝңеӨҡдәҺдёӯй—ҙж®өгҖӮжЈҖзҙўеңЁзІҫеәҰе’ҢжҲҗжң¬дёҠд»Қз„¶жҳҜиөўзҡ„гҖӮ
  • дё»жөҒй—ӯжәҗж——иҲ°жЁЎеһӢйҖҡеёёдёҚж”ҜжҢҒеҫ®и°ғгҖӮжӯӨж—¶еә”йҖүжӢ©ејҖжәҗжЁЎеһӢпјҲеҰӮ LlamaгҖҒ QwenгҖҒ MistralпјүиҝӣиЎҢеҫ®и°ғпјҢиҖҢйңҖж——иҲ°иғҪеҠӣзҡ„иҜ·жұӮеҲҷйҖҡиҝҮ API и°ғз”ЁгҖӮз¬¬ 2 иҠӮ йӮЈдёӘ branch жЁЎејҸпјҢе°ұжҳҜи®©иҝҷз§ҚвҖңеҲҶиҖҢжІ»д№ӢвҖқеңЁе·ҘзЁӢдёҠи·‘еҫ—иө·жқҘзҡ„е…ій”®гҖӮ

з”ҹдә§зҺҜеўғзҡ„йғЁзҪІжӢ“жү‘#

з”ҹдә§зә§ LLM жңҚеҠЎзҡ„ж•ҙдҪ“жһ¶жһ„е№¶ж— зү№ж®Ҡд№ӢеӨ„пјҢе…ій”®еңЁдәҺиҜҶеҲ«е“Әдәӣз»„д»¶жүҝжӢ…ж ёеҝғиҙҹиҪҪгҖӮд»Һе·ҰеҫҖеҸіиҜ»пјҡ

Edge пјҲиҫ№зјҳпјүгҖӮ CDN еҠ  WAFпјҢйҷҗжөҒиҰҒеӨҹзӢ гҖӮ LLM жҺҘеҸЈжһҒжҳ“иў«ж»Ҙз”ЁвҖ”вҖ”жҜҸж¬Ўи°ғз”ЁеқҮдә§з”ҹзӣҙжҺҘжҲҗжң¬вҖ”вҖ”еӣ жӯӨиҫ№зјҳйҷҗжөҒзӯ–з•Ҙеҝ…йЎ»жҜ”жҷ®йҖҡ API жӣҙдёҘж јпјҢе№¶еҹәдәҺз”ЁжҲ·иә«д»ҪпјҲиҖҢйқһд»… IP ең°еқҖпјүе®һж–ҪгҖӮ

API GatewayгҖӮи®ӨиҜҒгҖҒжҢүз§ҹжҲ·й…ҚйўқгҖҒиҜ·жұӮзә§е®Ўи®ЎгҖҒеҶіе®ҡдёҖдёӘиҜ·жұӮиө°еҲ°е“ӘдёӘзүҲжң¬е“ӘдёӘеә”з”Ёзҡ„и·Ҝз”ұи§„еҲҷгҖӮиҝҷдёҖеұӮзҡ„й…ҚйўқеҚ•дҪҚеә”иҜҘжҳҜ token жҲ–зҫҺе…ғпјҢиҖҢдёҚжҳҜж¬Ўж•°вҖ”вҖ”дёҖж¬Ў 10 дёҮ token зҡ„жҹҘиҜўеҸҜиғҪжҜ” 1000 ж¬ЎзҹӯжҹҘиҜўйғҪиҙөгҖӮ

еә”з”ЁжңҚеҠЎгҖӮж— зҠ¶жҖҒзҡ„ FastAPI / Node жңҚеҠЎпјҢжҢӮеңЁ HPA еҗҺйқўгҖӮе®ғд»¬иҙҹиҙЈи·‘з¬¬ 2 иҠӮ йҮҢзҡ„е·ҘдҪңжөҒпјҢдҪҶдёҚзӣҙжҺҘи°ғжЁЎеһӢвҖ”вҖ”е®ғд»¬и°ғ LLM GatewayгҖӮ

иҜӯд№үзј“еӯҳгҖӮеқҗеңЁ LLM Gateway еүҚйқўпјҢжҳҜж•ҙеҘ—жһ¶жһ„дёӯжҠ•иө„еӣһжҠҘзҺҮжңҖй«ҳзҡ„з»„д»¶гҖӮдёӨеұӮпјҡз¬¬дёҖеұӮжҳҜи§„иҢғеҢ–еҗҺзҡ„зІҫзЎ®еҢ№й…Қзј“еӯҳпјӣз¬¬дәҢеұӮжҳҜжҠҠз”ЁжҲ·й—®йўҳ embed д№ӢеҗҺжҹҘеҗ‘йҮҸзҙўеј•пјҢдҪҷејҰзӣёдјјеәҰи¶…иҝҮйҳҲеҖјпјҲ0.95 жҳҜдёҖдёӘз«ҷеҫ—дҪҸзҡ„й»ҳи®ӨеҖјпјүе°ұзӣҙжҺҘе‘ҪдёӯгҖӮеңЁе®ўжңҚгҖҒ FAQ еҸҠж•°жҚ®еҲҶжһҗзұ»еңәжҷҜдёӯпјҢиҜӯд№үзј“еӯҳе‘ҪдёӯзҺҮйҖҡеёёеҸҜиҫҫ 30вҖ“60%пјӣиҖҢеңЁејҖж”ҫжҖ§еҶ…е®№з”ҹжҲҗеңәжҷҜдёӯпјҢ 5вҖ“10% е·ІеұһиүҜеҘҪиЎЁзҺ°гҖӮ

LLM GatewayгҖӮиҙҹиҙЈжЁЎеһӢйҖүжӢ©гҖҒйҷҚзә§пјҲдё»жЁЎеһӢ 5xx жҲ–иў«йҷҗжөҒж—¶иҮӘеҠЁйҷҚеҲ°еӨҮжЁЎеһӢпјүгҖҒжҢүиҜ·жұӮ token йў„з®—жҺ§еҲ¶гҖҒжөҒејҸиҫ“еҮәгҖӮжҠҠиҝҷдёҖж‘ҠдәӢйӣҶдёӯеңЁдёҖдёӘжңҚеҠЎйҮҢпјҢиҖҢдёҚжҳҜж•ЈиҗҪеңЁеҗ„еә”з”Ёд»Јз ҒйҮҢпјҢжҳҜи®©дҪ вҖңдёҖдёӘдёӢеҚҲе°ұиғҪжҚўжЁЎеһӢдҫӣеә”е•ҶвҖқиҖҢдёҚжҳҜвҖңдёҖдёӘеӯЈеәҰйғҪжҚўдёҚе®ҢвҖқзҡ„е…ій”®гҖӮ

жЁЎеһӢжұ гҖӮе»әи®®дёүжЎЈпјҡиҮӘжңү GPU дёҠзҡ„е°ҸжЁЎеһӢпјҲеҗғдёӢжңҖй«ҳйў‘гҖҒжңҖе®№жҳ“зҡ„иҜ·жұӮпјүгҖҒејҖжәҗдёӯеһӢжЁЎеһӢпјҲйҖҡз”Ёд»»еҠЎпјүгҖҒж——иҲ° API пјҲзЎ¬йӘЁеӨҙпјүгҖӮжЁЎеһӢж—Ғиҫ№иҝҳиҰҒй…ҚдёҖдёӘд»Јз Ғ / е·Ҙе…·жІҷз®ұвҖ”вҖ”з»қеҜ№дёҚиҰҒжҠҠжЁЎеһӢз”ҹжҲҗзҡ„д»Јз Ғи·‘еңЁдҪ иҮӘе·ұзҡ„еә”з”ЁиҝӣзЁӢйҮҢгҖӮ

ејӮжӯҘйҖҡйҒ“гҖӮдёҖдёӘж¶ҲжҒҜйҳҹеҲ—еҠ дёҖз»„ workerпјҢеӨ„зҗҶжүҖжңүдёҚйңҖиҰҒеҗҢжӯҘиҝ”еӣһзҡ„дәӢпјҡж–ҮжЎЈж‘„е…ҘдёҺ embeddingгҖҒе®ҡж—¶иҜ„дј°гҖҒжҲҗжң¬дёҺз”ЁйҮҸжұҮжҖ»гҖҒжү№йҮҸз”ҹжҲҗгҖҒеҶҚи®ӯз»ғгҖӮ

йҒҘжөӢгҖӮжүҖжңүзӣ’еӯҗйғҪеҗҗ OpenTelemetry trace е’Ң Prometheus жҢҮж ҮгҖӮз¬¬ 6 иҠӮ з»Ҷи®ІгҖӮ

дёҖдёӘеёёи§Ғзҡ„иё©еқ‘пјҡи·іиҝҮ LLM GatewayпјҢи®©еә”з”Ёд»Јз ҒзӣҙжҺҘи°ғжЁЎеһӢгҖӮеңЁдҪ йңҖиҰҒжҚўдҫӣеә”е•ҶгҖҒеҠ йҷҚзә§гҖҒеҠ  token йў„з®—зҡ„йӮЈеӨ©д№ӢеүҚпјҢиҝҷйғҪжІЎй—®йўҳвҖ”вҖ”дҪҶйӮЈеӨ©д№ӢеҗҺпјҢжҜҸдёӘжңҚеҠЎйғҪеҫ—ж”№дёҖйҒҚгҖӮз¬¬дёҖеӨ©е°ұжҠҠ Gateway е»әиө·жқҘпјҢе“ӘжҖ•е®ғеҗҺйқўеҸӘжҢӮдёҖе®¶дҫӣеә”е•ҶгҖӮ

жҲҗжң¬дјҳеҢ–#

LLM жҲҗжң¬пјҢжҳҜеҜјиҮҙвҖңејҖеҸ‘зҺҜеўғи·‘еҫ—ж¬ўгҖҒиў«иҙўеҠЎдёҖеҲҖз ҚжҺүвҖқзҡ„еӨҙеҸ·еҺҹеӣ гҖӮиҝҷйҮҢжңүе…ӯж №жқ жқҶпјҢиҜ·жҢүд»ҘдёӢйЎәеәҸжӢүпјҢжҜҸдёҖж №йғҪе»әз«ӢеңЁеүҚйқўд№ӢдёҠгҖӮ

  1. Prompt еҺӢзј©гҖӮзІҫз®Җ Prompt еҸҜеңЁдёүж–№йқўйҷҚдҪҺжҲҗжң¬пјҡеҮҸе°‘иҫ“е…Ҙ token ж•°йҮҸгҖҒйҷҚдҪҺйҰ– token е»¶иҝҹгҖҒзј“и§ЈеҹәзЎҖи®ҫж–ҪеҺӢеҠӣпјҲеӣ  Attention и®Ўз®—еӨҚжқӮеәҰдёә O(nВІ)пјүгҖӮзІҫз®Җзі»з»ҹ Prompt дёӯзҡ„еҶ—дҪҷдҝЎжҒҜпјҢдјҳе…ҲдҪҝз”Ё ID иҖҢйқһе®һдҪ“еҗҚз§°пјҢе№¶иЈҒеүӘжЁЎеһӢе·Ійҡҫд»Ҙжңүж•ҲеҲ©з”Ёзҡ„еҺҶеҸІеҜ№иҜқгҖӮ
  2. е°ҸжЁЎеһӢи·Ҝз”ұгҖӮз¬¬ 2 иҠӮ зҡ„ branch жЁЎејҸпјҡжҜҸдёӘиҜ·жұӮе…ҲеҲҶзұ»пјҢеҸӘжҠҠзЎ¬йӘЁеӨҙйҖҒз»ҷиҙөжЁЎеһӢгҖӮж„ҸеӣҫеҲҶзұ»еҸҜз”ұиҪ»йҮҸзә§ 7B жЁЎеһӢе®ҢжҲҗпјҢеҚ•ж¬ЎжҺЁзҗҶжҲҗжң¬йҖҡеёёдҪҺдәҺ 0.01 зҫҺе…ғгҖӮ
  3. иҜӯд№үзј“еӯҳгҖӮз¬¬ 4 иҠӮ и®ІиҝҮгҖӮе…¶д»·еҖјеҸ–еҶідәҺе‘ҪдёӯзҺҮпјҢеӣ жӯӨйңҖеңЁзі»з»ҹдёҠзәҝйҰ–ж—ҘеҚіе®ҢжҲҗзӣ‘жҺ§еҹӢзӮ№гҖӮ
  4. жү№еӨ„зҗҶгҖӮ OpenAIгҖҒ AnthropicгҖҒеҮ д№ҺжүҖжңүеҺӮе•ҶйғҪжңү batch APIпјҢд»·ж јеӨ§зәҰжҳҜеҗҢжӯҘи°ғз”Ёзҡ„дёҖеҚҠпјҢ SLA 24 е°Ҹж—¶гҖӮжүҖжңүйқһе®һж—¶д»»еҠЎвҖ”вҖ”еҰӮеӨңй—ҙжү№йҮҸж‘ҳиҰҒгҖҒеҗ‘йҮҸеҢ–жӣҙж–°гҖҒжү№йҮҸиҜ„дј°вҖ”вҖ”еқҮеә”йҖҡиҝҮ Batch API жү§иЎҢгҖӮ
  5. INT8 / INT4 йҮҸеҢ–гҖӮиҮӘжүҳз®ЎжЁЎеһӢзҡ„иҜқпјҢGPTQ вҶ— пјҲFrantar зӯүпјҢ ICLR 2023пјүиҝҷдёҖзұ»жҠҖжңҜеңЁеӨ§еӨҡж•°д»»еҠЎдёҠиғҪеҒҡеҲ°зәҰ 2Г— зҡ„еҶ…еӯҳдёҺе»¶иҝҹиҠӮзңҒгҖҒиҙЁйҮҸжҚҹеӨұеҸҜеҝҪз•ҘгҖӮиҝҷжҳҜ GPU иө„жәҗжҲҗжң¬еҚ дё»еҜјж—¶еә”еҗҜз”Ёзҡ„дјҳеҢ–жүӢж®өгҖӮ
  6. и°ҲеҲӨжҲ–иҮӘжүҳз®ЎгҖӮеңЁеҺҳжё…иҮӘиә«жөҒйҮҸзү№еҫҒеҗҺпјҢжүҝиҜәж¶Ҳиҙ№жҠҳжүЈдёҺејҖжәҗжЁЎеһӢиҮӘжүҳз®ЎжүҚе…·еӨҮеҸҜиЎҢжҖ§гҖӮе’Ңж——иҲ° API зҡ„зӣҲдәҸе№іиЎЎзӮ№пјҢжҢүе·ҘдҪңиҙҹиҪҪз®—еӨ§жҰӮеңЁжҜҸжңҲ 5000 дёҮ вҖ“ 1 дәҝ token д№Ӣй—ҙгҖӮ

еҸідҫ§ж•ЈзӮ№еӣҫеұ•зӨәдәҶиҙЁйҮҸдёҺжҲҗжң¬зҡ„жқғиЎЎиҫ№з•ҢпјҲPareto frontierпјүгҖӮд»Һз”ҹдә§жөҒйҮҸи§ӮеҜҹеҲ°дёүд»¶дәӢпјҡжң¬ең°е°ҸжЁЎеһӢеңЁйҖҡз”Ёд»»еҠЎдёҠеҸҜиҫҫеҲ°ж——иҲ°жЁЎеһӢ 60вҖ“65% зҡ„жҖ§иғҪж°ҙе№іпјҢеңЁеһӮзӣҙйўҶеҹҹд»»еҠЎдёӯз”ҡиҮіжҺҘиҝ‘ 95%пјӣ INT8 йҮҸеҢ–еҸҜдҪҝжҲҗжң¬-жҖ§иғҪжӣІзәҝеҗ‘е·ҰдёӢж–№з§»еҠЁпјҲжҲҗжң¬йҷҚдҪҺгҖҒжҖ§иғҪжҚҹеӨұжһҒе°ҸпјүпјӣеңЁд»»дёҖжЁЎеһӢжЎЈдҪҚдёҠеҸ еҠ  RAGпјҢйҖҡеёёжҳҜдёҚжӣҙжҚўдҫӣеә”е•ҶеүҚжҸҗдёӢжңҖжҳҫи‘—зҡ„иҙЁйҮҸжҸҗеҚҮжүӢж®өгҖӮз”ЁдҪ иҮӘе·ұзҡ„иҜ„дј°з”»дёҖеј дҪ иҮӘе·ұзҡ„еӣҫпјҢз„¶еҗҺжӢҝжқҘеҒҡи·Ҝз”ұеҶізӯ–гҖӮ

е…ідәҺ benchmark зҡ„дёҖеҸҘеҝ е‘Ҡпјҡе…¬ејҖеҹәеҮҶжөӢиҜ•пјҲMMLUгҖҒ MT-BenchгҖҒ ArenaпјүеҜ№зү№е®ҡдә§е“ҒеңәжҷҜзҡ„жіӣеҢ–иғҪеҠӣжңүйҷҗгҖӮе»әи®®жһ„е»әе°ҸеһӢд»»еҠЎзә§иҜ„дј°йӣҶпјҲ50вҖ“500 жқЎж ·жң¬пјҢйҷ„еҸӮиҖғзӯ”жЎҲжҲ–иҜ„еҲҶж ҮеҮҶпјүпјҢжҜҸж¬ЎжЁЎеһӢеҲҮжҚўжҲ– Prompt и°ғж•ҙеҗҺеқҮйңҖйҮҚж–°иҝҗиЎҢпјӣеә”д»ҘиҮӘжңүиҜ„дј°з»“жһңдёәеҮҶпјҢиҖҢйқһе…¬ејҖеҹәеҮҶеҲҶж•°гҖӮ

еҸҜи§ӮжөӢжҖ§#

ж ҮеҮҶзҡ„еҸҜи§ӮжөӢжҖ§е·Ҙе…·ж ҲвҖ”вҖ”PrometheusгҖҒ OpenTelemetryгҖҒ Loki / Tempo / GrafanaвҖ”вҖ”еҜ№ LLM жңҚеҠЎйғҪйҖӮз”ЁпјҢеҸӘйңҖиҰҒеңЁ Google SRE зҡ„еӣӣйҮ‘дҝЎеҸ· вҶ— пјҲlatencyгҖҒ trafficгҖҒ errorsгҖҒ saturationпјүд№ӢдёҠеҶҚеҠ еӣӣдёӘ LLM дё“еұһдҝЎеҸ·гҖӮ

еҚ•иҜ·жұӮжҲҗжң¬ жҳҜдәӢе®һдёҠзҡ„з¬¬дә”дёӘйҮ‘дҝЎеҸ·гҖӮз”Ё histogram и·ҹиёӘгҖҒеҜ№ p95 жҠҘиӯҰгҖҒжҢүз§ҹжҲ· / жҺҘеҸЈ / жЁЎеһӢжӢҶеҲҶгҖӮиҝҷдёӘжҢҮж ҮдјҡеҸҳжҲҗиҙўеҠЎеңЁзҫӨйҮҢ @ дҪ зҡ„йӮЈжқЎж¶ҲжҒҜгҖӮ

е№»и§үзҺҮгҖӮжҠҪж · 1вҖ“5% зҡ„жөҒйҮҸеҒҡ LLM-as-judgeвҖ”вҖ”и®©дёҖдёӘжӣҙејәзҡ„жЁЎеһӢеҲӨж–ӯиҝҷжқЎеӣһзӯ”жҳҜеҗҰиў«жЈҖзҙўдёҠдёӢж–Үж”Ҝж’‘гҖӮз»қеҜ№еҖјжҳҜеҷӘеЈ°пјҢи¶ӢеҠҝдёҚжҳҜгҖӮ

Groundedness / еј•з”Ёе‘ҪдёӯзҺҮгҖӮеҜ№ RAG зі»з»ҹпјҡеӣһзӯ”йҮҢзҡ„дәӢе®һйҷҲиҝ°йҮҢпјҢжңүеӨҡе°‘иғҪиҝҪеӣһеҲ°дёҖдёӘиў«жЈҖзҙўеҲ°зҡ„ chunkпјҹиҝҷжҳҜз”ЁжҲ·ж„ҹзҹҘе№»и§үзҡ„е…ҲиЎҢжҢҮж ҮпјҢиҖҢдё”и®Ўз®—еҫҲдҫҝе®ңгҖӮ

жҠӨж Ҹи§ҰеҸ‘зҺҮгҖӮжіЁе…ҘиҝҮж»ӨеҷЁгҖҒ PII и„ұж•ҸгҖҒеҶ…е®№е®Ўж ёпјҢи§ҰеҸ‘йў‘зҺҮжҳҜеӨҡе°‘пјҹйҷЎеі°дёҖиҲ¬ж„Ҹе‘ізқҖжңүдәәжүҫеҲ°дәҶж–°з»•иҝҮпјӣзј“ж¶ЁдёҖиҲ¬ж„Ҹе‘ізқҖз”ЁжҲ·иЎҢдёәеҸ‘з”ҹдәҶжјӮз§»гҖҒиҝҮж»Өи§„еҲҷиҜҘйҮҚж–°и°ғгҖӮ

зј“еӯҳе‘ҪдёӯзҺҮдёҺйҷҚзә§зҺҮ ж”¶е°ҫгҖӮзј“еӯҳе‘ҪдёӯзҺҮе‘ҠиҜүдҪ иҜӯд№үзј“еӯҳжҳҜдёҚжҳҜеңЁжҢЈй’ұпјӣеҲҶдҫӣеә”е•Ҷзҡ„йҷҚзә§зҺҮе‘ҠиҜүдҪ з”ЁжҲ·иҝҳжІЎжқҘжҠұжҖЁд№ӢеүҚе“Әе®¶дёҠжёёжӯЈеңЁеҠЈеҢ–гҖӮ

д»»дҪ•зЁҚеҫ®еӨҚжқӮдёҖзӮ№зҡ„е·ҘдҪңжөҒпјҢеҲҶеёғејҸиҝҪиёӘйғҪдёҚжҳҜеҸҜйҖүйЎ№гҖӮDapper и®әж–Ү вҶ— пјҲSigelman зӯүпјҢ 2010пјүиҮід»Ҡд»ҚжҳҜжӯЈзЎ®зҡ„еҝғжҷәжЁЎеһӢгҖӮдёҖдёӘ RAG иҜ·жұӮзҡ„е…ёеһӢ trace еҢ…еҗ« retrieveгҖҒrerankгҖҒcompose_promptгҖҒllm.generate д»ҘеҸҠд»»дҪ•е·Ҙе…·и°ғз”Ёзҡ„ spanвҖ”вҖ”жІЎжңүиҝҷдәӣ spanпјҢвҖңиҝҷдёӘиҜ·жұӮж…ўдәҶвҖқ е°ұжҳҜдёҖеҸҘжІЎжі• debug зҡ„иҜқгҖӮ

дёҖдёӘеҖјеҫ—йҮҮзәізҡ„еҒҡжі•пјҡи®°ж—Ҙеҝ—еүҚеҜ№ prompt е’Ң response еҒҡе“ҲеёҢгҖӮиҝҗиҗҘжүҖйңҖзҡ„еҸҜи§ҒжҖ§йғҪиҝҳеңЁпјҲе“ӘдёӘжЁЎжқҝгҖҒеӣһзӯ”еӨҡй•ҝгҖҒе“ӘдёӘз”ЁжҲ·пјүпјҢдҪҶдҪ дёҚеҶҚйңҖиҰҒж— йҷҗжңҹең°жҠҠз”ЁжҲ·еҺҹж–Үеӯҳиҝӣзі»з»ҹгҖҒеңЁеҗҲи§„дёҠжүҝжӢ…йЈҺйҷ©гҖӮзңҹзҡ„йңҖиҰҒзңӢеҺҹж–Ү debug ж—¶пјҢиө°дёҖжқЎеҚ•зӢ¬зҡ„гҖҒиў«е®Ўи®Ўзҡ„и·Ҝеҫ„гҖӮ

дјҒдёҡйӣҶжҲҗ#

и®ІдёҖеҸҘдёҚйӮЈд№ҲеҘҪеҗ¬зҡ„е®һиҜқпјҡиҜ•зӮ№дёҠдёҚдәҶзәҝпјҢеҮ д№ҺйғҪдёҚжҳҜжЁЎеһӢиҙЁйҮҸзҡ„й”…вҖ”вҖ”жҳҜжЁЎеһӢе‘Ёеӣҙиҝҷе…ӯдёӘзӣ’еӯҗзҡ„й”…гҖӮжҜҸдёӘзӣ’еӯҗйғҪеҜ№еә”дёҖдёӘйҮҮиҙӯж–№дёҖе®ҡдјҡй—®зҡ„й—®йўҳпјҢдҪ еҝ…йЎ»жңүдёҖдёӘз«ҷеҫ—дҪҸзҡ„зӯ”жЎҲгҖӮ

Identity пјҲиә«д»ҪпјүгҖӮ SAML / OIDC еҚ•зӮ№зҷ»еҪ•гҖҒ SCIM иҮӘеҠЁеҢ–з”ЁжҲ·з”ҹе‘Ҫе‘ЁжңҹгҖӮжІЎжңүиҝҷдәӣпјҢжҜҸжҺҘдёҖдёӘе®ўжҲ·йғҪжҳҜдёҖдёӘеҮ е‘Ёзҡ„йЎ№зӣ®гҖӮ

Authorisation пјҲжҺҲжқғпјүгҖӮеҠҹиғҪеұӮз”Ё RBACпјҢж•°жҚ®еұӮз”Ё ABACгҖӮеҜ№ RAG иҖҢиЁҖж„Ҹе‘ізқҖпјҡж–ҮжЎЈзә§еҲ«зҡ„иҝҮж»Өеҝ…йЎ»еңЁжЈҖзҙўеұӮејәеҲ¶жү§иЎҢвҖ”вҖ”ж°ёиҝңдёҚиҰҒзӣёдҝЎвҖңжҲ‘еңЁ prompt йҮҢе‘ҠиҜүжЁЎеһӢдёҚиҰҒжҸҗж–ҮжЎЈ XвҖқгҖӮз§ҹжҲ·зңӢдёҚи§Ғзҡ„ж–ҮжЎЈпјҢжЈҖзҙўеҷЁе°ұдёҚиғҪиҝ”еӣһгҖӮ

Data residency пјҲж•°жҚ®й©»з•ҷпјүгҖӮеӯҳеӮЁе’ҢжҺЁзҗҶйғҪиҰҒж”ҜжҢҒеҢәеҹҹи·Ҝз”ұгҖӮж¬§зӣҹе®ўжҲ·иҰҒжұӮж¬§зӣҹжүҳз®Ўзҡ„жЁЎеһӢ + ж¬§зӣҹжүҳз®Ўзҡ„еҗ‘йҮҸеә“пјҢжІЎжңүи®Ёд»·иҝҳд»·з©әй—ҙгҖӮиҝҷжҳҜдёҖдёӘеҫҲйҡҫдәӢеҗҺиЎҘж•‘зҡ„жӢ“жү‘еҶізӯ–пјҢиҰҒеңЁз¬¬дёҖдёӘж¬§жҙІе®ўжҲ·иҗҪең°д№ӢеүҚе°ұе®ҡдёӢжқҘгҖӮ

BYOK дёҺ BYOMгҖӮе®ўжҲ·иҮӘе·ұ KMS йҮҢзҡ„еҜҶй’ҘпјӣеҸҜйҖүзҡ„иҮӘжүҳз®ЎжЁЎеһӢз«ҜзӮ№вҖ”вҖ”йҖҡиҝҮдҪ зҡ„е№іеҸ°и·Ҝз”ұпјҢдҪҶжҳҺж–Үж•°жҚ®д»ҺдёҚзҰ»ејҖе®ўжҲ· VPCгҖӮз¬¬ 4 иҠӮ йӮЈдёӘ LLM Gateway жҳҜ BYOM еңЁе·ҘзЁӢдёҠи·‘еҫ—йҖҡзҡ„еүҚжҸҗгҖӮ

Audit дёҺ DLPгҖӮдёҖд»ҪдёҚеҸҜзҜЎж”№зҡ„вҖңи°ҒеңЁд»Җд№Ҳж—¶еҖҷеҒҡдәҶд»Җд№ҲвҖқе®Ўи®Ўж—Ҙеҝ—пјҢеҠ дёҠдёҖйҒ“еңЁе…ҘеҸӮе’ҢеҮәеҸӮдёӨз«ҜеҒҡ PII и„ұж•Ҹзҡ„ DLPгҖӮдё»дҪ“и®ҝй—®иҜ·жұӮпјҲDSARпјүе·Ҙе…·вҖ”вҖ”иғҪжүҫеҮәе№¶еҲ йҷӨжҹҗдёӘз”ЁжҲ·зҡ„жүҖжңүж•°жҚ®вҖ”вҖ”жҳҜ GDPR е’ҢеҘҪеҮ жқЎзҫҺеӣҪе·һжі•зҡ„зЎ¬иҰҒжұӮпјӣж•°жҚ®жЁЎеһӢи®ҫи®Ўзҡ„ж—¶еҖҷе°ұиҰҒжҠҠе®ғеҒҡжҲҗдёҖеҸҘ SQL иғҪжҗһе®ҡпјҢиҖҢдёҚжҳҜдёҖж¬ЎиҖғеҸӨжҢ–жҺҳгҖӮ

Compliance пјҲеҗҲи§„пјүгҖӮ SOC 2 Type II жҳҜ B2B SaaS зҡ„е…ҘеңәеҲёгҖӮ ISO 27001гҖҒ HIPAAгҖҒ FedRAMP еҗ„иҮӘжү“ејҖзү№е®ҡеёӮеңәгҖӮе®ғд»¬йғҪдёҚжҳҜеӯӨз«Ӣзҡ„жҠҖжңҜйЎ№зӣ®пјҢиҖҢжҳҜиҜҒжҚ®ж”¶йӣҶйЎ№зӣ®вҖ”вҖ”жһ¶жһ„еҝ…йЎ»иғҪж’‘иө·иҝҷдәӣиҜҒжҚ®гҖӮеҘҪж¶ҲжҒҜжҳҜпјҡеүҚйқўдә”дёӘзӣ’еӯҗеҒҡжүҺе®һдәҶпјҢи®ӨиҜҒеӨ§йғЁеҲҶе°ұжҳҜж–Үд№Ұе·ҘдҪңгҖӮ
