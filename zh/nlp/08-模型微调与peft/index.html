  • дёәд»Җд№Ҳ еңЁеӨ§иҜӯиЁҖжЁЎеһӢпјҲLLMпјүж—¶д»ЈпјҢе…ЁйҮҸеҫ®и°ғжҳҜдёҖз§Қиө„жәҗжөӘиҙ№вҖ”вҖ”иҝҮеҸӮж•°еҢ–зҺ°иұЎдёҺеҶ…еңЁдҪҺз§©жҖ§зҡ„еҗҜзӨә
  • LoRA зҡ„е·ҘдҪңеҺҹзҗҶпјҡеҲҶи§Је…¬ејҸ $\Delta W = BA$ зҡ„еҗ«д№үгҖҒдёәдҪ• $B$ зҡ„еҲқе§ӢеҖји®ҫдёәйӣ¶гҖҒд»ҘеҸҠ $\alpha/r$ зј©ж”ҫеҰӮдҪ•еҪұе“Қжңүж•ҲеӯҰд№ зҺҮ
  • QLoRA зҡ„ж ёеҝғжҠҖжңҜпјҡ NF4 йҮҸеҢ–еҺҹзҗҶгҖҒеҸҢйҮҚйҮҸеҢ–жңәеҲ¶гҖҒеҲҶйЎөдјҳеҢ–еҷЁзҡ„и®ҫи®ЎжҖқи·ҜпјҢд»ҘеҸҠжҳҫеӯҳеҚ з”Ёзҡ„зІҫзЎ®дј°з®—ж–№жі•
  • Adapter е’Ң Prefix-Tuning зҡ„еә”з”Ёпјҡе®ғд»¬еңЁ Transformer жЁЎеқ—дёӯзҡ„е…·дҪ“дҪҚзҪ®гҖҒеңЁе“ӘдәӣеңәжҷҜдёӢиЎЁзҺ°дјҳејӮгҖҒеҸҲеңЁе“Әдәӣжғ…еҶөдёӢеӯҳеңЁдёҚи¶і
  • е®һйҷ…е·ҘзЁӢдёӯзҡ„еҶізӯ–зӮ№пјҡеҰӮдҪ•йҖүжӢ©еҗҲйҖӮзҡ„з§©гҖҒзӣ®ж ҮжЁЎеқ—зҡ„йҖүжӢ©зӯ–з•ҘгҖҒеӨҡ LoRA жңҚеҠЎзҡ„е®һзҺ°ж–№жі•гҖҒд»ҘеҸҠйҖҡиҝҮжҢҮд»Өеҫ®и°ғе’ҢеҹәдәҺдәәзұ»еҸҚйҰҲзҡ„ејәеҢ–еӯҰд№ пјҲRLHFпјүиҝӣиЎҢжЁЎеһӢеҜ№йҪҗзҡ„жңҖдҪіе®һи·ө

  • дәҶи§Ј GPT йЈҺж јзҡ„и§Јз ҒеҷЁеҺҹзҗҶпјҲеҸӮиҖғ з¬¬ 6 йғЁеҲҶ пјү

дёәд»Җд№ҲдёҚзӣҙжҺҘе…ЁйҮҸеҫ®и°ғпјҹ#

жҳҫеӯҳејҖй”Җзҡ„зҺ°е®һ#

е…ЁйҮҸеҫ®и°ғж„Ҹе‘ізқҖжүҖжңүеҸӮж•°йғҪиў«и§ЈеҶ»пјҢжҜҸдёӘжўҜеәҰйғҪиҰҒеӯҳеӮЁпјҢиҖҢдјҳеҢ–еҷЁпјҲйҖҡеёёжҳҜ AdamWпјүиҝҳдјҡдёәжҜҸдёӘеҸӮж•°йўқеӨ–з»ҙжҠӨдёӨдёӘ fp32 зҡ„зҠ¶жҖҒзј“еҶІеҢәгҖӮд»Ҙ 7B жЁЎеһӢеңЁж··еҗҲзІҫеәҰи®ӯз»ғдёӢзҡ„еҚ•жӯҘжҳҫеӯҳеҚ з”ЁдёәдҫӢпјҡ

еҶ»з»“еҫ®и°ғвҖ”вҖ”ж•Ҳжһңжңүйҷҗзҡ„еҹәзәҝж–№жі•#

еңЁ PEFT еҮәзҺ°еүҚпјҢжңҖз®ҖеҚ•зҡ„иҠӮзңҒжҲҗжң¬ж–№ејҸжҳҜеҶ»з»“жЁЎеһӢдё»дҪ“пјҢеҸӘи®ӯз»ғеҲҶзұ»еӨҙпјҢжҲ–д»…и§ЈеҶ»йЎ¶йғЁеҮ еұӮгҖӮдҫӢеҰӮпјҡ

еҰӮдҪ•йҖүжӢ©з§© $r$ #

е»әи®®д»Һ $r = 16$ ејҖе§ӢгҖӮд»ҺдёҠеӣҫеҸідҫ§еҸҜи§ҒпјҢеӨ§еӨҡж•°д»»еҠЎзҡ„ж”¶зӣҠжӢҗзӮ№иҗҪеңЁ $r = 8$ еҲ° $r = 32$ д№Ӣй—ҙпјҡ

  • з®ҖеҚ•еҲҶзұ»д»»еҠЎпјҡ$r = 8$ еҹәжң¬йҘұе’Ң
  • д»Јз Ғз”ҹжҲҗгҖҒйҖ»иҫ‘жҺЁзҗҶпјҡеҸҜеҸ—зӣҠдәҺ $r = 32$ з”ҡиҮі $r = 64$
  • йўҶеҹҹйҖӮй…Қд»»еҠЎпјҲеҰӮеҢ»з–—гҖҒжі•еҫӢпјүпјҡйҖҡеёё $r = 16$ вҖ“$32$ жӣҙеҗҲйҖӮ

иӢҘдёҚзЎ®е®ҡпјҢеҸҜеңЁ $r \in \{8, 16, 32\}$ дёӯеҒҡе°ҸиҢғеӣҙжҗңзҙўпјҢйҖүжӢ©иғҪй—ӯеҗҲйӘҢиҜҒйӣҶзӣ®ж ҮжҢҮж Үе·®и·қзҡ„жңҖе°ҸеҖјеҚіеҸҜгҖӮ

еҜ№йҪҗпјҡжҢҮд»Өеҫ®и°ғдёҺ RLHF#

PEFT жҳҜжқ жқҶпјҢиҖҢеҜ№йҪҗжүҚжҳҜдҪ йҖҡеёёжғіж’¬еҠЁзҡ„зӣ®ж ҮгҖӮзҺ°д»ЈеӨ§иҜӯиЁҖжЁЎеһӢзҡ„еҗҺи®ӯз»ғжөҒзЁӢдё»иҰҒз”ұдёӨдёӘйҳ¶ж®өдё»еҜјпјҡ

жңүзӣ‘зқЈжҢҮд»Өеҫ®и°ғгҖӮеңЁ (instruction, response) ж•°жҚ®еҜ№дёҠеҫ®и°ғеҹәзЎҖжЁЎеһӢпјҢдҪҝе…¶еӯҰдјҡйҒөеҫӘдәәзұ»зј–еҶҷзҡ„жҸҗзӨәгҖӮиҝҷдёҖйҳ¶ж®өпјҢиҙЁйҮҸиҝңиғңж•°йҮҸпјҡ1KвҖ“10K жқЎзІҫеҝғзӯӣйҖүзҡ„ж ·жң¬пјҢеҫҖеҫҖдјҳдәҺ 10 дёҮжқЎдј—еҢ…ж•°жҚ®гҖӮпјҲZhou зӯүдәәеңЁ 2023 е№ҙзҡ„ LIMA и®әж–Үдёӯз”ҡиҮіиҜҒжҳҺпјҢд»…з”Ё 1K жқЎй«ҳиҙЁйҮҸж ·жң¬пјҢе°ұиғҪи®©дёҖдёӘ 65B жЁЎеһӢиЎЁзҺ°еҮәиүІгҖӮпјү

class RewardModel(nn.Module):
    def __init__(self, base):
        super().__init__()
        self.base = base
        self.head = nn.Linear(base.config.hidden_size, 1)

    def forward(self, input_ids, attention_mask=None):
        out = self.base(input_ids=input_ids, attention_mask=attention_mask)
        last = out.last_hidden_state[:, -1, :]
        return self.head(last).squeeze(-1)

еғҸ vLLM е’Ң S-LoRA иҝҷзұ»жЎҶжһ¶жӣҙиҝӣдёҖжӯҘпјҡйҖҡиҝҮе ҶеҸ  LoRA еўһйҮҸпјҢеңЁеҚ•ж¬ЎеүҚеҗ‘дј ж’ӯдёӯжү№йҮҸеӨ„зҗҶжҢҮеҗ‘дёҚеҗҢйҖӮй…ҚеҷЁзҡ„иҜ·жұӮгҖӮжҳҫеӯҳдёӯеҸӘйңҖдёҖд»Ҫ 7B еҹәзЎҖжЁЎеһӢпјҢеҚіеҸҜй«ҳж•ҲжңҚеҠЎж•°зҷҫдёӘеҫ®и°ғзүҲжң¬гҖӮ

еёёи§Ғй—®йўҳ#

д»Җд№Ҳжғ…еҶөдёӢйҖӮеҗҲе…ЁйҮҸеҫ®и°ғпјҹ#

еҪ“дҪ жӢҘжңүе……и¶із®—еҠӣгҖҒвүҘ10 дёҮжқЎй«ҳиҙЁйҮҸж ·жң¬пјҢдё”еҜ№жҜҸдёӘе°Ҹж•°зӮ№зҡ„жҖ§иғҪжҸҗеҚҮйғҪй”ұй“ўеҝ…иҫғж—¶вҖ”вҖ”жҜ”еҰӮеҹәеә§жЁЎеһӢеҺӮе•ҶеҸ‘еёғж——иҲ°зә§жҢҮд»ӨжЁЎеһӢгҖӮеҜ№е…¶д»–дәәиҖҢиЁҖпјҡLoRA жҲ– QLoRA и¶ізҹЈгҖӮ

LoRA зҡ„з§©иҜҘеҰӮдҪ•йҖүжӢ©пјҹ#

д»Һ 16 ејҖе§ӢгҖӮиӢҘеҮәзҺ°ж¬ жӢҹеҗҲпјҲи®ӯз»ғжҚҹеӨұй«ҳдҪҚе№іеҸ°пјүпјҢеҸҜеўһиҮі 32 жҲ– 64пјӣиӢҘж•°жҚ®е°‘дё”иҝҮжӢҹеҗҲпјҢеҲҷйҷҚиҮі 4вҖ“8гҖӮеҲҶзұ»д»»еҠЎйҖҡеёё 8 е°ұеӨҹдәҶгҖӮ

еә”иҜҘи°ғж•ҙе“ӘдәӣжЁЎеқ—пјҹ#

q_proj е’Ң v_proj иғҪеёҰжқҘзәҰ 80% зҡ„ж”¶зӣҠгҖӮеҠ дёҠ k_proj е’Ң o_proj еҸҜиҰҶзӣ–еү©дҪҷйғЁеҲҶгҖӮеҸӘжңүеңЁз”ҹжҲҗд»»еҠЎеҚ дё»еҜјдё”жҳҫеӯҳе…Ғи®ёж—¶пјҢжүҚиҖғиҷ‘еҠ е…Ҙ FFN дёүд»¶еҘ—пјҲgate/up/down_projпјүпјҢеӣ дёәиҝҷдјҡдҪҝеҸҜи®ӯз»ғеҸӮж•°еўһеҠ дёүеҖҚгҖӮ

LoRA дјҡеҪұе“ҚжҺЁзҗҶйҖҹеәҰеҗ—пјҹ#

еҗҲе№¶еҗҺе®Ңе…ЁдёҚеҪұе“ҚгҖӮжңӘеҗҲе№¶еүҚпјҢжҜҸеұӮдјҡеӨҡдёҖж¬Ўеҫ®е°Ҹзҡ„зҹ©йҳөд№ҳжі•вҖ”вҖ”иҷҪеҸҜеҝҪз•ҘпјҢдҪҶзЎ®е®һеӯҳеңЁгҖӮ

QLoRA зҡ„иҙЁйҮҸдёӢйҷҚжҳҺжҳҫеҗ—пјҹ#

еңЁж ҮеҮҶеҹәеҮҶдёҠпјҢзӣёжҜ” fp16 LoRAпјҢQLoRA йҖҡеёёд»…дёӢйҷҚ 1вҖ“2 дёӘзӮ№пјҢеёёеңЁеҷӘеЈ°иҢғеӣҙеҶ…гҖӮеҜ№з»қеӨ§еӨҡж•°дәәжқҘиҜҙпјҢжҳҫеӯҳиҠӮзңҒзҡ„д»·еҖјиҝңи¶…иҝҷзӮ№жҚҹеӨұгҖӮ

йңҖиҰҒеӨҡе°‘жҢҮд»Өж•°жҚ®пјҹ#

LIMA иЎЁжҳҺпјҢ1K жқЎзІҫеҝғжҢ‘йҖүзҡ„ж ·жң¬е°ұи¶ід»Ҙи®©ејәеҹәеә§жЁЎеһӢеҸҳжҲҗиҝһиҙҜеҠ©жүӢгҖӮе®һи·өдёӯпјҢ1KвҖ“10K жқЎй«ҳиҙЁйҮҸж ·жң¬жҳҜеҗҲзҗҶдёӢйҷҗпјӣиҙЁйҮҸиҝңжҜ”ж•°йҮҸйҮҚиҰҒгҖӮ

PEFT ж–№жі•еҸҜд»Ҙз»„еҗҲдҪҝз”Ёеҗ—пјҹ#

еҸҜд»ҘгҖӮLoRA + Prompt Tuning жҳҜжңүж–ҮзҢ®ж”ҜжҢҒзҡ„з»„еҗҲпјҢQLoRA жң¬иә«д№ҹжҳҜжҠҖжңҜж ҲпјҲ4-bit еҹәеә§ + LoRA + еҲҶйЎөдјҳеҢ–еҷЁпјүгҖӮдҪҶеңЁеҗҢдёҖжЁЎеһӢдёӯеҗҢж—¶дҪҝз”Ё Adapter е’Ң LoRA еҲҷиҫғдёәзҪ•и§ҒгҖӮ

дёҖд»Ҫе…·дҪ“зҡ„еҫ®и°ғй…Қж–№пјҲ7B дёҠзҡ„ LoRAпјү#

д»ҘдёӢжҳҜжҲ‘е®һйҷ…дҪҝз”Ёе№¶дёҠзәҝзҡ„и¶…еҸӮж•°й…ҚзҪ®пјҢзңҹжӯЈеҪұе“Қж•Ҳжһңзҡ„е…ій”®йЎ№гҖӮ

ж•°жҚ®и§„жЁЎгҖӮеңЁ 7B еҹәеә§дёҠз”Ё LoRA ж—¶пјҢ1,000вҖ“5,000 жқЎй«ҳиҙЁйҮҸжҢҮд»Ө-е“Қеә”еҜ№е°ұиғҪиҺ·еҫ—еӨ§йғЁеҲҶж”¶зӣҠгҖӮе°‘дәҺ 500 жқЎж—¶пјҢfew-shot жҸҗзӨәжӣҙеҲ’з®—пјӣи¶…иҝҮ 20,000 жқЎеҲҷеӨ§жҰӮзҺҮйҷ·е…Ҙиҫ№йҷ…ж•ҲзӣҠйҖ’еҮҸпјҢйҷӨйқһд»»еҠЎжһҒе…¶еӨҚжқӮгҖӮ

Rank rгҖӮй»ҳи®Ө r=8пјҢдҪҶеҜ№д»Јз Ғз”ҹжҲҗгҖҒеӨҡжӯҘжҺЁзҗҶзӯүйҡҫйўҳпјҢеҸҜжҸҗеҚҮиҮі r=16 жҲ– r=32гҖӮr зҝ»еҖҚдјҡдҪҝеҸӮж•°йҮҸзҝ»еҖҚпјҢдҪҶи®ӯз»ғж—¶й—ҙеҮ д№ҺдёҚеҸҳгҖӮи¶…иҝҮ r=64 еҫҲе°‘жңүеё®еҠ©пјҢеҸҚиҖҢжҳ“иҝҮжӢҹеҗҲгҖӮ

AlphaгҖӮи®ҫ alpha = 2 * rпјҢеҚі r=8 вҶ’ alpha=16гҖӮalpha/r жҳҜ LoRA и·Ҝеҫ„дёҠзҡ„жңүж•ҲеӯҰд№ зҺҮзј©ж”ҫеӣ еӯҗпјҢдҝқжҢҒдёә 2 жҳҜеҸҜйқ й»ҳи®ӨгҖӮ

зӣ®ж ҮжЁЎеқ—гҖӮиӢҘжҳҫеӯҳе…Ғи®ёпјҢе»әи®®еҗҢж—¶еҜ№ q_proj, k_proj, v_proj, o_proj е’Ң gate_proj, up_proj, down_proj еә”з”Ё LoRAгҖӮд»…еҫ®и°ғжіЁж„ҸеҠӣеұӮпјҲеҺҹе§Ӣ LoRA и®ҫзҪ®пјүеңЁжҢҮд»Өи°ғдјҳдёӯдјҡжҚҹеӨұдёҚе°‘жҪңеҠӣгҖӮе®һжөӢиЎЁжҳҺпјҢжіЁж„ҸеҠӣ+MLP з»„еҗҲеңЁ Open LLM Leaderboard д»»еҠЎдёҠеҸҜйўқеӨ–жҸҗеҚҮ 2вҖ“4 еҲҶгҖӮ

еӯҰд№ зҺҮгҖӮLoRA жҺЁиҚҗд»Һ 2e-4 ејҖе§ӢгҖӮе…ЁйҮҸеҫ®и°ғйҖҡеёёз”Ё 2e-5пјҢиҖҢ LoRA еҸҜжүҝеҸ—й«ҳ 10 еҖҚзҡ„еӯҰд№ зҺҮпјҢеӣ дёәд»…е°‘йҮҸеҸӮж•°жӣҙж–°гҖӮй…ҚеҗҲ cosine и°ғеәҰе’Ң 3% warmupгҖӮ

Batch size е’ҢжўҜеәҰзҙҜз§ҜгҖӮзӯүж•Ҳ batch size жҺ§еҲ¶еңЁ 32вҖ“64 жқЎеәҸеҲ—гҖӮеҚ•еј  A100 80GBгҖҒеәҸеҲ—й•ҝ 2048 ж—¶пјҢmicro-batch йҖҡеёёдёә 2вҖ“4пјҢйңҖзҙҜз§Ҝиҫҫж ҮгҖӮ

EpochsгҖӮ1вҖ“3 иҪ®и¶ізҹЈгҖӮйӘҢиҜҒжҚҹеӨұйҖҡеёёеңЁз¬¬ 2 иҪ®и§Ұеә•пјҢи¶…иҝҮ 5 иҪ®еңЁе°Ҹж•°жҚ®йӣҶдёҠеҮ д№Һеҝ…з„¶иҝҮжӢҹеҗҲгҖӮ

еәҸеҲ—жү“еҢ…гҖӮе°ҶеӨҡдёӘзҹӯж ·жң¬жү“еҢ…жҲҗдёҖжқЎ 2048-token еәҸеҲ—пјҢе№¶жӯЈзЎ®и®ҫзҪ® attention maskгҖӮзӣёжҜ” paddingпјҢеҗһеҗҗйҮҸеҸҜжҸҗеҚҮ 2вҖ“3 еҖҚгҖӮtransformers зҡ„ SFTTrainer е·ІеҺҹз”ҹж”ҜжҢҒгҖӮ

еңЁ 5K ж ·жң¬гҖҒ7B еҹәеә§гҖҒеҚ•еј  A100 дёҠпјҢж•ҙиҪ®и®ӯз»ғзәҰйңҖ 2вҖ“4 е°Ҹж—¶пјҢз§ҹеҚЎжҲҗжң¬зәҰ $8вҖ“12гҖӮ

LoRA жӮ„жӮ„зҝ»иҪҰзҡ„ең°ж–№#

д»ҘдёӢжҳҜжҲ‘еңЁйЎ№зӣ®дёӯи§ҒиҝҮзҡ„дёүз§Қи®©еӣўйҳҹиё©еқ‘зҡ„еӨұиҙҘжЁЎејҸгҖӮ

еӨұиҙҘ 1пјҡзЁҖжңү token й•ҝе°ҫзҡ„зҒҫйҡҫжҖ§йҒ—еҝҳгҖӮLoRA иҷҪжҜ”е…ЁйҮҸеҫ®и°ғжӣҙиғҪдҝқжҠӨеҹәеә§жЁЎеһӢпјҢдҪҶд»Қдјҡжү°еҠЁиҫ“еҮәеҲҶеёғгҖӮжӣҫжңүдёҖж¬ЎпјҢеҜ№д»Јз ҒжЁЎеһӢеҒҡжҢҮд»Өеҫ®и°ғеҗҺпјҢиӢұж–ҮжҢҮд»ӨйҒөеҫӘиғҪеҠӣжҸҗеҚҮпјҢдҪҶ ErlangгҖҒHaskell зӯүзЁҖжңүиҜӯиЁҖзҡ„еӣ°жғ‘еәҰеҚҙйЈҷеҚҮ 30% д»ҘдёҠгҖӮи§ЈеҶіж–№жі•пјҡеңЁеҫ®и°ғж•°жҚ®дёӯж··е…Ҙ 5%вҖ“10% зҡ„еҹәеә§йЈҺж јж•°жҚ®дҪңдёәеӣһж”ҫгҖӮ

еӨұиҙҘ 2пјҡйқһиӢұиҜӯиҫ“еҮәзҡ„иҙЁйҮҸйҡҗжҖ§дёӢж»‘гҖӮеңЁиӢұж–Үдёәдё»зҡ„ SFT ж•°жҚ®дёҠеҒҡ LoRA жҢҮд»Өеҫ®и°ғпјҢеҚідҪҝеҹәеә§жҳҜеӨҡиҜӯиЁҖзҡ„пјҢдёӯж–ҮгҖҒйҹ©ж–ҮгҖҒж—Ҙж–Үиҫ“еҮәиҙЁйҮҸд№ҹдјҡдёӢйҷҚвҖ”вҖ”еӣ дёә LoRA еӯҰеҲ°дәҶвҖңиӢұж–ҮеҪўзҠ¶вҖқзҡ„еӣһзӯ”гҖӮеҜ№зӯ–пјҡSFT ж•°жҚ®дёӯиҮіе°‘еҢ…еҗ« 20% зҡ„зӣ®ж ҮиҜӯиЁҖж ·жң¬гҖӮ

еӨұиҙҘ 3пјҡеҗҲе№¶еҗҺзҡ„ LoRA вү  жҺЁзҗҶж—¶зҡ„ LoRAгҖӮеҪ“дҪ и°ғз”Ё model.merge_and_unload() е°Ҷ LoRA еҗҲе№¶иҝӣеҹәеә§жЁЎеһӢз”ЁдәҺйғЁзҪІж—¶пјҢеңЁ 4-bit йҮҸеҢ–и®ҫзҪ®дёӢпјҢеҗҲе№¶еҗҺзҡ„жЁЎеһӢжҖ§иғҪеҸҜиғҪжҳҺжҳҫеҠЈдәҺжңӘеҗҲе№¶зүҲжң¬гҖӮеҺҹеӣ еңЁдәҺпјҡеҗҲе№¶жү°еҠЁдәҶеҹәеә§жқғйҮҚпјҢиҖҢйҮҸеҢ–иҝҮзЁӢзҡ„йҮҚж–°еҸ–ж•ҙж”ҫеӨ§дәҶиҜҜе·®гҖӮи§ЈеҶіеҠһжі•пјҡиҰҒд№ҲжҺЁзҗҶж—¶дҝқжҢҒ LoRA еҲҶзҰ»пјҲиҪ»еҫ®е»¶иҝҹд»Јд»·пјүпјҢиҰҒд№ҲеңЁеҗҲе№¶еҗҺз”Ёйў„з•ҷйӣҶеҒҡйҮҸеҢ–ж ЎеҮҶгҖӮ
