DaaS — Documentation as a Skill

基础设施 · 文档管线 · Agent 工具链

把任何产品的文档变成 AI Agent 能直接用的 Skill。

给它一个文档目录，它读完所有内容，理出产品的能力结构，自动写两类 Skill：detail skill（每个能力的鉴权、端点、代码示例）与 intent skill（"部署模型"这种问题在多条路径里挑对的那条）。

company-skill.com 是配套的可视化控制台：Knowledge Base 看蒸馏出的 Skill 树，Ingest 投喂新文档，Workspaces 切换不同产品，Routing Playground 试 intent skill 的决策路径，Telemetry 跑对抗式召回评估。

面向任何公司——把整家公司蒸馏成一个 Skill，让 Agent 直接调用。没有人需要手写一行 Skill 内容。

AI4Science

架构 · Agent 控制平面
long-running observation

一套不知疲倦的自动化科研工作流。

跑在 OpenClaw Agent 运行时之上：抛一个研究问题进去，系统读论文、抽方法、跑实验、记中间状态、最后产出报告。共享记忆持久化每个 session，Harness 把失败的尝试压缩成下一轮的起点。

完整服务 内部跑 15 个模块——Chat / Overview / Files / Sessions / Logs / API Docs / Costs / Limits / Memory / Live Feed / Security / Sys Security / Config / Docker，覆盖端到端的实验编排、数据集、跨 Provider 的 token 与限流监控。

公开展示（llm4science.top）开放 Chat / Files / Sessions / Memory 四大可视化模块，让访客直接看到这条 Agent 链路在跑什么；编排逻辑、私有数据集与 Provider 凭证仍在内部。
