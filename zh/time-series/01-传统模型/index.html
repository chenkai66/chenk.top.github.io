时间序列模型（一）：传统统计模型

从一个统一的状态空间视角推导 ARIMA、SARIMA、VAR、GARCH、Prophet、指数平滑与卡尔曼滤波，配合 Box-Jenkins 工作流、ACF/PACF 识别与可运行的 Python 代码。

本章要点

  • 平稳性是 ARIMA 家族的入场券，差分是获取它的核心手段。
  • 像 Box-Jenkins 专家一样读懂 ACF 和 PACF 图：用“截尾”和“拖尾”规则确定 $p$ 和 $q$。
  • 深入掌握 ARIMA / SARIMA 的完整框架，特别是如何通过滞后 $s$ 算子融入季节性。
  • 将 VAR、GARCH、指数平滑、Prophet 和卡尔曼滤波 放在同一张图上对比：均值动态、方差动态和状态空间递推各自的特点与关系。
  • 提供一个决策准则：什么时候传统模型就够用了，什么时候需要转向本系列后续介绍的深度模型。

前置知识

  • 掌握基本的概率统计概念，比如均值、方差、协方差和相关系数。
  • 熟悉 NumPy 和 pandas 的时间索引操作。
  • VAR 和卡尔曼部分涉及一点线性代数，主要是矩阵乘法和特征值。

1. 为什么传统模型依然重要

深度学习兴起之前，时间序列分析的工具箱已经非常成熟。ARIMA 能捕捉线性自相关，SARIMA 加入了日历效应，VAR 支持多维数据，GARCH 专注于波动率建模，而卡尔曼滤波通过状态空间递归将这些方法统一起来。这些传统模型有三个显著特点，是深度学习模型无法轻松提供的：

  1. 可解释性强。 每个参数都有明确意义——“昨天的值对今天的影响权重是 $\phi_1$”，“两个月前的冲击以 $\theta_2$ 的权重衰减”。
  2. 不确定性校准可靠。 置信区间直接来自极大似然估计，而不是靠 Dropout 这种临时手段。
  3. 样本效率高。 几百个观测点就足够，完全不需要 GPU。

如果你的数据序列较短、趋势平滑，或者有清晰的日历结构，传统模型通常能轻松胜过 LSTM，而且调试起来也简单得多。在尝试更复杂的方法之前，请把本文当作 必须超越的基线。

2. 分解视角

建模之前，先在脑子里画个图：大多数时间序列都可以拆成三部分：

$$y_t = T_t + S_t + R_t,$$

$T_t$ 是缓慢变化的趋势，$S_t$ 是周期为 $s$ 的季节性成分，$R_t$ 是去掉结构后看起来像噪声的残差。经典加法分解就是把这个想法具体化。

图 1 – 经典加法分解。去掉趋势和季节性后，残差应该接近平稳白噪声。这正是 ARMA 类模型适用的场景。

整个 ARIMA 方法可以用一句话概括：把数据变换到平稳状态，然后用带自相关误差的线性模型拟合。

平稳性的形式定义

一个序列如果是弱平稳的，那么它的均值、方差和自协方差都不依赖时间 $t$：

$$\mathbb{E}[y_t] = \mu, \qquad \mathrm{Var}(y_t) = \sigma^2, \qquad \mathrm{Cov}(y_t, y_{t-k}) = \gamma_k.$$

但现实中，绝大多数序列都不是平稳的。它们要么有趋势，要么漂移不定，要么方差随水平增长。解决这个问题有两个常用方法：

  • 差分：$\nabla y_t = y_t - y_{t-1}$ 可以去掉线性趋势；$\nabla^2 y_t$ 可以去掉二次趋势。
  • 方差稳定化变换：比如 $\log y_t$ 或者 Box-Cox 变换，能压制乘性增长。

ADF（增广 Dickey-Fuller）检验提供了一种假设检验方法：原假设 $H_0$ 是“存在单位根”。如果 $p$ 值很小，就可以认为变换后的序列是平稳的。

3. AR、MA、ARMA – 三种记忆方式

ARIMA 模型由两个基本部分组成。它们看起来相似，但对“序列记住了什么”有不同的理解。

图 2 – AR 记住过去的值，MA 记住过去的冲击，ARMA 则两者兼顾。即使不计算统计量，从样本路径中也能看出明显的差异。

自回归模型：AR($p$)

$$y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \cdots + \phi_p y_{t-p} + \varepsilon_t.$$

今天的值是过去 $p$ 个值的线性组合，再加上一个白噪声冲击 $\varepsilon_t \sim \mathcal{N}(0, \sigma^2)$。持久性直接体现在系数 $\phi_k$ 中。当 $\phi_1$ 接近 1 时，AR(1) 会生成一条非常平滑且缓慢回归均值的路径。

移动平均模型：MA($q$)

$$y_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \cdots + \theta_q \varepsilon_{t-q}.$$

今天的值是最近 $q$ 个冲击的线性组合。记忆窗口有限：超过 $q$ 步之前的事件对今天完全没有直接影响。

组合模型：ARMA($p$, $q$)

$$\phi(B)\, y_t = \theta(B)\, \varepsilon_t,$$

其中 $B$ 是 滞后算子（$B y_t = y_{t-1}$），并且

$$\phi(B) = 1 - \phi_1 B - \cdots - \phi_p B^p, \qquad \theta(B) = 1 + \theta_1 B + \cdots + \theta_q B^q.$$

ARMA 的优势在于简洁性：用一组较小的 $(p, q)$ 就能模拟 MA($\infty$) 或 AR($\infty$) 的自相关结构。平稳性要求 $\phi(B) = 0$ 的所有根位于单位圆外；可逆性对 $\theta(B)$ 也有同样的要求。

从 ARMA 到 ARIMA

如果原始序列不平稳，可以先对序列取 $d$ 阶差分，然后在差分后的序列上拟合 ARMA 模型。完整的表示方法是 ARIMA($p, d, q$)：

$$\phi(B)\, (1-B)^d\, y_t = \theta(B)\, \varepsilon_t.$$

实际应用中，$d = 1$ 处理线性趋势，$d = 2$ 处理曲率变化，很少需要 $d > 2$。

4. ACF 和 PACF – 模型识别的显微镜

怎么选 $p$ 和 $q$？两张诊断图基本就能搞定。

  • 自相关函数 (ACF)：$\rho_k = \mathrm{Corr}(y_t, y_{t-k})$。
  • 偏自相关 (PACF)：去掉 $y_{t-1}, \ldots, y_{t-k+1}$ 的线性影响后，计算 $y_t$ 和 $y_{t-k}$ 的相关性。

Box-Jenkins 识别规则：

过程ACFPACF
AR($p$)拖尾（平滑衰减）滞后 $p$ 阶后截断
MA($q$)滞后 $q$ 阶后截断拖尾
ARMA($p$, $q$)拖尾拖尾

图 3 – AR(2) 的 PACF 在滞后 2 阶后降到零，尖峰模式是它的特征。对称地，MA(2) 的 ACF 在滞后 2 阶后降到零。如果不确定，优先选择简单模型，第 5 步的诊断测试会发现欠拟合问题。

如果两张图都没有明显的截断点，那多半就是 ARMA 过程。最直接的办法是对 $(p, q)$ 做网格搜索，选信息准则最小的一组。

AIC 和 BIC

$$\mathrm{AIC} = -2\,\ell(\hat{\theta}) + 2k, \qquad \mathrm{BIC} = -2\,\ell(\hat{\theta}) + k\log n,$$

其中 $\ell$ 是最大化的对数似然值，$k$ 是自由参数数量，$n$ 是样本量。BIC 对复杂度的惩罚更重，样本量小时用它更稳妥。

5. Box-Jenkins 工作流

ARIMA 不是一步到位的模型，而是 Box 和 Jenkins 在 1970 年提出的一个 迭代循环。后来的各种统计预测工具（比如 auto.arima）本质上只是把这四个步骤自动化了。

图 4 – 关键是那条反馈箭头。如果拟合后的残差通不过 Ljung-Box 检验，或者残差 ACF 还有结构，说明模型设定有问题。这时应该回去调整 $(p, d, q)$，而不是盲目相信预测结果。

  1. 识别
    先画出时间序列图，用 ADF 检验平稳性，分析 ACF 和 PACF 图，提出一个候选的 $(p, d, q)$。

  2. 估计
    使用极大似然估计或条件最小二乘法拟合模型。主流库都会自动完成这一步。

  3. 诊断检验
    检查残差是否是白噪声：

    • Ljung-Box 统计量 $Q = n(n+2)\sum_{k=1}^h \hat{\rho}_k^2 / (n-k)$ 应该无法拒绝 $H_0$。
    • 残差的 ACF 应该落在 $\pm 1.96/\sqrt{n}$ 的范围内。
    • 用 QQ 图和 Jarque-Bera 检验正态性，这对预测区间的准确性很重要。

  4. 预测
    只有当残差干净时，才能生成点预测和区间预测。

import numpy as np
import pandas as pd
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.tsa.stattools import adfuller
from statsmodels.stats.diagnostic import acorr_ljungbox

# 1. 模拟非平稳序列（带漂移的随机游走）
rng = np.random.default_rng(0)
n = 200
eps = rng.normal(0, 1.0, n)
y = np.zeros(n)
for t in range(1, n):
    y[t] = y[t - 1] + 0.05 + 0.6 * eps[t]

series = pd.Series(y, index=pd.date_range("2018-01-01", periods=n, freq="D"))

# 2. 识别：对原序列和一阶差分做 ADF 检验
print("ADF on level :", adfuller(series)[1])         # p 值大，说明非平稳
print("ADF on diff  :", adfuller(series.diff().dropna())[1])  # p 值小，说明平稳

# 3. 估计：用前 170 个点训练模型
train, test = series.iloc[:-30], series.iloc[-30:]
model = ARIMA(train, order=(2, 1, 1)).fit()

# 4. 诊断：检查残差是否为白噪声
lb = acorr_ljungbox(model.resid, lags=[10], return_df=True)
print(lb)  # p 值高，说明残差是白噪声

# 5. 预测：给出 95% 置信区间
fc = model.get_forecast(steps=30)
mean_fc = fc.predicted_mean
ci = fc.conf_int(alpha=0.05)

留出尾段的预测效果如下：

图 5 – 点预测很快回归长期漂移线，置信区间按 $\sigma\sqrt{h}$ 的速率随预测步长 $h$ 扩散。这个逐渐张开的喇叭口，正是线性模型对未来认知局限的真实写照。

7. 季节性：SARIMA

很多时间序列有 ARIMA 模型无法单独处理的周期性特征。比如，月度零售数据通常在 12 月出现峰值，日流量数据可能有周周期，小时负荷数据则可能呈现 24 小时的周期性。SARIMA($p, d, q$)($P, D, Q$)$_s$ 引入了周期为 $s$ 的季节滞后项：

$$\Phi(B^s)\, \phi(B)\, (1-B)^d (1-B^s)^D y_t \;=\; \Theta(B^s)\, \theta(B)\, \varepsilon_t.$$

我需要做两种差分：用常规差分 $(1-B)$ 去掉趋势，用季节差分 $(1-B^s)$ 去掉周期性。然后在常规滞后和 $s$ 的倍数滞后上分别添加 AR 和 MA 项。

model = SARIMAX(

    enforce_stationarity=False,

图 6 – 样本外的季节周期被清晰复现。季节差分算子 $(1-B^{12})$ 把年内的模式从残差中剥离，而季节 AR/MA 项又在正确的时间点将其加回到预测中。

季节部分识别小抄：对 季节差分后 的序列，在 $s, 2s, 3s, \ldots$ 这些滞后点查看 ACF 和 PACF 图。判断截尾或拖尾的规则与非季节性情况相同。

8. 超越 ARIMA：家族其他成员

上面提到的思想可以延伸到四个实用方向。它们不是孤立的领域，而是各自在 ARIMA 的某一维度上做了专门化。

8.1 VAR – 多元动态

如果有多条相互影响的时间序列（比如 GDP 和失业率、电力需求和气温），可以把标量 AR 升级为 向量自回归：

$$\mathbf{y}_t = \mathbf{c} + A_1 \mathbf{y}_{t-1} + A_2 \mathbf{y}_{t-2} + \cdots + A_p \mathbf{y}_{t-p} + \boldsymbol{\varepsilon}_t.$$

矩阵 $A_k$ 的每个元素都有明确意义：$(A_k)_{ij}$ 表示变量 $j$ 在滞后 $k$ 时对变量 $i$ 当前值的边际影响。这让 VAR 在宏观经济学中非常流行，因为 Granger 因果（“除了 $y$ 自己的历史，$x$ 是否还能帮助预测 $y$？"）是核心问题。

实际应用中有几点需要注意：如果有 $K$ 条序列和滞后阶数 $p$，模型会有 $K + pK^2$ 个自由参数。当 $K = 10, p = 4$ 时，参数数量已经达到 410 个，而观测数据可能只有几百个。因此，在高维场景下，贝叶斯 VAR 和因子模型等正则化方法应运而生。

8.2 GARCH – 方差动态

ARIMA 建模的是 均值，而 GARCH 建模的是 条件方差。基础的 GARCH(1,1) 模型如下：

$$\sigma_t^2 = \omega + \alpha\, \varepsilon_{t-1}^2 + \beta\, \sigma_{t-1}^2, \qquad \varepsilon_t = \sigma_t\, z_t,\quad z_t \sim \mathcal{N}(0, 1).$$

$\alpha$ 项让昨天的大冲击推高今天的方差（即 “ARCH 效应”），$\beta$ 项让方差具有持续性。平稳性要求 $\alpha + \beta < 1$，而在金融数据中，$\alpha + \beta$ 通常接近 0.95-0.99，说明波动率变化缓慢。

GARCH 是风险管理（VaR、期权定价）的标准工具，并且天然与 ARIMA 均值模型搭配使用：先用 ARIMA 拟合收益率，再用 GARCH 拟合残差平方。

8.3 指数平滑与 Holt-Winters

如果说 ARIMA 是 “显式随机” 的视角，指数平滑则是 “加权平均” 的视角。简单指数平滑假设存在一个随新信息漂移的水平 $\ell_t$：

$$\ell_t = \alpha\, y_t + (1-\alpha)\, \ell_{t-1}.$$

Holt 在此基础上增加了趋势 $b_t$；Holt-Winters 进一步加入了季节性 $s_t$。加法形式如下：

$$\ell_t = \alpha (y_t - s_{t-s}) + (1-\alpha)(\ell_{t-1} + b_{t-1}),\\ b_t = \beta(\ell_t - \ell_{t-1}) + (1-\beta) b_{t-1},\\ s_t = \gamma(y_t - \ell_t) + (1-\gamma) s_{t-s}.$$

这是 R 和 statsmodels 中 ETS 家族的核心方法，许多赢得 M 竞赛的方法都是它的改进版。

图 7 – 分解结构是 Holt-Winters 的亮点：每个分量都易于理解，平滑系数 ($\alpha, \beta, \gamma$) 决定了各分量的适应速度，递推形式让拟合复杂度仅为 $O(n)$。

8.4 Prophet

Prophet（Facebook，2017）是一个刻意保持简单的 加法分解 模型，专为业务分析师设计：

$$y(t) = g(t) + s(t) + h(t) + \varepsilon_t,$$

其中 $g(t)$ 是带 变点 的分段线性或逻辑趋势，$s(t)$ 用傅里叶展开表示多重季节性，$h(t)$ 编码用户提供的节假日信息。它通过 Stan 的 MAP 或全贝叶斯采样拟合，对外只暴露少量可调参数，对缺失值和离群点鲁棒。虽然在标准基准测试中表现并非顶尖，但其 API 设计和节假日处理能力让它成为产品分析中的默认选择。

8.5 卡尔曼滤波与状态空间视角

上述所有模型都可以看作 线性高斯状态空间模型 的特例：

$$\mathbf{x}_t = F_t \mathbf{x}_{t-1} + \mathbf{w}_t, \qquad \mathbf{w}_t \sim \mathcal{N}(0, Q_t),\\ \mathbf{y}_t = H_t \mathbf{x}_t + \mathbf{v}_t, \qquad \mathbf{v}_t \sim \mathcal{N}(0, R_t).$$

卡尔曼滤波 是精确的贝叶斯递推算法，用于在数据到达时维护 $p(\mathbf{x}_t \mid \mathbf{y}_{1:t}) = \mathcal{N}(\hat{\mathbf{x}}_t, P_t)$：

$$\hat{\mathbf{x}}_{t|t-1} = F_t\, \hat{\mathbf{x}}_{t-1},\\ P_{t|t-1} = F_t\, P_{t-1}\, F_t^\top + Q_t,\\ K_t = P_{t|t-1} H_t^\top (H_t P_{t|t-1} H_t^\top + R_t)^{-1},\\ \hat{\mathbf{x}}_t = \hat{\mathbf{x}}_{t|t-1} + K_t (\mathbf{y}_t - H_t \hat{\mathbf{x}}_{t|t-1}),\\ P_t = (I - K_t H_t)\, P_{t|t-1}.$$

为什么这很重要？ARIMA、指数平滑、动态线性回归、带季节哑元的结构模型 – 全部可以转写成上述形式，并免费继承卡尔曼递推的优势。这正是 statsmodels 的 SARIMAX 底层实现的方式，也是它能够轻松处理缺失观测和外生回归变量的原因。

9. 如何选择模型

模型适合场景不适合场景
ARIMA单序列、线性自相关、无日历效应强季节性、多输入变量
SARIMA季节周期单一且明显、序列长度适中多个季节性重叠
VAR多条平稳序列之间有反馈关系高维（$K \gtrsim 10$）
GARCH收益率或波动率数据，波动聚集平滑且方差恒定的序列
Holt-Winters / ETS季节性强、快速重新训练高度非线性的状态切换
Prophet带节假日、有缺失值的业务数据亚日级高频数据
Kalman / 状态空间在线更新、支持缺失数据、可自定义结构数据充足且能接受黑箱模型时

我的建议是：先用 ETS 或 SARIMA 建立基线。如果关注的是方差，可以加上 GARCH。只有当残差表现出明显的非线性结构，或者你有大量并行序列需要共享信息时，再考虑深度模型（LSTM、TCN、Transformer、N-BEATS、Informer – 这些后面会详细讲）。

10. 局限与下一步

传统模型在以下场景中会遇到瓶颈：

  • 动态是非线性的，比如机制切换、阈值效应或者硬饱和。
  • 需要处理几百甚至上千条并行序列，这些序列还得共享参数。
  • 上下文窗口很长，ARMA 模型的根无法简洁地表达。
  • 信号分布非高斯，连 Box-Cox 变换也无能为力。

这就是接下来七篇文章的出发点。我从 LSTM 开始，它是非线性序列模型的经典代表。然后逐步介绍 GRU、注意力机制、Transformer、TCN、N-BEATS，最后是针对超长序列的 Informer。每个模型都在扩展前面提到的某种线性模型。把 ARIMA / SARIMA / Holt-Winters 当成必须超越的基线，而不是可以绕过的过去。
