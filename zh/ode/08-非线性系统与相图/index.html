常微分方程（八）：非线性系统与相图

捕食-猎物循环、竞争排斥、Van der Pol 极限环、Hamilton 系统与 Poincare-Bendixson 定理——2D 非线性动力学的完整工具箱。

真实世界是非线性的。 捕食循环、心跳节律、神经元放电，线性方程无法描述。叠加原理失效时，世界出现新行为：极限环、多平衡点、双稳态、滞回。本章提供从 2D 相图直接读取这些行为的几何与分析工具。

本章要点

  • 非线性系统与线性系统本质不同
  • Lyapunov 稳定性：等位面、碗形、吸引域
  • 线性化与非线性全景（Hartman-Grobman 应用）
  • Lotka-Volterra：闭合轨道和守恒量
  • 竞争模型四种典型结果
  • Van der Pol 振子与极限环几何
  • 梯度系统与 Hamilton 系统
  • Poincare-Bendixson：2D 系统为何无混沌

从线性到非线性

线性系统满足叠加原理：若 $\mathbf{x}_1$、$\mathbf{x}_2$ 是解，则 $c_1\mathbf{x}_1 + c_2\mathbf{x}_2$ 也是。这是第 1-6 章工具箱的核心——指数试解、特征向量、基本矩阵的理论基础。

非线性系统不满足叠加，闭式解通常不存在。但它们有独特的特性：

  • 多个平衡点，各有稳定性类型
  • 极限环——孤立且稳定的周期轨道（线性系统中不存在）
  • 双稳态与滞回——对初始条件的记忆
  • 敏感依赖——3D 及以上可能出现混沌（第九章）

物理、生物、化学、神经科学、经济学中几乎所有有趣系统都是非线性的。

Lyapunov 稳定性的直观理解

Lyapunov 函数 $V(\mathbf{x})$ 是沿轨迹递减的标量（$\dot V \leq 0$）。几何上看，$V$ 的等值面像嵌套的"碗"，轨迹从外向内穿过。

左：五条轨迹从不同起点盘旋向内，穿过 $V(x,y) = x^2 + \tfrac12 y^2$ 的彩色等值面。右：$V(\mathbf{x}(t))$ 单调衰减（对数尺度）——原点渐近稳定的几何证明。

把 Lyapunov 稳定性看成"轨迹滑入碗底"，所有定理都清楚了：

  • $\dot V \leq 0$：轨迹不往上走 -> 困在初始的碗里（稳定）。
  • $\dot V < 0$：持续下滑 -> 滑到碗底（渐近稳定）。
  • $\dot V > 0$：轨迹往外爬 -> 不稳定。

LaSalle 推广：$\dot V$ 在某集合上为零时，轨迹趋于该集合的最大不变子集。

相图与零线

对 $x' = f(x,y),\ y' = g(x,y)$：

  • $x$ 零线是 $f(x,y) = 0$，轨迹垂直穿过。
  • $y$ 零线是 $g(x,y) = 0$，轨迹水平穿过。
  • 平衡点在两条零线交点处。
  • 各区域 $f$、$g$ 的正负决定流向。

手绘相图时，零线分析最省事。

线性化：局部为真，全局意外

双曲平衡点附近，Jacobian 特征值完全决定局部行为（Hartman-Grobman）。远离平衡点，情况完全不同。阻尼单摆展示了这一点：

左：真实单摆在每个 $n\pi$ 处有平衡点，存在分隔线和旋转区域。右：线性化 $\ddot\theta + \gamma\dot\theta + \omega_0^2\theta = 0$ 只显示一个稳定焦点。原点附近相同，全局差异显著。

线性特征值类型局部类型非线性稳定性
$\lambda_1 < \lambda_2 < 0$（实）稳定结点渐近稳定
$0 < \lambda_1 < \lambda_2$（实）不稳定结点不稳定
$\lambda_1 < 0 < \lambda_2$鞍点不稳定
$\alpha \pm \beta i,\ \alpha < 0$稳定螺旋渐近稳定
$\alpha \pm \beta i,\ \alpha > 0$不稳定螺旋不稳定
$\pm \beta i$中心不确定——非线性项决定

Lotka-Volterra 捕食模型

模型

$$x' = \alpha x - \beta xy, \qquad y' = \delta xy - \gamma y$$
  • $x$：猎物，$y$：捕食者
  • 平凡平衡点 $(0,0)$：鞍点
  • 共存平衡点 $(\gamma/\delta,\ \alpha/\beta)$：Jacobian 特征值 $\pm i\sqrt{\alpha\gamma}$（中心）
$$H(x,y) = \delta x - \gamma\ln x + \beta y - \alpha\ln y$$

确保轨道闭合。时间序列与相平面如下：

左：捕食者（红）滞后猎物（蓝）四分之一周期——经典生态循环。右：围绕中心 $(c/d,\ a/b)$ 的嵌套闭合轨道。不同初值永远停留在不同轨道上。

文字解读

  1. 猎物多 -> 捕食者繁盛 -> 捕食者增多。
  2. 捕食者多 -> 猎物减少 -> 猎物骤降。
  3. 猎物少 -> 捕食者饿死 -> 捕食者减少。
  4. 捕食者少 -> 猎物恢复 -> 回到第 1 步。

局限

  • 结构不稳定——额外项会破坏闭合轨道。
  • 无承载力——捕食者缺席时猎物无限增长。
  • 忽略空间与时滞。

这些缺陷促使我研究下一节更现实的模型。

竞争模型：四种结局

$$\begin{aligned} x' &= r_1 x\!\left(1 - \frac{x + \alpha_{12}y}{K_1}\right), \\ y' &= r_2 y\!\left(1 - \frac{y + \alpha_{21}x}{K_2}\right). \end{aligned}$$

乘积 $\alpha_{12}\,\alpha_{21}$ 决定最终的四种情况。

蓝红线是零线，黑点是平衡点，灰线是轨迹。零线交点决定初值的长期行为。

区域条件结局
弱干扰$\alpha_{12} < 1,\ \alpha_{21} < 1$稳定共存
强干扰$\alpha_{12} > 1,\ \alpha_{21} > 1$双稳态，胜者看初值
不对称$\alpha_{12} < 1,\ \alpha_{21} > 1$物种 1 胜
不对称$\alpha_{12} > 1,\ \alpha_{21} < 1$物种 2 胜

这就是数学形式的竞争排斥原理。

Van der Pol：非线性阻尼生成极限环

$$x'' - \mu(1 - x^2)x' + x = 0$$

关键在阻尼系数 $-\mu(1 - x^2)$：

  • $|x| < 1$：阻尼为负，系统注入能量。
  • $|x| > 1$：阻尼为正，能量耗散。

内部轨迹扩张，外部轨迹收缩，最终都趋于一个稳定极限环——孤立周期轨道，吸引所有点。

三个 $\mu$ 值。灰线是过渡过程，彩线是极限环本身。$\mu$ 增大时，极限环从近正弦变形为"弛豫振荡"，带突跳。

应用： 心脏起搏器细胞、神经元动作电位、电子振荡电路、间歇泉喷发、商业周期。

梯度系统与 Hamilton 系统：两种特殊世界

梯度系统：$\mathbf{x}' = -\nabla V$

  • 轨迹沿 $V$ 最陡下降。
  • $\dot V = -|\nabla V|^2 \leq 0$，势能总在降。
  • 没有周期轨道，绕一圈不能更低。
  • 机器学习的梯度下降是离散版。

Hamilton 系统：$x' = \dfrac{\partial H}{\partial y},\ y' = -\dfrac{\partial H}{\partial x}$

  • $H$ 沿轨道守恒，$\dot H = 0$。
  • 相空间体积守恒，Liouville 定理。
  • 轨道是 $H$ 的等值线。
  • 无阻尼单摆是典型例子。

两个系统分处耗散谱两端。

Poincare-Bendixson：2D 不能混沌

定理（Poincare-Bendixson）。 2D 光滑连续系统的有界轨迹，若不趋向平衡点，必收敛到闭合轨道。

简单说：2D 长期行为只有平衡或周期，混沌不存在。

关键在 Jordan 曲线定理——闭合轨道分割平面，困住轨迹。加一个维度，轨迹能从轨道上方逃逸，混沌就可能出现（第九章）。

Bendixson 判据（无闭合轨道）。 若 $\partial f/\partial x + \partial g/\partial y$ 在单连通区域符号恒定且非零，则区域内无闭合轨道。

def rk4_step(f, x, t, h):
    k1 = h * np.array(f(x, t))
    k2 = h * np.array(f(x + k1/2, t + h/2))
    k3 = h * np.array(f(x + k2/2, t + h/2))
    k4 = h * np.array(f(x + k3,    t + h))
    return x + (k1 + 2*k2 + 2*k3 + k4) / 6

练习题

概念题

  1. 为什么 $y' = y^2$ 不满足叠加原理？举个反例。
  2. 为什么 2D 连续系统不会出现混沌？
  3. 手绘 $x' = y,\ y' = -x + x^3 - 0.2y$（Duffing）的相图。

计算题

  1. 对 $x' = x - x^3,\ y' = -y$，找出所有平衡点并分类。
  2. 证明 $V = x^2 + y^2$ 是 $x' = -x + y^2,\ y' = -y$ 的 Lyapunov 函数。
  3. 验证 $H = \delta x - \gamma\ln x + \beta y - \alpha\ln y$ 在 Lotka-Volterra 中守恒。

编程题

  1. 复现图 5 的四种竞争状态，并为每个吸引域上色。
  2. 数值估算 $\mu \in \{0.1, 0.5, 1, 3, 10\}$ 时 Van der Pol 周期 $T(\mu)$。
  3. 比较 Euler 和 RK4 在 Van der Pol 方程上的精度，找出 Euler 失效的 $\mu$。

极限环的稳定性：Floquet 乘子

平衡点看 Jacobian 特征值。极限环 $\gamma(t)$ 看 Floquet 乘子——量化"偏离一点，绕一圈后放大或缩小多少"。

$$ \dot\delta = J\!\bigl(\gamma(t)\bigr)\,\delta. $$

积分一个周期得单值矩阵 $\Phi(T)$。其特征值是 Floquet 乘子 $\mu_i$。必有一个 $\mu_1 = 1$（相位漂移），其余决定横向稳定性：

  • 全部 $|\mu_i| < 1$ → 稳定极限环。
  • 至少一个 $|\mu_i| > 1$ → 不稳定。

数值方法简单：用 RK4 积分 $\Phi$ 的 ODE $\dot\Phi = J(\gamma(t))\,\Phi$，初值 $\Phi(0) = I$，跑一个周期求特征值。SciPy 没现成函数，但十几行代码搞定：

def monodromy(f, jac, gamma0, T, n=2000):
    from scipy.integrate import solve_ivp
    d = len(gamma0)
    def rhs(t, y):
        x, Phi = y[:d], y[d:].reshape(d, d)
        return np.concatenate([f(x), (jac(x) @ Phi).ravel()])
    y0 = np.concatenate([gamma0, np.eye(d).ravel()])
    sol = solve_ivp(rhs, (0, T), y0, dense_output=False, rtol=1e-10)
    return sol.y[d:, -1].reshape(d, d)

Van der Pol 在 $\mu = 1$ 时非平凡 Floquet 乘子约 $0.04$——强烈吸引，轨迹几圈内快速锁定。

数值常见坑：刚性、辛、漂移

坑一：刚性系统用显式法。 Lotka-Volterra 没问题。改成 $\dot x = -1000(x - \cos t) - \sin t$，显式 RK4 稳定步长 $h < 2/1000$，一秒要跑 500 步。换 LSODA 或 BDF 就好。判断方法：Jacobian 特征值跨数量级。

坑二：Hamilton 系统用非辛积分器。 单摆 $H = p^2/2 - \cos q$ 是守恒量。RK4 跑一万步，能量漂移明显。改 leapfrog 或 symplectic Euler，能量震荡但有界。PDE-ML 系列第五章讲辛网络，动机就在这。

坑三：长期积分的相位漂移。 极限环或准周期轨道，数值方法相位误差线性增长。想算 $10^4$ 周期后的相图，标准 RK4 不行。Geometric integrator 加大步长是常用办法。

接入 ML：Neural ODE 的相图就是网络行为

Neural ODE $\dot z = f_\theta(z, t)$ 把分类任务转为"数据点流向正确区域"。训练后的相图直接反映网络行为：

  • 拓扑限制真实存在。 Dupont et al. (2019) 指出，Neural ODE 无法让两条不相交曲线交叉——ODE 流是同胚。解决方法：升维到 $\mathbb{R}^{d+k}$，即 ANODE。这是相图理论的直接结果。
  • 吸引子可视化提供可解释性。 训练一个二维 Neural ODE 分类器后，画出 $f_\theta$ 的向量场，决策边界由不动点和稳定流形构成，无需 SHAP。
  • 稳定性正则化。 在损失中加 $\|J_\theta\|_F^2$，约束流的局部拉伸。这是隐式的对抗鲁棒性手段，来自相图理论，而非 ML 民间传说。
