常微分方程（七）：稳定性理论

桥梁能否扛住大风？生态系统能否从冲击中恢复？稳定性理论用 Lyapunov 函数、线性化和分岔分析回答这些问题——而且不需要解方程。

#ODE#稳定性#Lyapunov 理论#相空间#分岔
Read in English →

轻推系统一下，它会恢复平衡，漂走，还是直接崩溃？ 这个问题决定了桥梁抗风暴能力、生态系统旱后恢复力、经济危机反弹力。稳定性理论给出答案，且不用解微分方程。我学会从相平面几何看出系统命运。

本章要点

  • 三个定义：Lyapunov 稳定、渐近稳定、不稳定
  • Jacobian 线性化与 Hartman-Grobman 定理
  • Lyapunov 直接方法：用类能量函数证稳定性
  • LaSalle 不变集原理解决边界情况
  • 迹-行列式平面分类所有 2D 线性系统
  • 四种分岔：鞍-结点、跨临界、叉型、Hopf
  • 应用实例：单摆、捕食模型、倒立摆控制

前置知识

  • 第六章：线性系统、特征值、相图
  • 多元微积分：偏导数、Jacobian 矩阵

先看图：六张图讲完线性系统

稳定性本质是几何问题，描述相空间中轨迹如何流动。六张图就能说清 2D 线性系统的全部。

六种典范 2D 相图。实线是正向轨道，虚线是反向轨道。特征值决定具体图形。

非线性系统在每个平衡点附近局部呈现这六种图形之一。阻尼单摆和 Lotka-Volterra 捕食模型是典型例子：

阻尼单摆：θ 轴上稳定焦点与鞍点交替出现。Lotka-Volterra：闭合轨道环绕非双曲中心。

稳定性的精确定义

考虑 $\mathbf{x}' = \mathbf{f}(\mathbf{x})$，平衡点为 $\mathbf{x}^*$（即 $\mathbf{f}(\mathbf{x}^*) = \mathbf{0}$）。

$$\|\mathbf{x}(0) - \mathbf{x}^*\| < \delta \;\Longrightarrow\; \|\mathbf{x}(t) - \mathbf{x}^*\| < \varepsilon, \;\; \forall\, t > 0.$$

直觉：附近轨迹始终靠近。

渐近稳定。 Lyapunov 稳定且 $\mathbf{x}(t) \to \mathbf{x}^*$。 直觉：附近轨迹先靠近，再收敛。

不稳定。 不满足 Lyapunov 稳定。

吸引域是所有初值收敛到 $\mathbf{x}^*$ 的集合。渐近稳定是局部性质，吸引域反映局部范围。

为何分两种？ 中心（闭合轨道）是 Lyapunov 稳定但非渐近稳定——轨迹绕圈不收敛。Lotka-Volterra 是典型例子。

线性化：Jacobian 方法

$$\mathbf{x}' \;\approx\; J(\mathbf{x} - \mathbf{x}^*), \qquad J_{ij} = \frac{\partial f_i}{\partial x_j}\bigg|_{\mathbf{x}^*}.$$

Hartman-Grobman 定理

若 $J$ 的特征值实部全非零（双曲平衡点），非线性系统在 $\mathbf{x}^*$ 附近与 $\mathbf{u}' = J\mathbf{u}$ 拓扑等价。

  • 实部全负：渐近稳定
  • 存在正实部：不稳定
  • 纯虚特征值：线性化失效，改用 Lyapunov 方法

实例：阻尼单摆

$$\theta'' + \gamma\theta' + \omega_0^2\sin\theta = 0$$
平衡点Jacobian结论
$(0,0)$ 下垂$\begin{pmatrix}0 & 1 \\ -\omega_0^2 & -\gamma\end{pmatrix}$$\gamma>0$ 时实部全负，稳定焦点
$(\pi,0)$ 倒立$\begin{pmatrix}0 & 1 \\ \omega_0^2 & -\gamma\end{pmatrix}$一个正特征值，鞍点不稳定

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

gamma, omega0 = 0.3, 1.0
def pendulum(X, t):
    x, y = X
    return [y, -gamma*y - omega0**2*np.sin(x)]

fig, ax = plt.subplots(figsize=(10, 6))
t = np.linspace(0, 50, 2000)
for x0 in np.linspace(-3*np.pi, 3*np.pi, 15):
    for y0 in np.linspace(-3, 3, 5):
        sol = odeint(pendulum, [x0, y0], t)
        ax.plot(sol[:,0], sol[:,1], 'b-', linewidth=0.3, alpha=0.5)
ax.set_title('阻尼单摆相图')
ax.grid(True, alpha=0.3); plt.tight_layout(); plt.show()

Lyapunov 直接方法

核心思想

不解 ODE 也能证明稳定性。构造一个类能量标量函数 $V(\mathbf{x})$，证明它沿轨迹递减。

要求

  1. $V(\mathbf{x}^*) = 0$，其余 $V > 0$（正定）
  2. $\dot V = \nabla V \cdot \mathbf{f}(\mathbf{x}) \leq 0$ 沿轨道成立

稳定性定理

$\dot V$ 符号结论
$\dot V \leq 0$$\mathbf{x}^*$ Lyapunov 稳定
$\dot V < 0$（除 $\mathbf{x}^*$ 外）渐近稳定
$V > 0,\ \dot V > 0$不稳定（Chetaev）

几何直觉

轨迹从外向内穿越 $V$ 的等位面。$V$ 在 $\mathbf{x}^*$ 取最小值，轨迹无法逃出小等位面，并在严格递减时滑到底。

左：轨迹滑下碗 $V(x,y) = x^2 + \tfrac12 y^2$。右：$V$ 沿轨迹单调递减——渐近稳定性的几何证明。

怎么找 $V$

  • 物理能量：力学系统动能加势能
  • 二次型：$V = \mathbf{x}^T P \mathbf{x}$，$P$ 满足 Lyapunov 方程 $A^T P + PA = -Q$
  • 试错：从 $V = x^2 + y^2$ 开始，算 $\dot V$，调整系数

实例：单摆能量

$$V(\theta, \omega) = \tfrac{1}{2}\omega^2 + (1 - \cos\theta), \qquad \dot V = -\gamma\omega^2 \leq 0.$$

下垂位置稳定。LaSalle 原理将其升级为渐近稳定。

LaSalle 不变集原理

有时 $\dot V \leq 0$，但在整个集合上为零。标准 Lyapunov 只能保证稳定，无法保证收敛。

定理。 设 $E = \{\mathbf{x} : \dot V = 0\}$，$M$ 是 $E$ 的最大不变子集。每条有界轨迹最终进入 $M$。

若 $M = \{\mathbf{x}^*\}$，则 $\mathbf{x}^*$ 渐近稳定。

阻尼单摆中，$\dot V = 0$ 要求 $\omega = 0$。但 $\omega = 0$ 时，动力学使 $\dot\omega = -\omega_0^2 \sin\theta \neq 0$，除非 $\theta = 0$。于是 $M = \{(0,0)\}$，直接得到渐近稳定。

迹-行列式分类

对 $\mathbf{x}' = A\mathbf{x}$（2D），设 $\tau = \operatorname{tr}(A)$，$\Delta = \det(A)$。特征值公式为 $\lambda_{1,2} = \tfrac12(\tau \pm \sqrt{\tau^2 - 4\Delta})$。

区域类型
$\Delta < 0$鞍点
$\Delta > 0,\ \tau < 0,\ \tau^2 > 4\Delta$稳定结点
$\Delta > 0,\ \tau > 0,\ \tau^2 > 4\Delta$不稳定结点
$\Delta > 0,\ \tau < 0,\ \tau^2 < 4\Delta$稳定螺旋
$\Delta > 0,\ \tau > 0,\ \tau^2 < 4\Delta$不稳定螺旋
$\Delta > 0,\ \tau = 0$中心
$\tau^2 = 4\Delta$退化 / 反常结点

一张图概括了所有情况：

迹-行列式平面。抛物线 $\tau^2 = 4\Delta$ 分隔螺旋（上）和结点（下）。$\Delta$ 正半轴是中心轨迹。

分岔：图本身在变化

慢慢调整参数 $r$，平衡点会诞生、消失或交换稳定性。四个"标准型"描述了所有余维 1 的局部分岔。

实绿线：稳定平衡点。虚红线：不稳定。Hopf 图展示极限环半径 $\sqrt{\mu}$ 从零增长。

分岔类型标准型现象
鞍-结点$\dot x = r + x^2$两个平衡点在 $r = 0$ 撞击后湮灭
跨临界$\dot x = rx - x^2$两个平衡点穿过彼此并交换稳定性
叉型$\dot x = rx - x^3$一个平衡点分裂成三个（对称破缺）
Hopf复特征值穿越 $i\mathbb{R}$稳定焦点失稳，极限环出现

Hopf 是自然界自持振荡的核心机制——从心跳到变星脉动都源于此。

$$x' = ax - bxy, \qquad y' = -cy + dxy$$$$H(x,y) = dx - c\ln x + by - a\ln y$$

确保轨道闭合。系统表现为周期性种群波动（图 2 右）。

应用 2：倒立摆控制

倒立平衡点是鞍点。线性反馈 $u = -K\mathbf{x}$ 将闭环特征值移至左半平面，鞍点变为稳定焦点。

import numpy as np
from scipy.linalg import solve_continuous_are
from scipy.integrate import odeint

g, l, m, M_cart = 9.81, 0.5, 0.1, 1.0
A = np.array([[0,1,0,0],[0,0,-m*g/M_cart,0],
              [0,0,0,1],[0,0,(M_cart+m)*g/(M_cart*l),0]])
B = np.array([[0],[1/M_cart],[0],[-1/(M_cart*l)]])
Q = np.diag([10, 1, 100, 1]); R = np.array([[0.1]])
P = solve_continuous_are(A, B, Q, R)
K = np.linalg.inv(R) @ B.T @ P
print("闭环特征值：", np.linalg.eigvals(A - B @ K))

总结

概念核心要点
Lyapunov 稳定附近轨迹不远离
渐近稳定附近轨迹最终收敛到平衡点
线性化Jacobian 特征值决定局部行为（双曲情况）
Lyapunov 函数不需积分即可证明稳定性的标量函数
LaSalle 原理借助不变集将 $\dot V \leq 0$ 提升为渐近稳定
迹-行列式平面一张图概括所有 2D 线性系统分类
分岔鞍-结点、跨临界、叉型、Hopf——四种系统变化模式

练习题

基础题。

  1. 判断稳定性：(a) $x' = -x,\ y' = -2y$；(b) $x' = y,\ y' = -\sin x - 0.5y$。
  2. 用 $V = x^2 + y^2$ 分析 $x' = -x + y^2,\ y' = -y$。
  3. 找出 $\dot x = rx - x^3$ 的分岔点。

进阶题。

  1. 证明总能量是阻尼单摆的 Lyapunov 函数，再用 LaSalle 推导渐近稳定。
  2. 分析 Van der Pol 振子：原点不稳定，但存在稳定极限环。
  3. 对 Lotka-Volterra 竞争模型，推导共存与排斥条件。

编程题。

  1. 根据迹和行列式分类 2D 线性系统平衡点，复现图 3。
  2. 动画演示 Hopf 分岔：扫描 $\mu$，观察极限环变化。

数值实现：用 SciPy 解 Lyapunov 方程

$A^\top P + PA = -Q$ 是矩阵方程。手算只能解二维。三维以上，调 scipy.linalg.solve_continuous_lyapunov：

import numpy as np
from scipy.linalg import solve_continuous_lyapunov, eigvals

A = np.array([[0, 1], [-2, -3]])      # 阻尼振子
Q = np.eye(2)
P = solve_continuous_lyapunov(A, -Q)   # 符号约定：A P + P A^T = -Q
print("P =", P)
print("P 正定？", np.all(eigvals(P).real > 0))

底层用 Bartels-Stewart 算法：将 $A$ 化为 Schur 形式 $A = U T U^\top$，令 $\tilde P = U^\top P U$，转为上三角线性方程组，按列回代求解。复杂度 $O(n^3)$，比直接展开成 $n^2 \times n^2$ 的 Kronecker 系统快 $n^3$ 倍。

陷阱。 若 $A$ 有特征值在虚轴上，方程奇异，SciPy 抛 LinAlgError。这对应非双曲平衡点的几何特性——数值报错与理论现象一致。

当线性化失效：中心流形与高阶项

Hartman-Grobman 要求 Jacobian 特征值实部非零。特征值实部为 0 时，线性近似只给出中性方向，真实行为由非线性项决定。

例 $\dot x = -x^3$。Jacobian 在 $x = 0$ 处为 $0$，线性化无法判断稳定性。但 $V(x) = x^2/2$ 满足 $\dot V = -x^4 \le 0$，系统稳定。这是线性方法失效、Lyapunov 方法有效的典型例子。

中心流形定理。 系统在原点有 $n_s$ 个稳定方向、$n_c$ 个中性方向时，存在 $n_c$ 维不变流形 $W^c$。原点附近轨道指数收敛到 $W^c$。判稳定性只需将动力学约化到 $W^c$，分析降维系统即可。Hopf 分岔的标准型（下一节）由此导出。

实际编程时。 若 np.linalg.eigvals(J) 返回某个特征值的 abs(real) < 1e-8，不要直接判定稳定。改用 Lyapunov 函数或高阶 Taylor 展开。

接入 ML：Lyapunov 神经网络与稳定性约束训练

$$ V_\theta(x) > 0,\quad V_\theta(0) = 0,\quad \nabla V_\theta(x)^\top f(x) < 0. $$

实际操作两条经验：

  1. 正定性直接嵌入架构。 设 $V_\theta(x) = \|\phi_\theta(x)\|^2 + \epsilon \|x\|^2$，$\phi_\theta$ 是普通 MLP。这样 $V_\theta \ge \epsilon \|x\|^2$ 自然成立，不用加罚项。
  2. 下降条件靠 SMT 验证。 训练后用 dReal 或 Z3 检查 $\nabla V_\theta(x)^\top f(x) < 0$ 是否严格成立。Sample-based loss 不可靠，必须用反例驱动迭代（CEGIS）。

相关论文：Neural Lyapunov Control (Chang et al., 2019)、Learning Stability Certificates from Data (Boffi et al., 2020)。已在机器人腿足控制中落地。
