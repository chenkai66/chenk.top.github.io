常微分方程（十四）：传染病模型与流行病学

从第一性原理构建数学流行病学：SIR / SEIR 模型，R0 与群体免疫阈值的解析推导，以及包含无症状传播与时变干预的 COVID 风格情景。

#ODE#SIR 模型#SEIR 模型#流行病学#COVID-19
Read in English →

2020年初，全世界盯着几个常微分方程做决策。 “拉平曲线"不是口号，是一个具体方程的直观体现；“群体免疫阈值"不是猜测，是一行公式推出来的 $1 - 1/R_0$。Kermack和McKendrick在1927年写下的SIR模型，四行数学，精确到能影响万亿级决策。

本章从零开始搭建这套工具。先推导SIR模型的所有阈值和最终规模关系，再逐步加入现实因素：潜伏期（SEIR）、无症状传播、疫苗接种、时变干预（类似COVID情景）。目标不是"相信"预测，而是理解每个参数控制什么机制。

本章要点

  • SIR 模型：3 方程系统，含仓室、参数、基本再生数 $R_0 = \beta/\gamma$
  • 阈值定理：$R_0 > 1 \Leftrightarrow$ 爆发，最终规模关系 $S_\infty = S_0 e^{-R_0(1 - S_\infty)}$
  • 峰值高度 $I^* = 1 - 1/R_0 - \ln R_0/R_0$，群体免疫阈值 $1 - 1/R_0$
  • SEIR 变体：潜伏期拖慢初期增长率
  • COVID-19 扩展：无症状仓室、时变 $R_e(t)$、报告冰山
  • 网络模型：超级传播者的作用
  • 实战拟合：$R_0$、倍增时间、世代间隔的实际意义

先修知识：第 7 章 相平面分析、第 8 章 非线性稳定性、第 11 章 数值方法。

SIR 模型

人群分三个仓室：

  • $S$ —— 易感者
  • $I$ —— 感染者（正在传播）
  • $R$ —— 移除者（免疫、隔离或死亡）

质量作用律 + 指数恢复，得到 Kermack-McKendrick SIR 系统：

$$\boxed{\;\dot S = -\frac{\beta\,S\,I}{N},\qquad \dot I = \frac{\beta\,S\,I}{N} - \gamma I,\qquad \dot R = \gamma I.\;}$$

两个参数决定系统行为：

  • $\beta$ —— 传染系数：单位时间接触数 × 单次接触传染概率
  • $\gamma$ —— 移除率：$1/\gamma$ 是平均传染期

总和 $S + I + R \equiv N$ 守恒，系统实际是二维的。

基本再生数 $R_0$

$$\dot I \approx (\beta - \gamma) I,$$$$\boxed{\;R_0 \equiv \frac{\beta}{\gamma} > 1.\;}$$

阈值定理：无病平衡局部稳定 iff $R_0 < 1$。$R_0$ 的意义明确：一个感染者放进完全易感人群，预期造成二代感染数。每人不到一个 -> 链条衰减；超过一个 -> 初期指数爆发。

峰值由谁决定？

$$\frac{dI}{dS} = \frac{\gamma}{\beta S/N} - 1 = \frac{1}{R_0\,S/N} - 1.$$$$i(s) = i_0 + (s_0 - s) + \frac{1}{R_0}\ln\frac{s}{s_0}.$$$$\boxed{\;I^* \;\approx\; 1 - \frac{1}{R_0} - \frac{\ln R_0}{R_0}.\;}$$

峰值时间是 $S$ 首次降至 $1/R_0$ 的时刻。两量都仅由 $R_0$ 决定（$\gamma$ 设定时间尺度）。

最终规模：谁能逃过？

$$\boxed{\;S_\infty = S_0\,\exp\!\bigl[-R_0\,(1 - S_\infty/N)\bigr].\;}$$

$R_0 = 2.5$、$S_0 \approx N$ 时，方程给出 $S_\infty/N \approx 0.107$ —— 约 89% 被感染。结果反直觉，但是一行数学推论。

左上：经典 SIR 时序，$R_0 = 2.5,\ 1/\gamma = 10$ 天——$S$ 崩塌、$I$ 先涨后落、$R$ 趋于饱和。右上：相平面 $(S, I)$——轨道进入上半平面，越过竖线 $S = 1/R_0$ 时 $I$ 取得峰值，然后趋向 $S$ 轴。左下：累计感染 $1 - S$ 渐近到最终规模值（红色虚线）。右下：$S_\infty$ 与 $1 - S_\infty$ 作为 $R_0$ 的函数——曲线在 1 附近极陡，$R_0$ 微小变化 -> 最终影响巨大。

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir(t, y, beta, gamma, N):
    S, I, R = y
    return [-beta*S*I/N, beta*S*I/N - gamma*I, gamma*I]

N, gamma = 1.0, 1/10
fig, ax = plt.subplots(figsize=(10, 5))
for R0 in [1.5, 2.0, 2.5, 3.0]:
    beta = R0 * gamma
    sol = solve_ivp(sir, [0, 200], [N - 1e-3, 1e-3, 0],
                    args=(beta, gamma, N),
                    t_eval=np.linspace(0, 200, 600))
    ax.plot(sol.t, sol.y[1], lw=2, label=f'R0 = {R0}')
ax.set_xlabel('天'); ax.set_ylabel('感染比例')
ax.legend(); plt.tight_layout(); plt.show()

对 $R_0$ 的敏感性

$R_0$ 是决定结果的关键杠杆。$R_0$ 增加一倍，医院峰值需求可能翻五倍，峰值时间提前数周。这种敏感性还有解析形式，非常少见。

左：固定 $1/\gamma = 10$ 天，$R_0 \in \{1.2, 1.6, 2.0, 2.5, 3.5, 5.0\}$ 的 $I(t)$ 曲线族。$R_0$ 越大，峰值越高且越早。中：峰值感染比例（红）和峰值时间（蓝）随 $R_0$ 的变化。黑色虚线是解析公式 $1 - 1/R_0 - \ln R_0/R_0$，与数值结果完全吻合。右：最终感染比例（红）和群体免疫阈值 $1 - 1/R_0$（绿色虚线）。注意，最终规模总是超过 HIT。疫情不会在免疫达到阈值时立刻停止，而是在传播无法维持时停下，此时已发生大量"额外"感染。

疫苗接种与群体免疫

$$R_e = R_0 \cdot \frac{S(0)}{N} = R_0\,(1 - v).$$


要 $R_e \leq 1$，需要 $v \geq 1 - 1/R_0$。这就是 群体免疫阈值（HIT）。

$$v \geq \frac{1 - 1/R_0}{\mathrm{VE}}.$$


高 $R_0$ 加上低 VE，可能让群体免疫无法实现（$v > 1$）。

左上：覆盖率 $v = 0,\ 0.20,\ 0.40,\ HIT,\ 0.85$ 时的 $I(t)$。$v = HIT$ 时曲线几乎压平，$v = 0.85$ 完全无爆发。右上：峰值和额外感染随覆盖率变化；绿色区域是 HIT 之后。左下：HIT 与 $R_0$ 关系，标注典型疾病。右下：不同 $\mathrm{VE} \in \{0.5, 0.7, 0.9, 1.0\}$ 下覆盖率随 $R_0$ 的变化——红色"不可行"区域显示高 $R_0$ 加低 VE 时单靠疫苗无法达到群体免疫。

这张图解释了公共卫生为何紧盯 “最后一群人”：高 $R_0$ 疾病中，最后 10% 和最初 60% 同样重要。

SEIR 模型

许多疾病有潜伏期——感染但未具传染性。加个潜伏仓室 $E$：

$$\dot S = -\frac{\beta SI}{N}, \quad \dot E = \frac{\beta SI}{N} - \sigma E, \quad \dot I = \sigma E - \gamma I, \quad \dot R = \gamma I.$$

$\sigma$ 是转化率，$1/\sigma$ 是平均潜伏期。$R_0 = \beta/\gamma$ 不变。潜伏期到底有没有影响？

$$r^2 + (\sigma + \gamma)\,r + \sigma(\gamma - \beta) = 0.$$$$r_{\text{SEIR}} = \frac{1}{2}\!\left[-(\sigma + \gamma) + \sqrt{(\sigma + \gamma)^2 + 4\sigma\gamma(R_0 - 1)}\right] < r_{\text{SIR}} = \beta - \gamma.$$

潜伏期拖慢早期指数增长，即使 $R_0$ 不变。倍增时间取决于世代间隔 $T_g \approx 1/\sigma + 1/\gamma$，不只是 $R_0$。

左上：完整 SEIR 轨迹（$R_0 = 3,\ 1/\sigma = 5,\ 1/\gamma = 7$ 天）。右上：同 $R_0$ 下 SEIR vs SIR——SEIR 峰值更晚、稍低。左下：$1/\sigma \in \{0.5, 2, 5, 10, 20\}$ 天的 $I(t)$ 族，潜伏期越长峰值越晚越宽。右下：初期增长率 $r$ 与潜伏期关系，右轴是倍增时间 $\ln 2/r$。$\sigma \to \infty$（瞬时转化）时 SEIR 退化为 SIR。

一个 COVID 风格扩展

真实疫情比 SEIR 复杂得多。COVID-19 增加了四个数学特点：

  1. 无症状传播：比例 $p$ 的感染者不发病，但仍有传染性（降为 $\kappa$）。
  2. 时变 $\beta$：封锁、口罩、行为变化影响传播。
  3. 报告冰山：只检出部分病例；报告 $= \rho \cdot I_s$，$\rho < 1$。
  4. 变异株：新毒株用新的 $R_0$ 重启传播。
$$\dot S = -\frac{\beta(t)\,(I_s + \kappa I_a)\,S}{N}, \quad \dot E = \frac{\beta(t)\,(I_s + \kappa I_a)\,S}{N} - \sigma E,$$$$\dot I_a = p\,\sigma E - \gamma I_a, \quad \dot I_s = (1 - p)\sigma E - \gamma I_s, \quad \dot R = \gamma(I_a + I_s).$$$$R_e(t) = \frac{\beta(t)\,(1 - p + \kappa p)}{\gamma}\,\frac{S(t)}{N}.$$

目标是 $R_e(t) < 1$。两条路：降低 $\beta$（干预）或降低 $S/N$（免疫）。

左上：四阶段情景——基线 $\beta = 0.6$ 持续 30 天，封锁 $\beta = 0.18$ 持续 30 天，部分放开 $\beta = 0.30$，变异株 + 放开 $\beta = 0.40$。右上：有效 $R_e(t)$——封锁压低 $R_e$ 至 1 以下（绿区），第三波反弹超 1。左下：报告冰山——黑色虚线是监测报告（$\rho I_s$），远低于真实流行 $I_s + I_a$（紫色）。右下：反事实对比——有/无干预的累计感染。间距即"避免的病例数”，政策收益按人口比例计。

这不是真实数据拟合，而是拟合结构的简化示意图。同样方程，结合 Bayes 推断与实际报告数据，2020-2022 年驱动了多国官方预测。

网络与异质模型

$$R_0 = \frac{\beta}{\gamma}\,\frac{\langle k^2\rangle - \langle k\rangle}{\langle k\rangle}.$$

对于无标度网络（$P(k) \propto k^{-\alpha}$，$2 < \alpha < 3$），$\langle k^2\rangle$ 随系统规模发散，导致爆发阈值趋于零——即使传染力很低，也能维持流行。超级传播者主导疫情早期传播；针对他们进行接触追踪，效果显著。

两点教训：

  • 平均接触率低估风险，方差同样重要。
  • 平均覆盖浪费资源，优先处理高度节点。

数学的胜场与败场

数学能赢

  • 数量级预测：4 周后医院会爆吗？SEIR 加合理 $R_0$ 就够用。
  • 阈值推理：要打多少疫苗？公式是 $1 - 1/R_0$ 除以 VE，不用拟合。
  • 反事实比较：封锁救了多少命？跑两次模拟，差值就是答案（受 $R_0$ 不确定性限制）。

数学会输

  • 精确预测：$R_0$ 因场景、人群、天气、行为变化。几周后的病例数不准。
  • 忽略异质性：年龄、地理、家庭聚集都很重要。均场模型算的峰值时间不对。
  • 行为反馈：新闻一出，人就减少接触。这是闭环系统，忽略反馈会高估峰值。

正确用法：把模型当讨论情景的工具，别当成真能预测。

总结

概念关键公式
基本再生数$R_0 = \beta / \gamma$
爆发阈值$R_0 > 1$
峰值感染比例$I^* \approx 1 - 1/R_0 - \ln R_0 / R_0$
最终规模关系$S_\infty = S_0\,e^{-R_0(1 - S_\infty/N)}$
群体免疫阈值$1 - 1/R_0$
不完美疫苗$v \geq (1 - 1/R_0)/\mathrm{VE}$
SEIR 初期增长$r$ 满足 $r^2 + (\sigma + \gamma)r + \sigma(\gamma - \beta) = 0$
有效 $R$$R_e(t) = R_0(t)\,S(t)/N$
网络 $R_0$$\beta\,\langle k^2 - k\rangle / (\gamma\,\langle k\rangle)$

练习

概念题

  1. 为什么 SIR 最终规模会超过群体免疫阈值？给出直观和解析两方面理由。
  2. 定义世代间隔和串行间隔，解释二者差异。
  3. 两国病例倍增时间相同，$R_0$ 是否可能差别很大？用 SEIR 模型说明。

计算题

  1. 解 $R_0 = 1.05, 1.5, 3.0$ 的 SIR 系统，数值验证最终规模公式与超越方程一致。
  2. SEIR、$R_0 = 3$，画出世代间隔 $1/\sigma + 1/\gamma$ 和倍增时间关系图。固定 $R_0$，验证倍增时间随世代间隔线性增长。
  3. 实现无症状 COVID 模型，用 $\beta$ 和起始时间拟合本国第一波数据，算出 $R_e(t)$。

编程题

  1. 动画展示 SIR 相图随 $R_0$ 从 0.5 到 5.0 的变化。
  2. 实现随机 SIR（Gillespie 算法），对比确定性 ODE 在 $N = 10^2,\ 10^4,\ 10^6$ 下的均值轨迹。观察确定性极限何时生效。
  3. 实现两年龄段（儿童、成人）SIR 模型，加入 2x2 接触矩阵。计算下一代矩阵，求主特征值得到 $R_0$。
