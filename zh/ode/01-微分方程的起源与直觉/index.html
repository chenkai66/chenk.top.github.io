常微分方程（一）：微分方程的起源与直觉

微分方程为何存在？从冷却的咖啡和摆动的单摆出发，建立你对 ODE 的第一直觉，并用 Python 求解第一个微分方程。

CK
Chen Kai July 1, 2023 · 22 分钟 · 5308 字
#ODE#Mathematical Modeling#Python
Read in English →

你身边的一切都在变化。 咖啡会凉，人口会涨，单摆会动，病毒会传，股价会变，行星会转。这些系统没法用“某物等于多少”描述——只能用“某物变化多快”刻画。这就是微分方程的意义。学会读它，就是学会物理和生物的语言。

本章从头重建直觉。我从一杯咖啡出发，推导出与放射性衰变、电容放电相同形式的方程，再逐步讲方向场、分类，以及决定 ODE 是否有意义的存在唯一性定理。

本章要点

  • 微分方程到底是什么——不只是符号，更要理解本质
  • ODE 和 PDE、阶数、线性——这些区分都有实际意义
  • 三个经典模型：冷却、衰变、振动，后续会反复遇到
  • 方向场：不求解也能直观看出解的行为
  • 存在唯一性定理（Picard–Lindelöf），附具体反例说明
  • 用 Python 的 scipy 求解第一个微分方程，明确可信范围

前置知识

  • 一元微积分：导数、积分、链式法则
  • Python基础：NumPy数组、Matplotlib绘图

1. 什么是微分方程？从一杯咖啡说起

冲了杯 $T_0 = 90^\circ\mathrm{C}$ 的咖啡，放在 $T_\text{env} = 20^\circ\mathrm{C}$ 的房间。几分钟后凉了。两个问题很自然：

  1. 现在多少度？——代数问题。
  2. 现在凉得多快？——微积分问题。

第一个看似简单，但物理更关心第二个。牛顿在 1701 年写下答案：冷却速率与温差成正比。

$$\frac{dT}{dt} = -k\,\bigl(T - T_\text{env}\bigr).$$

这行公式就是微分方程——把函数 $T(t)$ 和它的导数关联起来。未知的是函数，不是某个数。

符号含义
$T(t)$时刻 $t$ 的温度
$dT/dt$温度瞬时变化率
$T_\text{env}$环境温度（$20^\circ\mathrm{C}$）
$k > 0$冷却常数，取决于杯子、表面积、空气

负号很重要：$T > T_\text{env}$ 时右端为负，$T$ 下降；$T < T_\text{env}$ 时右端为正，$T$ 上升。这条方程自带热力学第二定律。

解之前先「读」方程

不动笔，仅凭结构就能得出几条结论：

  • 温度越接近 $T_\text{env}$，温差越小，冷却越慢。冷却是减速的。
  • $T = T_\text{env}$ 时右端为零——咖啡达到室温后不再变。这是平衡点。
  • $k$ 越大（薄壁杯、通风）冷却越快；$k$ 越小（保温杯）冷却越慢。

这是第一课：好的 ODE 在解出前就告诉你定性行为。

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

def coffee_cooling(T, t, k, T_env):
    """牛顿冷却定律：dT/dt = -k (T - T_env)"""
    return -k * (T - T_env)

T0, T_env, k = 90.0, 20.0, 0.1
t = np.linspace(0, 60, 500)
T_sol = odeint(coffee_cooling, T0, t, args=(k, T_env))

plt.figure(figsize=(10, 5))
plt.plot(t, T_sol, color="#2563eb", linewidth=2, label="咖啡温度")
plt.axhline(T_env, color="#ef4444", linestyle="--", label=f"室温 = {T_env} °C")
plt.xlabel("时间 (分钟)"); plt.ylabel("温度 (°C)")
plt.title("牛顿冷却定律"); plt.legend(); plt.tight_layout(); plt.show()

print(f"10 分钟后: {T_sol[np.abs(t-10).argmin()][0]:.1f} °C")
print(f"30 分钟后: {T_sol[np.abs(t-30).argmin()][0]:.1f} °C")

闭合解

这条方程可以手算。令 $u = T - T_\text{env}$，则 $du/dt = -ku$，分离变量并代回得：

$$\boxed{\,T(t) = T_\text{env} + \bigl(T_0 - T_\text{env}\bigr)\,e^{-kt}.\,}$$

记住三点：

  • 偏离量 $T - T_\text{env}$ 按纯指数衰减。
  • 衰减速度由 $k$ 决定。$k$ 减半，冷却时间加倍。
  • $T \to T_\text{env}$ 是渐近的——理论上到不了，实际上很快物理不再关心。

图 1. 牛顿冷却律给出一个解族，每条曲线对应一个初始温度，但所有曲线最终汇聚到虚线（室温）。热咖啡、冷咖啡、冻酸奶——同一物理，同一终点。

2. 方向场：解之前先「看见」解

有个小技巧，数学界花了 200 年才真正重视。对任意一阶 ODE

$$\frac{dy}{dt} = f(t, y),$$

函数 $f$ 直接给出 $(t, y)$ 平面上每点的解斜率，完全不用解 $y$。在格点画出对应斜率的小箭头，就得到方向场（也叫斜率场或向量场）。解曲线就是与箭头相切的轨迹。

图 2. 衰减方程 $dy/dt = -y$ 的方向场。箭头表示局部斜率；紫色曲线是真实解；红色虚线是平衡点 $y = 0$。所有解都被推向零——平衡点是全局吸引子。

靠这种图，能分析那些写不出闭合解的系统。看图就能回答：

  • 有没有平衡点？（箭头在哪水平？）
  • 是吸引还是排斥？（箭头指向还是远离？）
  • 解会爆炸、振荡，还是稳定？

整个系列我会反复用方向场——这是理解非线性 ODE 最简单、最实在的方法。

3. 分类：ODE、PDE、阶数、线性

几个标签能省不少口舌。

ODE 与 PDE

类型自变量个数示例
常微分方程 (ODE)一个（通常是 $t$）$\dfrac{dy}{dt} = -ky$
偏微分方程 (PDE)多个（$x, y, t, \dots$）$\dfrac{\partial u}{\partial t} = k\,\dfrac{\partial^2 u}{\partial x^2}$

本系列只谈 ODE。PDE（热方程、波动方程、薛定谔、Navier–Stokes）留到第 13 章。

阶数 = 最高导数的阶

  • 一阶：$y' = f(t, y)$——放射性衰变、RC 电路、单种群增长。
  • 二阶：$y'' + p(t)y' + q(t)y = g(t)$——弹簧、单摆、RLC 电路、行星轨道。

阶数重要，因为唯一解需要恰好这么多初始条件。一阶 ODE 要 $y(0)$；二阶 ODE 要 $y(0)$ 和 $y'(0)$——位置和速度。这不是数学的怪癖，而是 $F = ma$ 的本质：预测粒子未来，必须知道初始位置和速度。

图 3. 左：$y' = -y$ 给出单参数解族——选起点，曲线就定了。右：$x'' + x = 0$ 给出双参数解族——要给初始位置和速度。阶数 = 宇宙要多少初始信息。

线性与非线性

如果 $y$ 及其导数只以一次幂出现且不相乘，方程就是线性的。$n$ 阶线性方程一般形式为

$$a_n(t)\,y^{(n)} + a_{n-1}(t)\,y^{(n-1)} + \dots + a_0(t)\,y = g(t).$$

只要出现 $y^2$、$\sin y$、$y\,y'$、$\sqrt{y}$ 等「不规矩」项，方程就是非线性的。

区分重要，因为线性是超能力：

  • 解可叠加（叠加原理）。
  • 完整解分解为齐次解 + 特解。
  • 线性代数工具（特征值、变换、Green 函数）直接可用。

非线性方程没这些便利。它们能做到线性做不到的事——极限环、混沌、有限时间爆破——但每个问题往往得单独处理。

图 4. 同样大小，不同世界。左：线性方程 $y' = -y + \sin t$ 所有解归于红色周期吸引子——未来「忘记」初始条件。右：非线性方程 $y' = y^2 - t$ 中，$y(0)$ 微小变化导致命运迥异，部分解在有限时间内逃逸到 $+\infty$。欢迎来到非线性世界。

4. 微分方程简史

微分方程不是「被发现」的，而是物理「逼出来」的。微积分诞生那会儿正好需要它。

年代人物里程碑
1666牛顿微积分 + 运动定律。$F = ma$ 是二阶 ODE
1690s伯努利家族悬链线、最速降线——把物理问题转成 ODE
1739欧拉系统化 ODE 理论；欧拉方法（现代积分器的起点）
1820s柯西解的存在性证明——解到底有没有？
1890皮卡、林德勒夫唯一性定理的现代形式
1880s+庞加莱定性理论：不追公式，研究解流的几何形状
1963洛伦兹混沌从三方程天气模型中诞生

这条弧线很清晰：从追公式到放弃，再到研究解流的几何。现代 ODE 理论更像庞加莱，不像牛顿。

5. 三个反复出现的经典模型

ODE 看似无处不在，其实是因为少数几个方程总以不同物理形式反复出现。以下是三个最经典的。

图 5. 科学中最常用的三条方程。每条只有一行，却共同描述了热交换、考古测年和机械工程中所有弹簧–质量系统。

5.1 放射性衰变——宇宙的秒表

每个放射性原子单位时间内衰变概率相同，总衰变率与剩余原子数成正比：

$$\frac{dN}{dt} = -\lambda N \qquad\Longrightarrow\qquad N(t) = N_0\,e^{-\lambda t}.$$

半衰期 $t_{1/2} = (\ln 2)/\lambda$ 是一个能概括整条曲线的数字。碳-14 的 $t_{1/2} = 5730\,\text{年}$，考古学家据此测年：测出残留 $^{14}\mathrm{C}$ 比例，取对数即可得年龄。同一条方程也描述 RC 电路电容放电和药物在血液中的清除。

5.2 马尔萨斯人口增长

若种群数量为 $P$，人均出生率减死亡率为 $r$，则

$$\frac{dP}{dt} = rP \qquad\Longrightarrow\qquad P(t) = P_0\,e^{rt}.$$

这与衰变方程完全相同，只是把 $-\lambda$ 换成 $r$。数学不关心东西是死是生，只关心速率常数的符号。

它预测的是无界指数增长——任何生物学家都会说这是胡扯，因为食物、空间或宿主迟早耗尽。修正方案是 Logistic 方程，它太重要，我直接把它和原版画在一起。

图 6. 左：马尔萨斯模型冲向无穷；Logistic 模型弯下腰，饱和于环境容量 $K$。右：增长率的「相视图」。Logistic 曲线有两个平衡点（$P = 0$ 和 $P = K$），中间增长，$K$ 之上衰减。第二章会细讲。

5.3 简谐振动

弹簧上的质点 $m$，受 Hooke 力 $F = -kx$ 和牛顿第二定律 $F = ma$ 支配：

$$m\,\frac{d^2 x}{dt^2} = -kx \qquad\Longleftrightarrow\qquad x'' + \omega_0^2\,x = 0, \quad \omega_0 = \sqrt{k/m}.$$

通解为 $x(t) = A\cos(\omega_0 t + \varphi)$——纯正弦波，自然角频率为 $\omega_0$。这条方程出现在小角度单摆、LC 电路、分子振动，以及几乎所有线性系统的本征模中。

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

omega0 = 2 * np.pi  # 自然频率：1 Hz
t = np.linspace(0, 3, 500)

def spring(state, t):
    x, v = state
    return [v, -omega0**2 * x]

sol = odeint(spring, [1.0, 0.0], t)  # x(0) = 1, v(0) = 0

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))
ax1.plot(t, sol[:, 0], color="#10b981", linewidth=2)
ax1.set_xlabel("时间 (s)"); ax1.set_ylabel("位移 x")
ax1.set_title("简谐振动")

ax2.plot(sol[:, 0], sol[:, 1], color="#7c3aed", linewidth=2)
ax2.set_xlabel("x"); ax2.set_ylabel("v = dx/dt")
ax2.set_title("相图——能量守恒给出一个闭合椭圆")
ax2.set_aspect("equal")
plt.tight_layout(); plt.show()

右图是一张相图，第一次展示二阶系统天然生活在 $(x, v)$ 二维空间里。能量守恒迫使轨迹成为闭合椭圆；这个几何事实将是第 7 章的核心。

6. 初值问题与解的存在性

一个裸 ODE 有无穷多解——一整族曲线（一参数或 $n$ 参数）。要挑出唯一解，需加初始条件：

$$\frac{dy}{dt} = f(t, y), \qquad y(t_0) = y_0.$$

二者合称初值问题（IVP）。接下来的问题，数学家花了两百年才认真研究：

IVP 是否真的有解？解是否唯一？

现代答案是 Picard–Lindelöf 定理。

定理（Picard–Lindelöf, 1890）. 若 $f(t, y)$ 在 $(t_0, y_0)$ 附近的矩形区域中对 $t$ 连续、对 $y$ 满足 Lipschitz 条件，则 IVP 在包含 $t_0$ 的某开区间上有唯一解。

「对 $y$ Lipschitz」指存在常数 $L$，使 $|f(t, y_1) - f(t, y_2)| \le L\,|y_1 - y_2|$。通俗讲，$f$ 沿 $y$ 方向的斜率有界。多数「老实」右端项（多项式、光滑函数）自动满足。

为什么重要？因为没有这个保证，模型本身就有歧义——这种情况确实存在。

图 7. 左： 对 $y' = -y$，$f$ 处处光滑；每点恰有一条解。红色曲线是从 $(0, 1)$ 出发的唯一解。右： 对 $y' = 3\,y^{2/3}$，$f$ 连续但在 $y = 0$ 处不 Lipschitz（斜率无穷大）。从原点出发，无穷多解都合法：$y(t) = 0$ 是一条，$y(t) = (t-c)^3$（任意 $c \ge 0$）也是一条，方程无法决定。物理上，模型失效。

结论明确：以 ODE 写下的「物理定律」，只有 Picard–Lindelöf 成立时才是真定律。 否则未来无法由现在唯一决定，决定论模型的意义就崩塌了。

7. 解析解 vs 数值解

解析解数值解
结果公式：$y(t) = C e^{-kt}$离散表 $(t_n, y_n)$
来源笔算、闭合积分时间步进：$y_{n+1} = y_n + h\,f(t_n, y_n) + \dots$
优点精确，揭示依赖关系能解任何写得出来的 ODE
缺点多数 ODE 没闭合解近似解，需控步长

研究生物理课上可能手算十几条 ODE。但工程和科研中，多数 ODE 靠数值解——这不丢人，现代建模基本都靠它。

最简单的数值方法是欧拉法，直接离散化导数定义：

$$y_{n+1} = y_n + h\,f(t_n, y_n).$$

import numpy as np
import matplotlib.pyplot as plt

def euler(f, y0, t):
    """前向欧拉法求解 y' = f(y, t)"""
    y = np.zeros(len(t)); y[0] = y0
    for i in range(len(t) - 1):
        h = t[i+1] - t[i]
        y[i+1] = y[i] + h * f(y[i], t[i])
    return y

t = np.linspace(0, 5, 100)
y_exact = np.exp(-t)
y_euler = euler(lambda y, t: -y, 1.0, t)

plt.figure(figsize=(8, 5))
plt.plot(t, y_exact, color="#2563eb", linewidth=2, label=r"精确解：$y = e^{-t}$")
plt.plot(t, y_euler, color="#ef4444", linewidth=2, linestyle="--", label="欧拉近似")
plt.xlabel("t"); plt.ylabel("y(t)")
plt.title("解析解 vs 数值解")
plt.legend(); plt.tight_layout(); plt.show()

欧拉法是高级方法的基础（Runge–Kutta、Adams–Bashforth、轨道传播用的辛积分器）。第 11 章会拆解 scipy.integrate.solve_ivp 的原理——底层逻辑不变：估导数、走一步、重复、控误差。

总结

概念核心要点
微分方程关联未知函数和其导数的方程
ODE vs PDE单个自变量 vs 多个自变量
阶数最高导数阶数，等于初始条件数
线性 vs 非线性线性能叠加、理论完整；非线性可能混沌
方向场先看「流」的方向，再解方程
IVP + Picard–Lindelöf初始条件 + Lipschitz 的 $f$ ⇒ 唯一解
解析解 vs 数值解能解析就解析，不能就数值（多数情况）

本章一句话：物理定律写成微分方程，因为自然讲的是「变化率」的语言；学 ODE，就是学会把这语言翻成图、公式和预测。

练习题

热身

  1. 验证 $y(t) = C e^{3t}$（任意常数 $C$）是 $y' = 3y$ 的解。
  2. 房间温度 $20^\circ\mathrm{C}$，咖啡从 $90^\circ\mathrm{C}$ 冷却到 $60^\circ\mathrm{C}$ 花了 10 分钟。求冷却常数 $k$，预测 $t = 30$ 分钟时的温度。
  3. 某放射性样品半衰期 10 年，100 g 样品 25 年后剩多少？

概念

  1. 证明：无论 $y(0)$ 取何值，$y' = -2y$ 的所有解都满足 $y \to 0$（当 $t \to \infty$）。
  2. 手绘 $y' = y(1 - y)$ 的方向场，找出平衡点，预测 $y(0) = 0.1$、$y(0) = 0.5$、$y(0) = 1.5$ 的长期行为。（这是 Logistic 方程。）
  3. 马尔萨斯人口模型为什么不现实？提出至少两种应加入的物理机制。
  4. 证明 IVP $y' = 3 y^{2/3}$、$y(0) = 0$ 有多个解，并指出违反了 Picard–Lindelöf 的哪条假设。

编程

  1. 实现欧拉法，用步长 $h = 0.5,\,0.1,\,0.01$ 求解 $y' = -y$、$y(0) = 1$ 在 $[0, 5]$ 上的解。在 log–log 坐标上画全局误差与 $h$ 的关系，斜率是多少？为什么？
  2. 画出 $y' = \sin t - y$ 的方向场和若干解曲线，猜测其长期行为并用数值验证。
  3. 重现图 6（Logistic vs 指数）：用 scipy.integrate.solve_ivp 同时求解两条 ODE 并叠图。Logistic 解何时达到 $K$ 的 99%？
