常微分方程（十二）：边值问题

边值问题在区间两端各给一部分信息。掌握打靶法、有限差分、配点法和 Sturm-Liouville 特征值问题——从梁的挠度到量子谐振子。

Read in English →

初值问题给定起始状态，要求向前推进。边值问题在两点提供部分信息，要求找到满足两端条件的解。措辞变化小，影响却大：边值问题可能有唯一解、无解或无穷多解。解决它需要一套全新工具——迭代、全局且与线性代数紧密相关。

这里 ODE 方法悄然过渡到 PDE 方法。离散化、特征值、配点等思想直接推广到高维椭圆型 PDE。

本章要点

  • BVP 比 IVP 质上更难，存在性和唯一性都可能出问题
  • 三种边界条件：Dirichlet、Neumann、Robin
  • 打靶法：将 BVP 转为求解初值的根问题
  • 有限差分：将 BVP 转为稀疏线性方程组
  • Sturm-Liouville 理论：BVP 化为矩阵特征值问题（通向量子力学和振动分析）
  • 配点法：用 scipy.integrate.solve_bvp 实现

前置知识：第 11 章 的数值方法。

1. 从 IVP 到 BVP：小改动，大影响

经典二阶 BVP：$y'' = f(x, y, y'), \quad y(a) = \alpha, \quad y(b) = \beta.$
IVP 指定$y(a)=\alpha, y'(a)=\alpha'$。数据量一样（两个标量），但分布很关键。

IVP 在轻度 Lipschitz 条件下，Picard 定理保证存在唯一性。流一定存在。BVP 没这种保障。

教科书常用例子

看$y'' + y = 0$加不同边界条件：

边界条件解
$y(0)=0,\;y(\pi)=0$无穷多解：$y = A\sin x$，$A$任意
$y(0)=0,\;y(\pi)=1$无解（$\sin x$在$x=\pi$处恒为零）
$y(0)=0,\;y(1)=1$唯一解：$y = \sin(x)/\sin(1)$

同一方程，三种不同结果。原因是$\sin x$在$x=\pi$处为零——齐次问题有非平凡解。按 Fredholm 二择一原理，非齐次问题此时要么超定，要么欠定。BVP 的存在唯一性取决于边界数据，不只看方程。

三类边界条件

  • Dirichlet（函数值）：$y(a)=\alpha,\;y(b)=\beta$。两端温度固定。最常见。
  • Neumann（导数值）：$y'(a)=\alpha,\;y'(b)=\beta$。规定热流或梁的自由端。
  • Robin（线性组合）：$\alpha_1 y(a) + \alpha_2 y'(a) = \gamma$。Newton 冷却定律、部分吸收。

实际中常混合使用（一端 Dirichlet，一端 Neumann）。下面的数值方法稍改即可处理三种情况。

2. 打靶法

最简单的想法：把 BVP 转成参数化的 IVP。二阶 BVP 已知 $y(a)=\alpha$，但不知道 $y'(a)$。猜一个 $s$，正向积分 $y'' = f(x, y, y')$，初值 $y(a) = \alpha, y'(a) = s$，到 $x=b$ 看残差 $F(s) := y(b; s) - \beta$。要求 $F(s)=0$，这是个一维求根问题，用二分、割线、Newton 或 brentq 都行。

左：BVP $y''+y=0,\;y(0)=0,\;y(\pi/2)=1$。五个初始斜率对应五条轨迹，只有 $s=1$ 击中目标。右：残差 $F(s) = y(\pi/2; s) - 1$。打靶把 BVP 化为找该曲线的零点。

几何上就是炮兵测距：发射不同仰角的炮弹，调整直到命中目标。这也是方法名字的由来。

import numpy as np
from scipy.integrate import solve_ivp
from scipy.optimize import brentq

def shooting_method(f, a, b, alpha, beta, s_low=-10, s_high=10):
    """打靶法解 y'' = f(x, y, y'), y(a)=alpha, y(b)=beta。"""
    def residual(s):
        sol = solve_ivp(lambda x, Y: [Y[1], f(x, Y[0], Y[1])],
                        [a, b], [alpha, s], dense_output=True,
                        rtol=1e-9, atol=1e-12)
        return sol.sol(b)[0] - beta

    # 必要时扩张区间
    while residual(s_low) * residual(s_high) > 0:
        s_low, s_high = s_low * 2, s_high * 2

    s_opt = brentq(residual, s_low, s_high, xtol=1e-10)
    sol = solve_ivp(lambda x, Y: [Y[1], f(x, Y[0], Y[1])],
                    [a, b], [alpha, s_opt], dense_output=True,
                    rtol=1e-9, atol=1e-12)
    x_out = np.linspace(a, b, 200)
    return x_out, sol.sol(x_out)[0]

打靶法什么时候管用，什么时候不管用？

底层 IVP 良态时，打靶法极好：$s$ 小变化只引起 $y(b)$ 小变化，求根目标干净。但如果 IVP 指数放大误差，打靶就糟糕——奇异摄动附近或含增长模的刚性问题里常见。试试在 $[0,10]$ 上打靶 $y'' - 100y = 0$（$y(0)=0, y(10)=1$）：$\sinh$ 模污染一切，残差 $F(s)$ 跨越约 $10^{43}$ 量级，求根精度无从谈起。

标准对策是多重打靶：把 $[a, b]$ 切成 $M$ 段，各段独立打靶（两端均为未知），段间加匹配方程。结果是非线性方程组变大，但每段短而良态。

3. 有限差分法

$$y'(x_i) \approx \frac{y_{i+1} - y_{i-1}}{2h}, \qquad y''(x_i) \approx \frac{y_{i+1} - 2y_i + y_{i-1}}{h^2}.$$


代入 BVP 的每个内部节点$i=1,\ldots,N-1$，得到$N-1$个方程，对应$N-1$个未知数$y_1,\ldots,y_{N-1}$（边界值$y_0=\alpha, y_N=\beta$已包含在右端）。

线性 BVP$y'' + p(x)y' + q(x)y = r(x)$的矩阵是三对角的，只有上下两条非零对角线。$n\times n$三对角系统求解开销$\mathcal{O}(n)$，远低于通用 Gauss 消元的$\mathcal{O}(n^3)$。


左：$y''=-\pi^2\sin(\pi x)$（$y(0)=y(1)=0$）数值解，精确解为$\sin(\pi x)$。三个高亮点显示局部三点模板。右：$h^2A$结构，仅三条非零对角线，其余严格为零。内存与 CPU 开销均为$\mathcal{O}(N)$。

非线性 BVP 离散后得非线性代数系统，用 Newton 迭代求解。Newton-Jacobi 矩阵保持稀疏性，每次迭代开销仍是$\mathcal{O}(N)$。

import numpy as np
from scipy.linalg import solve_banded

def finite_difference_linear(p, q, r, a, b, alpha, beta, N):
    """中心差分求解 y'' + p(x)y' + q(x)y = r(x), y(a)=alpha, y(b)=beta。"""
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    n = N - 1
    pi = np.array([p(xi) for xi in x[1:-1]])
    qi = np.array([q(xi) for xi in x[1:-1]])
    ri = np.array([r(xi) for xi in x[1:-1]])

    main = -2 / h**2 + qi
    upper = 1 / h**2 + pi[:-1] / (2 * h)
    lower = 1 / h**2 - pi[1:]  / (2 * h)

    rhs = ri.copy()
    rhs[0]  -= (1 / h**2 - pi[0]  / (2 * h)) * alpha
    rhs[-1] -= (1 / h**2 + pi[-1] / (2 * h)) * beta

    ab = np.zeros((3, n))
    ab[0, 1:] = upper
    ab[1, :]  = main
    ab[2, :-1] = lower

    y_int = solve_banded((1, 1), ab, rhs)
    return x, np.concatenate([[alpha], y_int, [beta]])

收敛性

中心差分是二阶方法，截断误差$\mathcal{O}(h^2)$，$h$减半时误差降为四分之一。要四阶精度可用紧致（Pade）差分或 Numerov 方法（专为$y''=f(x,y)$设计）。要谱精度可用 Chebyshev 配点，但代价是稠密矩阵。

4. 特征值问题

$$-y'' = \lambda y, \quad y(0) = y(\pi) = 0.$$


直接计算得特征值 $\lambda_n = n^2$，特征函数为 $\sin(nx)$，$n=1,2,3,\ldots$。

用有限差分离散后，算子 $-y''$ 变成对称三对角矩阵。其特征值近似 $\{\lambda_n\}$，低模（长波长）精度高，高模（网格无法分辨小于 $2h$ 的振荡）误差逐渐增大。


左：前五个数值特征模（彩色实线）与精确 $\sin(nx)$（黑虚线），错位显示更清晰。数值特征值精确到小数四位，匹配 $1, 4, 9, 16, 25$。右：低模严格落在 $\lambda_n=n^2$ 曲线上；系统色散误差随 $n$ 接近网格极限而增大。

from scipy.linalg import eigh_tridiagonal
import numpy as np

# -y'' = lambda y, y(0) = y(pi) = 0, N 段
N = 400
h = np.pi / N
n = N - 1
main = (2 / h**2) * np.ones(n)
off  = (-1 / h**2) * np.ones(n - 1)
eigvals, eigvecs = eigh_tridiagonal(main, off, select='i',
                                    select_range=(0, 9))
print(eigvals.round(4))    # [1.0000  4.0000  9.0000 16.0000 25.0000 ...]

5. Sturm-Liouville 理论

一般二阶自共轭特征值问题：$-\bigl(p(x)\,y'\bigr)' + q(x)\,y = \lambda\,w(x)\,y, \quad y(a) = y(b) = 0,$ 其中 $p>0, w>0$。这是 Sturm-Liouville 形式，也是从 ODE 到数学物理的桥梁。

性质（有限区间上的正则 SL 问题）：

  1. 特征值$\lambda_1 < \lambda_2 < \lambda_3 < \cdots \to \infty$，实数、单重、有序。
  2. 正交性：$\int_a^b w(x) y_m(x) y_n(x)\,dx = 0$（$m\ne n$）。
  3. 完备性：满足边界条件的光滑函数可展为$f(x) = \sum_n c_n y_n(x)$，其中$c_n = \int wf y_n / \int w y_n^2$。
  4. 振荡定理：第$n$个特征函数$y_n$在$(a,b)$内有$n-1$个零点。

这些性质让 Fourier 级数、Bessel 展开、Legendre 级数和量子力学束缚态都能用。它们是思考"特征函数展开"的正确方式。

实例：量子谐振子

无量纲 Schrödinger 方程：$-\psi''(x) + x^2\,\psi(x) = E\,\psi(x), \quad \psi(\pm\infty) = 0.$ 这是$p \equiv 1, q(x) = x^2, w \equiv 1$的 SL 问题（无界域）。精确特征值：$E_n = 2n + 1$（$n=0,1,2,\ldots$），特征函数为 Hermite 函数。

数值解法：将域截断到$[-L, L]$（$L$足够大使$\psi$在边界接近零），中心差分离散，求矩阵的最低几个特征值。

黑线为势能$V(x)=x^2$。彩色特征函数画在能量$E_n$高度上，呈现经典结构：每升高一模多一个零点；波函数在回弹点间振荡，外侧指数衰减。数值特征值与精确$2n+1$匹配到小数四位。

这一模板——“离散，eigh，按能量画”——基本是一维束缚态量子力学的整套工具。同样方法适用于振动弦、鼓、分子的简正模。

6. 配点法：scipy.integrate.solve_bvp

现代 BVP 求解器多用配点法，不用打靶或有限差分。思路：假设解是网格上的分段多项式，要求它在每段的若干"配点"上严格满足 ODE。SciPy 的 solve_bvp 实现了 4 阶 Lobatto IIIa 格式，支持自适应网格细化。

用户提供：

  • fun(x, y)：右端函数，向量化的一阶系统。
  • bc(ya, yb)：边界条件残差，返回数组长度等于系统大小。
  • x 和 y_init：初始网格和初始猜测。

from scipy.integrate import solve_bvp
import numpy as np

# Bratu 问题：y'' + lambda * exp(y) = 0, y(0) = y(1) = 0, lambda = 2
def fun(x, y):
    return np.vstack([y[1], -2 * np.exp(y[0])])

def bc(ya, yb):
    return np.array([ya[0], yb[0]])      # 两端为零

x_init = np.linspace(0, 1, 10)
y_init = np.zeros((2, x_init.size))
y_init[0] = 0.5 * np.sin(np.pi * x_init)        # 凸起形猜测

sol = solve_bvp(fun, bc, x_init, y_init, tol=1e-10)
assert sol.success

solve_bvp 优势：

  • 自适应网格细化，节点放在解变化最快处。
  • 原生支持非线性 BVP，通过 Newton 迭代求解配点残差。
  • 结果 sol.sol 是连续可微样条，任意点可求值。

弱点：需要合理初始猜测，糟糕的猜测可能让 Newton 收敛到错误分支或不收敛。

7. 一题多解：同一问题的三种方法

著名的 Bratu 问题 $y'' + 2 e^y = 0,\;y(0)=y(1)=0$ 在低分支有两个解，一个小的，一个较大的。我关注小的。

左：打靶 + brentq、有限差分 + Newton、solve_bvp 配点——三种算法、三条数值流水线、一个解。右：两两差异在 $10^{-7}$ 量级，由各方法容差决定。

这种冗余让人放心。两种独立 BVP 方法在容差外不一致时，必有一个错了。通常是打靶撞上敏感性墙，或 solve_bvp 被坏初值困在错误分支。

8. 通向何方：应用

  • 梁的挠度。Euler-Bernoulli 梁$EI\,y^{(4)} = w(x)$，搭配固定/自由/简支端条件。特征值是固有振动频率，特征函数是模态形状。
  • 稳态热传导。$(k(x)T'(x))' + Q(x) = 0$，用 Dirichlet（指定温度）或 Neumann（指定热流）边界。典型的 Sturm-Liouville 问题；分离变量后特征函数展开给出时间依赖解。
  • 量子束缚态。类似上述。换成有限井、周期势或类氢库仑尾——流程不变，只需改$V(x)$。
  • Thomas-Fermi、Bratu、Falkner-Skan、Blasius、Painleve。经典非线性 BVP，历史丰富。给个合理初值，solve_bvp 能搞定。
  • 两点最优控制。Pontryagin 极大值原理将最优控制化为状态-协态系统的 BVP。多重打靶法是主力。

有限差分与配点思想扩展到二维、三维椭圆 PDE（Laplace、Poisson、Helmholtz）。矩阵不再三对角但保持稀疏；迭代求解器（multigrid、Krylov）处理代数问题。BVP 的知识全都能放大用。

9. 方法选择速查

方法适用注意
打靶良态 IVP，简单问题敏感性 / 刚性爆炸
多重打靶敏感问题、两点最优控制设置复杂，非线性系统更大
有限差分线性问题、特征值问题阶数限二，提阶需额外工作
谱 / Chebyshev光滑解、高精度基底选择需谨慎
配点（solve_bvp）通用非线性 BVP初值猜测要合理

实用建议：新 BVP 直接用 solve_bvp。失败通常是初猜问题，不是方法问题。

习题

  1. 存在唯一性实验。用 solve_bvp 解$y''+y=0,\;y(0)=0,\;y(\pi)=1$。记录结果，用 Fredholm 二择一原理解释。

  2. 有限差分收敛性。取$N\in\{20,40,80,160,320\}$，解$y''=-\pi^2\sin(\pi x),\;y(0)=y(1)=0$。log-log 图画最大误差对$h$，验证斜率为 2。

  3. 量子势井。在$[-10, 10]$上求$-\psi''+V(x)\psi=E\psi$的最低五个特征值与特征函数，$V(x)=-10e^{-x^2/2}$。高斯井有几个束缚态（$E<0$）？

  4. 多重打靶。单打靶解$y''=100y,\;y(0)=0,\;y(1)=1$（应失败或精度差）。将$[0,1]$分 5 段，用多重打靶验证结果是否匹配$y=\sinh(10x)/\sinh(10)$。

  5. 非线性示例。用 solve_bvp 求 Bratu 问题$y''+\lambda e^y=0,\;y(0)=y(1)=0$（$\lambda=2$）的两个分支。提示：试两种不同凸起幅度的初猜。
