2. Neural ODE ── 深度即时间

2018 年一篇 NeurIPS 论文 “Neural Ordinary Differential Equations”（Chen, Rubanova, Bettencourt, Duvenaud）让深度学习圈开始翻 ODE 教材。方法干净得有点过分。

残差网络更新隐状态公式：

$$h_{t+1} = h_t + f(h_t,\; \theta_t).$$

把层索引当作连续时间变量，步长缩小到极限，离散更新变成：

$$\frac{d h(t)}{dt} = f\!\bigl(h(t),\; t,\; \theta\bigr).$$

这就是可学习的 ODE。前向传播是解 ODE；网络有 自适应深度；反向传播用 伴随方法 把内存开销降到 $O(1)$（和控制理论第 16 章 Pontryagin 方程类似）。

左上：离散 ResNet（左侧蓝色块）随 $N \to \infty$ 变成连续深度 ODE（右侧渐变条）。右上：轨迹由学习到的向量场 $f_\theta(h,t)$ 决定——这是网络在"思考"。左下：伴随方法反向积分 ODE 得梯度，无需存中间激活。右下：ResNet 层数增加时离散轨迹逐步逼近同一条连续轨迹；ODE 求解器自动选步长。

# Neural ODE 训练循环（伪代码概念示例）
# 实际实现：torchdiffeq、jax.experimental.ode、diffrax

import torch, torch.nn as nn
from torchdiffeq import odeint_adjoint as odeint

class ODEFunc(nn.Module):
    def __init__(self, dim=2, hidden=64):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(dim, hidden), nn.Tanh(),
            nn.Linear(hidden, dim))
    def forward(self, t, h):
        return self.net(h)

f = ODEFunc()
h0 = torch.randn(32, 2)                      # 初始隐状态批次
t  = torch.linspace(0., 1., 20)
h_traj = odeint(f, h0, t)                    # shape (20, 32, 2)
loss   = ((h_traj[-1] - target) ** 2).mean()
loss.backward()                              # 伴随法计算梯度

Neural ODE 天然适合处理 不规则采样的时间序列（医疗记录、传感器日志），还催生了用于密度建模的 连续归一化流。哲学上它是机器学习重新发现微积分传统的最佳样本——积分器不再是工具，它就是模型。

3. 时滞微分方程 ── 带"记忆"的系统

很多真实系统不只响应当前状态，还响应过去的某个状态。配送车队对今天的价格变化，反应的是两周前的订单量。今天的红细胞计数反映几天前骨髓的状态。激光腔反馈的是皮秒前发出的光。

一阶时滞方程的一般形式：

$$\dot x(t) = f\bigl(x(t),\; x(t - \tau)\bigr).$$

状态空间是无限维的：初始数据需要整段历史 $\{x(s) : s \in [t-\tau, t]\}$，而不是单个数值。

Hutchinson 方程

带延迟的 Logistic 模型：

$$\dot N(t) = r N(t)\,\Bigl(1 - \frac{N(t-\tau)}{K}\Bigr).$$

无延迟（$\tau = 0$）时是平滑的 Verhulst S 形曲线。有延迟时可能失稳，产生极限环。具体来说，当 $r\tau > \pi/2$ 时，平衡点通过 Hopf 分岔失稳，出现振荡。

import numpy as np, matplotlib.pyplot as plt
from collections import deque

def solve_dde(f, x0, t_span, tau, dt=0.01):
    """简易 Euler 时滞解算器——t_span[0] 之前为常值历史。"""
    n_delay = int(round(tau / dt))
    n_steps = int((t_span[1] - t_span[0]) / dt)
    history = deque([x0]*(n_delay + 1), maxlen=n_delay + 1)
    t_vals, x_vals = [t_span[0]], [x0]
    t = t_span[0]
    for _ in range(n_steps):
        x_new = history[-1] + dt * f(t, history[-1], history[0])
        history.append(x_new)
        t += dt
        t_vals.append(t); x_vals.append(x_new)
    return np.array(t_vals), np.array(x_vals)

r, K = 1.0, 1.0
for tau in [0.5, 1.5, 2.0, 2.5]:    # 后两个穿越 r*tau = pi/2
    t, N = solve_dde(lambda t, n, nd: r*n*(1 - nd/K), 0.5, [0, 60], tau)
    plt.plot(t, N, label=f'tau = {tau}')
plt.axhline(1, color='k', ls='--', alpha=0.4)
plt.legend(); plt.show()

Mackey-Glass 方程 在 $\tau = 17$ 时产生低维混沌——一个由单一时滞反馈环生成的奇异吸引子。时滞常见于流行病学（潜伏期）、经济学（生产周期）和所有带传输延迟的控制回路。

4. 随机微分方程 ── 噪声成为主角

真实系统常受 随机 激励，比如分子热扰动、市场微结构噪声、突变。最自然的描述是 Ito 随机微分方程

$$dX_t = f(X_t, t)\,dt + g(X_t, t)\,dW_t,$$

其中 $W_t$ 是标准 Wiener 过程（布朗运动），$dW_t$ 是形式高斯增量，均值 0，方差 $dt$。

两个经典例子：

几何布朗运动（Black-Scholes 资产模型）

$$dS = \mu S\,dt + \sigma S\,dW, \qquad S(t) = S_0\exp\!\Bigl[(\mu - \tfrac12\sigma^2)\,t + \sigma W(t)\Bigr].$$

Ornstein-Uhlenbeck 过程（均值回复）

$$dX = \theta(\mu - X)\,dt + \sigma\,dW.$$

SDE 的概率密度 $\rho(x, t)$ 满足 Fokker-Planck 方程

$$\partial_t \rho = -\partial_x(f\rho) + \tfrac12\,\partial_x^2(g^2\rho),$$

这是一个确定性 PDE。单条轨迹的 SDE 对应群体的 PDE，随机与确定性世界由此相连。

import numpy as np, matplotlib.pyplot as plt
np.random.seed(0)
mu, sig, S0, T, n = 0.08, 0.30, 100.0, 1.0, 1000
dt = T/n
for _ in range(20):
    dW = np.random.normal(0, np.sqrt(dt), n)
    W = np.concatenate([[0], np.cumsum(dW)])
    t = np.linspace(0, T, n+1)
    S = S0 * np.exp((mu - 0.5*sig**2)*t + sig*W)
    plt.plot(t, S, lw=0.9, alpha=0.7)
plt.title('Geometric Brownian motion'); plt.show()

SDE 是金融数学、统计物理、神经科学的基石，也是现代生成式 AI 的基础。下一节会回到这个话题。

5. 分数阶微分方程 ── 阶数为 0.7 的导数

导数的阶数可以是连续参数吗？$0 < \alpha < 1$ 时，Caputo 分数阶导数 ${}^C\!D^\alpha$ 类似普通导数的"卷积平滑"：

$${}^C\!D^\alpha f(t) \;=\; \frac{1}{\Gamma(1-\alpha)}\int_0^t \frac{f'(\tau)}{(t-\tau)^\alpha}\,d\tau.$$

分数阶松弛方程

$${}^C\!D^\alpha y(t) = -\lambda\,y(t)$$

解为 Mittag-Leffler 函数

$$y(t) = E_\alpha\!\bigl(-\lambda t^\alpha\bigr) = \sum_{k=0}^\infty \frac{(-\lambda t^\alpha)^k}{\Gamma(1 + k\alpha)}.$$

$\alpha = 1$ 时退化为 $e^{-\lambda t}$。$\alpha < 1$ 时，早期衰减快于指数，但长尾是幂律 $\sim t^{-\alpha}/\Gamma(1-\alpha)$——系统记得自己的历史。

这种记忆性让分数阶 ODE 成为以下场景的天然语言：

  • 粘弹性材料 ── 蠕变既非胡克也非牛顿。
  • 反常扩散 ── 均方位移 $\langle x^2\rangle \propto t^\alpha$，$\alpha \neq 1$（多孔介质、生物细胞、某些尺度上的金融收益率）。
  • 幂律松弛 ── 介电材料、玻璃、生物组织。

6. ODE-ML 连接 ── 不只是一时风潮

引言提到过 Neural ODE 和 PINN。核心图景是：ODE 和 SDE 是现代生成模型的核心。

左：Neural ODE 用学习到的漂移 $f_\theta$ 拟合轨迹。中：PINN 结合稀疏数据与 ODE/PDE 残差，在无数据区域外推。右：Score-based 扩散模型用前向 SDE 把数据变噪声，再学 score $\nabla \log p_t$ 驱动时间反向 SDE，从噪声还原数据。下：五个适合用 ODE/SDE 理解的 ML 模型家族。

记住三点：

  • Score-based 扩散（Stable Diffusion 的引擎）是带学习漂移的 SDE。图像生成 = 反向求解随机 ODE。
  • 连续归一化流 用 ODE 变换基础密度；变量替换靠 $\partial_t \log p = -\nabla \cdot f$。
  • 最优传输 / flow matching 训练向量场，沿直线轨迹形变分布——ODE 是生成 AI 的几何骨架。

实用结论：2025 年的 ML 实践者 必须 能读懂 $dX = f\,dt + g\,dW$。

7. 方法选择流程图

任何到我桌上的 ODE，下面是实战决策树。

自上而下读：先问"它线性吗？"，再进入对应分支。同一个物理系统通常需要 两个 入口——定性分析（相图 / 稳定性）+ 数值方法（RK45、BDF、辛积分）。

几条流程图无法显示的实战提示：

  • 务必先检查刚性。 刚性系统用 RK45 跑会无声地走巨多小步。看到快速暂态就换 BDF 或 Radau。
  • Hamiltonian 系统 优先用辛积分（Verlet）——长时间近似守恒能量，通用 RK 会漂移。
  • BVP 推荐 scipy 的 solve_bvp；刚性 BVP 可能需要 collocation。
  • PDE 先离散空间（线法），再在时间上调 ODE 求解器。

10. 系列旅程

走过的路。基础 → 动力学与混沌 → 计算 → 应用 → 超越。波浪起伏：每章对应一个核心想法的高峰或低谷。

每个标签都值得停一下。起源教会我把 $F = ma$ 写成方程。一阶给出四个手算技巧，解决大部分问题。线性理论和拉氏把 LTI 系统的 $n$ 阶行为化为代数和特征多项式。级数打开特殊函数的大门，超越多项式和指数。

方程组推广到向量；相平面用几何代替封闭解；稳定性和混沌说明预测崩塌时结构依然美丽。分岔展示定性行为如何随参数变化。

数值教工程实现：RK4、BDF、辛积分。BVP 和 PDE 拓宽空间维度。应用覆盖流行病学、生态、控制、力学、电子、流体，验证五步语法的普适性。

终章进入现代：Neural ODE 把积分器变成可学习模型；SDE 让噪声站上舞台；分数阶导数填补整数阶空白；PINN 和扩散模型将 ODE 嵌入当代 AI 引擎。

11. 结语

微分方程是变化的法则。 它描述物体下落、种群增长、电流振荡、国家迈向群体免疫、神经网络学习、宇宙膨胀。一切动力学现象，形式化后都是这个家族的方程。

你读完了 18 章。见到了 Newton、Laplace、Lyapunov、Lorenz、Lotka、Volterra、Bode、Riccati、Chen-Rubanova-Bettencourt-Duvenaud。你可能写过 Python，画出干净的相图时感到惊讶。看到摆动的门或卡顿的 App，你或许已经默默写下它的方程。

如果是这样，目标就达到了。数学不是背一堆公式，而是一种 看见 的习惯。ODE 比几乎所有学科都更能训练这种习惯，因为它是严谨与现实交汇的地方。

去用它。预测、设计、控制、学习。

旅程到此结束，工作从现在开始。
