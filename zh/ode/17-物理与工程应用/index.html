常微分方程（十七）：物理与工程应用

ODE 在物理和工程中的真实应用：非线性单摆、RLC 电路与谐振、开普勒轨道与守恒律、多自由度结构振动与调谐质量阻尼器、流体从 Poiseuille 流到旋涡脱落——五个经典案例配完整 Python 仿真。

微分方程不是纯数学游戏——它是理解物理世界的语言。 天体运动、电路响应、单摆摆动、桥缆后方的涡脱落，所有动力系统都用 ODE “说话”。

本章精心梳理了五个典型应用。每个例子都会用到第 1-16 章的全部工具：相平面、特征值、拉普拉斯变换、模态分析、守恒律、数值积分、控制。这些案例都不是玩具问题，而是真实的物理场景，写得紧凑，结构清晰可见。

本章要点

  • 非线性单摆：小角度线性化与完整方程，周期和振幅，分界线（separatrix）
  • RLC 电路：三种阻尼状态，谐振，Q 因子，调谐
  • 开普勒轨道：万有引力，偏心率，能量角动量守恒，Bertrand 定理
  • 多自由度结构振动：模态分析，谐振，调谐质量阻尼器（Tacoma 大桥教训）
  • 流体力学：Poiseuille 流，雷诺数，旋涡脱落，Stokes 沉降

前置知识

  • 第 1-3 章：一阶和二阶线性理论
  • 第 6-7 章：线性系统与相图
  • 第 8 章：非线性稳定性与能量方法
  • 第 11 章：数值方法（贯穿全书）

1. 非线性单摆 ── 非线性 ODE 的"Hello World"

刚性长度 $L$、重力场 $g$ 中的质点单摆满足

$$\ddot\theta + \frac{g}{L}\sin\theta = 0.$$

小角度下用 $\sin\theta \approx \theta$ 替换，得到简谐振动，周期 $T_0 = 2\pi\sqrt{L/g}$，与振幅无关。这是伽利略利用的等时性奇迹。

完整方程不等时，周期依赖振幅，精确表达式涉及第一类完全椭圆积分：

$$T(\theta_0) = \frac{4}{\omega_0}\,K\!\bigl(\sin(\theta_0/2)\bigr), \qquad \omega_0 = \sqrt{g/L}.$$

左上：从 $\theta_0 = 1.5$ rad 开始，线性方程和完整方程前半个周期一致，之后漂移。线性模型先丢相位，再丢幅值。右上：数值积分得到的 $T(\theta_0)$；0.5 rad 以下小角度等时几乎完美，$\theta_0 \to \pi$ 时周期发散，因达到不稳定平衡。下：完整 $(\theta, \dot\theta)$ 相图。分界线内小幅摆动，外为翻转，沿 $\theta$ $2\pi$ 周期重复。

from scipy.integrate import solve_ivp
import numpy as np

g, L = 9.81, 1.0

def pendulum(t, s):
    theta, omega = s
    return [omega, -(g/L)*np.sin(theta)]

# 不同初角速度：小幅摆动 vs 翻转
for v0 in [1.5, 4.0, 6.5, 7.5]:    # 后两者越顶
    sol = solve_ivp(pendulum, (0, 10), [0.0, v0],
                    t_eval=np.linspace(0, 10, 2000),
                    rtol=1e-10, atol=1e-12)
    # sol.y 中可见闭合环或 theta 单调

为什么重要。 单摆是最简单的非平凡保守非线性系统，相图包含第 7-8 章所有概念：稳定 / 不稳定平衡、分界线、能量-周期关系、可积性。所有其他振子——分子振动、Josephson 结、等离子体波——都借用这个骨架。

2. RLC 电路 ── 同一个方程，换成铜线

串联 $R$-$L$-$C$ 回路在电压 $V(t)$ 驱动下，电容电荷 $q$ 满足：

$$L\ddot q + R\dot q + \frac{1}{C}\,q = V(t).$$

对照机械系统的弹簧-质量-阻尼方程 $m\ddot x + c\dot x + kx = F(t)$。两者完全相同，阻尼比和固有频率的直觉直接适用。

定义 $\omega_n = 1/\sqrt{LC}$（固有频率），$\zeta = (R/2)\sqrt{C/L}$（阻尼比）。从 $V$ 到 $q$ 的传递函数为：

$$H(s) = \frac{1/L}{s^2 + (R/L)\,s + 1/(LC)},$$

阶跃响应仍是第 16 章的三种情况：欠阻尼、临界阻尼、过阻尼。

左上：经典串联 RLC。右上：三种阻尼比下的阶跃响应——欠阻尼振荡，临界阻尼最快无超调，过阻尼缓慢。左下：$\zeta$ 小时 $|H(j\omega)|$ 出现尖锐谐振峰。右下：$R$ 增大时 Q 因子 $Q = (1/R)\sqrt{L/C}$ 下降，3-dB 带宽 $\Delta\omega = R/L$ 变宽——选择性与速度的权衡。

无阻尼振子在共振驱动下幅值线性增长。1940 年 Tacoma Narrows 大桥事件正是这一现象的历史注脚（详见第 4 节）。

3. 行星轨道 ── 牛顿遇见开普勒

牛顿万有引力在二维下给出公式：

$$\ddot{\mathbf r} \;=\; -\frac{GM\,\mathbf r}{|\mathbf r|^3}.$$

守恒量包括总能量 $E = \tfrac12|\mathbf v|^2 - GM/|\mathbf r|$ 和角动量 $\mathbf L = \mathbf r \times \mathbf v$。它们直接推导出开普勒三定律，无需解析求解 ODE。

from scipy.integrate import solve_ivp
import numpy as np

GM = 1.0
def kepler(t, s):
    x, y, vx, vy = s
    r = np.hypot(x, y)
    return [vx, vy, -GM*x/r**3, -GM*y/r**3]

# 圆、椭圆、近抛物线轨道
for v0 in [1.0, 0.85, 0.55]:
    sol = solve_ivp(kepler, (0, 15), [1, 0, 0, v0],
                    t_eval=np.linspace(0, 15, 4000),
                    method='DOP853', rtol=1e-10)

左：从 $(1,0)$ 出发，改变切向速度 $v_y$，轨道从圆过渡到狭长椭圆，正是开普勒观察到的家族。右上：思想实验——将 $1/r^2$ 改为 $1/r^3$，轨道不再闭合，而是螺旋坠入中心（符合 Bertrand 定理：仅 $1/r$ 和谐振势能让所有束缚轨道闭合）。右下：高阶数值积分在多次公转后仍把守恒量保持在 $10^{-7}$ 量级。

Bertrand 定理 是个震撼的事实：所有球对称势中，只有 $\propto 1/r$ 和 $\propto r^2$ 能让所有束缚轨道闭合。这解释了太阳系为何存在稳定闭合行星轨道。数学先行，天文随后。

4. 结构振动 ── 楼宇、桥梁、调谐质量阻尼器

多层楼可简化为刚性弹簧（梁柱）连接的质量串。两层楼，质量 $m_1, m_2$，刚度 $k_1, k_2$：

$$M\ddot{\mathbf x} + C\dot{\mathbf x} + K\mathbf x = \mathbf F(t), \qquad M = \begin{pmatrix} m_1 & 0 \\ 0 & m_2 \end{pmatrix}, \; K = \begin{pmatrix} k_1+k_2 & -k_2 \\ -k_2 & k_2 \end{pmatrix}.$$

模态分析。 特征值问题 $K\,\boldsymbol\phi = \omega^2 M\,\boldsymbol\phi$ 给出固有频率 $\omega_i$ 和模态形状 $\boldsymbol\phi_i$。代入 $\mathbf x = \sum_i q_i(t)\,\boldsymbol\phi_i$，方程组解耦为 $n$ 个独立单自由度振子，多自由度问题化为标量问题。

左：两个模态形状——模态 1 同相（整楼摇晃），模态 2 反相（楼层反向运动）。中列：二楼初始位移激发的时间响应（上）、含小阻尼的频率响应有两个谐振峰（下）。右列：上——共振驱动无阻尼时幅值线性增长，典型 Tacoma Narrows 现象；下——加小型 调谐质量阻尼器（TMD），单一高峰拆成两个低峰，是摩天大楼防风抗震常用方法（台北 101 顶部 660 吨金球）。

这个套路通用：系统是琴弦、机翼、涡轮叶片还是千禧桥，步骤都是 组装 $M, C, K$ → 解特征值 → 模态解耦 → 加阻尼。

5. 流体力学 ── 从稳态管流到旋涡脱落

稳态管流（Poiseuille）

不可压粘性流在长直圆管中，稳态 Navier-Stokes 方程退化为径向 ODE：

$$\frac{1}{r}\frac{d}{dr}\!\Bigl(r\,\frac{du}{dr}\Bigr) = -\frac{1}{\mu}\frac{dp}{dz},$$

边界条件 $u(R) = 0$（无滑移），$u(0)$ 有限。解为抛物线轮廓 $u(r) = u_{\max}(1 - r^2/R^2)$，体积流率 $Q = \pi R^4 \Delta p / (8\mu L)$ 即 Hagen-Poiseuille 律。流量对半径的四次方依赖解释了动脉扩张降血压的原理：半径加倍，流量增 16 倍。

雷诺数与层流-湍流转捩

无量纲数 $\mathrm{Re} = \rho U L / \mu$ 衡量惯性与粘性效应之比。$\mathrm{Re} \lesssim 2300$ 时为层流，$Q \propto \Delta p$ 成立；超过此值，湍涡耗散能量，同样压差下流量减小。

旋涡脱落（Strouhal 数）

钝体绕流后，涡以频率 $f$ 交替脱落，使 Strouhal 数 $\mathrm{St} = f D / U$ 在宽 Re 范围内基本恒定（约 0.21）。当频率匹配结构模态时，Tacoma 大桥共振再现。

沉降小球（Stokes 阻力）

致密小球在粘性流体中下落满足：

$$m\dot v = (\rho_p - \rho_f) V g - 6\pi\mu R\,v,$$

一阶线性 ODE，终端速度 $v_t = (\rho_p - \rho_f) V g / (6\pi\mu R)$，时间常数 $\tau = m/(6\pi\mu R)$。

左上：Hagen-Poiseuille 抛物速度剖面。右上：$Q$-$\Delta p$ 关系展示层流线性如何转为湍流的 $\sqrt{\Delta p}$ 标度。左下：Strouhal 数在宽 Re 范围内的曲线——亚临界区平台 $\approx 0.21$ 解释缆绳和烟囱振荡频率可预测的原因。右下：三种半径小球在重力下指数达到终端速度，$v_t \propto R^2$。

6. 共同的语法

本章每个领域都遵循同样的五步语法：

  1. 写牛顿 / 基尔霍夫 / Navier-Stokes 律，针对无穷小元。
  2. 化为 ODE，用对称性、量纲分析或模态投影。
  3. 找平衡点，线性化，求固有频率和阻尼比。
  4. 加激励，找谐振，决定加阻尼或 调频。
  5. 数值积分，处理非线性主导的情况（大幅、湍流、多体）。

掌握这个循环，物理与工程的大片内容就能贯通。

小结

微分方程是 物理学的通用语言。前 16 章搭建了语言框架，本章用几个经典模型——单摆、RLC、开普勒、多自由度楼宇、管流——展示了从本科物理到工程实践的跨越。没用新数学，只是识别结构后选对了工具。

下一章是终章，跳出经典范畴，探讨 Neural ODE、随机与分数阶方程，以及 ODE 和现代机器学习的联系。

7. 带单位的实战例：调谐 RLC 带通滤波器

不带单位的方程是数学，带单位才是工程。设计一个中心频率 $f_0 = 1\,\text{kHz}$、带宽 $\Delta f = 100\,\text{Hz}$ 的串联 RLC 带通滤波器。

第一步：指标转 ODE 系数。

$$ \omega_0 = 2\pi f_0 = 6283\ \text{rad/s},\quad Q = \frac{f_0}{\Delta f} = 10. $$

对 $L\ddot q + R\dot q + q/C = V(t)$，$\omega_0 = 1/\sqrt{LC}$，$Q = \omega_0 L / R$。

$$ C = \frac{1}{\omega_0^2 L} = \frac{1}{(6283)^2 \cdot 0.01} \approx 2.53\,\mu\text{F}. $$$$ R = \frac{\omega_0 L}{Q} = \frac{6283 \cdot 0.01}{10} = 6.28\,\Omega. $$

第三步：检查单位。 $\omega_0^2 L \cdot C = (\text{s}^{-2})(\text{H})(\text{F}) = (\text{s}^{-2})(\text{V}\cdot\text{s/A})(\text{A}\cdot\text{s/V}) = 1$。无量纲，正确。

第四步：物理意义。 $Q = 10$ 表示输入断开后，振幅每 $Q/(\pi)\approx 3.2$ 个周期衰减到 $1/e$——约 3.2 ms。数学上，阻尼比 $\zeta = 1/(2Q) = 0.05$，欠阻尼明显。

第五步：能量视角。 共振时，电感储能 $\frac12 L I_{\max}^2$ 和电容储能 $\frac12 C V_{\max}^2$ 周期互换，每周期损失 $\pi/Q$ 比例的能量。$Q$ 越高，峰值越窄越尖。

四个数字 ($f_0, \Delta f, L, R$)，把抽象二阶 ODE 变成能装进面包板的实物。

8. 极限情况：小参数让方程退化

非线性单摆 $\ddot\theta + (g/\ell)\sin\theta = 0$，当 $|\theta| \to 0$ 时退化为谐振子，周期 $2\pi\sqrt{\ell/g}$ 与振幅无关。这是经典的小角度近似。

但 ML 时代更有意义的是反向问题：参数 $\to 0$ 或 $\to \infty$ 时，方程性质突变，数值求解更棘手。

例 1：高雷诺数极限。 Navier-Stokes 方程中粘性系数 $\nu \to 0$，形式上退化为 Euler 方程，但规则解几乎都会失稳成湍流。数值上，$\nu$ 过小导致网格 Péclet 数爆炸，必须用迎风格式或激波捕捉方法。

例 2：硬刚度极限。 弹簧常数 $k \to \infty$ 时，质点被约束在某个流形上。直接积分要求时间步长 $h \sim 1/\sqrt k$，计算成本不可接受。物理上的解决办法是改写为带 Lagrange 乘子的约束 ODE/DAE。scipy.integrate.solve_ivp 的 LSODA 或 BDF 能应对中等刚性问题；更高刚性需要 Hairer-Wanner 的隐式辛方法。

例 3：低马赫数极限。 可压缩流方程在 $M \to 0$ 时，声波和对流时间尺度相差 $1/M$ 倍。显式法的 CFL 条件被声波主导，浪费 99% 计算资源。低马赫预条件是工业 CFD 的核心技巧。

记住：参数趋零未必简化数值计算，反而可能让问题更难解。这是 PDE-ML 第二章 Neural Operator 想绕开的核心矛盾。

9. 工程系统的数据驱动建模：SINDy 与 Koopman

第六节提到"一个 ODE 框架描述五种现象"。但实际工程中，新系统出现时方程可能推不出来。这时靠数据反推。

$$ \dot x = \Theta(x)\,\xi,\quad \min \|\dot x - \Theta(x)\xi\|_2^2 + \lambda \|\xi\|_1. $$

$\xi$ 的大部分分量归零，剩下的就是方程结构。我用 pysindy 跑过 Lorenz 数据，1000 个采样点能恢复出右端三项及其系数 $\sigma, \rho, \beta$。前提是候选库包含正确项。

Koopman 算子方法。 把非线性映射 $x_{t+1} = F(x_t)$ 提升到观测函数空间的线性算子 $\mathcal{K}\,g = g \circ F$。结合 DMD（动态模式分解）算 Koopman 谱，直接读出主导振荡模态和衰减常数。Schmid 的 DMD with Control 已用于卡门涡街的 PIV 数据。

何时用哪个：

  • 数据干净、变量物理意义明确、想得到方程 → SINDy。
  • 数据嘈杂、需要预测/控制、不关心方程形式 → Koopman/DMD。
  • 想结合两者 → 在 Koopman 不变子空间里用 SINDy 做稀疏识别（Otto & Rowley, 2019）。
