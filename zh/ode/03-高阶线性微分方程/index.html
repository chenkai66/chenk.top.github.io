常微分方程（三）：高阶线性微分方程

从弹簧到 RLC 电路，系统讲清高阶线性 ODE 的全部理论：叠加原理、Wronskian 行列式、特征方程、待定系数法、参数变易法，以及共振现象。

CK
Chen Kai August 4, 2023 · 20 分钟 · 4534 字
#ODE#Higher-Order ODE#Characteristic Equation#Spring Oscillation#RLC Circuit
Read in English →

一阶 ODE 只记一个数；二阶 ODE 记两个。 这点额外自由度，让同一个方程能描述吉他弦振动、汽车悬挂、FM 收音机的 LC 谐振电路、风中高楼摆动。背后总是三种状态：振荡、略带过冲回平衡、缓慢爬回。决定走哪条路的，永远是同一个代数工具——特征方程。

本章一次性搭建整套工具。先证明结构定理，再推导标准解，然后看同一幅图在不同物理场景反复出现。

本章要点

  • 惯性加回复力直接导致二阶方程
  • 三条结构定理：叠加原理、Wronskian、$y = y_h + y_p$
  • 特征多项式解常系数齐次方程（实根、重根、复根）
  • 阻尼比 $\zeta$ 和欠阻尼/临界/过阻尼三种情况
  • 待定系数法：外力函数简单时用
  • 参数变易法：外力复杂时用
  • 共振现象与振幅曲线分析
  • 同一方程的电路形式（RLC），任意 $n$ 阶 ODE 化为一阶系统的方法

前置知识

  • 第二章：一阶方法 ——变量分离、积分因子
  • 线性代数基础：$2\times 2$ 行列式、复数运算

1. 二阶为什么是"自然单位"

系统只要有惯性（动能、动量、电感电流）和回复力（弹簧、重力、电容电压），牛顿第二定律 $F=ma$ 就会给出位置的二阶导数和位置本身：

$$m\,\ddot x \;=\; -\,k x \;-\; b\,\dot x \;+\; F_\text{ext}(t).$$

两边除以 $m$，用标准归一化记号表示，得到本章核心公式：

$$\boxed{\;\ddot x + 2\zeta\omega_0\,\dot x + \omega_0^2\,x \;=\; f(t)\;}$$

其中 $\omega_0 = \sqrt{k/m}$ 是自然频率，$\zeta = b/(2\sqrt{mk})$ 是阻尼比。本章例子基本都是它的特例。

二阶的意义在哪？ 一阶方程 $\dot x = F(x,t)$ 只需一个数 $x(t)$ 确定状态；二阶方程要两个初始条件 $x(0), \dot x(0)$——可以独立指定位置和速度。系统因此能存储并交换两类"东西"（动能与势能、电压与电流……）。振荡就是这种交换的外在表现。

2. 解空间的结构

2.1 标准形式

$n$ 阶线性 ODE 写成标准形式是

$$y^{(n)} + p_{n-1}(x)\,y^{(n-1)} + \cdots + p_1(x)\,y' + p_0(x)\,y \;=\; g(x),$$

$g \equiv 0$ 是齐次方程，否则是非齐次。

2.2 三条核心定理

(T1) 叠加原理。 $y_1, y_2$ 是齐次解，则 $c_1 y_1 + c_2 y_2$ 也是。导数线性性直接验证。

(T2) 解空间维数。 系数 $p_i$ 连续时，$n$ 阶齐次线性 ODE 恰有 $n$ 个线性无关解 $y_1, \dots, y_n$，通解为

$$y_h(x) \;=\; c_1 y_1(x) + c_2 y_2(x) + \cdots + c_n y_n(x).$$

这是存在唯一性定理的另一种表述：$n$ 个初始条件 $y(x_0), y'(x_0), \dots, y^{(n-1)}(x_0)$ 唯一确定 $n$ 维向量空间中的元素。

(T3) 非齐次结构。 $L[y] = g$ 的通解是

$$y \;=\; y_h \;+\; y_p,$$

其中 $y_p$ 是任意特解。若 $y, \tilde y$ 都满足，则 $L[y - \tilde y] = 0$，差是一个齐次解。

2.3 Wronskian：行列式检验无关性

如何验证 $n$ 个候选解是否线性无关？对它们求到 $n-1$ 阶导数，构造 Wronskian 行列式：

$$W(y_1, \dots, y_n)(x) \;=\; \det \begin{pmatrix} y_1 & y_2 & \cdots & y_n \\ y_1' & y_2' & \cdots & y_n' \\ \vdots & \vdots & & \vdots \\ y_1^{(n-1)} & y_2^{(n-1)} & \cdots & y_n^{(n-1)} \end{pmatrix}.$$

二阶情形：$W(y_1, y_2) = y_1 y_2' - y_2 y_1'$。

判据。 区间内只要有一点 $x_0$ 满足 $W(x_0) \neq 0$，这些解就线性无关；阿贝尔等式保证 $W$ 在区间上处处非零。

$$W = \sin x \cdot (-\sin x) - \cos x \cdot \cos x = -1 \neq 0,$$

处处非零，构成 $y'' + y = 0$ 的一组基。

例（相关）。 $y_1 = \sin x, y_2 = 2\sin x$，$W = 2\sin x\cos x - 2\sin x\cos x \equiv 0$。两者是数乘关系，张成空间维数为 1。

Wronskian 判别线性无关。左：$\sin x$ 与 $\cos x$ 无关，$W \equiv -1$；右：$\sin x$ 与 $2\sin x$ 互为数乘，$W \equiv 0$。

3. 常系数情形：特征方程

3.1 关键技巧

对常系数齐次方程

$$a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y \;=\; 0,$$

试 $y = e^{rx}$。每求一次导数就提出一个 $r$，化为特征多项式：

$$P(r) \;\equiv\; a_n r^n + a_{n-1} r^{n-1} + \cdots + a_1 r + a_0 \;=\; 0.$$

每个根贡献一块基础解。根 $\to$ 基础解是本节核心。

3.2 三种情形

$$y \;=\; c_1 e^{r_1 x} + c_2 e^{r_2 x} + \cdots + c_n e^{r_n x}.$$

例。 $y'' - 5y' + 6y = 0 \Rightarrow (r-2)(r-3) = 0$，得 $y = c_1 e^{2x} + c_2 e^{3x}$。

$$y \;=\; (c_1 + c_2 x + c_3 x^2 + \cdots + c_k x^{k-1})\,e^{rx}.$$

原因：若 $P(r)$ 有二重根，算子分解为 $(D - r)^2 \cdot Q(D)$，直接验证 $(D-r)^2[x e^{rx}] = 0$。

例。 $y'' - 4y' + 4y = 0 \Rightarrow (r-2)^2 = 0$，得 $y = (c_1 + c_2 x)e^{2x}$。

$$y \;=\; e^{\alpha x}\bigl(c_1 \cos\beta x + c_2 \sin\beta x\bigr).$$

为何成立。 复根对原本贡献 $C_1 e^{(\alpha+i\beta)x} + C_2 e^{(\alpha-i\beta)x}$，由欧拉公式 $e^{i\beta x} = \cos\beta x + i\sin\beta x$ 重新组合即得实形式。直观上，$\alpha$ 控制指数包络，$\beta$ 控制振荡频率。

例。 $y'' + 2y' + 5y = 0 \Rightarrow r = -1 \pm 2i$，得 $y = e^{-x}(c_1\cos 2x + c_2\sin 2x)$——一段被收缩指数包络困住的正弦波。

3.3 在复平面上"看"特征根

特征根在复平面的位置，就是解的定性行为：

  • 左半平面（$\Re(r) < 0$）：解衰减，稳定。
  • 右半平面（$\Re(r) > 0$）：解发散，不稳定。
  • 虚轴上（$\Re(r) = 0$）：纯振荡，临界。
  • 离开实轴（$\Im(r) \neq 0$）：含频率为 $|\Im(r)|$ 的振荡分量。

三种典型分布。上排：复 $r$ 平面上的位置，绿色背景标出稳定的左半平面；下排：对应时域解。重根产生 $x e^{rx}$ 的"小拐"，复根对给出阻尼正弦。

4. 阻尼振动：三分法的图解

看标准化形式的二阶齐次方程：

$$\ddot x + 2\zeta\omega_0\,\dot x + \omega_0^2\,x \;=\; 0.$$

特征方程 $r^2 + 2\zeta\omega_0 r + \omega_0^2 = 0$ 的判别式是 $4\omega_0^2(\zeta^2 - 1)$。$\zeta^2 - 1$ 的正负决定三种情况：

状态条件特征根行为
欠阻尼$0 < \zeta < 1$$-\zeta\omega_0 \pm i\omega_d$，$\omega_d = \omega_0\sqrt{1-\zeta^2}$包络 $e^{-\zeta\omega_0 t}$ 下振荡
临界阻尼$\zeta = 1$$-\omega_0$（二重根）$(c_1 + c_2 t)e^{-\omega_0 t}$，无振荡最快回归
过阻尼$\zeta > 1$两个负实根两个衰减指数和，缓慢回归

$\ddot x + 2\zeta\omega_0 \dot x + \omega_0^2 x = 0$ 的三种阻尼状态。左图虚线是 $\pm e^{-\zeta\omega_0 t}$——虚部定振荡，实部定衰减。

工程旁注。 汽车悬挂调到 $\zeta \approx 0.6$–$0.7$，略低于临界阻尼：回稳快，微小过冲人无感。纯 $\zeta = 1$ 显得"死板"，末段回归太慢；$\zeta = 0.1$ 过减速带后弹跳不停。门吸故意设计成过阻尼（$\zeta > 1$），避免关门声大。

5. 非齐次方程：待定系数法

5.1 流程

解 $L[y] = f(x)$（常系数）：

  1. 解 $L[y] = 0$，得 $y_h$。
  2. 根据 $f$ 的形式猜 $y_p$。
  3. 代入匹配系数，解出未知常数。
  4. 合成：$y = y_h + y_p$。

5.2 猜解对照表

外力 $f(x)$试探解 $y_p$
$e^{\alpha x}$$A e^{\alpha x}$
$\cos\beta x$ 或 $\sin\beta x$$A\cos\beta x + B\sin\beta x$
$n$ 次多项式$n$ 次多项式
$e^{\alpha x}\cos\beta x$$e^{\alpha x}(A\cos\beta x + B\sin\beta x)$
上述乘积 / 求和对应乘积 / 求和

共振修正规则。 若试探解本身是齐次解，乘 $x$（双重共振乘 $x^2$）。否则代入会矛盾。

5.3 一个完整例子

解 $y'' + y' + y = e^{-x/2}\cos x$。

齐次部分。 $r^2 + r + 1 = 0$ 的根为 $r = -\tfrac12 \pm \tfrac{\sqrt 3}{2} i$，故

$$y_h \;=\; e^{-x/2}\bigl(c_1\cos\tfrac{\sqrt 3}{2}x + c_2\sin\tfrac{\sqrt 3}{2}x\bigr).$$

试探形式。 外力 $e^{-x/2}\cos x$ 的指数包络与 $y_h$ 相同，但内层频率 $\beta = 1$ 不等于 $\sqrt 3 / 2$，不共振，直接试

$$y_p \;=\; e^{-x/2}(A\cos x + B\sin x).$$

求解 $A, B$。 代入整理 $\cos$ 和 $\sin$ 系数，得线性方程组 $\tfrac14 A + \tfrac32 B = 1,\ -\tfrac32 A + \tfrac14 B = 0$，解得 $A = \tfrac{2}{37},\ B = \tfrac{24}{37}$。

待定系数法三步流程：① 识别外力；② 构造试探基；③ 求解小线性方程组——绿色虚线验证 $y_p'' + y_p' + y_p$ 完全重现 $f$。

5.4 共振：试探解落在 $y_h$ 内

对无摩擦振子，按自然频率施力：

$$\ddot x + \omega_0^2 x \;=\; F_0\cos\omega_0 t.$$

朴素试探 $A\cos\omega_0 t + B\sin\omega_0 t$ 已是齐次解，故乘 $t$：

$$x_p \;=\; \frac{F_0}{2\omega_0}\,t\,\sin\omega_0 t.$$

振幅随时间线性增长——能量每周期泵入而不消散。加入阻尼后，无界增长变为有限峰值，位置 $\omega_r = \omega_0\sqrt{1 - 2\zeta^2}$，峰高与 $1/\zeta$ 成正比。

稳态振幅随驱动频率变化，取 $\zeta = 0.05, 0.1, 0.2, 0.4, 1/\sqrt 2$。$\zeta \to 0$ 时峰值无界增长（无阻尼共振极限）；$\zeta > 1/\sqrt 2$ 时峰消失——系统无共振。

6. 参数变易法：当猜不出来时

6.1 为什么需要它

待定系数法只适用于指数、多项式、正弦余弦及其乘积。遇到 $\sec x, \tan x, \ln x, e^{x^2}$ 这类外力，就需要一种对任意连续 $f$ 都有效的办法——参数变易法。

6.2 公式（二阶）

给定 $y'' + p(x)y' + q(x)y = f(x)$，已知齐次解的基 $y_1, y_2$。设特解形式为 $y_p = u_1(x)\,y_1(x) + u_2(x)\,y_2(x)$，将齐次解中的"常数"提升为函数。补充约束 $u_1' y_1 + u_2' y_2 = 0$ 后代入，得到线性方程组

$$\begin{pmatrix} y_1 & y_2 \\ y_1' & y_2' \end{pmatrix} \begin{pmatrix} u_1' \\ u_2' \end{pmatrix} = \begin{pmatrix} 0 \\ f \end{pmatrix},$$

其行列式为 Wronskian $W$。用 Cramer 法则可得闭式公式

$$\boxed{\;y_p \;=\; -\,y_1 \int \frac{y_2\,f}{W}\,dx \;+\; y_2 \int \frac{y_1\,f}{W}\,dx\;}.$$

6.3 例：$y'' + y = \sec x$

齐次基为 $y_1 = \cos x,\ y_2 = \sin x,\ W = 1$。被积函数化简后很干净：

$$\frac{y_2\,f}{W} = \sin x\,\sec x = \tan x, \qquad \frac{y_1\,f}{W} = \cos x\,\sec x = 1.$$

积分得

$$y_p \;=\; -\cos x\int\tan x\,dx + \sin x\int 1\,dx \;=\; \cos x\,\ln|\cos x| + x\,\sin x.$$

参数变易法在 $y'' + y = \sec x$ 上的演示。上排：齐次基 $y_1, y_2$ 与奇异外力 $\sec x$；下排：原函数 $u_1(x) = \ln|\cos x|$ 与 $u_2(x) = x$，以及拼合得到的特解 $y_p = \cos x \ln|\cos x| + x\sin x$。

何时用谁。 待定系数法适用时更快，只需猜形式并解小线性方程组。参数变易法对任意连续 $f$ 都成立，但需计算两个积分，可能较难。指数 / 三角 / 多项式外力首选待定系数法；其余情况用参数变易法。

7. RLC 电路：同一个方程，换了电气外衣

串联 RLC 电路接电压源 $V(t)$，满足基尔霍夫电压定律：

$$L\,\ddot q \;+\; R\,\dot q \;+\; \frac{q}{C} \;=\; V(t),$$

$q(t)$ 是电容上的电荷，$\dot q = i(t)$ 是电流。与力学标准形式逐项对比：

力学量电学对应
质量 $m$电感 $L$
阻尼 $b$电阻 $R$
劲度 $k$$1/C$
位移 $x$电荷 $q$
外力 $F(t)$电压 $V(t)$

标准参数变为 $\omega_0 = 1/\sqrt{LC}$（谐振角频率）、$\zeta = \tfrac{R}{2}\sqrt{C/L}$（阻尼比）、$Q = 1/(2\zeta)$（品质因数；$Q$ 大则共振峰窄而高）。欠阻尼、临界、过阻尼的分析完全适用。

左：串联 RLC 电路示意；右：$t=0$ 突然合上电压 $V$ 后的阶跃响应。三种状态再次出现，只是物理量换了单位。调收音机本质上是欠阻尼且 $Q$ 极高的情形。

8. 化为一阶系统

数值求解器和第六章 的矩阵理论只认一阶系统。任何 $n$ 阶 ODE 都能通过引入新变量降阶。比如，

$$y'' + a\,y' + b\,y = f(x), \qquad y_1 := y,\ y_2 := y',$$

对应的一阶系统是

$$\begin{pmatrix} y_1' \\ y_2' \end{pmatrix} \;=\; \begin{pmatrix} 0 & 1 \\ -b & -a \end{pmatrix} \begin{pmatrix} y_1 \\ y_2 \end{pmatrix} + \begin{pmatrix} 0 \\ f(x) \end{pmatrix}.$$

矩阵 $\bigl(\begin{smallmatrix} 0 & 1 \\ -b & -a \end{smallmatrix}\bigr)$ 的特征值正好是特征多项式 $r^2 + ar + b$ 的根。本章讨论的特征根问题，其实就是二维线性代数的变形。第六章会深入利用这一点。

9. 小结

方法选择

方程类型首选方法
常系数齐次特征方程
常系数非齐次，外力"简单"特征方程 + 待定系数
变系数或任意连续外力先求 $y_h$，再用参数变易
数值求解化为一阶系统

根 → 基础解 速查表

根解空间基底
不同实根 $r$$e^{rx}$
实根 $r$，重数 $k$$e^{rx},\ x e^{rx},\ \dots,\ x^{k-1} e^{rx}$
复根对 $\alpha \pm i\beta$$e^{\alpha x}\cos\beta x,\ e^{\alpha x}\sin\beta x$

核心要点

  • 二阶 ODE 是惯性加回复力的天然描述。
  • 特征根在 $\mathbb{C}$ 上的位置决定解的定性行为。
  • Wronskian 判别候选解是否张满解空间。
  • 共振是试探解与齐次解重合的结果。乘 $x$ 修复代数，解释无界增长。
  • 力学和电学二阶系统本质相同，直觉通用。

习题

热身。

  1. 求 $y'' - 3y' + 2y = 0$ 的通解。
  2. 求 $y'' - 4y' + 4y = 0$ 的通解，重根情况。
  3. 解初值问题 $y'' + y = 0,\ y(0) = 1,\ y'(0) = 0$。
  4. 解 $y'' + 4y = \sin 2x$，指出共振修正项。

练习。

  1. 用参数变易法求 $y'' + y = \tan x$ 在 $(-\pi/2, \pi/2)$ 上的特解。
  2. RLC 电路参数：$R = 100\,\Omega,\ L = 0.5\,\text{H},\ C = 10\,\mu\text{F}$。计算 $\omega_0$、$\zeta$、$Q$，判断阻尼状态。
  3. 证明 $\ddot x + 2\zeta\omega_0\dot x + \omega_0^2 x = F_0\cos\omega t$ 的稳态振幅为 $A(\omega) = F_0/\sqrt{(\omega_0^2 - \omega^2)^2 + (2\zeta\omega_0\omega)^2}$。当 $\zeta < 1/\sqrt 2$，峰值出现在 $\omega_r = \omega_0\sqrt{1 - 2\zeta^2}$。

编程。

  1. 复现三种阻尼图：用 scipy.integrate.solve_ivp 绘制 $\zeta = 0.1, 0.3, 0.7, 1.0, 2.0$、$\omega_0 = 2\pi$ 的阶跃响应。
  2. 模拟双摆（非线性耦合二阶系统），数值验证初始条件相差 $10^{-6}$ 的两条轨迹几秒后分离。这是第九章 （混沌）的初步体验。
