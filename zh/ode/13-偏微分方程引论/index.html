常微分方程（十三）：偏微分方程引论

当未知函数依赖多于一个自变量时，ODE 世界分裂为更广阔的偏微分方程世界。本章系统地讲三大经典方程：热方程、波动方程、Laplace 方程。

#ODE#PDE#热方程#波动方程#Python
Read in English →

一个量依赖多个自变量时，ODE 世界就扩展为更丰富的 PDE 世界。 金属棒的温度是位置和时间的函数；振动弦在空间和时间中变化；静电势分布在三维空间中。ODE 技术不再是答案，而是工具——分离变量法将 PDE 转化为一族 ODE，这些 ODE 的本征值构成算子谱，叠加原理再把结果拼回去。

本章不是百科全书，而是实战导览。我聚焦三大经典方程：热方程、波动方程、Laplace 方程。任何二阶线性 PDE 在二变量下，从精确意义上说，都与其中之一相似。

本章要点

  • PDE 和 ODE 的本质区别，三大正则型为何涵盖几乎所有线性二阶问题
  • 分类定理：判别式 $B^2 - AC$ 判断抛物型 / 双曲型 / 椭圆型
  • 分离变量法的固定套路：PDE -> Sturm-Liouville 本征问题 -> Fourier 级数
  • d’Alembert 公式与光锥：有限传播速度 vs 形式上的"无限速度"
  • 有限差分法（FTCS、Crank-Nicolson、leapfrog）及其稳定性（CFL 条件、von Neumann 分析）
  • 极值原理 与 Gauss-Seidel 迭代解 Laplace 方程
  • 适定性 初探：哪些初/边值数据让 PDE 问题可解

先修知识：第 11 章 数值方法，第 12 章 边值问题，以及多元微积分（梯度、Laplace 算子、链式法则）。

从 ODE 到 PDE

ODE 中，未知函数 $y(t)$ 只依赖一个变量。每个解由有限个常数确定，数量等于方程阶数，初值条件能唯一确定解。

PDE 需要一个多变量函数。确定解需要在低维曲面（2D 曲线、3D 曲面）上提供数据，解空间是真正无限维的。同一方程，某种数据下适定，换种数据可能不适定——ODE 没有这种现象。

三大经典方程

方程公式类型物理意义
热方程$u_t = \alpha\,u_{xx}$抛物型不可逆扩散
波动方程$u_{tt} = c^2\,u_{xx}$双曲型时间可逆传播
Laplace 方程$u_{xx} + u_{yy} = 0$椭圆型平衡 / 无时间演化

为什么是这三个？任意二变量二阶线性 PDE，通过坐标变换都能化为这三种正则形式之一。判别式决定具体类型。

分类定理

$$A\,u_{xx} + 2B\,u_{xy} + C\,u_{yy} + (\text{低阶项}) = 0,$$

定义判别式 $\Delta = B^2 - AC$：

$\Delta$类型正则代表实特征曲线
$> 0$双曲型波动方程两族
$= 0$抛物型热方程一族（退化）
$< 0$椭圆型Laplace 方程无（复）

实特征曲线的数目，决定扰动传播方式、问题接受的数据类型和数值方法的选择。

左上：$(AC, B^2)$ 平面被直线 $B^2 = AC$ 切分——上方是双曲型，下方是椭圆型，线上是抛物型。中上：抛物型基本解——原点处的 $\delta$ 源迅速展开为高斯分布，形式上瞬间非零（“无限速度”）。右上：双曲型——局部扰动严格限制在光锥 $|x|\leq ct$ 内。下排：三大正则方程并列，列出关键性质。

物理直觉

  • 双曲型：池塘投石——涟漪有限速传播；远处信号需要时间到达。
  • 抛物型：触摸钢棒一端——理论上另一端立刻升温，但指数级微弱。
  • 椭圆型：稳态问题——时间已积出，只剩源汇平衡。

抛物型的"无限速度"虽不严格，但误差衰减极快，实际模型仍然非常有效。

热方程

一段话推导

$$\rho c_p \frac{\partial u}{\partial t} = -\frac{\partial q}{\partial x},$$$$u_t = \alpha\,u_{xx}, \qquad \alpha = \frac{k}{\rho c_p}.$$


扩散系数 $\alpha$ 的量纲是 $\text{长度}^2/\text{时间}$，这是 $k, \rho, c_p$ 唯一自然的组合。

分离变量法

$$\frac{T'}{\alpha T} = \frac{X''}{X} = -\lambda \quad (\text{常数}).$$


左边只含 $t$，右边只含 $x$，相等只能是同一个常数。于是

$$X'' + \lambda X = 0, \qquad T' + \alpha\lambda\,T = 0.$$

Dirichlet 边界给出本征值 $\lambda_n = (n\pi/L)^2$，本征函数 $X_n(x) = \sin(n\pi x/L)$，时间因子 $T_n(t) = e^{-\alpha\lambda_n t}$。叠加得完整解：

$$\boxed{\;u(x, t) = \sum_{n=1}^\infty b_n\,\sin\!\frac{n\pi x}{L}\,e^{-\alpha (n\pi/L)^2 t},\quad b_n = \frac{2}{L}\int_0^L f(x)\sin\!\frac{n\pi x}{L}\,dx.\;}$$

两条结构性教训：

  1. 热方程是 Fourier 滤波器。高模态（大 $n$）衰减快得多——模态 $n$ 半衰期 $\propto 1/n^2$。初值尖锐特征瞬间磨平，平滑特征慢慢褪去。
  2. 边界条件选择基函数：Dirichlet -> 正弦；Neumann -> 余弦；周期 -> 复指数。本征函数正交，系数即内积。这就是 Laplace 算子的谱分解。

左：$u(x, t)$ 六个时刻快照——分段常数初值不到一秒被磨平，趋向平衡 $u\equiv 0$。中：同一解的时空热图——明亮结构沿时间轴迅速褪色。右：Fourier 模态 log 衰减；模态 $n=10$ 比 $n=1$ 快上千倍。热方程是低通滤波器。

分离变量过程的可视化

把"配方"中的每个原料分别画出来更直观。本征函数是空间驻波；时间因子是衰减包络；它们的张量积是基本解；叠加重构初值。

左上：空间本征模 $X_n(x) = \sin(n\pi x/L)$，自动满足两端为零。右上：对应时间因子 $T_n(t)$，$n$ 越大衰减越快。中排：基本解 $u_n(x, t) = X_n T_n$ 在三个时刻形态——短波模态在 $t=2$ 时几乎消失。下排：用更多模态重建方波初值。三个模态时形状勉强可见；三十个模态时除跳跃点外几乎处处吻合，跳跃点处过冲是 Gibbs 现象。

热方程的有限差分

离散化 $x_j = j\,\Delta x$、$t^n = n\,\Delta t$。三种主力格式，每种一行公式加一段稳定性分析：

$$u_j^{n+1} = u_j^n + r\,(u_{j+1}^n - 2u_j^n + u_{j-1}^n), \qquad r = \frac{\alpha\,\Delta t}{\Delta x^2}.$$


von Neumann 分析（代入 $u_j^n = \xi^n e^{ikj\Delta x}$，要求 $|\xi|\leq 1$）给出 $r \leq 1/2$。$\Delta x$ 减半要求 $\Delta t$ 减为四分之一——精度便宜，稳定性昂贵。

BTCS（后向时间，中心空间）：隐式，$O(\Delta t, \Delta x^2)$，无条件稳定——任何 $r > 0$ 都满足 $|\xi|\leq 1$。每步求解一个三对角系统。

Crank-Nicolson：FTCS 与 BTCS 平均；$O(\Delta t^2, \Delta x^2)$，无条件稳定，是线性热方程的"金标准"。

import numpy as np
from scipy.linalg import solve_banded

def crank_nicolson_heat(f, alpha, L, T_end, Nx=101, Nt=400):
    """Crank-Nicolson 求解 u_t = alpha u_xx，Dirichlet 边界 u(0)=u(L)=0。"""
    x  = np.linspace(0, L, Nx); dx = x[1] - x[0]
    dt = T_end / Nt
    r  = alpha * dt / dx**2
    u  = f(x); u[0] = 0; u[-1] = 0
    n  = Nx - 2
    ab = np.zeros((3, n))
    ab[0, 1:] = -r/2
    ab[1, :]  = 1 + r
    ab[2, :-1] = -r/2
    for _ in range(Nt):
        ui  = u[1:-1]
        rhs = (1 - r) * ui
        rhs[1:]  += (r/2) * ui[:-1]
        rhs[:-1] += (r/2) * ui[1:]
        u[1:-1] = solve_banded((1, 1), ab, rhs)
    return x, u

波动方程

d’Alembert 与光锥

$$\boxed{\;u(x, t) = \frac{1}{2}\bigl[f(x - ct) + f(x + ct)\bigr] + \frac{1}{2c}\int_{x - ct}^{x + ct} g(s)\,ds.\;}$$

两点观察重新定义了"波"的含义：

  1. $(x, t)$ 处解只依赖区间 $[x - ct, x + ct]$ 内数据。区间外扰动不可见——有限信号速度。
  2. $f(x \pm ct)$ 是行波：刚体平移，分别向右、左，速度为 $c$。静止脉冲分裂成两个半高脉冲，反向以 $c$ 传播。

有限弦上的驻波

$$u(x, t) = \sum_{n=1}^\infty \bigl[a_n \cos(n\pi c\,t/L) + b_n \sin(n\pi c\,t/L)\bigr]\sin(n\pi x/L).$$

第 $n$ 模态频率 $\omega_n = n\pi c/L$。$n=1$ 是基频，$n\geq 2$ 是泛音。单簧管和小提琴音色不同，因为泛音幅度不同。

数值格式与 CFL 条件

$$u_j^{n+1} = 2u_j^n - u_j^{n-1} + \sigma^2(u_{j+1}^n - 2u_j^n + u_{j-1}^n), \qquad \sigma = \frac{c\,\Delta t}{\Delta x}.$$

von Neumann 分析给出 CFL 条件 $\sigma \leq 1$。几何解释优美：数值依赖区域必须包含物理依赖区域 $[x - ct, x + ct]$。若 $\sigma > 1$，物理信号快于网格传递，数值解爆炸。

上排：(a) 高斯脉冲分裂为两个半高脉冲反向离开；(b) 固定弦上 $n=3$ 的驻波——三段、两个内节点（红点处恒为零）；(c) 时空热图，虚线是光锥 $x = \pm ct$，外侧恒等于零。下排：(d) leapfrog 在 $\sigma=0.7$ 时干净传播；(e) 同一格式 $\sigma=1.05$，半个时间单位内出现数值湍流；(f) 特征曲线网 $x \pm ct = \text{常数}$——信息沿这些曲线传递。

Laplace 方程

时间无关的平衡

$\nabla^2 u = u_{xx} + u_{yy} = 0$ 描述稳态分布——板内温度（边界长期固定后），引力势在质量外，无旋流体的速度势。没有时间演化，只有边界条件。

两条结构性结果

$$\min_{\partial\Omega} u \leq u(\mathbf{x}) \leq \max_{\partial\Omega} u, \quad \forall \mathbf{x}\in\Omega.$$

内部不会比边界最热点更热，也不会比最冷点更冷。证明只需两行，基于均值性质：$u(\mathbf{x}_0)$ 等于以 $\mathbf{x}_0$ 为中心、完全包含于 $\Omega$ 的任意圆盘上 $u$ 的平均。内部最大值等于邻域值，连通性将其传播到边界。

唯一性：两个解差出一个边界为零的调和函数。由极值原理，差恒为零。Laplace BVP 恰有一个解。

数值方法：五点格式

$$u_{i, j} = \frac{1}{4}\bigl(u_{i+1, j} + u_{i-1, j} + u_{i, j+1} + u_{i, j-1}\bigr).$$

离散极值原理就是"取邻居平均"。算法自然得出：从任意初值开始，反复用邻居平均替换内部值（Jacobi / Gauss-Seidel）。$N \times N$ 网格需 $O(N^2)$ 次迭代；带最优松弛因子的 SOR 降到 $O(N\log N)$。实际工作中常用多重网格或稀疏直接解法，但迭代法是极值原理的直接体现。

左：$\nabla^2 u = 0$ 的解，上边界为热高斯凸起，其他三边为零。热从上往下、向两侧扩散。中：等势线 $u = \text{常数}$ 与负梯度 $-\nabla u$（依 Fourier 定律为热流方向）。右：600 个内部随机采样点的 $u$ 值——全部位于边界最大值之下、最小值之上，极值原理直观验证。

Poisson 方程

加上源项：$\nabla^2 u = -f$。五点格式变为 $u_{i,j} = \tfrac{1}{4}(\text{邻居和}) + \tfrac{\Delta x^2}{4} f_{i,j}$。Green 函数把连续问题化为与基本解 $G(\mathbf{x}, \mathbf{y}) = -\frac{1}{2\pi}\ln|\mathbf{x} - \mathbf{y}|$（二维）的卷积——它是热方程高斯基本解的"源驱动型"对应物。

总结：方法地图

方程类型初/边值数据数值方法稳定性
$u_t = \alpha u_{xx}$抛物型$u(x, 0)$ + 2 个 BCFTCS / Crank-Nicolson$r \leq 1/2$（FTCS），无条件（CN）
$u_{tt} = c^2 u_{xx}$双曲型$u(x, 0),\ u_t(x, 0)$ + 2 个 BCleapfrogCFL $\sigma \leq 1$
$\nabla^2 u = 0$椭圆型仅 BCGauss-Seidel / SOR / 多重网格始终稳定

面对新 PDE 的清单：

  1. 判型。线性二阶直接算 $B^2 - AC$。
  2. 挑数据。双曲、抛物要初值；椭圆不要。
  3. 几何简单用分离变量，复杂时离散化。
  4. 先查稳定性，再看精度。不稳定就没精度。

PDE 世界很大——非线性守恒律、流体力学、广义相对论、量子场论——我们刚入门。但这三大方程是语法，其他都是词汇。

练习

概念题

  1. 为什么热方程会忘掉初值，波动方程不会？用 Fourier 模态解释。
  2. 证明 $\Delta = B^2 - AC$ 在 $(x, y)$ 旋转下不变。分类是几何的，与坐标无关。
  3. 从物理角度说明椭圆型问题为何不能提初值问题。

计算题

  1. 实现 Crank-Nicolson 方法解热方程，同时减半 $\Delta t$ 和 $\Delta x$，验证二阶收敛。
  2. 数值研究波动方程在 $\sigma = 1$ 的行为：$\sigma = 1$ 时格式有何特点？略大于 1 时呢？
  3. 验证 $G(x, t) = (4\pi\alpha t)^{-1/2}\,e^{-x^2/(4\alpha t)}$ 对 $t > 0$ 满足热方程。

编程题

  1. 解正方形上的 Laplace 方程：三边为 0，上边为 $\sin(\pi x/L)$。比较迭代解和解析解 $u(x, y) = \sin(\pi x/L)\sinh(\pi y/L)/\sinh(\pi)$。
  2. 动画展示三角脉冲初值的 d’Alembert 解，观察其分裂为两个等高三角形。
  3. 重现 Gibbs 现象：用 $K = 5, 20, 100$ 模态绘制方波的 Fourier 部分和，测量跳跃处的过冲幅度。
