实验代码说明

本目录包含"变分推断的PDE理论：从Fokker-Planck到Langevin动力学"一文的四个实验代码。

实验列表

  1. experiment1_fokker_planck.py: 一维Fokker-Planck演化可视化
  2. experiment2_langevin_sampling.py: Langevin动力学采样（多峰分布）
  3. experiment3_vi_vs_mcmc.py: VI vs MCMC收敛性对比
  4. experiment4_svgd.py: SVGD粒子轨迹与密度估计

依赖

pip install numpy matplotlib scipy torch

运行

每个实验都可以独立运行：

python experiment1_fokker_planck.py
python experiment2_langevin_sampling.py
python experiment3_vi_vs_mcmc.py
python experiment4_svgd.py

输出

每个实验会生成多个可视化图片： - 静态图（PNG格式） - 动画（GIF格式，部分实验） - 数据图（KL散度、轨迹等）
