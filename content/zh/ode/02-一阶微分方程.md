## 1. 可分离方程：最自然的形式

### 1.1 形态识别

一阶 ODE 是**可分离的**，当右端能写成 "$x$ 的函数" 乘以 "$y$ 的函数"：

$$\frac{dy}{dx} \;=\; g(x)\,h(y).$$

这种分解让我能把 $y$ 和 $x$ 分开：

$$\frac{dy}{h(y)} \;=\; g(x)\,dx
\qquad\Longrightarrow\qquad
\int \frac{dy}{h(y)} \;=\; \int g(x)\,dx + C.$$

常数 $C$ 是族参数，每个值对应一条解曲线。比如 $\frac{dy}{dx} = -x/y$ 分离后得 $y\,dy + x\,dx = 0$，积分是 $x^2 + y^2 = C$，表示一族同心圆。方向场的小段处处与这些圆相切——这就是解的几何意义。

![dy/dx = -x/y 的解曲线是同心圆，处处与方向场相切。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig1_separable_solution_curves.png)
*分离变量把二维方向场化为一族一维曲线 $x^2+y^2=C$。*

### 1.2 例题：指数增长与衰减

求解 $\dfrac{dy}{dx} = ky$，其中 $k$ 是常数。

1. **分离：** $\;\dfrac{dy}{y} = k\,dx$
2. **两边积分：** $\;\ln|y| = kx + C_1$
3. **取指数：** $\;y = A e^{kx}$，其中 $A = \pm e^{C_1}$ 吸收符号和常数

$k$ 控制时间尺度，不决定形状。两种情况：

- $k > 0$：每隔 $\ln 2 / k$ 翻倍——无限制种群、连续复利、链式反应失控
- $k < 0$：半衰期 $\ln 2 / |k|$——放射性衰变、电容放电、药物代谢

### 1.3 应用：药物代谢与半衰期法则

多数药物遵循一阶清除：清除速率正比于血液浓度：

$$\frac{dC}{dt} = -kC, \qquad C(t) = C_0 e^{-kt}.$$

**半衰期** $t_{1/2}$ 是浓度降到一半的时间。代入 $C_0/2 = C_0 e^{-k t_{1/2}}$ 解出公式：

$$t_{1/2} = \frac{\ln 2}{k} \approx \frac{0.693}{k}.$$

布洛芬 $t_{1/2} \approx 2$ 小时。从 400 mg 开始：

| 时间 | 残余 | 含义 |
|------|------|------|
| 0 h  | 400 mg | 峰浓度 |
| 2 h  | 200 mg | 一个半衰期 |
| 4 h  | 100 mg | 两个半衰期 |
| 6 h  | 50 mg  | 已低于治疗窗口 |

这解释了布洛芬每 4–6 小时服用一次的原因：再长浓度不足，再短会蓄积。"五个半衰期清除完毕"（约 99%）也是这个公式的推论。

### 1.4 应用：Logistic 方程

纯指数增长 $P' = rP$ 不现实：现实中没有无限增长。Verhulst 在 1838 年引入**环境容纳量** $K$，得到数学生态学基石方程：

$$\frac{dP}{dt} = r P\left(1 - \frac{P}{K}\right).$$

这是可分离方程。左边用部分分式：

$$\int \!\left(\frac{1}{P} + \frac{1/K}{1 - P/K}\right)\,dP \;=\; \int r\,dt,$$

取指数得 **Logistic 曲线**：

$$P(t) \;=\; \frac{K}{1 + \left(\dfrac{K}{P_0} - 1\right)e^{-rt}}.$$

不必解方程，直接看结构就知道三件事：

- $P \ll K$：括号近似为 $1$，接近指数增长
- $P = K/2$：右端等于 $rK/4$，是最大增速
- $P \to K$：括号趋于 $0$，种群饱和

下图右侧画出 $\dot P$ 随 $P$ 的变化——一条向下抛物线，零点在 $P=0$ 和 $P=K$，峰值在 $K/2$。

![不同初值下的 Logistic 曲线，以及增长率 dP/dt 关于 P 的抛物线。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig6_logistic_growth.png)
*左：所有解收敛到 $K$。右：增长率 $\dot P = rP(1-P/K)$ 在 $P=K/2$ 处最大。*
## 2. 一阶线性方程：积分因子法

### 2.1 标准形式

$$\frac{dy}{dx} + P(x)\,y \;=\; Q(x).$$

这是应用数学的主力工具。某个量的变化率正比于自身，再加上外部驱动时，都会归结为这种形式。

### 2.2 关键技巧

两边同乘 **积分因子**

$$\mu(x) \;=\; e^{\int P(x)\,dx}.$$

设计 $\mu$ 的目的是让左边变成一个完整导数：

$$\frac{d}{dx}\bigl[\mu(x)\,y\bigr] \;=\; \mu(x)\,Q(x).$$

直接积分一次，再除以 $\mu$：

$$\boxed{\,y(x) \;=\; \frac{1}{\mu(x)} \left[\int \mu(x)\,Q(x)\,dx + C\right].\,}$$

整套方法就这么简单。为什么管用？因为 $\mu$ 满足 $\mu' = \mu P$，正好符合乘积法则。

### 2.3 例题：$y' + 2xy = x$

| 步骤 | 计算 | 结果 |
|------|------|------|
| 1. 识别 | $P = 2x$，$\;Q = x$ | — |
| 2. 积分因子 | $\mu = \exp\!\int 2x\,dx$ | $\mu = e^{x^2}$ |
| 3. 乘 $\mu$ | 左端化为 $\frac{d}{dx}[e^{x^2}y]$ | 右端 $= x e^{x^2}$ |
| 4. 积分 | $\int x e^{x^2} dx = \tfrac12 e^{x^2}$ | $e^{x^2}y = \tfrac12 e^{x^2} + C$ |
| 5. 解出 | 两边除以 $e^{x^2}$ | $y = \tfrac12 + C e^{-x^2}$ |

当 $x \to \pm\infty$，$C e^{-x^2} \to 0$，所有解最终汇入平衡值 $y = 1/2$。积分常数 $C$ 只在原点附近起作用。

### 2.4 积分因子的作用：三幅图说清楚

取 $y' + 2y = 4$（于是 $P = 2$，$\mu = e^{2x}$，平衡点 $y=2$）：

![y' + 2y = 4 的方向场与解族、积分因子 e^{2x}、以及左端坍缩为完整导数。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig2_integrating_factor.png)
*左：每条解 $y = 2 + Ce^{-2x}$ 都迅速被吸引到平衡 $y=2$。中：积分因子 $e^{2x}$ 是一条干净的指数。右：乘上 $\mu$ 后，$\mu(x)y(x)$ 与 $\int 4\mu\,dx$ 完全吻合——左端成了一个原函数。*

### 2.5 应用：RC 电路充电

电阻 $R$ 与电容 $C$ 串联，外加电压源 $V_s$，按基尔霍夫电压定律：

$$RC\,\frac{dV_c}{dt} + V_c \;=\; V_s.$$

对照标准形式：$P(t) = 1/RC$，$Q(t) = V_s/RC$。积分因子 $\mu(t) = e^{t/RC}$。当 $V_c(0)=0$ 时，结果是

$$V_c(t) \;=\; V_s\bigl(1 - e^{-t/\tau}\bigr), \qquad \tau = RC.$$

时间常数 $\tau$ 是每个电气工程师都熟悉的：

| 时刻 | $V_c/V_s$ | 工程上的读法 |
|------|-----------|----------------|
| $\tau$  | 63.2% | "一个时间常数" |
| $3\tau$ | 95.0% | 在大多数逻辑电路里可视为充满 |
| $5\tau$ | 99.3% | 工程意义上"完全充满" |

把符号翻一下，令 $V_s = 0$，同一个公式描述了**放电过程**——也解释了相机闪光灯为什么需要等一两秒才能再次拍照。
## 3. 恰当方程

### 3.1 几何思想

把 ODE 写成微分形式：

$$M(x,y)\,dx + N(x,y)\,dy = 0.$$

假设存在势函数 $F(x,y)$，满足

$$\frac{\partial F}{\partial x} = M, \qquad \frac{\partial F}{\partial y} = N.$$

那么 $M\,dx + N\,dy$ 就是 $dF$。方程 $dF = 0$ 表示 $F$ 沿解曲线为常数。解族为

$$F(x,y) \;=\; C.$$

解曲线是 $F$ 的**等高线**。这个观点很深刻：求解 ODE 转化为找一个标量函数 $F$，其等高线图就是解族。

### 3.2 恰当性判定

势函数存在的充要条件是混合偏导相等（即 $F_{xy} = F_{yx}$）：

$$\boxed{\,\dfrac{\partial M}{\partial y} \;=\; \dfrac{\partial N}{\partial x}.\,}$$

不满足时，方程不是恰当的——但可以尝试修正（见 §3.4）。

### 3.3 例题：恰当方程的求解步骤

考虑 $(2x + y)\,dx + (x + 2y)\,dy = 0$。先验证：$M_y = 1 = N_x$，恰当。然后还原 $F$：

1. 对 $M$ 关于 $x$ 积分：$\;F = x^2 + xy + g(y)$。
2. 对 $y$ 求导并与 $N$ 比对：$\;x + g'(y) = x + 2y \Rightarrow g'(y) = 2y \Rightarrow g(y) = y^2$。
3. 得到 $F(x,y) = x^2 + xy + y^2$，解族为 $x^2 + xy + y^2 = C$。

几何上这是一族倾斜椭圆。下图叠加梯度场 $\nabla F = (M, N)$ 和等高线；梯度处处与等高线**垂直**。这就是"$dF = 0$ 沿解成立"的几何含义：梯度指"上山方向"，解曲线保持高度不变。

![F(x,y) = x^2 + xy + y^2 的等高线与梯度场，二者处处垂直。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig3_exact_level_curves.png)
*每条等高线是一条解。灰色箭头是 $\nabla F$，与等高线垂直——这是"沿解 $dF = 0$"的几何表达。*

### 3.4 救场：把非恰当方程变恰当

若 $M_y \neq N_x$，乘以 $\mu(x,y)$ 修正。两种常见情况有显式公式：

| 条件 | 取 |
|------|-----|
| $(M_y - N_x)/N$ 仅依赖 $x$ | $\mu(x) = \exp\!\int \dfrac{M_y - N_x}{N}\,dx$ |
| $(N_x - M_y)/M$ 仅依赖 $y$ | $\mu(y) = \exp\!\int \dfrac{N_x - M_y}{M}\,dy$ |

这与线性方程积分因子构造相同。线性方程方法是此法的特例。
## 4. 伯努利方程

### 4.1 形态

$$\frac{dy}{dx} + P(x)\,y \;=\; Q(x)\,y^{n}, \qquad n \neq 0, 1.$$

$n = 0$ 时是线性方程，$n = 1$ 时可分离。其余情况，$y^n$ 破坏了线性。

### 4.2 替换方法

令 $v = y^{1-n}$，则 $\dfrac{dv}{dx} = (1-n)\,y^{-n}\,\dfrac{dy}{dx}$。将原方程乘以 $(1-n) y^{-n}$ 后整理：

$$\frac{dv}{dx} + (1-n)P(x)\,v \;=\; (1-n)Q(x).$$

这是关于 $v$ 的线性方程。用第 2 节的积分因子求解 $v$，再通过 $y = v^{1/(1-n)}$ 回代。

### 4.3 替换的几何含义

取 $y' + y = y^2$（$n = 2$，$v = 1/y$）。替换后变为 $v' - v = -1$，解是绕平衡点 $v = 1$ 的指数族。在 $v$-平面中，轨迹规整笔直；在原 $y$-平面中，轨迹弯曲且可能在有限时间内发散。

![y' + y = y^2 在原 (x, y) 平面与替换后 (x, v) 平面（v = 1/y）的对比。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig4_bernoulli_transform.png)
*五条轨迹对比。左 $(x, y)$：非线性，$y=1$ 是不稳定平衡。右 $(x, v)$：线性，$v=1$ 是稳定平衡（对应 $y=1$ 从下方趋近）。*

同样方法能还原 **Logistic 方程**：它是 $n = 2$、$P(t) = -r$、$Q(t) = -r/K$ 的伯努利方程。§1.4 的部分分式积分本质上只是伯努利替换加线性求解。
## 5. 综合应用：水箱里的盐

![常微分方程（二）：一阶微分方程的求解方法 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-first-order-methods/illustration_2.jpg)

经典"房室模型"，骨架通用。流行病学、药代动力学、化学工程里都常见。

**问题设定。** 1000 L 水箱装纯水。盐水浓度 $2$ g/L，流入速率 $5$ L/min。充分搅拌，流出速率相同，体积恒定。求盐量 $Q(t)$。

**建模。** 流入减流出：

$$\underbrace{\frac{dQ}{dt}}_{\text{变化率}}
= \underbrace{5 \cdot 2}_{\text{流入: g/min}}
\,-\, \underbrace{5 \cdot \tfrac{Q}{1000}}_{\text{流出: g/min}}
= 10 - \tfrac{Q}{200},
\qquad Q(0) = 0.$$

**求解。** 线性方程：$Q' + \tfrac{1}{200}Q = 10$。积分因子 $\mu = e^{t/200}$，解得

$$Q(t) \;=\; 2000\bigl(1 - e^{-t/200}\bigr).$$

**读懂答案。** 平衡值 $Q^\ast = 2000$ g，对应浓度 $2$ g/L，等于流入浓度。时间常数 $\tau = 200$ min 决定收敛速度。初值 $Q(0) > Q^\ast$ 时，描述的是冲淡偏咸水箱——下图展示两种情形，最终都趋于同一平衡。

![两种初值下盐量 Q(t) 都收敛到 2000 g，附带浓度面板。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig5_mixing_tank.png)
*两种情形，一个平衡。从 $Q(0)=0$ 灌满或从 $Q(0)=3000$ 冲淡，最终都到 $2$ g/L；仅偏离方向不同。*
## 6. 数值方法预览

$f(x,y)$ 稍复杂，闭式技巧就失效——实际问题大多如此。解决办法是：沿方向场一步步推进。

### 6.1 欧拉方法

最简单的思路：每步用切线代替曲线。

$$y_{n+1} \;=\; y_n + h\,f(t_n, y_n).$$

- 单步局部误差：$O(h^2)$
- 全局误差：$O(h)$，称"一阶方法"

简单、易写，但精度常不足。

### 6.2 四阶 Runge-Kutta（RK4）

数值 ODE 的主力方法。每步计算四次斜率，加权平均：

$$k_1 = f(t_n, y_n), \qquad
k_2 = f\!\left(t_n + \tfrac{h}{2},\, y_n + \tfrac{h}{2}k_1\right),$$

$$k_3 = f\!\left(t_n + \tfrac{h}{2},\, y_n + \tfrac{h}{2}k_2\right), \qquad
k_4 = f(t_n + h,\, y_n + h k_3),$$

$$y_{n+1} \;=\; y_n + \frac{h}{6}\bigl(k_1 + 2k_2 + 2k_3 + k_4\bigr).$$

全局误差 $O(h^4)$——同样步长下比欧拉高四个数量级，代价是多三次函数求值。

### 6.3 看方向场，预判解的形态

写代码前，先画右端项。表面相似的方程，长期行为可能完全不同：

![线性方程 y'=y-x 与伯努利方程 y'=y-y^3 的方向场对比。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/02-一阶微分方程/fig7_slope_field_comparison.png)
*左（线性）：唯一直线解 $y=x+1$，其余指数发散。右（伯努利）：三个平衡点 $y=-1, 0, +1$，轨迹被压向 $\pm 1$，与初值符号无关。同属一阶方程，几何形态迥异。*

### 6.4 SciPy 三行搞定

牛顿冷却定律：$T' = -k(T - T_{\text{env}})$，$T(0) = 90$°C，$T_{\text{env}} = 20$°C，$k = 0.1$ /min。

```python
from scipy.integrate import solve_ivp
import numpy as np

def cooling(t, T, k=0.1, T_env=20):
    return -k * (T - T_env)

sol = solve_ivp(cooling, [0, 60], [90.0], method="RK45", dense_output=True)

t = np.linspace(0, 60, 300)
T = sol.sol(t)[0]
print(f"t=10 min 时: {T[50]:.1f} C")
print(f"t=30 min 时: {T[150]:.1f} C")
```

`solve_ivp` 自适应步长。多数问题无需手写 RK4；需关注的是：**标准求解器何时吃力**（刚性方程——见第 11 章）。
## 7. 总结：四问筛选法

遇到一阶方程，按顺序过一遍下面的表。第一个“是”就告诉你用什么方法。

| 提问 | 若是 | 方法 |
|------|------|------|
| 能写成 $y' = g(x)\,h(y)$ 吗？ | 可分离 | 移项后两边积分 |
| 能写成 $y' + P(x)y = Q(x)$ 吗？ | 线性 | 积分因子 $\mu = e^{\int P\,dx}$ |
| 能写成 $M\,dx + N\,dy = 0$ 且 $M_y = N_x$ 吗？ | 恰当 | 还原势函数 $F$；解为 $F = C$ |
| 能写成 $y' + Py = Qy^n$（$n \notin \{0,1\}$）吗？ | 伯努利 | 替换 $v = y^{1-n}$，再走第二行 |
| 都不行？ | — | 数值方法（第 11 章），或找一个巧妙换元 |

这四种方法并不独立。线性方程的积分因子是修正非恰当方程积分因子的特例。伯努利替换把非线性问题化为线性问题。学好这四种方法，本质是反复练习一个核心思想——**找到变量替换或乘子，把方程化成一眼能看出积分形式**——只是在四种场景下分别应用。
