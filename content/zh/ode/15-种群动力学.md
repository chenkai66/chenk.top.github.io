**为什么猞猁和雪兔的数量每 10 年精确波动？** 引入一个外来物种为何有时会让整个生态系统崩溃？相似竞争者为何有时共存、有时却互相灭绝？答案不在物种本身，而在描述物种关系的方程里。本章梳理数学生态学经典模型：从单种群 Logistic 和 Allee 效应，到二种群竞争、捕食-被捕食振荡，再到年龄结构与空间扩散。

## 本章要点

- 单种群三件套：**Malthus**（指数）、**Logistic**（饱和）、**Allee 效应**（灭绝阈值）
- **Lotka-Volterra** 捕食模型：闭轨周期、守恒量、富营养化悖论
- **Holling 功能反应**：打破严格周期性
- **二种群竞争**四种结局：共存、排斥（两种）、双稳
- **年龄结构**：Leslie 矩阵、主特征值为长期增长率、稳定年龄分布
- **集合种群**（Levins）：斑块占用动力学、灭绝阈值

## 单种群增长

设 $N(t)$ 为种群数量。

**Malthus**（1798）：$\dot N = rN$，解得 $N(t) = N_0 e^{rt}$。数学简洁，生物学上仅几代内成立。

**Logistic**（Verhulst 1838）：加入密度依赖的拥挤效应：
$$\boxed{\;\dot N = r N\!\left(1 - \frac{N}{K}\right).\;}$$环境承载力 $K$ 是稳定点，$0$ 不稳定。闭式解是 S 曲线 $N(t) = K/(1 + ((K - N_0)/N_0) e^{-rt})$，最大增长率 $rK/4$ 出现在 $N = K/2$。

**Allee 效应**（1931）：某些物种需要最低密度才能增长——配偶寻找、群体防御、合作捕食。强 Allee 形式：$$\dot N = rN\!\left(1 - \frac{N}{K}\right)\!\left(\frac{N}{A} - 1\right).$$此时 $0$ 稳定，$A$ 是不稳定阈值，$K$ 稳定。低于 $A$ 灭绝，高于 $A$ 趋向 $K$。这是双稳态：一个方程两个吸引盆。

Allee 效应解释了濒危物种栖息地修复后仍难恢复的原因，也说明小群放归为何常失败。数学与力学三次势相同，动力学是双井势中的梯度下降。

![Allee 效应：人均/总增长率、不同初值轨迹、能量景观。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-种群动力学/fig3_allee_effect.png)

*左上：人均增长率 $\dot N/N$ 随密度变化。Logistic（蓝）在 $K$ 前为正；强 Allee（红）在 $A$ 下为负。右上：$\dot N$ vs $N$——强 Allee 有三个不动点。左下：不同初值轨迹——低于 $A$ 滑向灭绝，高于 $A$ 爬到 $K$。右下：对应"势" $V(N)$，满足 $\dot N = -V'(N)$。两稳态（灭绝、$K$）是山谷，$A$ 是势垒。带噪声系统可在两盆间跳跃，给出生态学机制切换的清晰模型。*
## 捕食模型：Lotka-Volterra

被食者 $x$，捕食者 $y$：$$\boxed{\;\dot x = \alpha x - \beta x y, \qquad \dot y = \delta x y - \gamma y.\;}$$- $\alpha$：无捕食时被食者增长率  
- $\gamma$：无食物时捕食者死亡率  
- $\beta, \delta$：相遇率参数  

导数为零得两个不动点：灭绝 $(0, 0)$（鞍点），共存 $(\gamma/\delta,\ \alpha/\beta)$（中性中心）。

### 一个守恒量

定义$$H(x, y) = \delta x - \gamma\ln x + \beta y - \alpha\ln y.$$直接计算 $\dot H = 0$。每条 LV 轨道都在 $H$ 的等位线上。等位线是绕中心的闭曲线，解周期性运动，中心真正中性，非渐近稳定。

这种守恒律很脆弱，小扰动就会破坏。真实生态系统不精确满足 LV，现实中的周期通常是极限环（见下文 Holling 反应），不是 LV 的闭轨族。

### 雪兔-猞猁数据

哈德逊湾公司皮毛记录（1845-1930）显示 10 年雪兔-猞猁周期。LV 定性上能复现。定量上，生态学家认为周期实际由雪兔-植被相互作用驱动，猞猁只是跟随，但历史吻合度让 LV 声名大噪。

![Lotka-Volterra：时序、相图闭轨族、守恒量、富营养化悖论。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-种群动力学/fig1_lotka_volterra.png)

*左上：经典异相振荡——被食者先达峰，捕食者滞后四分之一周期。右上：五条不同初值闭轨围绕 $(\gamma/\delta,\ \alpha/\beta)$，向量场指示旋转方向。左下：单条轨道上 $H$ 数值漂移可忽略——真守恒律。右下：换成 Holling Type II + Logistic 被食者，提高承载力 $K$ 让平衡点失稳，出现极限环——富营养化悖论。*

```python
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

def lv(t, s, a=1.0, b=0.1, d=0.05, g=0.5):
    x, y = s
    return [a*x - b*x*y, d*x*y - g*y]

sol = solve_ivp(lv, (0, 60), [10, 2], t_eval=np.linspace(0, 60, 4000))
fig, ax = plt.subplots(1, 2, figsize=(12, 4))
ax[0].plot(sol.y[0], sol.y[1]); ax[0].set_xlabel('被食者'); ax[0].set_ylabel('捕食者')
ax[1].plot(sol.t, sol.y[0], label='被食者')
ax[1].plot(sol.t, sol.y[1], label='捕食者'); ax[1].legend()
plt.tight_layout(); plt.show()
```

### 功能反应（Holling）

真实捕食者会饱和——狼一天吃不下无穷多兔子。Holling 分三类：

- **Type I**：线性 $g(x) = ax$（LV 默认；罕见）。  
- **Type II**：$g(x) = ax/(1 + ahx)$，饱和到 $1/h$，最常见。  
- **Type III**：$g(x = ax^2/(1 + ahx^2)$，sigmoid，低密度时切换捕食对象。  

把 LV 相互作用换成 Holling-II 得 **Rosenzweig-MacArthur** 模型，平衡点随承载力 $K$ 增大经 Hopf 分岔失稳。$K$ 大 -> 食物多 -> 捕食者多 -> 被食者崩塌 -> 极限环大 -> 灭绝。食物增多反而是坏事。

### 富营养化悖论

参数固定时，Hopf 分岔在 $K$ 越过阈值时发生，之后极限环幅度 $\propto \sqrt{K - K_c}$。最终极限环逼近坐标轴，小噪声推某一种群到零。这不是抽象反直觉，湖泊富营养化实验中已观察到。
## 二种群竞争

两个物种竞争同一资源池：$$\dot N_1 = r_1 N_1\!\left(1 - \frac{N_1 + \alpha_{12} N_2}{K_1}\right), \qquad \dot N_2 = r_2 N_2\!\left(1 - \frac{N_2 + \alpha_{21} N_1}{K_2}\right).$$无量纲**竞争系数** $\alpha_{ij}$ 表示物种 $j$ 的一个个体对物种 $i$ 人均增长率的压制程度，相对于 $i$ 自身。

系统最多有四个不动点：

- $(0, 0)$ —— 一般不稳定
- $(K_1, 0)$ —— 物种 1 胜
- $(0, K_2)$ —— 物种 2 胜
- 内部 $(\hat N_1, \hat N_2)$ —— 共存（若存在）

稳定性看"另一物种在承载力时，第一种能否入侵"。比较增长率得出**四种典型结局**：

| 条件 | 结局 |
|---|---|
| $\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$ | **稳定共存** |
| $\alpha_{12} < K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$ | 物种 1 胜 |
| $\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} < K_2/K_1$ | 物种 2 胜 |
| $\alpha_{12} > K_1/K_2$ 且 $\alpha_{21} > K_2/K_1$ | **双稳**：奠基者效应 |

**竞争排斥原理**（Gause 1934）是推论：生态位完全相同（$\alpha_{12} = \alpha_{21} = 1$，$K_1 = K_2$）的物种无法稳定共存。共存需要**生态位分化**——种内竞争强于种间竞争。

![四张图：稳定共存、物种 1 胜、物种 2 胜、双稳奠基者效应；零线、向量场、轨迹。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-种群动力学/fig2_competition_model.png)

*每张图含零线（蓝/红）、向量场（灰箭）、五条彩色轨道、不动点（黑点/星）。左上：共存——零线交叉方向正确，内部星稳定。右上 + 左下：排斥——内部不动点不存在，轨道汇向坐标轴。右下：双稳——内部不动点为**鞍点**，分界面是其稳定流形；起在哪侧哪方胜。*
## 年龄结构模型

真实种群有年龄结构。**Leslie 矩阵**模型离散化年龄和时间，$n_t = (n_t^{(0)}, n_t^{(1)}, \dots, n_t^{(m)})$：$$\boxed{\;n_{t+1} = L\,n_t,\quad L = \begin{pmatrix} F_0 & F_1 & \cdots & F_m \\ P_0 & 0 & \cdots & 0 \\ 0 & P_1 & \cdots & 0 \\ \vdots & & \ddots & \vdots \\ 0 & 0 & P_{m-1} & 0 \end{pmatrix}.\;}$$- $F_i$ = 年龄 $i$ 的生育率（每步后代数）
- $P_i$ = 年龄 $i$ 存活到 $i+1$ 的概率

迭代公式 $n_{t+1} = L\,n_t$ 的长期行为由 $L$ 的主特征值 $\lambda$ 决定：
- $\lambda > 1$：几何增长，速率为 $\lambda$
- $\lambda = 1$：稳态
- $\lambda < 1$：衰减至灭绝

对应右特征向量是稳定年龄分布——各年龄段比例的极限值。任何初分布最终收敛到它（除非 $L$ 有等模复特征值；非负不可约 Leslie 矩阵被 Perron-Frobenius 排除）。

### 为什么 $\lambda$ 是关键

净再生率 $R_0 = \sum_i \ell_i F_i$，$\ell_i = \prod_{j<i} P_j$ 是存活到 $i$ 的概率。$R_0 = 1 \Leftrightarrow \lambda = 1$。种群增长不仅取决于生育数量，还取决于生育时机。早生的孩子贡献更大，因为他们更早繁殖。这种效应被特征值自动捕捉。

![Leslie 矩阵热图、特征值谱、几何增长轨迹、稳定年龄分布。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-种群动力学/fig4_age_structured_leslie.png)

*左上：Leslie 矩阵 $L$ 热图——首行是 $F_i$，次对角线是 $P_i$。右上：特征值分布；主实特征值（红星）在单位圆外，种群增长。左下：从 1000 新生儿出发的轨迹。10-15 年后进入几何增长。右下：稳定年龄分布（红条）与 $t = 30$ 实际分布（黑点）吻合。*

```python
import numpy as np

F = np.array([0.0, 0.5, 2.5, 1.5, 0.2])           # 生育率
P = np.array([0.6, 0.7, 0.85, 0.5])               # 存活率
L = np.zeros((5, 5)); L[0, :] = F
for i in range(4): L[i + 1, i] = P[i]

lam, vec = np.linalg.eig(L)
i = np.argmax(np.abs(lam))
print('长期增长率:', lam[i].real)
print('稳定年龄分布:', np.abs(vec[:, i].real / vec[:, i].real.sum()))
```
## 集合种群：很多斑块

物种分布在斑块（森林碎片、池塘、岛屿）而非连续栖息地时，关键不是局部种群，而是**斑块占用比例**。Levins（1969）提出最简模型：$$\boxed{\;\dot p = c\,p\,(1 - p) - e\,p,\;}$$$p$ 是占用比例，$c$ 是定殖率，$e$ 是灭绝率。平衡点 $p^* = 1 - e/c$，要求 $c > e$。

模型结构与 SIS 传染病模型一致：斑块是"个体"，定殖是"传播"。集合种群持续条件是 $c > e$，即区域**灭绝阈值**。

保护启示：单个斑块健康不够，定殖率（廊道、扩散）太低仍会区域灭绝。修路、设栅栏切断 $c$，即使每块碎片看似完好。
## 空间扩散：Fisher-KPP

![常微分方程（十五）：种群动力学 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-population-dynamics/illustration_2.png)

**Fisher 方程**（Fisher 1937；Kolmogorov-Petrovskii-Piskunov 1937）结合 Logistic 增长与扩散：$$\boxed{\;\partial_t N = D\,\partial_x^2 N + r N\!\left(1 - \frac{N}{K}\right).\;}$$初值紧支集时，种群前沿以确定渐近速度推进：$$c_{\min} = 2\sqrt{D r}.$$
两行证明：前沿（$N \ll K$）线性化；行波 ansatz $N = e^{-\lambda(x - ct)}$ 要求 $c = D\lambda + r/\lambda$，对 $\lambda$ 极小化得 $c_{\min} = 2\sqrt{Dr}$。实际选择的速度就是线性极小值——著名的 **KPP 选择原理**。

这个公式在生态学中无处不在：预测植物范围在气候变化下的扩张速度、20 世纪 20 年代麝鼠入侵欧洲（实测约 $\sqrt{Dr}$）、遗传学中有利等位基因固定波速。

![Levins 集合种群、灭绝阈值、Fisher-KPP 行波快照、前沿位置-时间关系。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/15-种群动力学/fig5_metapopulation.png)

*左上：不同灭绝率 $e$ 下的 Levins $p(t)$。$e$ 越大 $p^*$ 越低；$e > c$ 区域灭绝。右上：$p^*$ 与 $e/c$ 关系——$e = c$ 处灭绝阈值。左下：Fisher-KPP 行波 $t = 0, 20, 40, 60, 80$ 快照，恒形前沿以 $c_{\min} = 2\sqrt{Dr}$ 推进。右下：数值跟踪前沿位置；线性拟合斜率与 $2\sqrt{Dr}$ 误差在 2% 以内。*

### 超越 Fisher：图灵模式

两个物种（如激活子 + 抑制子，或捕食者 + 被食者）扩散速率不同，均匀平衡可能失稳——自发产生模式。这就是 **Turing 不稳定**（1952）。该机制被假定支撑动物花纹、半干旱区植被条带、贝床动力学。它是数学形态发生学的核心。
## 总结

| 模型 | 方程 | 核心特征 |
|---|---|---|
| Malthus | $\dot N = rN$ | 无界指数增长 |
| Logistic | $\dot N = rN(1 - N/K)$ | 饱和于 $K$ |
| 强 Allee | $\dot N = rN(1 - N/K)(N/A - 1)$ | 灭绝阈值 $A$，双稳态 |
| Lotka-Volterra | $\dot x = \alpha x - \beta xy,\ \dot y = \delta xy - \gamma y$ | 守恒量，中性周期 |
| Holling Type II | $g(x) = ax/(1 + ahx)$ | 捕食饱和，富营养化悖论 |
| LV 竞争 | $\dot N_i = r_i N_i (1 - (N_i + \alpha_{ij} N_j)/K_i)$ | 四种典型结果 |
| Leslie | $n_{t+1} = L n_t$ | 主特征值为增长率，稳定年龄分布 |
| Levins | $\dot p = cp(1 - p) - ep$ | 区域灭绝阈值 $e/c$ |
| Fisher-KPP | $\partial_t N = D\partial_x^2 N + rN(1 - N/K)$ | 波速 $2\sqrt{Dr}$ |

数学生态学方程少，行为多。几条简单规则——扩散加反应、单种群加年龄结构、均场加异质——撑起整个领域。
