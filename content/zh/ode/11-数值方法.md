## 1. 为什么需要数值方法？

前几章讲的解析方法——分离变量、积分因子、Laplace 变换、特征展开——虽然强大但很**脆弱**。只对少数方程有效，真实问题稍复杂就失效。比如$\frac{dy}{dx} = \sin(xy)$，没有封闭解。Navier-Stokes 方程、三体问题、多物种化学反应网络、Lorenz 系统，全都无法用符号方法解决。只能在网格点$x_n = x_0 + nh$上算**离散近似**$y_n \approx y(x_n)$。

接下来要解决三个问题：

- 更新规则$y_{n+1} = \Phi(y_n, h, f, \ldots)$怎么选？
- 误差随$h$如何变化？
- 方程"刚性"时（小$h$为稳定性而非精度），方法表现如何？

我会逐一解答。
## 2. 前向欧拉：几何思想

给定$\dot{y} = f(x, y)$、$y(x_0) = y_0$，最简单的方法是用切线代替曲线：$y_{n+1} = y_n + h\,f(x_n, y_n).$这就是**前向欧拉**，也是显式单步法的原型。直观理解：每一步读取当前斜率，沿该方向走一小步$h$，再更新斜率。像雾中爬坡，只依赖脚下坡度。

### 精度阶

真解 Taylor 展开为：$y(x_n + h) = y(x_n) + h\,y'(x_n) + \tfrac{h^2}{2}y''(x_n) + \mathcal{O}(h^3).$与欧拉公式$y_{n+1} = y_n + h f(x_n, y_n)$相减，得**局部截断误差**$\mathcal{O}(h^2)$/步。固定区间$[x_0, X]$上走$N=(X-x_0)/h$步，**全局误差**为$\mathcal{O}(h)$。步长减半，误差也减半，但代价翻倍。

### 线性稳定性

将欧拉法用于测试方程$\dot{y} = \lambda y$（$\mathrm{Re}\,\lambda<0$）：$y_{n+1} = (1 + h\lambda)\,y_n.$放大因子$R(z) := 1 + z$（$z = h\lambda$）。稳定性要求$|R(z)| \le 1$。实负$\lambda$时需$h < 2/|\lambda|$；纯虚$\lambda$时，**任何$h>0$都不稳定**。欧拉法**至多条件稳定**，对纯振荡问题无能为力。

```python
import numpy as np

def euler(f, x0, y0, x_end, h):
    """前向欧拉。f(x, y) -> dy/dx。"""
    xs, ys = [x0], [y0]
    x, y = x0, y0
    while x < x_end - 1e-12:
        h_eff = min(h, x_end - x)
        y = y + h_eff * f(x, y)
        x = x + h_eff
        xs.append(x); ys.append(y)
    return np.array(xs), np.array(ys)
```
## 3. Heun、中点法、Runge-Kutta 家族

提升欧拉法精度的关键是**每步多算几次斜率**，再巧妙组合。显式$s$级 Runge-Kutta 的通用形式：  
$k_i = f\!\left(x_n + c_i h,\; y_n + h\textstyle\sum_{j<i} a_{ij}k_j\right),\quad y_{n+1} = y_n + h\sum_i b_i k_i.$  
系数$\{a_{ij}, b_i, c_i\}$排成 **Butcher 表**。

### Heun（改进欧拉），二阶

两步预测-校正方法：

1. 预测：$\tilde y = y_n + h\,f(x_n, y_n)$
2. 校正：$y_{n+1} = y_n + \tfrac{h}{2}\bigl[f(x_n, y_n) + f(x_n + h, \tilde y)\bigr]$

局部误差$\mathcal{O}(h^3)$，全局误差$\mathcal{O}(h^2)$。步长减半，误差降为四分之一。效果明显改善。

### 经典 RK4，四阶

四级经典方法：

$$k_1 = f(x_n, y_n)$$$$k_2 = f\!\left(x_n + \tfrac{h}{2},\; y_n + \tfrac{h}{2}k_1\right)$$$$k_3 = f\!\left(x_n + \tfrac{h}{2},\; y_n + \tfrac{h}{2}k_2\right)$$$$k_4 = f(x_n + h,\; y_n + h k_3)$$$$y_{n+1} = y_n + \tfrac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
权重$(1,2,2,1)/6$与 Simpson 积分公式一致（并非巧合）。全局误差$\mathcal{O}(h^4)$，每步调用$f$四次——科学计算中的经典性价比之选。对光滑右端的非刚性问题，这个算法已稳坐工程默认值百年。

```python
def rk4(f, x0, y0, x_end, h):
    xs, ys = [x0], [y0]
    x, y = x0, y0
    while x < x_end - 1e-12:
        h_eff = min(h, x_end - x)
        k1 = f(x, y)
        k2 = f(x + h_eff/2, y + h_eff*k1/2)
        k3 = f(x + h_eff/2, y + h_eff*k2/2)
        k4 = f(x + h_eff, y + h_eff*k3)
        y = y + h_eff*(k1 + 2*k2 + 2*k3 + k4)/6
        x = x + h_eff
        xs.append(x); ys.append(y)
    return np.array(xs), np.array(ys)
```

### 对比：欧拉 vs RK4

![欧拉 vs RK4 在测试方程$\dot y=-2y$上各取若干步长。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-数值方法/fig1_euler_vs_rk4.png)  
*同样步长、同样问题（$\dot y=-2y$, $y(0)=1$）。$h=0.5$时欧拉法甚至出现负值——越过平衡点，因$h\lambda=-1$接近稳定边界。相同$h$下，RK4 与精确解几乎重合。*

差距显著。从一阶到四阶，固定$h$下的误差预算被平方两次。若要欧拉法在$h=0.1$达到 RK4 精度，需$h\approx 10^{-5}$——代价十万倍。
## 4. 收敛阶可视化

检验方法阶数最干净的办法：取一系列步长，计算终点全局误差，画 log-log 图。斜率就是阶数。

![Log-log 收敛：欧拉$\mathcal{O}(h)$，Heun$\mathcal{O}(h^2)$，RK4$\mathcal{O}(h^4)$。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-数值方法/fig2_error_convergence.png)
*参考虚线斜率分别为 1、2、4。误差紧贴对应直线，直到舍入误差（RK4 最小$h$）或稳定性（欧拉最大$h$）接管。*

两个实用教训：

- **阶数是硬上限**：$p$阶方法收敛速度超不过$\mathcal{O}(h^p)$。想更快只能换高阶方法或用外推。
- **舍入下界真实存在**：光滑问题上 RK4 在$h=10^{-3}$已接近机器精度。再降$h$只会让舍入累积——**反而更差**。

## 5. 自适应步长控制

真实解不均匀光滑，有平台、尖峰和缓尾。固定$h$在平台浪费，在尖峰危险。解决办法是**自适应步长**：动态调整$h$，使局部误差接近用户设定容差。

标准方法是**嵌入式 Runge-Kutta**：两个$p$阶和$p+1$阶方法共享级求值，差值估计局部误差$E_n$。若$E_n>\text{tol}$，拒绝该步，缩小$h$重试；否则接受，并按$h_\text{new} = 0.9\,h\,\bigl(\text{tol}/E_n\bigr)^{1/(p+1)}$选下一步。0.9 是安全因子，指数来自$p+1$阶渐近特性。最常用嵌套对是 **Dormand-Prince RK4(5)**——`solve_ivp` 内部的 "RK45"。每步 6 次函数求值，同时给出 4 阶和 5 阶估计。

![自适应 RK45 在尖锐瞬态上的步长位置与步长历史。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-数值方法/fig3_adaptive_step_size.png)
*上：含$t=1$附近 Gauss 脉冲的强迫线性 ODE。红点为积分器实际取步。下：步长在尖峰处缩小约 100 倍追踪，随后恢复。固定$h$若小到追上尖峰，平滑段会多走数千步。*

## 6. 刚性问题：显式方法的失败模式

![常微分方程（十一）：数值方法 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-numerical-methods/illustration_2.png)

有些方程在慢演化上叠加少量快动力学。快部分衰减后，想用大步长——但显式方法不让。它要求 $h \lesssim 1/|\lambda_{\max}|$ 永远成立，只为数值稳定。这类问题叫**刚性**。

教科书例子是高非线性的 van der Pol 振子：$\ddot{y} - \mu(1-y^2)\dot{y} + y = 0.$ $\mu=1000$ 时，慢尺度 $\sim \mu$，快尺度 $\sim 1/\mu$，比值 $10^6$。用 RK45 积分就是灾难。

![刚性问题：显式 RK45 比隐式 BDF 多用约 100 倍步数。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-数值方法/fig4_stiffness.png)  
*同一条轨迹，两种方法。BDF（隐式，蓝）整段积分只需几千步；RK45（显式，红）需要数十万步还吃力。这种模式在所有刚性问题上普适。*

出路是**隐式**方法。最简单的是**后向欧拉**：$y_{n+1} = y_n + h\,f(x_{n+1}, y_{n+1}),$ 在新点求斜率。代价是每步要解关于 $y_{n+1}$ 的代数方程，通常用 Newton 法。回报是后向欧拉的稳定区是整个左半平面（A-稳定）。无论多刚性，$h$ 只受精度约束。

### 稳定区可视化

![显式与隐式方法在复平面上的稳定区。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/11-数值方法/fig5_implicit_vs_explicit.png)  
*左：显式方法的稳定区是原点附近的有限椭圆。RK4（蓝）覆盖负实轴 $|h\lambda|\le 2.78$ 左右——较大但有界。右：后向欧拉与梯形规则覆盖整个左半平面，这是 A-稳定的定义。*

刚性问题特征值沿负实轴拉得很远，显式区是窄走廊。要么把 $h$ 缩到能装下（巨大代价），要么用 A-稳定方法（无 $h$ 约束）。

工业级隐式家族：

- **BDF**（后向差分公式，1-5 阶）。LSODE、CVODE、SciPy `'BDF'` 的默认刚性求解器。
- **Radau IIA**（3、5、9 阶）。强稳定性的隐式 RK；SciPy `'Radau'`。
- **隐式中点 / 梯形规则**。A-稳定，二阶，对 Hamilton 系统能量守恒。

实用准则：若 `solve_ivp(..., method='RK45')` 永远跑不完或拒绝容差，换 `'BDF'` 或 `'Radau'`。若刚/非刚区域交替，`'LSODA'` 自动检测。
## 7. 还应该知道的几种方法

### 多步法：Adams-Bashforth 和 Adams-Moulton

多步法不重复计算$f$，而是复用前几步的值：$y_{n+1} = y_n + h \sum_{j=0}^{k-1} \beta_j f(x_{n-j}, y_{n-j}).$ Adams-Bashforth 是显式，Adams-Moulton 是隐式。预测-校正对结合 AB 预测和 AM 校正。优点是启动后每步只需一次$f$求值；缺点是对不连续点敏感，且需要单独启动。

### Hamilton 流的辛积分器

能量守恒系统（行星轨道、分子动力学、格点规范理论）用常规方法会引发**能量漂移**：守恒量因舍入累积逐渐偏离。辛积分器保持相空间的辛 2-形式，虽不能严格守恒能量，但能把误差限制在真值附近的有界振荡内，即便积分上百万周期也如此。

最简例子是$\ddot q = -\nabla V(q)$的 **Stormer-Verlet（leapfrog）**：  
$p_{n+1/2} = p_n - \tfrac{h}{2}\nabla V(q_n),\quad q_{n+1} = q_n + h\,p_{n+1/2},\quad p_{n+1} = p_{n+1/2} - \tfrac{h}{2}\nabla V(q_{n+1}).$  
二阶精度，时间可逆，辛。N 体天体物理的标准工具。
## 8. SciPy 实战

```python
from scipy.integrate import solve_ivp
import numpy as np

# 非刚性问题：默认 RK45 就行。
sol = solve_ivp(lambda t, y: -2*y, [0, 5], [1.0],
                t_eval=np.linspace(0, 5, 100),
                rtol=1e-8, atol=1e-10)

# 刚性问题：用 BDF 或 Radau。
def vdp(t, y, mu=1000):
    return [y[1], mu*(1 - y[0]**2)*y[1] - y[0]]
sol = solve_ivp(vdp, [0, 3000], [2.0, 0.0],
                method='BDF',
                rtol=1e-6, atol=1e-9)

# 不确定刚性？交给 LSODA 决定。
sol = solve_ivp(rhs, [0, T], y0, method='LSODA')
```

`solve_ivp` 常用选项：

- `method`：`'RK45'`（默认）、`'RK23'`、`'DOP853'`（八阶高精度）、`'Radau'`、`'BDF'`、`'LSODA'`。
- `rtol`、`atol`：相对/绝对容差，按分量设置。默认值较松（$10^{-3}, 10^{-6}$）；正式计算收紧到$10^{-8}, 10^{-10}$或更低。
- `dense_output=True`：返回插值器 `sol.sol(t)`，任意$t$可查，不限积分步。
- `events`：传函数，求解器自动找零点，适合碰撞、反弹、阈值穿越。
- `jac`：提供解析 Jacobi 矩阵，隐式方法提速明显。

### 可靠性 checklist

1. **两套容差各跑一次**。`rtol` 缩小 100 倍后结果变化大，说明松的那次不准。
2. **画步长历史图**。求解器卡 `min_step` 说明有问题。`nfev` 爆表通常是显式方法碰刚性问题。
3. **检查 `success`**。`solve_ivp` 失败时默认沉默，必须手动确认。
4. **长积分用稠密输出 + 粗网格复核**。单步检查看不到漂移，但$10^6$周期后会很明显。

## 9. 方法选择速查表

| 方法 | 阶 | 类型 | 刚性稳定？ | 适用场景 |
|---|---|---|---|---|
| 前向欧拉 | 1 | 显式 | 否 | 教学用，别上生产 |
| Heun（改进欧拉） | 2 | 显式 | 否 | 玩具问题 |
| 经典 RK4 | 4 | 显式 | 否 | 光滑非刚性问题 |
| Dormand-Prince RK4(5) | 4(5) | 显式自适应 | 否 | 默认工作马 |
| DOP853 | 8(5,3) | 显式自适应 | 否 | 高精度非刚性 |
| 后向欧拉 | 1 | 隐式 | 是（A-稳定） | 最简刚性求解器 |
| 梯形 / Crank-Nicolson | 2 | 隐式 | 是（A-稳定） | 轻度刚性，保守 |
| BDF（1-5 阶） | 至 5 | 隐式多步 | 是 | 工业刚性首选 |
| Radau IIA | 3, 5, 9 | 隐式 RK | 是（L-稳定） | 极刚 + 高精度 |
| Stormer-Verlet | 2 | 辛 | — | Hamilton / 轨道 |

**一句话建议**：先试 `solve_ivp(method='RK45')`；慢或不稳定换 `'BDF'`；关心守恒量就看辛方法。
## 习题

1. **阶数验证**。实现 `euler`、`heun`、`rk4`。解$\dot y=-2y$，初值$y(0)=1$，区间$[0,3]$，步长$h\in\{0.5, 0.25, 0.125, 0.0625\}$。画 log-log 全局误差图，确认斜率分别为 1、2、4。

2. **稳定边界实测**。用欧拉法解$\dot y=\lambda y$（$\lambda=-10$）。找出失稳临界步长$h_*$，验证其与$2/|\lambda|$一致。

3. **刚性系统两种解法**。解 Robertson 化学动力学方程：$$\dot y_1 = -0.04 y_1 + 10^4 y_2 y_3,\quad \dot y_2 = 0.04 y_1 - 10^4 y_2 y_3 - 3\times 10^7 y_2^2,\quad \dot y_3 = 3\times 10^7 y_2^2,$$
初值$y(0)=(1,0,0)$，区间$[0,10^{11}]$。分别用 `'RK45'` 和 `'BDF'` 求解，比较运行时间和步数。

4. **自适应 vs 固定**。用 `solve_ivp` 解$\dot y=-y+100\,e^{-(t-1)^2/0.005}$，区间$[0,3]$，相对容差 `rtol=1e-6`。再用固定步长 RK4 达到相同精度，记录步数并对比。

5. **辛 vs RK4 在 Kepler 轨道上**。用 leapfrog 和 RK4 同步长积分圆 Kepler 轨道$10^4$周期。画出两种方法的能量随时间变化曲线。
