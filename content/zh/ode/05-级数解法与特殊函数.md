**有些 ODE 的解写不成初等函数。** Bessel 方程描述圆柱热传导和鼓面振动，Legendre 方程出现在球坐标分离变量中，Airy 方程刻画量子隧穿。它们的解定义了全新的特殊函数。本章介绍用幂级数和 Frobenius 法找解的统一方法，解释为何同一组特殊函数反复出现在物理与工程中。

## 1. 为什么需要幂级数？

研究圆柱体内热传导，分离变量后得到  
$$x^2 y'' + x y' + (x^2 - n^2) y = 0,$$  
这是 **Bessel 方程**。常系数特征方程、积分因子、参数变易法都用不上，因为系数是$x$的函数，不是常数。更麻烦的是，$x = 0$处奇异，但圆柱轴正好在那里，物理上必须有解。

**对策。** 假设解是级数形式，代入方程，让 ODE 自己生成系数。需要两种形式：

1. **常点**用普通 Taylor 级数$\sum a_k (x - x_0)^k$。
2. **正则奇点**用 Frobenius 级数$(x - x_0)^r \sum a_k (x - x_0)^k$，指数$r$由方程决定。

这套方法能解出 Bessel、Legendre、Hermite、Airy、Chebyshev、Laguerre、超几何函数——古典特殊函数全家族。
## 2. 常点处的幂级数解

### 2.1 算法

方程写成标准形式：
$$y'' + P(x)\,y' + Q(x)\,y = 0.$$
若$P, Q$在$x_0$解析，则$x_0$是**常点**。步骤如下：

1. 验证$x_0$是否为常点。
2. 设$y = \sum_{k=0}^{\infty} a_k (x - x_0)^k$。
3. 逐项求导后代入 ODE。
4. 收集$(x - x_0)^k$系数并置零。
5. 解出$a_k$关于$a_0, a_1$的递推式。

Fuchs 定理保证：级数在以$x_0$为中心、不含$P, Q$奇点的最大开圆盘内收敛——*半径延伸至复平面上最近奇点*。这说明复分析对实系数 ODE 也重要。

![常点附近，幂级数解的收敛半径直达复平面上最近的奇点。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-级数解法与特殊函数/fig1_radius_of_convergence.png)
*在常点$x_0$附近，幂级数解收敛于一个最大圆盘，边界由$P, Q$在复平面上的最近奇点决定。例如$P(x) = 1/(1 + x^2)$的奇点在$x = \pm i$，所以$x_0 = 0$的收敛半径$R = 1$。*

### 2.2 热身：重新发现$e^x$

用幂级数解$y' = y$。设$y = \sum a_k x^k$，则$y' = \sum (k+1) a_{k+1} x^k$，匹配系数得$$(k+1)\,a_{k+1} = a_k, \qquad a_k = \frac{a_0}{k!},$$于是$$y = a_0 \sum_{k=0}^{\infty} \frac{x^k}{k!} = a_0\,e^x.$$级数法"重新发现"了指数函数。意义在于：即使已知答案，递推仍是机械可靠的过程。

### 2.3 Airy 方程
$$y'' - x\,y = 0.$$
波（光、量子概率幅、水波）遇到"局部波数从实变虚"的转折点时，都会出现此方程；经典例子是线性势垒附近的量子隧穿。

$P(x) = 0, Q(x) = -x$均为整函数，故$x = 0$是常点。代入$y = \sum a_k x^k$得递推$$a_{k+2} = \frac{a_{k-1}}{(k+1)(k+2)} \qquad (k \geq 1), \qquad a_2 = 0.$$系数分成两条独立链，分别由$a_0, a_1$起始；得到两个线性独立解，即 **Airy 函数**$\text{Ai}(x)$与$\text{Bi}(x)$。$\text{Ai}$在$x > 0$（经典禁区）指数衰减，在$x < 0$（经典允许区）振荡——量子隧穿现象一目了然。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.special import airy

x = np.linspace(-15, 5, 1000)
Ai, Aip, Bi, Bip = airy(x)

plt.figure(figsize=(10, 5))
plt.plot(x, Ai, 'b-', linewidth=2, label='Ai(x)')
plt.plot(x, Bi, 'r-', linewidth=2, label='Bi(x)')
plt.axhline(0, color='k', linewidth=0.5)
plt.title("Airy 函数：y'' - xy = 0 的解")
plt.legend(); plt.ylim(-0.6, 1.2)
plt.grid(True, alpha=0.3); plt.tight_layout()
plt.show()
```
## 3. 正则奇点与 Frobenius 方法

### 3.1 普通 Taylor 级数为何失效

考虑方程  $$4 x^2 y'' + y = 0.$$  
直接验证，$y = \sqrt{x}$ 是解。但 $\sqrt{x}$ 在原点切线竖直，不解析。形如 $\sum a_k x^k$ 的级数无法复现它。必须引入分数次幂。

![普通 Taylor 级数无法复现原点尖点；Frobenius 待定式可以。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-级数解法与特殊函数/fig2_frobenius_vs_taylor.png)  
*左：过原点的多项式在 $x = 0$ 处斜率有限，无法匹配 $\sqrt{x}$ 尖点。右：Frobenius 待定式 $y = x^{1/2}\sum a_k x^k$ 精确还原。指标方程 $4x^2 y'' + y = 0$ 有重根 $r = 1/2$。*

### 3.2 正则奇点定义

若 $P$ 或 $Q$ 在 $x_0$ 奇异，但缓和后的乘积  
$$(x - x_0)\,P(x), \qquad (x - x_0)^2\,Q(x)$$  
均在 $x_0$ 解析，则称 $x_0$ 为**正则奇点**。直观上，$P$ 至多单极，$Q$ 至多二阶极。

### 3.3 Frobenius 待定式

试取广义级数  
$$y = (x - x_0)^r \sum_{k=0}^{\infty} a_k (x - x_0)^k, \qquad a_0 \neq 0,$$  
其中 $r$ 待定。代入 ODE，令最低次 $(x - x_0)$ 系数为零，得**指标方程**  
$$r(r - 1) + p_0\,r + q_0 = 0,$$  
其中 $p_0 = \lim_{x \to x_0}(x-x_0) P(x), q_0 = \lim_{x \to x_0}(x-x_0)^2 Q(x)$。两根 $r_1 \geq r_2$ 决定独立解形式：

| 情形 | 两个独立解 |
|------|------------|
|$r_1 - r_2 \notin \mathbb{Z}$ | 两个 Frobenius 级数，各对应一个根 |
|$r_1 = r_2$（重根） | 一个 Frobenius 级数，加一个含 $\ln(x - x_0)$ 的解 |
|$r_1 - r_2 \in \mathbb{Z}_{>0}$ | 一个 Frobenius 级数；第二解可能含对数项 |

更高次幂系数匹配给出 $a_k$ 递推关系，逐项求解即可。
## 4. Bessel 函数

![常微分方程（五）：级数解法与特殊函数 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-laplace-transform/illustration_2.jpg)

### 4.1 Bessel 方程

$$x^2 y'' + x y' + (x^2 - n^2) y = 0.$$
这个方程在圆柱物理中很重要，常见于以下三种场景：

- **圆形鼓面振动**——亥姆霍兹方程$\nabla^2 u + k^2 u = 0$的径向部分。
- **圆柱波导**——金属管内的电磁模式。
- **圆柱体热扩散**——金属棒的瞬态冷却。

$x = 0$是正则奇点，$p_0 = 1, q_0 = -n^2$。指标方程$r^2 - n^2 = 0$给出$r = \pm n$。

### 4.2 第一类 Bessel 函数

取$r = n$（$n \geq 0$），递推得到$$\boxed{\; J_n(x) = \sum_{m=0}^{\infty} \frac{(-1)^m}{m!\,\Gamma(m + n + 1)} \left(\frac{x}{2}\right)^{2m+n}.\;}$$
常用三条性质：

- **递推关系：**$\displaystyle J_{n-1}(x) + J_{n+1}(x) = \frac{2n}{x}\,J_n(x)$，可在不同阶间切换。
- **大$x$渐近：**$\displaystyle J_n(x) \approx \sqrt{\frac{2}{\pi x}}\cos\!\left(x - \frac{n\pi}{2} - \frac{\pi}{4}\right)$，远端像衰减余弦。
- **零点：**$J_n$的正零点$j_{n,1} < j_{n,2} < \dots$间距约$\pi$，*量子化了圆膜共振频率*。

![第一类 Bessel 函数，前几个$J_0$零点已标出。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-级数解法与特殊函数/fig3_bessel_first_kind.png)
*前五个 Bessel 函数$J_0, \dots, J_4$。标出的零点$j_{0,1} \approx 2.405, j_{0,2} \approx 5.520, \ldots$是单位圆膜对称振动模态的径向波数。*

### 4.3 从零点到鼓面模态

单位圆鼓极坐标分离变量，归一化模态为$$u_{n,m}(r,\varphi,t) = J_n(j_{n,m}\,r)\,\cos(n\varphi)\,\cos(c\,j_{n,m}\,t),$$其中$j_{n,m}$是$J_n$的第$m$个正零点。整数$n$表示**角向**节线条数，$m$表示**径向**节圆个数（含边界）。

![圆形鼓面的二维振动模态，由 Bessel 函数与角向余弦组合而成。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-级数解法与特殊函数/fig4_drumhead_modes.png)
*单位圆鼓四个本征模态的瞬态图。黑色虚线是节线（不动的位置）。$(0,1)$是基础"呼吸"模态；$(0,2)$多一道径向节；$(1,1)$多一道角向节；$(2,1)$多两道角向节。*

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.special import jv, jn_zeros

x = np.linspace(0.01, 20, 1000)
plt.figure(figsize=(8, 4))
for n in range(5):
    plt.plot(x, jv(n, x), linewidth=2, label=f'$J_{n}(x)$')
plt.axhline(0, color='k', linewidth=0.5)
plt.title('第一类 Bessel 函数')
plt.legend(fontsize=9); plt.grid(True, alpha=0.3)
plt.tight_layout(); plt.show()
```

### 4.4 第二类 Bessel 函数

当$n$是非负整数时取负根$r = -n$，递推失效，需引入对数项。得到的线性独立解叫**第二类 Bessel 函数**$Y_n(x)$。$Y_n$在原点附近行为是$Y_n(x) \sim -(2/\pi)(n-1)!(x/2)^{-n}$，会发散。定义域**包含轴心**的问题（实心鼓、实心圆柱）只能用$J_n$；定义域是**圆环域**的问题（同轴电缆）需同时用$J_n$和$Y_n$。

## 5. Legendre 多项式

### 5.1 Legendre 方程
$$(1 - x^2)\,y'' - 2x\,y' + n(n+1)\,y = 0.$$
球坐标分离变量时总能碰到它：拉普拉斯算子角向部分、氢原子波函数、引力和静电多极展开、天线辐射方向图。这里 $x$ 代表 $\cos\theta$，定义域自然就是 $[-1, 1]$。

$x = \pm 1$ 是正则奇点（首项系数 $1 - x^2$ 的零点），$x = 0$ 是常点。

### 5.2 为什么是多项式？

$x = 0$ 处的递推关系是$$a_{k+2} = \frac{k(k+1) - n(n+1)}{(k+1)(k+2)}\,a_k.$$
$n$ 是非负整数时，分子在 $k = n$ 归零，级数截断成多项式。另一解仍是无穷级数，在 $x = \pm 1$ 发散。要求解在球两极正则，就只能选多项式，参数 $n$ 自然量子化。

按 $P_n(1) = 1$ 归一化：

|$n$|$P_n(x)$|
|-----|-----------|
| 0 |$1$|
| 1 |$x$|
| 2 |$\frac{1}{2}(3x^2 - 1)$|
| 3 |$\frac{1}{2}(5x^3 - 3x)$|
| 4 |$\frac{1}{8}(35x^4 - 30x^2 + 3)$|

### 5.3 Rodrigues 公式与正交性

紧凑闭式：$$P_n(x) = \frac{1}{2^n n!}\,\frac{d^n}{dx^n}(x^2 - 1)^n.$$
Legendre 展开的核心恒等式：$$\int_{-1}^{1} P_m(x)\,P_n(x)\,dx = \frac{2}{2n+1}\,\delta_{mn}.$$
$[-1, 1]$ 上的合理函数都能展成 $f(x) = \sum_n c_n P_n(x)$，系数 $c_n = \frac{2n+1}{2}\int_{-1}^{1} f(x) P_n(x)\,dx$。这是电磁学和引力中多极展开的数学基础。

![$[-1, 1]$ 上的 Legendre 多项式，正交区间已用浅色标出。](./05-级数解法与特殊函数/fig5_legendre_polynomials.png)
*Legendre 多项式 $P_0, \dots, P_5$。归一化保证它们都过 $(1, 1)$，奇偶性保证它们都过 $(-1, (-1)^n)$。每个 $P_n$ 在 $(-1, 1)$ 内恰有 $n$ 个简单零点——Gauss-Legendre 数值积分用的就是这些根。*

## 6. Hermite 多项式与量子谐振子

### 6.1 Hermite 方程
$$y'' - 2x\,y' + 2n\,y = 0.$$
$n$为非负整数时，解是 **Hermite 多项式** $H_n(x)$：$$H_0 = 1, \quad H_1 = 2x, \quad H_2 = 4x^2 - 2, \quad H_3 = 8x^3 - 12x, \ldots$$
正交性公式：$$
\int_{-\infty}^{\infty} H_m(x) H_n(x) e^{-x^2} dx = \sqrt{\pi}\,2^n n!\,\delta_{mn}
$$权函数是高斯分布，正好对应波函数包络。

### 6.2 量子谐振子

抛物势中粒子的薛定谔方程（无量纲形式）：$$
-\tfrac{1}{2}\psi'' + \tfrac{1}{2} x^2 \psi = E\,\psi.
$$
代入 $\psi(x) = H_n(x) e^{-x^2/2}$，化为 Hermite 方程。要求 $\psi$ 可归一化，$n$ 必须是非负整数。本征函数和本征值为：$$
\psi_n(x) = \frac{1}{\sqrt{2^n n!}\,\pi^{1/4}}\,H_n(x)\,e^{-x^2/2}, \qquad E_n = n + \tfrac{1}{2}\quad(\text{以 }\hbar\omega\text{ 为单位}).
$$
两条物理结论：

- **能量量子化。** $n = 0, 1, 2, \ldots$ 才有可归一化解，能级分立。
- **零点能。** 基态 $E_0 = \tfrac{1}{2}\hbar\omega \neq 0$，抛物势中粒子无法静止。

![量子谐振子前几个 Hermite 本征态，按本征能叠在抛物势上。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/05-级数解法与特殊函数/fig6_qho_wavefunctions.png)

*前六个本征态 $\psi_n$，画在各自能量 $E_n = n + \tfrac{1}{2}$ 处，叠加在抛物势 $V(x) = \tfrac{1}{2} x^2$ 上。$\psi_n$ 的节点数等于 $n$，这是 Hermite 多项式的特征。*

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.special import hermite
from math import factorial

x = np.linspace(-4, 4, 500)
plt.figure(figsize=(8, 6))
for n in range(5):
    Hn = hermite(n)
    norm = 1 / np.sqrt(2**n * factorial(n)) * (1/np.pi)**0.25
    psi = norm * Hn(x) * np.exp(-x**2 / 2)
    plt.plot(x, psi + n, linewidth=2, label=f'n={n}')
    plt.fill_between(x, n, psi + n, alpha=0.2)
plt.title('量子谐振子波函数')
plt.grid(True, alpha=0.3); plt.tight_layout(); plt.show()
```
## 7. Sturm-Liouville 统一框架

上述特殊函数全是自伴 **Sturm-Liouville 问题** 的本征函数：$$\frac{d}{dx}\!\left[p(x)\,\frac{dy}{dx}\right] + \bigl[q(x) + \lambda\,w(x)\bigr]\,y = 0,$$
配上合适的边界条件。调整 $p, q, w$ 和区间，就能得到不同特殊函数：

| 函数 | $p(x)$ | $w(x)$ | 区间 |
|------|--------|--------|------|
| Legendre $P_n$ | $1 - x^2$ | $1$ | $[-1, 1]$ |
| Bessel $J_n$ | $x$ | $x$ | $[0, a]$ |
| Hermite $H_n$ | $e^{-x^2}$ | $e^{-x^2}$ | $(-\infty, \infty)$ |
| Laguerre $L_n$ | $x e^{-x}$ | $e^{-x}$ | $[0, \infty)$ |
| Chebyshev $T_n$ | $\sqrt{1 - x^2}$ | $1/\sqrt{1 - x^2}$ | $[-1, 1]$ |

每族函数都自带三个性质：

1. **实本征值**——算子在权 $w$ 下对称。
2. **正交性**——$\int y_m y_n\,w\,dx = 0$（$m \neq n$）。
3. **完备性**——任意函数可展成广义 Fourier 级数。

这就是为什么"球谐展开""Bessel 模态展开""Hermite 态展开"感觉一样：它们本质是同一条定理，只是权重不同。

## 总结

| 概念 | 核心思想 |
|------|----------|
| 常点幂级数 | 假设 $y = \sum a_k x^k$，推导 $a_k$ 的递推关系 |
| 收敛半径 | 等于复平面上 $P, Q$ 最近奇点的距离 |
| 正则奇点 | Frobenius 法：$y = x^r \sum a_k x^k$ |
| 指标方程 | 关于 $r$ 的二次方程，确定首项指数 |
| Bessel 函数 | 圆柱几何；零点决定鼓膜振动频率 |
| Legendre 多项式 | 球几何；多极展开的基函数 |
| Hermite 多项式 | 量子谐振子波函数 |
| Sturm-Liouville 理论 | 统一框架，多种正交本征函数 |

**基础题**

1. 用幂级数解 $y'' + y = 0$，验证结果为 $\sin x$ 和 $\cos x$。
2. 证明 $x = 0$ 是 $2x^2 y'' + 3x y' - (1 + x) y = 0$ 的正则奇点，求其指标方程。
3. 用 Rodrigues 公式计算 $P_4(x)$ 和 $P_5(x)$，验证递推关系 $(n+1) P_{n+1}(x) = (2n+1) x P_n(x) - n P_{n-1}(x)$。

**进阶题**
