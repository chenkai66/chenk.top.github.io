湖泊清澈几十年，一个夏天突然变浑。电网平稳运行，几秒内级联崩溃。钢柱在递增载荷下笔直挺立，直到突然弯折。

这不是预测失败，而是动力系统遵循**分岔理论**所刻画的固有规律。当参数越过临界值，相空间的拓扑结构发生突变，原本不可能的状态变为必然出现。本章目标是分类这些"重排"。结果发现，范式屈指可数，一旦掌握其特征，便能在各类系统中辨识它们。

## 1. 什么是分岔？

"Bifurcation"（拉丁文 *furca* = 叉子）由庞加莱在 1885 年前后提出。直觉是几何的。单参数系统$\dot{x}=f(x,\mu)$的长期行为像一幅依赖$\mu$的连续画面。大多数$\mu$值下，小扰动只改变画面位置，不改变本质特征：平衡点数量、稳定性、极限环都保持不变。这种$\mu$叫**结构稳定**。

分岔则是反例：在$\mu_c$处，画面形状在任意小扰动下发生**不连续**变化。新平衡点突然出现，两个平衡点碰撞湮灭，稳定焦点变成极限环，周期解周期翻倍。

变化是定性的，不是定量的。水管缓慢变窄不属于分岔；而水管突然爆裂——才属于典型的分岔现象。

### 一个有用的心智模型

把$f(x,\mu)$看作依赖$\mu$的**势能景观**：平衡点是导数为零的位置，稳定性由二阶导决定。$\mu$变化时，景观连续变形。但当山头被夷平成鞍点再翻转成谷地的瞬间，**那就是分岔**。在两次分岔之间，势能景观仅发生连续形变。
## 2. 四种余维-1 范式

分岔理论有个奇迹：任意单参数分岔附近，动力学局部等价于四个标准方程之一。这些就是**范式**。

![四种余维-1 范式分岔图全景。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-分岔理论/fig5_normal_forms_overview.png)
*四种余维-1 分岔在同一张图中。蓝色实线 = 稳定，红色虚线 = 不稳定，绿色带 = 极限环振幅。单参数 ODE 的每个分岔局部都是这四种之一。*

为什么叫"余维-1"？因为每种分岔需要调节一个参数。要让两种同时发生，得调两个参数。转动一个旋钮时，遇到的通常是余维-1 事件。

### 2.1 鞍-结点分岔（fold / 折叠）

**范式：**$\dot{x} = \mu - x^2.$令$\dot{x}=0$得$x^* = \pm\sqrt{\mu}$。

|$\mu$范围 | 平衡点 |
|---|---|
|$\mu < 0$| **无** |
|$\mu = 0$| 一个（半稳定，在$x=0$） |
|$\mu > 0$| 两个：$+\sqrt{\mu}$稳定，$-\sqrt{\mu}$不稳定 |

线性化$f_x = -2x$直接给出稳定性：$+\sqrt{\mu}$处$f_x<0$（稳定），$-\sqrt{\mu}$处$f_x>0$（不稳定）。两个平衡点随$\mu$穿越 0 成对诞生。

![鞍-结点分岔：左为相函数，右为带流向的分岔图。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-分岔理论/fig1_saddle_node_bifurcation.png)
*左：抛物线$\mu - x^2$随$\mu$增大向上平移，零点在折叠点$\mu=0$处诞生。右：分岔图。灰色区域没有平衡点——某种状态不存在。*

**应用案例**

- **激光器**：低于阈值泵浦电流，唯一稳态是"无光"；高于阈值出现相干发射态。
- **神经元（I 类）**：低于刺激电流最低限，静息态是唯一平衡点；超过后静息态与鞍点碰撞消失，神经元开始放电。
- **湖泊从清变浑**：清水平衡随营养盐负荷穿越临界点在折叠分岔中消失。

折叠标志是湮灭前双稳态：临界值前两个平衡点共存，临界值后一个都没有。系统必须跑去别处，往往剧烈。

### 2.2 跨临界分岔

**范式：**$\dot{x} = \mu x - x^2.$两个平衡点恒存在：$x^*=0$和$x^*=\mu$。它们不消失，只在$\mu=0$处交换稳定性。

|$\mu$范围 |$x^*=0$|$x^*=\mu$|
|---|---|---|
|$\mu < 0$| 稳定 | 不稳定 |
|$\mu > 0$| 不稳定 | 稳定 |

这是系统因对称性或定义保留"平凡"平衡点（$x=0$）时出现的分岔。

![跨临界分岔：轨迹收敛到当前稳定的那条分支。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-分岔理论/fig2_transcritical_bifurcation.png)
*左：$\mu=-0.8$时轨迹收敛到 0；$\mu=+0.8$时收敛到$\mu$。右：两条分支构成"X"形，在交点处稳定性互换。*

**应用**

- **流行病学**：无病平衡点恒存在。基本再生数$R_0$穿越 1 时，它把稳定性让给地方性流行平衡点——正是跨临界分岔。
- **种群动力学**：灭绝态总是平衡点；环境质量参数穿越阈值时，灭绝态把稳定性让给共存态。

### 2.3 超临界叉型分岔

**范式：**$\dot{x} = \mu x - x^3.$平衡点恒为$x^*=0$，且当$\mu>0$时多出$x^*=\pm\sqrt{\mu}$。平凡分支失稳，同时两个对称稳定分支诞生。

这是普适的对称破缺分岔。方程在$x \to -x$下不变，新平衡点必成对出现。临界前系统位于对称解，临界后必须在两个非对称解中任选其一。

### 2.4 亚临界叉型分岔（危险的那一种）

**范式：**$\dot{x} = \mu x + x^3.$平凡分支失稳时没有附近的稳定分支接住系统：$\mu<0$时$x^*=0$稳定，并有两条不稳定分支$\pm\sqrt{-\mu}$；$\mu>0$时只剩不稳定的平凡分支。轨迹冲向无穷。

实际系统中高阶项最终会重新稳定一切：加入$-x^5$项给出经典滞后模型$\dot{x} = \mu x + x^3 - x^5,$它在$-\tfrac14 \le \mu \le 0$区间内同时存在大幅稳定分支和平凡稳定态。准静态提升$\mu$复现著名滞后回路：$\mu$从下方穿越 0 时系统突然跳到大振幅，要让它跳回来必须把$\mu$拉到$-\tfrac14$以下。

![叉型分岔：左为温和的超临界叉型，右为带滞后回路的危险亚临界叉型。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-分岔理论/fig3_pitchfork_bifurcation.png)
*左：超临界叉型。新分支从零连续生长，温和、可逆。右：被$-x^5$稳定的亚临界叉型。紫色箭头标出$\mu=0$处的灾难性向上跳跃和$\mu=-\tfrac14$处的延迟向下跳跃。双稳窗口宽度就是滞后回路。*

**为什么这事关重大**

- **欧拉屈曲**：受压细长柱发生超临界叉型分岔——临界载荷下小幅可逆地往左或往右弯。
- **薄壳骤崩**（圆柱壳的 von Karman 屈曲）：叉型是亚临界的。壳保持笔直，然后砰地坍塌成大变形构型。航空工程师用 3-10 倍安全系数计算屈曲载荷的原因。
- **铁磁体过 Curie 点**：超临界叉型，磁化强度从零连续生长。
- **气候转换**：冰期-间冰期跃迁常被建模为温度-反照率系统的亚临界叉型（或折叠）。滞后窗口意味着——一旦翻转，仅仅把 CO$_2$恢复到原值不能让系统恢复。

### 总结表

| 分岔 | 范式 | 现象 | "软"还是"硬"？ |
|---|---|---|---|
| 鞍-结点 |$\dot{x}=\mu-x^2$| 平衡点成对生灭 | 硬（状态消失） |
| 跨临界 |$\dot{x}=\mu x-x^2$| 两条分支交换稳定性 | 软 |
| 超临界叉型 |$\dot{x}=\mu x-x^3$| 对称分裂，分支从 0 生长 | 软 |
| 亚临界叉型 |$\dot{x}=\mu x+x^3$| 对称分裂向外扩张；跳跃 + 滞后 | 硬 |

## 3. Hopf 分岔：焦点孕育出环

![常微分方程（十）：分岔理论 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-bifurcation-theory/illustration_2.png)

前面讲的都是标量分岔。第一个真正**二维**的分岔是 **Hopf**（严格说是 Andronov-Hopf）。它让振荡得以出现。

**极坐标范式：**  
$\dot{r} = \mu r - r^3, \qquad \dot{\theta} = \omega.$  
径向方程就是超临界叉型分岔。

- $\mu \le 0$：唯一吸引子是$r=0$，稳定焦点。
- $\mu > 0$：原点变成不稳定焦点，半径$r=\sqrt{\mu}$的**稳定极限环**包围它。

直角坐标形式：  
$\dot{x} = \mu x - \omega y - x(x^2+y^2),$  
$\dot{y} = \omega x + \mu y - y(x^2+y^2).$  
原点处 Jacobi 矩阵$\bigl(\begin{smallmatrix}\mu & -\omega \\ \omega & \mu\end{smallmatrix}\bigr)$，特征值$\lambda = \mu \pm i\omega$。当$\mu$从负穿越 0 时，复共轭对横截虚轴——这就是 **Hopf 条件**。

![Hopf 分岔：分岔前、临界、分岔后的相图，加上极限环振幅的三维抛物面。](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/ode/10-分岔理论/fig4_hopf_bifurcation.png)  
*上排：$\mu=-0.4$时轨迹螺旋收敛到原点；$\mu=0$时悬停不定；$\mu=+0.4$时收敛到半径$\sqrt{0.4}\approx 0.63$的极限环。下：极限环族构成向右开口的抛物面$r=\sqrt{\mu}$。*

**Hopf 定理（Hopf 1942）**  
设$\mathbf{x}_0$是$\dot{\mathbf{x}}=\mathbf{f}(\mathbf{x},\mu)$在$\mu=\mu_c$处的平衡点，Jacobi 矩阵有复数对$\lambda(\mu)=\alpha(\mu)\pm i\beta(\mu)$满足：

1. $\alpha(\mu_c)=0$（在虚轴上）
2. $\beta(\mu_c)\ne 0$（确为复数）
3. $\alpha'(\mu_c)\ne 0$（横截穿越，非仅相切）

假设三次 **Lyapunov 系数**$\ell_1 \ne 0$。则一族周期轨从$\mathbf{x}_0$在$\mu_c$处涌出，周期约为$2\pi/\beta(\mu_c)$。$\ell_1 < 0$时极限环**稳定**（超临界 Hopf），$\ell_1 > 0$时**不稳定**（亚临界 Hopf）。

**亚临界 Hopf** 是亚临界叉型的"环"版本：临界前毫无征兆，临界时直接把系统抛到远处吸引子。飞机机翼颤振、心律失常突发、激光模式跳变——都是典型的亚临界 Hopf。

```python
import numpy as np
from scipy.integrate import odeint

def hopf(s, t, mu, omega=1.0):
    x, y = s
    r2 = x*x + y*y
    return [mu*x - omega*y - x*r2,
            omega*x + mu*y - y*r2]

t = np.linspace(0, 60, 6000)
sol = odeint(hopf, [0.05, 0.05], t, args=(0.4,))
# 轨迹螺旋向外，最终落在极限环 r = sqrt(0.4) ~ 0.632 上
```
## 4. 余维与普适性

分岔的**余维 $k$** 指需要调节 $k$ 个独立参数才能通用地实现。余维-1 事件在参数空间中形成曲线；余维-2 事件出现在两条曲线相交的孤立点。

常见余维-2 分岔：

- **尖点（cusp）**：两条鞍-结点曲线相切相遇。展开式为 $\dot{x} = \mu_1 + \mu_2 x - x^3$，包含由两条鞍-结点曲线包围的滞后区。
- **Bogdanov-Takens（BT）**：双重零特征值。展开式中鞍-结点、Hopf 和同宿曲线交于一点。若参数图中 Hopf 曲线和折叠曲线靠近，交点处可能找到 BT。
- **Bautin（广义 Hopf）**：第一 Lyapunov 系数过零，标记超临界 Hopf 和亚临界 Hopf 的边界。极限环在此发生"鞍-结点-环"折叠。

这些分类的存在源于**中心流形定理**和**范式方法**：在余维-$k$ 分岔附近，只有少数变量（中心方向）携带慢动力学，其余被它们支配。经多项式坐标变换后，慢动力学化简为普适范式。这使得分岔理论近乎完备——类似元素周期表。
## 5. 全局分岔：远离平衡点的拓扑变化

局部分岔重排单点附近的相空间。全局分岔大尺度重排，通常通过重连不变流形。

### 同宿分岔

从鞍点沿不稳定流形出发，沿稳定流形回到同一鞍点的轨迹叫**同宿轨**。它只在孤立参数值存在（余维-1）。靠近同宿分岔时，邻近周期轨周期任意长——轨道在鞍点附近爬行时间越来越久。普适标度律为 $T \sim -\log|\mu - \mu_c|$。

**Shilnikov 定理**：到鞍-焦点的同宿环（特征值比满足条件）暗示邻域内有可数无穷多周期轨——即**混沌**。

### 异宿分岔

思路类似，但轨迹连接**不同**鞍点。异宿环产生慢振荡，每个鞍点附近有近乎停滞的长阶段。它是神经回路中"无赢家竞争"的标准模型。

### SNIC（不变圆上的鞍-结点）

发生在**闭不变曲线**上的鞍-结点分岔。临界前曲线被鞍-结点对切断；临界后这对湮灭，曲线恢复为极限环。极限环以**无限周期**诞生（系统在原平衡点位置爬行），周期标度律为 $T \sim 1/\sqrt{\mu_c - \mu}$。SNIC 是神经元从静息到放电的第二种标准路径（I 类兴奋性），也是 El Niño 振子模型的关键机制。
## 6. 通往混沌之路：周期倍化

极限环也会分岔。最著名的是**周期倍化级联**：周期$T$的稳定环失稳，产生周期$2T$的稳定环；接着倍化为$4T$、$8T$、$16T$，在有限参数值处累积，越过该点进入混沌。

Mitchell Feigenbaum（1978）发现，相邻倍化间隔$\Delta_n = \mu_n - \mu_{n-1}$以几何级数收缩，比率为$\delta = \lim_{n \to \infty} \frac{\Delta_n}{\Delta_{n+1}} = 4.6692016\ldots$。这个值与具体系统无关，只要系统有光滑二次极大值。滴水、Rayleigh-Bénard 对流、电子电路中的周期倍化都受同一个常数支配。

最简模型是 **Logistic 映射**$x_{n+1} = r x_n(1 - x_n)$：

|$r$范围 | 行为 |
|---|---|
|$0 < r < 1$| 灭绝：$x \to 0$|
|$1 < r < 3$| 稳定不动点在$1 - 1/r$|
|$3 \le r < 3.449$| 周期 2 |
|$3.449 \le r < 3.544$| 周期 4 |
|$\vdots$| 周期 8, 16, 32, ... |
|$r > 3.5699$| 混沌（含周期窗） |

Sharkovsky 定理（1964）和推论"**周期 3 蕴含混沌**"（Li-Yorke 1975）补全了理论：任何含周期-3 轨的连续区间映射都有所有周期轨道，以及不可数多条非周期轨。

深入讨论见 [第 9 章](/zh/ode/09-混沌理论与洛伦兹系统/)。
## 7. 数值检测与延拓

实际中很少有解析范式，通常只有向量场 $\mathbf{f}(\mathbf{x},\mu)$。标准工具是**延拓方法**：

1. **追踪平衡分支**：从已知平衡点出发，用 Newton 法解 $\mathbf{f}(\mathbf{x},\mu)=0$，逐步递增 $\mu$。用**伪弧长延拓**处理折叠，按弧长参数化分支，算法能"拐弯"。
2. **监视 Jacobi 特征值**：实特征值穿越 0 表示鞍-结点、跨临界或叉型分岔（靠对称性和二阶系数区分）；复数对穿越虚轴表示 Hopf 分岔。
3. **切换分支**：在分岔点用范式系数计算新分支的切方向。

工业级工具包括 **AUTO-07p**（金标准，Fortran/C）、**MATCONT**（MATLAB）、**PyDSTool** 和 **BifurcationKit.jl**。研究级分岔分析基本离不开这些工具；自己写延拓算法费时费力且容易出错。

```python
import numpy as np
from scipy.linalg import eigvals

def detect_bifurcations(f, fx, x_branch, mu_grid):
    """沿平衡分支追踪特征值穿越事件"""
    events = []
    prev = None
    for mu, x in zip(mu_grid, x_branch):
        eigs = eigvals(fx(x, mu))
        if prev is not None:
            # 实特征值穿越 -> 折叠/跨临界/叉型
            for a, b in zip(np.sort(prev.real), np.sort(eigs.real)):
                if a * b < 0 and abs(a.imag) + abs(b.imag) < 1e-8:
                    events.append({'kind': 'real_crossing', 'mu': mu})
            # 复对穿越 -> Hopf
            prev_pair = prev[np.iscomplex(prev)]
            curr_pair = eigs[np.iscomplex(eigs)]
            if prev_pair.size and curr_pair.size:
                if prev_pair[0].real * curr_pair[0].real < 0:
                    events.append({'kind': 'hopf', 'mu': mu,
                                   'omega': abs(curr_pair[0].imag)})
        prev = eigs
    return events
```
## 8. 这一切意味着什么

分岔理论的核心是：**光滑的因能引发突变的果，但只通过少数规范机制**。怀疑系统逼近临界点时，可以问几个具体问题：

- **哪种分岔？** 方差增大、扰动恢复变慢（**临界减速**）指向折叠或叉型；振荡增强指向 Hopf。
- **超临界还是亚临界？** 若接近时双稳态，就是亚临界——跳跃大且可能不可逆。
- **有滞后吗？** 如果有，别以为反向调参就能复原。
- **有早期预警信号吗？** 折叠分岔中，$\mu \to \mu_c$时小扰动恢复率按$\sim |\mu-\mu_c|^{1/2}$衰减至零。这方法已在生态、气候和流行病学中用于预报临界跃迁。

**分类简单，现象普遍。**
## 习题

1. **不完美叉型**。证明$\dot{x} = h + \mu x - x^3$在$(\mu, h)$平面有鞍-结点曲线，尖点在原点。画出$h>0$、$h=0$、$h<0$的分岔图。

2. **$\dot{x} = \mu - x - e^{-x}$的分岔**。用$\mu = x + e^{-x}$隐式求平衡点。证明$\mu_c=1, x_c=0$是鞍-结点分岔，判断各分支稳定性。

3. **Logistic 倍化数值实验**。计算 Logistic 映射的倍化参数$r_n$到$n=6$。用$\delta_n := (r_n-r_{n-1})/(r_{n+1}-r_n)$估计 Feigenbaum 常数。

4. **捕食者-被捕食者中的 Hopf**。对$\dot{x} = x(1-x) - \tfrac{xy}{a+x},\;\dot{y} = -dy + \tfrac{xy}{a+x}$，找共存平衡点 Hopf 分岔的参数组合，数值判断超/亚临界。

5. **带$x^5$饱和的亚临界叉型**。对$\dot{x} = \mu x + x^3 - x^5$，推导$\mu = -1/4$处鞍-结点折叠位置及滞后区间。绘出准静态升降$\mu$时的回路。
