## 为什么需要无穷维分析

在 $\mathbb{R}^n$ 里，线性代数就够用了：所有范数等价，所有线性映射连续，闭有界集都是紧的。但函数空间是无穷维的，这些好性质全部失效。泛函分析的核心任务就是在无穷维空间里重新建立分析的基础设施。

起点很朴素：给集合配一个距离，看看能推出什么。

## 度量空间

**度量空间** $(X, d)$ 是集合 $X$ 配上满足三条公理的函数 $d: X \times X \to [0, \infty)$：

$$d(x, y) = 0 \iff x = y, \quad d(x, y) = d(y, x), \quad d(x, z) \le d(x, y) + d(y, z).$$

三角不等式是关键——它保证了开球确实是开集，极限唯一，拓扑行为良好。

**例 1：上确界度量。** $C[0,1]$（连续函数空间）配距离：

$$d(f, g) = \sup_{t \in [0,1]} |f(t) - g(t)|.$$

这个距离下的收敛就是一致收敛。连续函数的一致极限仍是连续函数——空间是完备的。

**例 2：$L^1$ 度量。** 同样是 $C[0,1]$，但换成：

$$d_1(f, g) = \int_0^1 |f(t) - g(t)|\, dt.$$

现在收敛变弱了。Cauchy 列的"极限"可能是不连续的函数，根本不在 $C[0,1]$ 里。空间不完备。

同一个集合，不同的度量，分析性质完全不同。度量的选择至关重要。

## 赋范空间

度量空间没有代数结构。**赋范空间**在向量空间上定义范数 $\|\cdot\|: X \to [0, \infty)$：

$$\|x\| = 0 \iff x = 0, \quad \|\alpha x\| = |\alpha|\,\|x\|, \quad \|x + y\| \le \|x\| + \|y\|.$$

每个范数自然诱导度量 $d(x, y) = \|x - y\|$，但反过来不成立。范数增加了齐次性和平移不变性——几何在每一点看起来都一样。

**$\ell^p$ 空间。** 对 $1 \le p < \infty$：

$$\ell^p = \left\{ (x_n)_{n=1}^\infty : \sum_{n=1}^\infty |x_n|^p < \infty \right\}, \quad \|x\|_p = \left(\sum_{n=1}^\infty |x_n|^p\right)^{1/p}.$$

对 $p = \infty$：

$$\ell^\infty = \left\{ (x_n) : \sup_n |x_n| < \infty \right\}, \quad \|x\|_\infty = \sup_n |x_n|.$$

$\ell^1$ 的单位球有棱角，$\ell^2$ 的单位球是圆的，$\ell^\infty$ 的单位球像方块。有限维里这只是外观差异；无穷维里，它决定了哪些算子有界、哪些泛函连续、哪些定理适用。

**定理（范数不等价）。** 无穷维空间上存在不等价的范数。取 $x^{(n)} = (1, 1, \ldots, 1, 0, 0, \ldots)$（前 $n$ 个分量为 1），则 $\|x^{(n)}\|_1 = n$ 而 $\|x^{(n)}\|_\infty = 1$。不存在常数 $C$ 使得 $\|\cdot\|_1 \le C\|\cdot\|_\infty$ 对所有 $\ell^1$ 中元素成立。

## 完备性与 Banach 空间

每个 Cauchy 列都收敛的赋范空间叫做 **Banach 空间**：

$$\text{Banach 空间} = \text{完备的赋范空间}。$$

完备性不是可选项——它是分析能否进行的门槛。不动点定理、Baire 纲定理、开映射定理，全部要求完备性。

**定理（$\ell^p$ 的完备性）。** 对 $1 \le p \le \infty$，$\ell^p$ 是 Banach 空间。

*证明思路（$p < \infty$）。* 设 $(x^{(k)})$ 在 $\ell^p$ 中 Cauchy。对固定 $n$，数列 $(x^{(k)}_n)_k$ 在 $\mathbb{R}$ 中 Cauchy（因为 $|x^{(k)}_n - x^{(m)}_n| \le \|x^{(k)} - x^{(m)}\|_p$），收敛到某个 $x_n$。定义 $x = (x_n)$。固定 $\varepsilon > 0$，选 $K$ 使得 $k, m \ge K$ 时 $\|x^{(k)} - x^{(m)}\|_p < \varepsilon$。对任意有限 $N$，$\sum_{n=1}^N |x^{(k)}_n - x^{(m)}_n|^p < \varepsilon^p$，令 $m \to \infty$ 得 $\sum_{n=1}^N |x^{(k)}_n - x_n|^p \le \varepsilon^p$。对所有 $N$ 成立，故 $\|x^{(k)} - x\|_p \le \varepsilon$。$\square$

## 一个重要的反例

$c_{00}$（有限支撑序列空间）配 $\ell^2$ 范数。这是赋范空间但不是 Banach 空间：序列 $x^{(n)} = (1, 1/2, 1/3, \ldots, 1/n, 0, 0, \ldots)$ 是 Cauchy 的，但极限 $(1, 1/2, 1/3, \ldots)$ 不在 $c_{00}$ 中。

每个赋范空间都有**完备化**——包含它作为稠密子空间的最小 Banach 空间。$c_{00}$ 在 $\|\cdot\|_2$ 下的完备化就是 $\ell^2$。这和 $\mathbb{Q}$ 完备化得到 $\mathbb{R}$ 是完全类比的。

## Baire 纲定理：完备性的第一个回报

**定理（Baire）。** 完备度量空间不能写成可数个无处稠密集的并。

等价地说：可数个稠密开集的交仍然稠密。听起来抽象，但它是后面三个大定理的动力引擎。

**应用。** $C[0,1]$ 中"大多数"连续函数（在 Baire 纲意义下）处处不可微。可微函数构成一个第一纲集——拓扑意义上的"极少数"。
