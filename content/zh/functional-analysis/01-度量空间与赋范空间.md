## 为什么需要无穷维分析

在 $\mathbb{R}^n$ 里，线性代数就够用了：所有范数等价，所有线性映射连续，闭有界集都是紧的。但函数空间是无穷维的，这些好性质全部失效。泛函分析的核心任务就是在无穷维空间里重新建立分析的基础设施。

我想把失效的原因说清楚。$\mathbb{R}^n$ 中 Heine-Borel 定理告诉我们闭有界集是紧的。但在任何无穷维赋范空间中，闭单位球永远不是紧的。这个失效会层层传导——本科分析中大量定理暗中依赖紧性，一旦紧性消失，你需要全新的工具。

起点很朴素：给集合配一个距离，看看能推出什么。我们构建的层级——度量空间、赋范空间、Banach 空间——代表了结构递增的环境，越高层能支撑越强的定理。

## 度量空间

**度量空间** $(X, d)$ 是集合 $X$ 配上满足三条公理的距离函数 $d: X \times X \to [0, \infty)$：

$$d(x, y) = 0 \iff x = y, \quad d(x, y) = d(y, x), \quad d(x, z) \le d(x, y) + d(y, z).$$

三角不等式是核心——它保证了开球确实是开集，极限唯一，拓扑行为良好。没有它的话，"距离"可能出现绕道比直达更短的荒谬情况。

**定义（收敛）。** $(X, d)$ 中序列 $(x_n)$ 收敛到 $x$，当且仅当 $d(x_n, x) \to 0$。

**定义（Cauchy 列）。** 序列 $(x_n)$ 是 Cauchy 的，如果对任意 $\varepsilon > 0$，存在 $N$ 使得 $n, m \ge N$ 时 $d(x_n, x_m) < \varepsilon$。

每个收敛列都是 Cauchy 的（三角不等式直接给出）。反过来——每个 Cauchy 列都收敛——就是完备性，它是区分能用的空间和病态空间的关键性质。

**例 1：上确界度量。** $C[0,1]$（$[0,1]$ 上连续函数的集合）配距离：

$$d(f, g) = \sup_{t \in [0,1]} |f(t) - g(t)|.$$

这个距离下的收敛就是一致收敛。连续函数的一致极限仍是连续函数——空间是完备的。关键事实是实分析中的定理：连续函数列一致收敛则极限连续。

**例 2：$L^1$ 度量。** 同样是 $C[0,1]$，但换成：

$$d_1(f, g) = \int_0^1 |f(t) - g(t)|\, dt.$$

现在收敛变弱了。取 $f_n(t) = t^n$——这列在 $L^1$ 意义下是 Cauchy 的（$\int_0^1 |t^n - t^m| dt \to 0$），但逐点极限在 $[0,1)$ 上为 0、在 $t=1$ 处为 1，是不连续函数，不在 $C[0,1]$ 中。空间在这个度量下不完备。

同一个集合，不同的度量，分析性质完全不同。度量决定了什么叫"接近"，这决定了一切后续结论。

**例 3（离散度量）。** 在任何集合 $X$ 上定义 $d(x,y) = 1$（$x \ne y$时）和 $d(x,x) = 0$。这满足度量公理，而且总是完备的（Cauchy 列必须最终为常数列）。但它在分析上毫无用处——生成的拓扑是离散拓扑，每个集合都是开集。我提到它只是为了强调：光满足公理不保证有趣的结构。

## 赋范空间

度量空间只有距离，没有代数结构。**赋范空间**在向量空间 $X$ 上定义范数 $\|\cdot\|: X \to [0, \infty)$：

$$\|x\| = 0 \iff x = 0, \quad \|\alpha x\| = |\alpha|\,\|x\|, \quad \|x + y\| \le \|x\| + \|y\|.$$

每个范数自然诱导度量 $d(x, y) = \|x - y\|$，但反过来不成立。范数增加了齐次性和平移不变性——空间的几何在每一点看起来都一样，而且缩放是均匀的。这是真正的限制条件：离散度量就不可能来自任何范数（把向量乘以 $1/2$ 应该让距离减半，但离散度量只取 0 和 1）。

从度量空间到赋范空间的关键收获是：我们可以做加法、数乘，且这些运算与距离兼容。这让我们能谈论线性映射、子空间、商空间——线性代数的工具箱与拓扑的结合。

**$\ell^p$ 空间。** 对 $1 \le p < \infty$：

$$\ell^p = \left\{ (x_n)_{n=1}^\infty : \sum_{n=1}^\infty |x_n|^p < \infty \right\}, \quad \|x\|_p = \left(\sum_{n=1}^\infty |x_n|^p\right)^{1/p}.$$

对 $p = \infty$：

$$\ell^\infty = \left\{ (x_n) : \sup_n |x_n| < \infty \right\}, \quad \|x\|_\infty = \sup_n |x_n|.$$

证明这些确实是范数需要一些功夫。$\ell^p$ 的三角不等式就是 **Minkowski 不等式**，它需要 Holder 不等式作为前置：

$$\|x + y\|_p \le \|x\|_p + \|y\|_p.$$

这些空间有截然不同的几何。$\ell^1$ 的单位球有棱角（$\sum |x_n| \le 1$ 定义的集合）。$\ell^2$ 的单位球是圆的（$\sum x_n^2 \le 1$）。$\ell^\infty$ 的单位球像方块（$\sup |x_n| \le 1$）。有限维里这只是外观差异；无穷维里，它决定了哪些算子有界、哪些泛函连续、哪些定理适用。

**定理（无穷维空间范数不等价）。** $\ell^1$ 上的 $\|\cdot\|_1$ 与 $\|\cdot\|_\infty$ 不等价。

*证明。* 取 $x^{(n)} = (1, 1, \ldots, 1, 0, 0, \ldots)$（前 $n$ 个分量为 1）。则 $\|x^{(n)}\|_1 = n$ 而 $\|x^{(n)}\|_\infty = 1$。若范数等价，则存在 $C > 0$ 使 $\|x\|_1 \le C\|x\|_\infty$ 对所有 $x \in \ell^1$ 成立。但 $n = \|x^{(n)}\|_1 \le C \cdot 1 = C$ 对所有 $n$ 成立，矛盾。$\square$

有限维中所有范数等价（通过单位球面上的紧性论证）。无穷维的失效意味着范数的选择有真正的分析后果——不是方便与否的问题。

## 完备性与 Banach 空间

每个 Cauchy 列都收敛的赋范空间叫做 **Banach 空间**：

$$\text{Banach 空间} = \text{完备的赋范空间}.$$

完备性不是可选项——它是分析能否进行的分水岭。不动点定理、Baire 纲定理、开映射定理，全部要求完备性。没有完备性，你不能随意取极限，整个理论会崩塌。

**定理（$\ell^p$ 的完备性）。** 对 $1 \le p \le \infty$，$\ell^p$ 是 Banach 空间。

*证明（$p < \infty$）。* 设 $(x^{(k)})$ 在 $\ell^p$ 中 Cauchy。对固定 $n$，$(x^{(k)}_n)_k$ 在 $\mathbb{R}$ 中 Cauchy（因为 $|x^{(k)}_n - x^{(m)}_n| \le \|x^{(k)} - x^{(m)}\|_p$），收敛到某个 $x_n$。定义 $x = (x_n)$。

要证 $x \in \ell^p$ 且 $\|x^{(k)} - x\|_p \to 0$。固定 $\varepsilon > 0$，取 $K$ 使 $k, m \ge K$ 时 $\|x^{(k)} - x^{(m)}\|_p < \varepsilon$。对任意有限 $N$：

$$\sum_{n=1}^N |x^{(k)}_n - x^{(m)}_n|^p \le \|x^{(k)} - x^{(m)}\|_p^p < \varepsilon^p.$$

令 $m \to \infty$：$\sum_{n=1}^N |x^{(k)}_n - x_n|^p \le \varepsilon^p$。对所有 $N$ 成立，故 $\|x^{(k)} - x\|_p \le \varepsilon$（$k \ge K$）。特别地，$x = x^{(K)} + (x - x^{(K)})$ 是两个 $\ell^p$ 元素之和，所以 $x \in \ell^p$。$\square$

**例 4。** $C[0,1]$ 配上确界范数是 Banach 空间（连续函数一致收敛则极限连续）。配 $L^1$ 范数则不完备——其完备化是 $L^1[0,1]$。

**例 5（重要的函数空间）。** 对 $1 \le p \le \infty$ 和可测 $\Omega \subseteq \mathbb{R}^n$，$L^p(\Omega)$ 是 Banach 空间。证明用 Riesz-Fischer 定理，结构类似 $\ell^p$ 的证明。这些空间是 PDE 理论的主力。

## 一个重要的反例：完备化

$c_{00}$（有限支撑序列空间）配 $\ell^2$ 范数。这是赋范空间但不是 Banach 空间：序列 $x^{(n)} = (1, 1/2, 1/3, \ldots, 1/n, 0, 0, \ldots)$ 是 Cauchy 的（可验证 $\|x^{(n)} - x^{(m)}\|_2^2 = \sum_{k=m+1}^n 1/k^2 \to 0$），但极限 $(1, 1/2, 1/3, \ldots) \notin c_{00}$（无穷多非零项）。

每个赋范空间都有**完备化**——包含它作为稠密子空间的最小 Banach 空间。$c_{00}$ 在 $\|\cdot\|_2$ 下的完备化是 $\ell^2$。在 $\|\cdot\|_\infty$ 下的完备化是 $c_0$（收敛到 0 的序列空间）。不同范数，不同完备化，不同的 Banach 空间。

这和 $\mathbb{Q}$ 完备化得到 $\mathbb{R}$ 完全类比——两种情况都是在补充"缺失的极限"。

**定理（完备化的存在性）。** 每个赋范空间 $(X, \|\cdot\|)$ 都有完备化：存在 Banach 空间 $\hat{X}$ 和等距线性嵌入 $J: X \hookrightarrow \hat{X}$，使得 $J(X)$ 在 $\hat{X}$ 中稠密。完备化在等距同构意义下唯一。

## Baire 纲定理：完备性的第一个回报

**定理（Baire）。** 完备度量空间中，可数个稠密开集的交仍然稠密。

等价表述：完备度量空间不能写成可数个无处稠密集的并。如果一个空间是可数个无处稠密集之并，它叫**第一纲集**（meager）。Baire 定理说完备度量空间永远不是第一纲的——它们"太大"了，无法这样分解。

*证明思路。* 设 $(U_n)$ 是稠密开集。给定非空开集 $V$，要在 $V \cap \bigcap U_n$ 中找到点。$U_1$ 稠密，所以 $V \cap U_1$ 非空且开。取闭球 $\overline{B_1} \subseteq V \cap U_1$，半径 $\le 1$。$U_2$ 稠密，取 $\overline{B_2} \subseteq B_1 \cap U_2$，半径 $\le 1/2$。如此下去：得到嵌套闭球，半径趋于 0。它们的中心构成 Cauchy 列。完备性给出极限点，属于所有 $U_n$ 和 $V$。$\square$

听起来抽象，但这是第 4 篇三个大定理的动力引擎。Baire 纲定理把完备性转化为组合工具。

**应用 1。** $C[0,1]$ 中"大多数"连续函数（Baire 纲意义下）处处不可微。可微函数构成第一纲集——拓扑上的"极少数"。Weierstrass 给出了第一个显式例子；Baire 定理无需构造就证明了存在性。

**应用 2。** 可以把 Baire 论证形式化为二人博弈（Banach-Mazur game）：A 试图把交集困在无处稠密集中，B 逃脱。完备性给 B 必胜策略。

## 等价范数与同构

两个范数 $\|\cdot\|_a$ 和 $\|\cdot\|_b$ 在空间 $X$ 上**等价**，如果存在常数 $c, C > 0$ 使得：

$$c\|x\|_a \le \|x\|_b \le C\|x\|_a \quad \forall x \in X.$$

等价范数给出相同拓扑：相同开集、相同收敛列、相同 Cauchy 列、相同完备性。有限维中所有范数等价；无穷维中不然——而且失效方式很重要。

**例 6。** $C^1[0,1]$ 上，$\|f\|_\infty = \sup |f|$ 和 $\|f\|_{C^1} = \sup |f| + \sup |f'|$ 不等价。空间在 $\|\cdot\|_{C^1}$ 下是 Banach 的，在 $\|\cdot\|_\infty$ 下不完备（$C^1$ 函数列一致收敛的极限不一定是 $C^1$ 的）。范数的选择决定空间是否完备。
