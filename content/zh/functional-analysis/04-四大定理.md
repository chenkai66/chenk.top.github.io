## 理论的脊梁

泛函分析有四个基础定理。其中三个（开映射、闭图、一致有界性）使用 Baire 纲定理，要求完备性。第四个（Hahn-Banach）是代数性的，不需要完备性但需要选择公理（或 Zorn 引理）。它们合起来构成结构骨架——几乎每个深层结果都能追溯到这四个之一。

统一的主题是：把定性信息转化为定量信息。"每个点有界"变成"有一致界"。"算子满射"变成"算子是开映射"。"图是闭的"变成"算子有界"。这种炼金术——从软假设得到硬结论——就是完备性赋予你的能力。

## Hahn-Banach 定理

**定理（Hahn-Banach，延拓形式）。** 设 $X$ 是赋范空间，$M \subseteq X$ 是子空间，$f: M \to \mathbb{R}$ 是有界线性泛函，$\|f\|_M = C$。则存在延拓 $F: X \to \mathbb{R}$，$F|_M = f$ 且 $\|F\|_X = C$。

范数精确保持——不增不减。这是惊人之处。延拓线性泛函很容易（用 Hamel 基），保范延拓需要 Zorn 引理的全部力量。

*证明。* 分两步。第一步，延拓一维：若 $x_0 \notin M$，要在 $M \oplus \mathbb{R}x_0$ 上定义 $F(m + tx_0) = f(m) + tc$。约束 $|F(m + tx_0)| \le C\|m + tx_0\|$ 对所有 $m \in M$，$t \in \mathbb{R}$ 必须成立。$t > 0$ 时：$c \le C\|m/t + x_0\| - f(m/t)$。$t < 0$ 时：$c \ge -C\|m/|t| - x_0\| + f(m/|t|)$。上界的下确界不小于下界的上确界（这里用到 $\|f\|_M \le C$），所以合法的 $c$ 存在。

第二步，超限归纳。考虑所有保持范数界 $C$ 的 $f$ 的延拓，按延拓关系偏序。每条链有上界（取定义域的并）。Zorn 引理给出极大延拓 $F$。若定义域是真子空间，还能延拓一维——矛盾。故定义域是全 $X$。$\square$

复数版从实数版推出：利用 $\text{Im}\,f(x) = -\text{Re}\,f(ix)$，复泛函由实部决定，延拓实部保持复范数。

**推论 1（赋范泛函）。** 对赋范空间中任何 $x \ne 0$，存在 $f \in X^*$ 满足 $f(x) = \|x\|$ 且 $\|f\| = 1$。

*证明。* 在一维子空间 $\mathbb{C}x$ 上定义 $g(\alpha x) = \alpha\|x\|$，$\|g\| = 1$。Hahn-Banach 延拓。$\square$

这说对偶空间足够丰富，能"看见"每个非零向量。也给出 $\|x\| = \sup_{\|f\| \le 1} |f(x)|$。

**推论 2（典范嵌入）。** 映射 $J: X \to X^{**}$，$(Jx)(f) = f(x)$ 是等距嵌入。若 $J$ 满射，$X$ 称为**自反**的。$\ell^p$、$L^p$（$1 < p < \infty$）自反；$\ell^1$、$\ell^\infty$、$L^1$、$c_0$ 不自反。

**推论 3（分离子空间）。** 若 $M$ 是闭子空间，$x_0 \notin M$，则存在 $f \in X^*$，$f|_M = 0$ 且 $f(x_0) \ne 0$。

**例 1（Banach 极限）。** 在 $\ell^\infty$ 上，收敛序列的子空间有泛函 $\text{LIM}(x) = \lim x_n$。Hahn-Banach 延拓到全 $\ell^\infty$：存在"广义极限" $L: \ell^\infty \to \mathbb{R}$，线性、正、平移不变，且延拓普通极限。Banach 极限给每个有界序列赋一个值，与普通极限一致（普通极限存在时）。

## 一致有界性原理（Banach-Steinhaus）

**定理。** 设 $X$ 是 Banach 空间，$Y$ 是赋范空间，$\{T_\alpha\}_{\alpha \in A} \subseteq B(X, Y)$ 是有界算子族。若 $\sup_\alpha \|T_\alpha x\| < \infty$ 对每个 $x \in X$ 成立，则 $\sup_\alpha \|T_\alpha\| < \infty$。

逐点有界蕴含一致有界。每个向量分别给出依赖向量的有限界，但不知怎的合谋产生了不依赖向量的统一界。这应该让人惊讶——它真的是一个定理，不是同义反复。

*证明（Baire）。* 定义 $F_n = \{x \in X : \sup_\alpha \|T_\alpha x\| \le n\}$。每个 $F_n$ 闭（是 $\bigcap_\alpha \{x: \|T_\alpha x\| \le n\}$，由 $T_\alpha$ 连续性闭）。由假设 $X = \bigcup F_n$。$X$ 是 Banach（Baire 空间），某个 $F_N$ 有非空内部：存在 $x_0$ 和 $r > 0$ 使 $B(x_0, r) \subseteq F_N$。

对 $\|x\| \le 1$：$x_0 + rx \in F_N$，所以 $\sup_\alpha \|T_\alpha(x_0 + rx)\| \le N$。三角不等式：

$$\sup_\alpha \|T_\alpha(rx)\| \le N + \sup_\alpha \|T_\alpha x_0\| \le N + N = 2N.$$

故对所有 $\|x\| \le 1$：$\sup_\alpha \|T_\alpha x\| \le 2N/r$，给出 $\sup_\alpha \|T_\alpha\| \le 2N/r$。$\square$

**逆否命题（奇性凝聚）。** 若 $\sup_\alpha \|T_\alpha\| = \infty$，则 $\{x : \sup_\alpha \|T_\alpha x\| = \infty\}$ 是稠密 $G_\delta$ 集——不仅非空，而且"大多数"点都爆破。

**例 2（Fourier 级数的发散）。** $C[-\pi, \pi]$（上确界范数）上部分和算子 $S_n f = \sum_{k=-n}^n \hat{f}(k) e^{ikt}$ 满足 $\|S_n\| = L_n$（Lebesgue 常数），$L_n \sim (4/\pi^2)\log n \to \infty$。Banach-Steinhaus 告诉我们：必存在 $f \in C[-\pi, \pi]$ 使其 Fourier 级数在某点发散。实际上这样的 $f$ 构成稠密 $G_\delta$——"大多数"连续函数的 Fourier 级数都会在某点发散。这是 du Bois-Reymond 定理，无需构造显式反例。

**例 3（逐点收敛蕴含一致有界）。** 若 $T_n \to T$ 逐点（即对所有 $x$ 有 $T_n x \to Tx$），则 $\sup_n \|T_n\| < \infty$。这是 Banach-Steinhaus 的直接推论。

## 开映射定理

**定理（Banach，Schauder）。** 设 $X, Y$ 是 Banach 空间，$T: X \to Y$ 是满射有界线性算子。则 $T$ 是开映射：对每个开集 $U \subseteq X$，像 $T(U)$ 在 $Y$ 中是开的。

**推论（有界逆定理）。** 若 $T: X \to Y$ 是 Banach 空间间的双射有界线性算子，则 $T^{-1}$ 自动有界。逆的连续性免费获得。

*证明。* 只需证 $T(B_X(0,1))$ 包含 $Y$ 中原点的某个球。

第一步（Baire）：$T$ 满射，所以 $Y = \bigcup_n T(B_X(0, n))$。Baire 给出 $\overline{T(B_X(0,1))}$ 有非空内部，对称性和平移给 $\overline{T(B_X(0,1))} \supseteq B_Y(0, 2\delta)$。

第二步：证 $T(B_X(0,2)) \supseteq B_Y(0, \delta)$。给定 $\|y\| < \delta$：$y \in \overline{T(B_X(0,1))}$，找 $x_1$，$\|x_1\| < 1$，$\|y - Tx_1\| < \delta/2$。然后 $y - Tx_1 \in \overline{T(B_X(0,1/2))}$，找 $x_2$，$\|x_2\| < 1/2$，$\|y - Tx_1 - Tx_2\| < \delta/4$。继续：得 $(x_n)$，$\|x_n\| < 2^{1-n}$。和 $x = \sum x_n$ 在 Banach 空间 $X$ 中绝对收敛（总范数 $< 2$），$Tx = y$。$\square$

**例 4。** 恒等映射 $I: (C[0,1], \|\cdot\|_\infty) \to (C[0,1], \|\cdot\|_1)$ 有界且双射，但逆无界（取 $f_n(t) = t^n$：$\|f_n\|_\infty = 1$ 但 $\|f_n\|_1 = 1/(n+1) \to 0$）。不矛盾：$(C[0,1], \|\cdot\|_1)$ 不是 Banach 的。两个空间的完备性都是本质的。

**应用。** 若同一向量空间上两个 Banach 范数满足 $\|x\|_1 \le C\|x\|_2$，则范数等价（$\|x\|_2 \le C'\|x\|_1$ 自动成立）。"完备性 + 连续嵌入 = 等价"——强大的自动正则性。

## 闭图定理

**定理。** 设 $X, Y$ 是 Banach 空间，$T: X \to Y$ 线性。若图 $\Gamma(T) = \{(x, Tx) : x \in X\}$ 在 $X \times Y$ 中是闭的，则 $T$ 有界。

闭图意味着：$x_n \to x$ 且 $Tx_n \to y$ 时，必有 $y = Tx$。这弱于连续性（连续要求 $x_n \to x$ 就推出 $Tx_n \to Tx$，不用预设 $Tx_n$ 的极限存在）。

*证明。* 图 $\Gamma(T)$ 是 Banach 空间 $X \times Y$ 的闭线性子空间，自身是 Banach 的。投影 $\pi_1: \Gamma(T) \to X$，$(x, Tx) \mapsto x$ 有界且双射。开映射推论给出 $\pi_1^{-1}: X \to \Gamma(T)$ 有界。$T = \pi_2 \circ \pi_1^{-1}$（$\pi_2$ 是到 $Y$ 的有界投影），$T$ 有界。$\square$

**例 5（微分算子）。** $D: C^1[0,1] \to C[0,1]$，$Df = f'$，$C^1[0,1]$ 配上确界范数。图是闭的：$f_n \to f$ 一致且 $f_n' \to g$ 一致，则 $f' = g$（微积分标准定理）。但 $D$ 无界（$\|t^n\|_\infty = 1$ 而 $\|(t^n)'\|_\infty = n$）。不矛盾：$C^1[0,1]$ 配上确界范数不是 Banach 空间。定理要求定义域完备。

**例 6（实践中的闭图）。** PDE 理论中许多算子通过验证闭图条件来证明有界性，而不是直接估计范数。定理把序列条件（容易验证）转化为范数估计（真正需要的）。

## 四大定理的关系

它们不是独立的：

- **闭图 $\Leftrightarrow$ 开映射**：互相推出。$T$ 的开映射等价于 $T^{-1}$ 的闭图。同一枚硬币的两面。
- **两者需要 Baire**：开映射和闭图要求定义域和值域都完备（通过 Baire 纲定理）。
- **一致有界性只需定义域完备**：值域 $Y$ 不需要是 Banach 的。
- **Hahn-Banach 不同**：不需要完备性，用 Zorn 引理。代数/几何性质，而非拓扑性质。

模式：**完备性把定性信息转化为定量信息。** 逐点界 $\to$ 一致界（Banach-Steinhaus）。满射 $\to$ 开性（开映射）。图闭 $\to$ 有界（闭图）。每次都是"软"假设产生"硬"结论，引擎是 Baire。

**定理（奇性凝聚原理）。** 若 $\{T_n\} \subseteq B(X,Y)$，$\|T_n\| \to \infty$，则 $\{x \in X : \sup_n \|T_n x\| = \infty\}$ 是 $X$ 中稠密 $G_\delta$ 集。爆破是"大多数"的行为，不是例外。
