## 困扰工程师的那个问题

方程没错，算法也没错，为什么算出来的结果完全不对？

问题通常出在一个叫**条件数**的数字上。它用来衡量线性系统的“敏感度”——输入的一点小变化会不会被放大成输出的大错误。要理解条件数，首先得知道怎么衡量向量和矩阵的“大小”，这就是范数的作用。

## 向量范数：衡量“大小”

### 什么是范数？

向量空间上的**范数** $\|\cdot\|$ 是满足以下三条规则的函数：

1. **非负性。** $\|\vec{x}\| \geq 0$，且只有当 $\vec{x} = \vec{0}$ 时等号成立。
2. **齐次性。** $\|c\vec{x}\| = |c|\,\|\vec{x}\|$。
3. **三角不等式。** $\|\vec{x} + \vec{y}\| \leq \|\vec{x}\| + \|\vec{y}\|$。

这三条规则和我们日常对“大小”的理解完全一致：任何东西的大小都不可能是负数；放大两倍，尺寸也变两倍；绕远路永远比直线距离长。

### 必须掌握的三种范数

对于 $\vec{x} \in \mathbb{R}^n$：

**$L^1$ 范数 —— 曼哈顿距离。**  
$\|\vec{x}\|_1 = |x_1| + |x_2| + \cdots + |x_n|$。  
想象一辆出租车在网格状街道上行驶，从原点到 $(3, 4)$ 需要走 $3 + 4 = 7$ 个街区。

**$L^2$ 范数 —— 欧几里得距离。**  
$\|\vec{x}\|_2 = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}$。  
这是直线距离，比如乌鸦从原点飞到 $(3, 4)$ 的距离是 $\sqrt{9+16}=5$。

**$L^\infty$ 范数 —— 切比雪夫距离。**  
$\|\vec{x}\|_\infty = \max_i |x_i|$。  
国际象棋中的国王每次可以朝八个方向移动一步。从 $(0,0)$ 到 $(3, 4)$ 最少需要 $\max(3,4) = 4$ 步——因为国王可以同时沿横竖和斜线移动。

### 单位球的几何形状

每种范数都定义了一个“单位球” $\{\vec{x} : \|\vec{x}\| \leq 1\}$：

- **$L^1$：** 二维是菱形，三维是八面体——尖角正好落在坐标轴上。
- **$L^2$：** 二维是圆，三维是球——完全旋转对称。
- **$L^\infty$：** 二维是正方形，三维是立方体——平面与坐标轴平行。

随着 $p$ 从 $1$ 增大到 $\infty$，单位球从菱形逐渐变成圆形，再扩展成正方形。$L^1$ 球的尖角正是 LASSO 回归能产生稀疏解的原因：损失函数的等高线往往最先碰到 $L^1$ 球的尖角，而这些尖角位于坐标轴上——意味着某些坐标会被强制为零。

![向量范数：单位球的形状随 p 变化](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig1_norm_unit_balls.png)

### 范数等价定理

**定理。** 在有限维空间中，所有范数都是**等价**的：存在常数 $c, C > 0$，使得  
$$c\|\vec{x}\|_a \leq \|\vec{x}\|_b \leq C\|\vec{x}\|_a.$$
不同范数就像不同的尺子，测量的是同一个向量。如果一个序列在某个范数下收敛，那么它在其他范数下也会收敛。选择哪种范数主要取决于物理意义或计算方便——但一旦涉及收敛**速度**，范数的选择就变得重要了。
## 矩阵范数：这个变换有多“猛”？

### Frobenius 范数：把矩阵拉平成一个长向量

对于 $m \times n$ 的矩阵 $A$， Frobenius 范数定义为：$$\|A\|_F = \sqrt{\sum_{i,j} a_{ij}^2}.$$简单来说，就是把矩阵的所有元素排成一列，计算它们的 $L^2$ 范数。

**图像类比**  
可以把矩阵 $A$ 想象成一张灰度图，每个元素代表一个像素的亮度。 Frobenius 范数就是这张图的“总能量”。无论是衡量图像压缩误差，还是神经网络权重更新的幅度，它都是工业界的默认选择。

**与奇异值的关系**  $$\|A\|_F = \sqrt{\sigma_1^2 + \sigma_2^2 + \cdots + \sigma_r^2}.$$换句话说， Frobenius 范数是所有奇异值的平方和再开根号——它关心的是矩阵在每个方向上的拉伸。

### 诱导范数：最大放大倍数

诱导范数（也叫算子范数）衡量的是矩阵 $A$ 对单位向量的最大拉伸倍数：$$\|A\| = \max_{\|\vec{x}\|=1} \|A\vec{x}\|.$$
可以把 $A$ 想象成一个放大镜，诱导范数问的就是：这个放大镜最多能把东西放大多少倍？如果 $\|A\|_2 = 3$，说明存在某个输入方向，经过 $A$ 变换后长度变成了原来的 3 倍。

最常见的三种诱导范数如下：

| 范数 | 公式 | 怎么算 |
|---|---|---|
| $\|A\|_1$（列和范数） | $\max_j \sum_i |a_{ij}|$ | 每列绝对值求和，取最大值 |
| $\|A\|_2$（谱范数） | $\sigma_{\max}(A)$ | 最大奇异值 |
| $\|A\|_\infty$（行和范数） | $\max_i \sum_j |a_{ij}|$ | 每行绝对值求和，取最大值 |

**记忆口诀**： 1 → 列和，$\infty$ → 行和。

### 谱范数：最常用的范数

谱范数 $\|A\|_2 = \sigma_1$ 是最有用的矩阵范数：

- 它等于**最大奇异值**。
- 几何上，矩阵 $A$ 把单位球映射成椭球，$\|A\|_2$ 就是椭球最长半轴的长度。
- 它表示矩阵 $A$ 的**最大放大倍数**。

![算子范数：椭圆最长半轴](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig2_operator_norm.png)

上面两张图讲清楚了整个故事。左边是单位圆和两个极端方向 $v_{\max}$、$v_{\min}$。右边是经过 $A$ 变换后的椭圆——橙色箭头落在最长半轴上，长度是 $\sigma_{\max} = \|A\|_2$；绿色箭头被压缩到长度 $\sigma_{\min}$。

### Frobenius 范数 vs 谱范数：两种“大小观”

这两种范数回答的是同一个矩阵的不同问题。谱范数关注的是**峰值拉伸**，适合用来分析最坏情况。 Frobenius 范数关注的是**总体拉伸**，适合用来评估整体行为。

![Frobenius 范数 vs 谱范数](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig3_frob_vs_spectral.png)

图中那个矩阵，$\|A\|_2 = \sigma_1 \approx 2.78$，而 $\|A\|_F \approx 2.93$。两者接近是因为 $\sigma_2$ 相对较小；如果矩阵的奇异值都差不多大，两者的差距会更明显。

### 次乘性

对于诱导范数，有以下性质：$$\|AB\| \leq \|A\|\,\|B\|.$$两次变换叠加的总放大倍数，不会超过两次单独放大倍数的乘积。这条简单的不等式是分析迭代算法收敛性的核心工具，也是理解深层神经网络梯度爆炸/消失问题的关键。
## 条件数：线性系统的健康报告

### 定义

对于可逆矩阵 $A$，条件数定义为：$$\kappa(A) = \|A\|\,\|A^{-1}\|.$$通常用的是**谱条件数**：$$\kappa_2(A) = \frac{\sigma_{\max}}{\sigma_{\min}}.$$也就是最大奇异值与最小奇异值的比值。

**用过敏来类比。**
- $\kappa \approx 1$：免疫系统正常，吹点风完全没事。
- $\kappa \approx 10^{10}$：严重过敏体质，一粒花粉就能引发休克。

![条件数：良态 vs 病态](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig4_condition_number.png)

良态矩阵会把输入的圆映射成一个接近圆形的椭圆（虚线灰圆和绿色椭圆几乎重合）。病态矩阵则会把圆压成一根针：输入空间中有些方向上，矩阵的作用几乎可以忽略不计。为了求逆，我必须把这些被压扁的方向重新放大，但这样一来，噪声也会随之放大。

### 关键性质

- **下界**：$\kappa(A) \geq 1$。只有当 $A$ 是正交矩阵的标量倍时，等号才成立。
- **正交不变性**：如果 $Q$ 是正交矩阵，那么 $\kappa(QA) = \kappa(A)$。旋转和反射不会影响条件数。
- **缩放不变性**：$\kappa(\alpha A) = \kappa(A)$。决定条件数的是椭圆的形状，而不是大小。
- **求逆对称性**：$\kappa(A^{-1}) = \kappa(A)$。解方程和求逆的难度是一样的。

### 一句话总结几何意义

条件数就是输出椭圆的**扁率**：$$\kappa = \frac{\text{最长半轴}}{\text{最短半轴}}.$$$\kappa = 1$ 表示椭圆是完美的圆；$\kappa = \infty$ 表示椭圆塌缩成一条线段（矩阵不可逆）。

**简单例子**：单位矩阵 $A = I$ 的条件数是 $\kappa = 1$。矩阵 $B = \begin{pmatrix} 1 & 0 \\ 0 & 10^{-5} \end{pmatrix}$ 的条件数是 $\kappa = 10^{5}$，它在某个方向上压缩了 $10^{5}$ 倍。
## 病态矩阵：数值计算的噩梦

![线性代数（十）：矩阵范数与条件数 -- 数值计算的健康体检 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/illustration_2.png)

### Hilbert 矩阵

说到病态矩阵，最典型的例子就是 **Hilbert 矩阵**：$$H_{ij} = \frac{1}{i + j - 1}.$$它的条件数增长速度快得吓人。

| 阶数 $n$ | $\kappa(H_n)$ | 双精度下可信位数 |
|---|---|---|
| 5  | $\sim 10^{5}$  | 大约 10 位 |
| 10 | $\sim 10^{13}$ | 大约 3 位 |
| 12 | $\sim 10^{16}$ | 基本为 0 |
| 15 | $\sim 10^{18}$ | 完全没用 |

```python
import numpy as np
from scipy.linalg import hilbert

n = 12
H = hilbert(n)
x_true = np.ones(n)
b = H @ x_true
x_hat = np.linalg.solve(H, b)

print(f"kappa(H_{n})    = {np.linalg.cond(H):.2e}")
print(f"相对误差       = {np.linalg.norm(x_true - x_hat) / np.linalg.norm(x_true):.2e}")
```

当 $n = 12$ 时，相对误差已经达到 1 的量级。算法本身没错，但结果可能完全错误，误差达到 100%。

### 看看“爆炸”长什么样

![奇异值谱与 Hilbert 矩阵的 κ(n) 增长](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig6_singular_values.png)

左图展示了两个 $20 \times 20$ 矩阵的奇异值分布：一个表现正常，奇异值平稳下降；另一个病态严重，奇异值跨越了六个数量级。右图在对数坐标下绘制了 $\kappa_2(H_n)$ 的变化趋势——几乎是一条直线，每增加两阶，条件数就翻一番。红色虚线是双精度计算的极限 $10^{16}$，一旦超过这条线， IEEE 754 双精度算法也无能为力。

### 病态从哪里来？

- **多项式拟合。** 高次拟合生成的 Vandermonde 矩阵，条件数随阶数指数级增长。
- **细网格。** 有限元网格细化 $h$ 倍，条件数大约变为原来的 $h^{-2}$ 倍。
- **正规方程。** $\kappa(A^TA) = \kappa(A)^2$——把最小二乘问题转成正规方程，条件数直接平方，业内戏称为“数值线性代数的原罪”。
- **接近奇异。** 两行或两列“几乎”线性相关。

## 误差分析：条件数如何放大误差

### 右端项扰动

假设我求解 $A\vec{x} = \vec{b}$，但右端项 $\vec{b}$ 被一个小扰动 $\delta\vec{b}$ 污染了。解会有多大变化？

经典上界是：$$\frac{\|\delta\vec{x}\|}{\|\vec{x}\|} \;\leq\; \kappa(A)\,\frac{\|\delta\vec{b}\|}{\|\vec{b}\|}.$$
条件数就是相对误差的**最大放大倍数**。

具体来看：
- $\kappa = 10$：输入误差 1%，输出误差最多 10%。
- $\kappa = 10^{10}$：输入误差 1%，输出误差可能达到 $10^{8}$%——结果完全不可信。
- $\kappa = 10^{16}$：即使机器精度（约 $10^{-16}$）的舍入误差，也足以让答案彻底崩坏。

这个现象可以通过实验观察到。下图展示了对 $b$ 施加幅度为 2%、方向各异的扰动后，解 $x + \delta x$ 的分布情况。

![扰动敏感性：微小 δb，巨大 δx](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig5_perturbation.png)

当 $\kappa \approx 2$ 时，解的分布是一个围绕真值 $x$ 的小圆环，放大倍数接近 1。而当 $\kappa \approx 200$ 时，同样的 2% 扰动会让解沿着一条细长的对角线滑动——最坏情况下，放大倍数接近 $\kappa$ 本身。

### 矩阵扰动

如果矩阵 $A$ 本身也有扰动 $\delta A$，公式变为：$$\frac{\|\delta\vec{x}\|}{\|\vec{x}\|} \;\leq\; \kappa(A)\!\left(\frac{\|\delta A\|}{\|A\|} + \frac{\|\delta\vec{b}\|}{\|\vec{b}\|}\right).$$
还是同一个条件数，同样的惩罚规则——矩阵元素的误差也会被 $\kappa(A)$ 放大。

### 经验法则：损失多少有效数字

如果 $\kappa(A) \approx 10^k$，用双精度求解 $A\vec{x} = \vec{b}$ 时，大约会损失 $k$ 位有效数字。

| $\kappa(A)$ | 损失位数 | 可靠位数 |
|---|---|---|
| $10^{4}$  | 4  | ≈ 12 |
| $10^{8}$  | 8  | ≈ 8  |
| $10^{12}$ | 12 | ≈ 4  |
| $10^{16}$ | 16 | 0 （完全不可信）|

## 谱半径与迭代收敛

### 定义

**谱半径**是矩阵特征值模的最大值：$$\rho(A) = \max_i |\lambda_i|.$$任何矩阵范数都满足 $\rho(A) \leq \|A\|$。

### 收敛条件

固定点迭代公式 $\vec{x}_{k+1} = B\vec{x}_k + \vec{c}$，从任意初始值出发都能收敛的充要条件是 $\rho(B) < 1$。

**用水温调节来类比。** 如果系统稳定（$\rho < 1$），每次调节都会让水温更接近理想值；如果系统不稳定（$\rho \geq 1$），水温就会在冰凉和滚烫之间来回波动。

**收敛速度。** 每次迭代后误差大约缩小到原来的 $\rho(B)$ 倍。要达到误差 $\epsilon$，大约需要 $\log\epsilon / \log\rho(B)$ 次迭代。$\rho$ 越小，收敛速度越快。

### Neumann 级数

当 $\rho(A) < 1$ 时，$$(I - A)^{-1} = I + A + A^2 + A^3 + \cdots,$$这是标量公式 $\frac{1}{1-x} = 1 + x + x^2 + \cdots$ 的矩阵形式。这个公式在摄动分析和小矩阵逆的近似计算中非常实用。
## 数值稳定性：如何选择合适的算法

### 为什么正规方程不靠谱

最小二乘问题 $\min_{\vec{x}} \|A\vec{x} - \vec{b}\|$ 的教科书解法是通过**正规方程** $A^TA\hat{\vec{x}} = A^T\vec{b}$。但这里有个致命的问题：$$\kappa(A^TA) = \kappa(A)^2.$$如果 $A$ 的条件数是 $10^6$，那么正规方程的条件数就会变成 $10^{12}$——这简直就是自己给自己挖坑。

### QR 分解：稳定的选择

把 $A$ 分解成 $A = QR$，然后求解 $R\hat{\vec{x}} = Q^T\vec{b}$。因为 $Q$ 是正交矩阵，系统的条件数保持在 $\kappa(A)$，不会被平方放大。现代几乎所有的最小二乘实现都默认用这种方法。

### SVD 分解：最稳但最贵

通过伪逆公式 $\hat{\vec{x}} = V\Sigma^+U^T\vec{b}$ 求解。 SVD 不仅能优雅处理秩亏问题，还能直接给出最小范数解，顺带还提供了奇异值作为额外信息。不过代价也不小，计算开销大约是 QR 的 2 到 3 倍。

### 三种方法的实际对比

下图展示了同一个 $50 \times 50$ 系统 $Ax = b$，用三种方法求解时，误差随条件数从 $10^2$ 增加到 $10^{14}$ 的变化情况：

![数值稳定性：三种解法的误差 vs 条件数](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/10-矩阵范数与条件数/fig7_numerical_stability.png)

有三点值得注意。(1) QR 和 SVD 的曲线基本重合，沿着 $\varepsilon_{\text{mach}} \cdot \kappa$ 这条线一路增长。(2) 正规方程的曲线在双对数图上斜率是它们的两倍，这就是 $\kappa^2$ 带来的代价。(3) 当条件数达到 $\kappa \approx 10^8$ 时，正规方程的结果已经完全不可信，而 QR 和 SVD 依然能保证八位有效数字。**数据进来之前，就要选好算法。**
## 预条件：解决病态系统的关键

### 核心思想

解方程 $A\vec{x} = \vec{b}$ 时，$\kappa(A)$ 很大是个麻烦。我找一个容易求逆的矩阵 $M$，然后改解$$M^{-1}A\vec{x} = M^{-1}\vec{b}.$$如果 $M \approx A$，那么 $M^{-1}A \approx I$，新系统的 $\kappa(M^{-1}A) \approx 1$。

**类比一下**。用厨房秤称大象显然不现实——但如果我大概知道大象有多重，只需要用秤测偏差就行。预条件的作用就是把问题从不适合的尺度调整到工具能处理的范围内。

### 常见预条件子

| 预条件子 | $M$ | 优点 | 缺点 |
|---|---|---|---|
| Jacobi | $\text{diag}(A)$ | 实现简单，完全并行 | 效果有限 |
| Gauss–Seidel | $A$ 的下三角部分 | 比 Jacobi 更强 | 不易并行化 |
| 不完全 LU （ILU） | 稀疏近似 $LU$ | 强大且通用 | 需要控制填充元 |
| 不完全 Cholesky （ICC） | 稀疏近似 $LL^T$ | 存储需求减半 | 只适用于对称正定 |

权衡很简单：更强的预条件子减少迭代次数，但每次迭代的成本更高。最优的预条件子通常依赖问题本身的结构（比如 PDE 用几何多重网格，并行计算用域分解等）。
## 应用拾遗

### 有限元分析

在结构力学中，刚度矩阵 $K$ 的条件数满足 $\kappa(K) \propto h^{-2}$，其中 $h$ 是网格尺寸。细化网格可以捕捉更多几何细节，但会让矩阵变得更难求解。如果网格分布不均匀，或者材料属性差异很大（比如钢筋混凝土，钢和混凝土的刚度相差几个数量级），问题会更加棘手。这种情况下，预条件处理几乎是必不可少的。

### 图像去模糊

图像去模糊问题通常表示为 $\vec{y} = B\vec{x} + \text{noise}$，其中模糊算子 $B$ 往往极度病态。直接求逆会放大高频噪声，导致结果不可用。**Tikhonov 正则化**通过优化问题$$\min_{\vec{x}} \|B\vec{x} - \vec{y}\|^2 + \lambda\|\vec{x}\|^2$$来解决这个问题。正则化参数 $\lambda$ 在保真度和稳定性之间找到平衡，实际上将条件数中的 $\sigma_{\min}$ 替换为 $\sqrt{\sigma_{\min}^2 + \lambda}$。即使 $\lambda$ 很小，也能显著改善问题的数值性质。

### 深度学习：梯度消失与爆炸

权重矩阵的谱范数决定了信号的传播特性：
- 如果多层网络中 $\|W\|_2 > 1$，前向信号和反向梯度会指数级增长。
- 如果多层网络中 $\|W\|_2 < 1$，信号会指数级衰减，导致梯度消失。

为了解决这些问题，研究者提出了多种方法，比如批归一化、精心设计的初始化（Xavier、 He）、残差连接和梯度裁剪。这些方法本质上都在努力让网络雅可比矩阵的有效谱范数接近 $1$。

### 机器学习中的正则化

岭回归通过在 $X^TX$ 上加上 $\lambda I$ 来改善数值稳定性：$$\kappa(X^TX + \lambda I) = \frac{\sigma_1^2 + \lambda}{\sigma_n^2 + \lambda}.$$
即使 $\lambda$ 不大，也能让条件数下降几个数量级。类似的思想在优化领域反复出现，比如 Levenberg–Marquardt 方法、信赖域算法和 weight decay。

### 对比最小二乘解法

```python
def compare_methods(cond_target=1e8, n=50, seed=0):
    rng = np.random.default_rng(seed)
    Q1, _ = np.linalg.qr(rng.standard_normal((n, n)))
    Q2, _ = np.linalg.qr(rng.standard_normal((n, n)))
    s = np.logspace(0, -np.log10(cond_target), n)
    A = Q1 @ np.diag(s) @ Q2

    x_true = rng.standard_normal(n)
    b = A @ x_true

    x_normal = np.linalg.solve(A.T @ A, A.T @ b)        # 风险较大的方法
    Q, R = np.linalg.qr(A); x_qr = np.linalg.solve(R, Q.T @ b)
    x_svd = np.linalg.lstsq(A, b, rcond=None)[0]

    print(f"kappa(A)        : {np.linalg.cond(A):.2e}")
    print(f"正规方程误差    : {np.linalg.norm(x_normal - x_true):.2e}")
    print(f"QR 方法误差     : {np.linalg.norm(x_qr - x_true):.2e}")
    print(f"SVD 方法误差    : {np.linalg.norm(x_svd - x_true):.2e}")

compare_methods()
```

## 实用指南

| $\kappa(A)$ | 风险 | 建议 |
|---|---|---|
| $< 10^{4}$ | 低 | 直接用标准方法就行 |
| $10^{4}$–$10^{8}$ | 中 | 检查结果，优先选 QR 或 SVD，别用正规方程 |
| $10^{8}$–$10^{12}$ | 高 | 加正则化，或者用预条件方法 |
| $> 10^{12}$ | 极高 | 别硬刚，退一步重新设计问题 |

## 本章总结

| 概念 | 关键公式 | 理解 |
|---|---|---|
| 向量范数 | $\|\vec{x}\|_p = (\sum |x_i|^p)^{1/p}$ | 这个向量有多大？ |
| Frobenius 范数 | $\|A\|_F = \sqrt{\sum a_{ij}^2}$ | 矩阵的总能量有多大 |
| 谱范数 | $\|A\|_2 = \sigma_{\max}$ | 最大放大能力 |
| 条件数 | $\kappa = \sigma_{\max}/\sigma_{\min}$ | 误差放大的上限 |
| 谱半径 | $\rho(A) = \max|\lambda_i|$ | 控制迭代是否收敛 |
| 正规方程 | $\kappa(A^TA) = \kappa(A)^2$ | 为什么正规方程可能带来问题 |
