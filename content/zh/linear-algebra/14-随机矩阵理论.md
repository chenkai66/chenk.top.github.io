## 1. 第一个惊喜：随机矩阵为何“不随机”

拿一个 $1000 \times 1000$ 的矩阵，用独立的标准高斯分布填满它的元素，对称化后除以 $\sqrt{n}$。计算特征值，画出直方图。你会发现，无论怎么换随机种子，直方图总是呈现出 $[-2, 2]$ 区间上的同一个半圆。换成均匀分布的 $\{-1, +1\}$ 元素，结果还是那个半圆。换成复埃尔米特高斯分布，半圆依然不变。

这并不是什么魔法，而是经典中心极限定理在高维空间中的体现。当你对上百万个弱相关的随机变量做某种“平均”时（一个特征值其实是矩阵元素的复杂平均），微观分布会被抹平，只剩下少数几个粗粒度的统计量——均值、方差和对称性——决定了最终的极限。半圆律就是**谱的中心极限定理**。

在矩阵的世界里，变化的只是极限对象本身：不再是单一线上的高斯分布，而是一整条**特征值密度曲线**，再加上关于**间距**的精确描述和**边缘**处的涨落规律。RMT 就是研究这三者的数学工具。

## 2. 两大模型家族

### 2.1 Wigner 矩阵：对称噪声

一个 **Wigner 矩阵** $\mathbf{W} \in \mathbb{R}^{n\times n}$ 满足以下条件：

- 对角元 $w_{ii}$ 独立同分布，均值为 0，方差为 $\sigma_d^2$；
- 上三角元素 $w_{ij}$（$i < j$）独立同分布，均值为 0，方差为 $\sigma^2$；
- 下三角元素由对称性决定，即 $w_{ji} = w_{ij}$。

如果所有元素都服从高斯分布，这就是 **高斯正交系综**（GOE）。名字中的“正交”来源于它的分布对任何正交矩阵 $\mathbf{O}\mathbf{W}\mathbf{O}^\top$ 的共轭作用保持不变。对于复埃尔米特矩阵，对应的是 **GUE**（酉不变性）；对于四元数矩阵，则是 **GSE**（辛不变性）。GOE、GUE 和 GSE 这三个模型分别对应时间反演对称性是否存在、是否被磁场破坏、或者是否因半整数自旋而破坏——这正是为什么这些矩阵在物理学中反复出现。

**直观理解。** 把 Erdős–Rényi 社交网络的边权重 $w_{ij}$ 想象成随机生成的“亲密度”分数。这个亲密度矩阵的特征值能告诉你它的全局社群结构。如果亲密度完全是随机的，那么这种结构就是我们接下来要研究的普适规律。

### 2.2 Wishart 矩阵：样本协方差

设 $\mathbf{X} \in \mathbb{R}^{n \times p}$ 的元素独立同分布，均值为 0，方差为 1。**Wishart 矩阵**（也叫样本协方差矩阵）定义为：

$$\mathbf{S} \;=\; \frac{1}{n}\,\mathbf{X}^\top \mathbf{X} \;\in\; \mathbb{R}^{p\times p}.$$

当 $n \gg p$ 时，$\mathbf{S}$ 是真实协方差的一个良好估计。但如果 $n$ 和 $p$ 都很大且它们的比值固定，$\mathbf{S}$ 会偏离真实值——偏离的方式是有规律的，可以用 Marchenko–Pastur 定律描述。

**直观理解。** 假设跟踪 $p = 500$ 只股票一年的每日收益率（$n \approx 252$ 个交易日）。你计算出的协方差矩阵有 $125{,}000$ 个自由参数，但只有约 $126{,}000$ 个数据点。结果得到的估计本质上是一个随机矩阵，即使真实协方差是单位阵，特征值也会按照随机矩阵理论（RMT）的规律分布开来。

## 3. Wigner 半圆律

### 3.1 定理陈述

设 $\mathbf{W}$ 是一个 $n \times n$ 的 Wigner 矩阵，非对角元的方差为 $\sigma^2$。构造归一化矩阵：

$$\hat{\mathbf{W}} \;=\; \frac{\mathbf{W}}{\sigma\sqrt{n}}.$$

当 $n \to \infty$ 时，**经验谱分布** $\frac{1}{n}\sum_i \delta_{\lambda_i(\hat{\mathbf{W}})}$ 几乎必然弱收敛到**半圆密度**：

$$f(x) \;=\; \frac{1}{2\pi}\sqrt{4 - x^2}, \qquad x \in [-2, 2],$$

在 $[-2, 2]$ 区间外，$f(x) = 0$。

### 3.2 为什么是半圆？三种视角

**1) 矩方法（严格的数学推导）。** 计算 $m_k = \mathbb{E}[\frac{1}{n}\operatorname{tr}\hat{\mathbf{W}}^k]$。迹展开后对应的是 $n$ 个顶点上长度为 $k$ 的闭合路径，每条路径的权重由对应的高斯矩乘积决定。独立性和零均值条件使得只有**两两配对且非交叉的路径**存活下来——这正是 Catalan 数 $C_{k/2}$ 所计数的结构。而 Catalan 数恰好是半圆分布的偶阶矩，因此极限分布必然是半圆。

**2) Coulomb 气体（物理学家的直觉）。** GOE 特征值的联合分布为：
$$\rho(\lambda_1, \dots, \lambda_n) \;\propto\; \prod_{i<j} |\lambda_i - \lambda_j|\;\exp\!\Big(-\tfrac{n}{4}\sum_i \lambda_i^2\Big).$$
这个公式描述了 $n$ 个带电粒子在直线上受到对数排斥（Vandermonde 因子）和谐振子势约束的平衡分布。排斥力与约束力的平衡结果就是半圆分布。

**3) 自由 CLT（代数视角）。** 对称随机矩阵可以看作大量“自由”的秩 1 扰动之和。在自由概率中，“独立变量求和”的对应概念是**自由加法卷积**，它的中心极限定理给出的结果是半圆分布，而不是高斯分布。第 7 节会详细讨论这一点。

### 3.3 为什么底层分布无关紧要？

半圆律具有**普适性**：无论矩阵元素的分布是高斯、均匀分布、$\pm 1$，还是任何均值为 0 且方差有限的分布，最终的极限分布都是同一个半圆。这与经典中心极限定理类似——归一化后的和总是收敛到高斯分布，与具体分布无关。

背后的机制也是一样的：归一化操作只保留了二阶矩，其他细节都被平均掉了。

### 3.4 代码验证

```python
import numpy as np
import matplotlib.pyplot as plt

n, repeats = 1500, 30
all_eigs = []
for _ in range(repeats):
    a = np.random.randn(n, n)
    w = (a + a.T) / np.sqrt(2 * n)        # 方差归一化：标准半圆
    all_eigs.append(np.linalg.eigvalsh(w))
all_eigs = np.concatenate(all_eigs)

x = np.linspace(-2, 2, 600)
plt.hist(all_eigs, bins=90, density=True, alpha=0.55, label="Empirical")
plt.plot(x, np.sqrt(np.maximum(4 - x**2, 0)) / (2 * np.pi),
         lw=2.5, label="Semicircle")
plt.legend(); plt.show()
```

实际上，当 $n=200$ 时，模拟结果已经与理论半圆分布几乎完全吻合。

## 4. Marchenko-Pastur 定律

![线性代数（十四）：随机矩阵理论 -- 混沌中的秩序 — visual](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/14-随机矩阵理论/illustration_2.png)

### 4.1 定义

假设 $\mathbf{X}$ 是一个 $n \times p$ 的矩阵，其元素独立同分布，均值为 $0$，方差为 $1$。定义 $\gamma = p/n$（**纵横比**）。当 $n, p \to \infty$ 且 $\gamma$ 固定时，$\mathbf{S} = \frac{1}{n}\mathbf{X}^\top\mathbf{X}$ 的经验谱分布会收敛到 **Marchenko-Pastur 密度**：

$$f(\lambda) \;=\; \frac{1}{2\pi\gamma\,\lambda}\sqrt{(\lambda_+ - \lambda)(\lambda - \lambda_-)},\qquad \lambda \in [\lambda_-, \lambda_+],$$

其中边界点为：

$$\lambda_\pm \;=\; (1 \pm \sqrt{\gamma})^2.$$

如果 $\gamma > 1$，矩阵的秩为 $n < p$，除了区间 $[\lambda_-, \lambda_+]$ 内的特征值外，还会出现 $p - n$ 个精确为零的特征值。

### 4.2 这条密度的意义

即使总体协方差矩阵是单位阵，有限样本也会让特征值分散开来：

- $\gamma = 0.1$：边界为 $[0.47, 1.69]$——扩展幅度较小。
- $\gamma = 0.5$：边界为 $[0.09, 2.91]$——最大特征值是理论值的**三倍**。
- $\gamma = 1.0$：边界为 $[0, 4]$——谱范围触及零，矩阵接近奇异状态。

这条密度曲线是随机矩阵理论（RMT）在统计学中所有定量应用的基础：任何严格大于 $\lambda_+$ 的样本特征值都被认为是**显著的**；而落在 $[\lambda_-, \lambda_+]$ 区间内的特征值则与纯噪声一致。

![不同纵横比下的 Marchenko-Pastur 密度：随着 $\gamma$ 增大，谱范围变宽；当 $\gamma=1$ 时，谱范围触及零](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/14-随机矩阵理论/fig2_marchenko_pastur.png)

### 4.3 代码

```python
import numpy as np, matplotlib.pyplot as plt

n, p, repeats = 1000, 500, 30
gamma = p / n
all_eigs = []
for _ in range(repeats):
    x = np.random.randn(n, p)
    all_eigs.append(np.linalg.eigvalsh(x.T @ x / n))
all_eigs = np.concatenate(all_eigs)

lam_minus, lam_plus = (1 - np.sqrt(gamma))**2, (1 + np.sqrt(gamma))**2
xs = np.linspace(lam_minus + 1e-3, lam_plus - 1e-3, 500)
mp = np.sqrt((lam_plus - xs) * (xs - lam_minus)) / (2 * np.pi * gamma * xs)

plt.hist(all_eigs, bins=100, density=True, alpha=0.5, label="Empirical")
plt.plot(xs, mp, "r", lw=2, label="MP theory")
for e in (lam_minus, lam_plus):
    plt.axvline(e, ls="--", color="g")
plt.legend(); plt.show()
```
## 5. 精细结构：排斥与边缘

半圆分布和 MP 分布描述的是**整体**——宏观形状。随机矩阵理论（RMT）在两个更精细的尺度上也给出了精确答案。

### 5.1 间距分布：特征值互相排斥

选一个主体区域内的普通点，观察它到最近邻特征值的距离分布，并用局部平均间距归一化。对于 GOE 矩阵，这个间距分布可以用 **Wigner 猜测**很好地近似：

$$p(s) \;=\; \frac{\pi s}{2}\,\exp\!\Big(-\frac{\pi s^2}{4}\Big).$$

关键点是 $p(0)=0$：特征值不会重叠。它们主动**排斥**，对于 GOE，在 $s=0$ 处以二次方式趋零（GUE 是三次——不同对称类有不同的排斥指数）。

再看看**独立**能级的情况：它们的间距分布是指数形式 $p(s)=e^{-s}$，在 $s=0$ 处达到**最大值**。独立性导致的聚集现象，正是特征值坚决避免的。

![特征值间距 vs 泊松：GOE 直方图在零间距处归零，泊松曲线却在那里取最大值](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/14-随机矩阵理论/fig3_eigenvalue_repulsion.png)

### 5.2 边缘：Tracy-Widom 分布

对于 $n\times n$ 的 GOE 矩阵，最大特征值 $\lambda_{\max}$ 在大 $n$ 时接近 $2$，但它的波动非常小——量级是 $n^{-2/3}$，而不是直觉上的 $n^{-1/2}$。具体来说，

$$n^{2/3}\big(\lambda_{\max}-2\big) \;\xrightarrow{d}\; \mathrm{TW}_1,$$

其中 $\mathrm{TW}_1$ 是 $\beta=1$ 的 **Tracy-Widom 分布**。这个分布高度不对称：左尾超指数衰减（几乎看不到 $\lambda_{\max}$ 明显小于 2），右尾按 $\exp(-\frac{2}{3}t^{3/2})$ 衰减（极端大值会出现，但很稀有）。同一个 Tracy-Widom 分布还出现在随机置换的最长上升子序列、晶体生长的高度涨落，以及现代统计中大随机矩阵的最大奇异值。

### 5.3 随机谱 vs 确定谱

一张图就能说明一切：随机 Wigner 谱是一个**平滑且强相关**的整体；而确定谱则是**几个尖峰簇**，每个簇内部只有独立的采样噪声。

## 6. 应用

### 6.1 无线通信：MIMO 容量随天线数量线性增长

一个 MIMO 信道有 $n_t$ 根发射天线和 $n_r$ 根接收天线，其容量公式为：

$$C \;=\; \sum_i \log_2\!\Big(1 + \frac{\mathrm{SNR}}{n_t}\,\lambda_i\Big),$$

其中 $\lambda_i$ 是矩阵 $\mathbf{H}\mathbf{H}^\dagger$ 的特征值，$\mathbf{H}$ 是随机信道矩阵。MP 律描述了 $\lambda_i$ 的极限分布，对 $\log_2(1+\mathrm{SNR}\cdot\lambda)$ 积分后可以得到遍历容量。结论很明确：容量与 $\min(n_t, n_r)$ 成**线性**关系。这就是为什么现代手机、基站和 Wi-Fi 路由器都采用多天线设计的理论依据。

```python
import numpy as np

def mimo_capacity(n_r, n_t, snr_db, trials=500):
    snr = 10 ** (snr_db / 10)
    caps = []
    for _ in range(trials):
        H = (np.random.randn(n_r, n_t)
             + 1j * np.random.randn(n_r, n_t)) / np.sqrt(2)
        eigs = np.linalg.eigvalsh(H @ H.conj().T)
        caps.append(np.sum(np.log2(1 + snr / n_t * eigs)))
    return float(np.mean(caps))

print(f"4x4 @ 10 dB: {mimo_capacity(4, 4, 10):.2f} bits/s/Hz")
print(f"8x8 @ 10 dB: {mimo_capacity(8, 8, 10):.2f} bits/s/Hz")
```

### 6.2 金融：清洗样本协方差矩阵

跟踪 $p$ 只股票在 $n$ 个交易日的数据，实际场景中纵横比 $\gamma = p/n$ 远离 0。MP 边界能精确区分哪些样本特征值是噪声。标准做法如下：

1. 对协方差矩阵 $\mathbf{S}$ 做特征分解：$\mathbf{S} = \mathbf{U}\,\mathrm{diag}(\lambda_1, \dots, \lambda_p)\,\mathbf{U}^\top$。
2. 估计噪声方差 $\sigma^2$，比如取 MP 支撑范围内的特征值均值。
3. 计算边界值 $\lambda_\pm = \sigma^2(1 \pm \sqrt{\gamma})^2$。
4. 把 $[\lambda_-, \lambda_+]$ 范围内的特征值替换为它们的均值（或者用更复杂的收缩方法）。
5. 重构清洗后的协方差矩阵 $\tilde{\mathbf{S}} = \mathbf{U}\,\mathrm{diag}(\tilde\lambda_i)\,\mathbf{U}^\top$。

用清洗后的 $\tilde{\mathbf{S}}$ 构建投资组合，样本外夏普比率通常能提升 10%-30%，同时显著降低换手率。

![样本协方差、RMT 清洗后的协方差、真实单位阵协方差三张热力图对比](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/14-随机矩阵理论/fig6_covariance_estimation.png)

```python
import numpy as np

def clean_covariance(returns):
    """把 MP 主体内的特征值替换为均值。"""
    n, p = returns.shape
    gamma = p / n
    S = np.cov(returns, rowvar=False)
    eigs, vecs = np.linalg.eigh(S)
    sigma2 = np.mean(eigs)
    lo, hi = sigma2 * (1 - np.sqrt(gamma))**2, sigma2 * (1 + np.sqrt(gamma))**2
    bulk = (eigs >= lo) & (eigs <= hi)
    eigs[bulk] = eigs[bulk].mean()
    return vecs @ np.diag(eigs) @ vecs.T
```

### 6.3 高维 PCA：保留多少主成分？

在 PCA 流程中，当 $p \approx n$ 时，传统方法很难回答“保留多少主成分”的问题。RMT 提供了一个清晰的答案：**保留所有大于 $\lambda_+ = (1 + \sqrt{\gamma})^2$ 的特征值。** 凡是低于 MP 边界的特征值，都可以认为是独立同分布的噪声。

```python
import numpy as np

def pca_signal_count(X):
    n, p = X.shape
    gamma = p / n
    Xs = (X - X.mean(0)) / X.std(0)
    eigs = np.linalg.eigvalsh(Xs.T @ Xs / n)
    return int(np.sum(eigs > (1 + np.sqrt(gamma))**2))
```

### 6.4 尖峰协方差与 BBP 相变

上面的 PCA 准则在信号较弱时不够准确。更贴近实际的模型是**尖峰协方差**：总体协方差矩阵为 $\Sigma = \mathbf{I} + \sum_{k=1}^{r} (s_k - 1) \mathbf{v}_k\mathbf{v}_k^\top$，即单位阵加上 $r$ 个尖峰。**Baik-Ben Arous-Péché（BBP）相变**揭示了以下规律：

- 如果总体尖峰 $s_k > 1 + \sqrt{\gamma}$，对应的样本特征值会脱离 MP 主体，落在 $s_k + \gamma s_k/(s_k - 1)$，并且样本特征向量与 $\mathbf{v}_k$ 的夹角余弦为正。
- 如果 $s_k \le 1 + \sqrt{\gamma}$，尖峰会被埋进 MP 主体中，PCA 无法还原它，无论算法多么复杂。

这是高维 PCA 的一条硬性不可能定理。它明确告诉我什么时候需要更多样本，什么时候问题在当前 $n, p$ 下统计上无解。

### 6.5 神经网络初始化

初始化权重时，让 $\mathbf{W}^\top\mathbf{W}$ 的特征值集中在 $1$ 附近，可以逐层稳定激活的方差。Xavier/Glorot 初始化法则 $\mathrm{Var}(w_{ij}) = 2 / (n_\text{in} + n_\text{out})$ 恰好让 $\mathbf{W}$ 的奇异值落在以 $1$ 为中心的 MP 支撑范围内。正交初始化更进一步——它完全消除了奇异值的展宽，在深线性网络中信号传播更加稳定。

## 7. 工具箱

### 7.1 Stieltjes 变换

每个定义在 $\mathbb{R}$ 上的概率测度 $\mu$ 都有一个 **Stieltjes 变换**：

$$m_\mu(z) \;=\; \int \frac{d\mu(\lambda)}{\lambda - z}, \qquad z \in \mathbb{C}^+.$$

通过反演公式可以恢复密度：
$$f(\lambda) \;=\; -\frac{1}{\pi}\lim_{\eta \to 0^+}\operatorname{Im}\,m_\mu(\lambda + i\eta).$$

为什么用它？因为经验谱分布的 Stieltjes 变换正好等于 $\frac{1}{n}\operatorname{tr}(\mathbf{M} - z\mathbf{I})^{-1}$，而预解算子（resolvent）代数上很好操作。现代随机矩阵理论（RMT）的证明基本都在 Stieltjes 变换层面完成：写出 $m(z)$ 的自洽方程，求解，再反演。

对于半圆律，方程是 $m(z)^2 + zm(z) + 1 = 0$，解得 $m(z) = (-z + \sqrt{z^2 - 4})/2$。Marchenko-Pastur 分布的方程也很简洁。

### 7.2 自由概率论一句话讲清楚

自由概率论里，"非交换随机变量" $a, b$ 是**自由独立**的，当它们的交替混合矩满足特定的迹分解规则。Voiculescu 的核心定理指出：**大尺寸独立随机矩阵在渐近意义下是自由的**。卷积的自由版本——**自由加法卷积** $\boxplus$——把 $\mathbf{A}$ 和 $\mathbf{B}$ 的谱分布映射到 $\mathbf{A} + \mathbf{B}$ 的谱分布。对应的中心极限定理给出半圆分布。实际应用中，所有对标量独立随机变量和的操作（均值、方差、CLT、Berry-Esseen），都可以找到矩阵谱的自由对应物。

![自由加法卷积：两个自由半圆之和仍是半圆，方差相加](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/linear-algebra/14-随机矩阵理论/fig5_free_convolution.png)

### 7.3 半圆律的证明骨架（矩方法）

计算 $m_k = \mathbb{E}[\frac{1}{n}\operatorname{tr}\hat{\mathbf{W}}^k]$。展开迹后，得到一个关于 $\{1, \dots, n\}$ 上长度为 $k$ 的闭路径的求和，权重是 $\prod_e \mathbb{E}[w_e]$。独立性加上零均值会消去所有包含奇数次某条边的路径。当 $k$ 为偶数时，主导贡献来自**两两配对且配对非交叉**的路径——这些路径的数量恰好是 Catalan 数 $C_{k/2}$，归一化后权重为 $1$。Catalan 数正是半圆分布的偶阶矩，所以极限分布必然是半圆分布。

## 9. 本章总结

**模型：** Wigner 矩阵用来描述对称噪声，Wishart 矩阵用来描述样本协方差。高斯版本的 GOE、GUE 和 GSE 分别对应不同的对称类别。

**主体规律：** Wigner 谱服从**半圆律**，Wishart 谱服从 **Marchenko-Pastur 分布**。这两者都具有普适性：只要元素分布的方差有限，具体分布形态并不重要。

**精细结构：** 特征值之间存在**排斥效应**，遵循已知规律（Wigner 猜想）；最大特征值的波动尺度为 $n^{-2/3}$，服从 **Tracy-Widom 分布**。

**工具：** Stieltjes 变换将谱问题转化为代数方程；**自由概率**把经典中心极限定理（CLT）风格的推理推广到独立矩阵的谱分析中。

**为什么重要：** 随机矩阵理论（RMT）为 MIMO 容量、协方差清理、PCA 阈值选择、尖峰模型的 BBP 相变以及神经网络初始化提供了精确且通常无需参数的预测。核心启示始终如一：高维随机性背后隐藏着确定性的骨架，而我可以利用它。
