前两章建立了一个清晰的二分法。第二章介绍了*第一基本形式* $\mathrm{I}$ —— 内蕴度量，即表面上的蚂蚁可以测量的东西。第三章介绍了*第二基本形式* $\mathrm{II}$ 和形状算子 —— 外在数据，描述表面如何在 $\mathbb{R}^3$ 中弯曲。从 $\mathrm{II}$ 计算了高斯曲率 $K = \det S$ 和平均曲率 $H = \mathrm{tr}\,S/2$。乍看之下，$K$ 和 $H$ 都应该依赖于嵌入方式。弯曲表面（不拉伸）你可能会期望两者都会改变。

对于 $H$，这是正确的：圆柱的 $H = -1/(2r)$，平面的 $H = 0$，尽管它们是等距的。

对于 $K$，发生了奇迹般的事情：$K$ 不会改变。圆柱的 $K = 0$，平面也是如此。两者都有常数零高斯曲率，尽管只有其中一个看起来是平的。这就是*惊人定理* —— 拉丁语为“显著定理” —— 由 Gauss 在 1827 年证明，它是经典微分几何的核心结果。

从惊人定理中得出了两个巨大的后果。首先，表面上的内蕴几何足够丰富，几乎可以在不提及嵌入到 $\mathbb{R}^3$ 的情况下完成所有想要的几何。可以测量距离，定义“直线”（测地线），并根据曲率对表面进行分类，全部从内部完成。其次，相同的内蕴装置无缝推广到更高维流形 —— 即广义相对论和现代几何理论中的空间。

本文将介绍惊人定理及其周围的装置：Christoffel 符号、测地线方程、平行移动、内蕴/外在区别的精确化。

## Christoffel 符号：曲坐标下的记账

在标准坐标下的 $\mathbb{R}^n$ 中，基向量 $\mathbf{e}_1, \ldots, \mathbf{e}_n$ 是常数。在表面上，每个切平面的自然基向量 —— $\mathbf{x}_u$ 和 $\mathbf{x}_v$ —— *逐点变化*。当我们对用这个基表示的向量场求导时，不能只对分量求导；我们必须考虑基本身的变化。

这就是 Christoffel 符号的作用。

形式化地说，给定一个表面上的图 $\mathbf{x}(u_1, u_2)$（我将使用 $u_1, u_2$ 而不是 $u, v$ 以使符号更简洁），二阶导数 $\mathbf{x}_{ij} = \partial^2\mathbf{x}/\partial u_i\partial u_j$ 是 $\mathbb{R}^3$ 中的向量。将它们分解为切向和法向分量：
$$\mathbf{x}_{ij} = \Gamma^1_{ij}\mathbf{x}_1 + \Gamma^2_{ij}\mathbf{x}_2 + L_{ij}\mathbf{n},$$
其中 $L_{ij}$ 是第二基本形式的系数（因此在之前的记号中 $L_{11} = L$, $L_{12} = M$, $L_{22} = N$），而 *Christoffel 符号* $\Gamma^k_{ij}$ 是切向分量。

由于混合偏导数的对称性，$\Gamma^k_{ij} = \Gamma^k_{ji}$。

![Christoffel 符号编码基框架如何转动](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_1_christoffel.png)

这很关键，因为 Christoffel 符号描述了当你在表面上移动时，基 $\{\mathbf{x}_u, \mathbf{x}_v\}$ 如何“扭曲”。它们将出现在测地线方程、平行移动公式中，并且在证明惊人定理时至关重要。

仅用 $\mathrm{I}$ 表示的公式：
$$\Gamma^k_{ij} = \frac{1}{2}\sum_l g^{kl}\bigl(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij}\bigr),$$
其中 $g_{ij}$ 是 $\mathrm{I}$ 的分量（因此 $g_{11} = E$, $g_{12} = F$, $g_{22} = G$），$g^{kl}$ 是 $\mathrm{I}^{-1}$ 的分量。

这个公式是关键。它表明：*Christoffel 符号仅依赖于第一基本形式*。尽管 Christoffel 符号是通过嵌入的二阶导数定义的，但最终可以从度量单独计算出来。

从 $\mathbf{x}_{ij} = \sum_k\Gamma^k_{ij}\mathbf{x}_k + L_{ij}\mathbf{n}$ 开始，取与 $\mathbf{x}_l$ 的内积：
$$\mathbf{x}_{ij}\cdot\mathbf{x}_l = \sum_k\Gamma^k_{ij}g_{kl}.$$

现在使用 $\partial_i g_{jl} = \partial_i(\mathbf{x}_j\cdot\mathbf{x}_l) = \mathbf{x}_{ij}\cdot\mathbf{x}_l + \mathbf{x}_j\cdot\mathbf{x}_{il}$。置换指标并结合：
$$\mathbf{x}_{ij}\cdot\mathbf{x}_l = \frac{1}{2}\bigl(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij}\bigr).$$

与前面的结合：$\sum_k\Gamma^k_{ij}g_{kl} = \frac{1}{2}(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$。乘以 $g^{lm}$（逆度量）并对 $l$ 求和得到上面的公式。

这个计算将在第 10 章中以稍微不同的记号重复进行。该公式是 *Levi-Civita 连接*。

## 示例：球面上的 Christoffel 符号

使用单位球上的球坐标 $\mathbf{x}(\theta, \varphi)$，其中 $\mathrm{I} = \mathrm{diag}(\sin^2\varphi, 1)$。因此 $g_{11} = \sin^2\varphi$, $g_{22} = 1$, $g_{12} = 0$, $g^{11} = 1/\sin^2\varphi$, $g^{22} = 1$, $g^{12} = 0$。

度量的非零偏导数：
- $\partial_\varphi g_{\theta\theta} = \partial_\varphi(\sin^2\varphi) = 2\sin\varphi\cos\varphi = \sin 2\varphi$。

其他都为零。应用公式：
- $\Gamma^\theta_{\theta\varphi} = \Gamma^\theta_{\varphi\theta} = \frac{1}{2}g^{\theta\theta}\partial_\varphi g_{\theta\theta} = \frac{1}{2\sin^2\varphi}\cdot 2\sin\varphi\cos\varphi = \cos\varphi/\sin\varphi = \cot\varphi$。
- $\Gamma^\varphi_{\theta\theta} = \frac{1}{2}g^{\varphi\varphi}(-\partial_\varphi g_{\theta\theta}) = -\frac{1}{2}\cdot 2\sin\varphi\cos\varphi = -\sin\varphi\cos\varphi$。

其他分量为零。因此：
$$\Gamma^\theta_{\theta\varphi} = \Gamma^\theta_{\varphi\theta} = \cot\varphi,\qquad \Gamma^\varphi_{\theta\theta} = -\sin\varphi\cos\varphi.$$

这些 Christoffel 符号编码了球坐标的所有“内在弯曲”：当你沿着一个纬圈（$\theta$ 变化）移动时，基向量 $\mathbf{x}_\theta$ 在切平面内旋转，这种旋转被 $\Gamma^\varphi_{\theta\theta}$ 及其同伴捕获。

## 测地线方程

*测地线* 是表面上的“直线” —— 曲线尽可能直。

定义（变分）。曲面 $S$ 上的一条曲线 $\gamma$ 是测地线，如果它局部最小化弧长：沿表面测量的 $\gamma(s_1)$ 和 $\gamma(s_2)$ 之间的距离等于 $\gamma$ 在这些参数之间的长度，对于 $s_1, s_2$ 足够接近。

定义（微分）。曲面 $S$ 上一条以恒定速度参数化的曲线 $\gamma(t) = \mathbf{x}(u_1(t), u_2(t))$ 是测地线当且仅当其加速度垂直于 $S$：
$$\gamma''(t) \parallel \mathbf{n}(\gamma(t)).$$

这两个定义是等价的，并且都导致*测地线方程*在图坐标下：
$$\ddot{u_k} + \sum_{i,j}\Gamma^k_{ij}\dot{u_i}\dot{u_j} = 0\quad\text{对于 } k = 1, 2.$$

这是一个关于 $u_1(t), u_2(t)$ 的两个耦合二阶 ODE 系统。给定初始条件 $(u_i(0), \dot{u_i}(0))$，存在唯一的测地线（由 ODE 存在唯一性定理保证）。

![局部坐标下的测地线方程](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_2_geodesic_eq.png)

测地线是内蕴几何中的“直线”。它们将欧几里得直线推广到弯曲空间。一个粒子在表面上无外力作用下（除了约束在表面上）运动时，轨迹是一条测地线 —— 这就是测地线方程的物理内容。在广义相对论中，时空中的测地线是惯性观察者的轨迹；空间的弯曲取代了牛顿引力的概念。

Christoffel 符号输入测地线方程。由于它们是内蕴的，测地线也是内蕴的：表面上的蚂蚁可以计算它们。

### 示例：球面上的测地线

使用单位球上的 $\mathbf{x}(\theta,\varphi)$。测地线方程：
$$\ddot\theta + 2\cot\varphi\,\dot\theta\dot\varphi = 0,\qquad \ddot\varphi - \sin\varphi\cos\varphi\,\dot\theta^2 = 0.$$

寻找一类特殊情况：沿着经线的曲线，$\theta = $ 常数，有 $\dot\theta = 0$。第一个方程自动满足。第二个变为 $\ddot\varphi = 0$，即 $\varphi$ 与 $t$ 成线性关系。因此经线（通过极点的大圆），$\varphi$ 与弧长成线性变化，是测地线。

赤道呢？$\varphi = \pi/2$，$\dot\varphi = 0$。第二个方程变为 $-\sin(\pi/2)\cos(\pi/2)\dot\theta^2 = 0$，自动满足。第一个变为 $\ddot\theta + 2\cot(\pi/2)\dot\theta\dot\varphi = \ddot\theta + 0 = 0$，所以 $\theta$ 与 $t$ 成线性关系。赤道上 $\theta$ 与弧长成线性变化是测地线。

通过球面的旋转对称性，*每条大圆都是测地线*。事实上，这些是唯一的测地线。**球面上的测地线恰好是大圆**。这恢复了球面三角学中熟悉的经典事实。

![球面上的测地线是大圆](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_3_sphere_geodesic.png)

它没有恢复的是：球面上的测地线不是唯一的（在对跖点之间有无限多条；否则两条），不全局最小化（绕远路也是一条测地线但不是最短路径），并且一般的存在定理是局部的。

### 示例：圆柱上的测地线

圆柱 $\mathbf{x}(u, v) = (\cos u, \sin u, v)$ 有 $\mathrm{I} = I_2$。所有 Christoffel 符号都消失（度量是常数）。测地线方程：$\ddot u = \ddot v = 0$。解是 $u(t) = a t + b$, $v(t) = c t + d$，这在圆柱上描绘了一条*螺旋线*（或如果 $c = 0$，则是一个圆；或如果 $a = 0$，则是一条垂直线）。

螺旋线、垂直线和水平圆。圆柱上最直的路径是这些。注意：圆柱在内蕴上是平坦的（与平面相同），在平面上测地线是直线 $u = at+b$, $v = ct+d$。将平面卷成圆柱的映射将直线映射成螺旋线（以及特殊情况 —— 圆和垂直线）。等距变换保留测地线。

### 示例：环面

环面具有可变的高斯曲率，因此测地线可以做奇怪的事情。环面上的测地线方程通常不可积（对于任意初始条件没有封闭形式解），但有两种特殊情况存在：经线（围绕管的小圆）和“穿过内赤道的长路径”（仅在环面内部）。

最有趣的是：大多数环面上的测地线在环面上是稠密的。选择一个一般的初始方向，测地线会在一段时间后覆盖一个开集。相比之下，周期测地线仅在初始斜率为有理数（在平坦环面坐标下）时存在。

![平坦环面上不同有理和无理斜率的测地线](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_6_torus_geodesic.png)

这种二分法 —— 有理斜率的闭轨道，无理斜率的稠密轨道 —— 是现代环面动力系统理论的原型。

## 惊人定理

现在是经典曲面理论的高潮。

下面这个结论是核心：曲面的高斯曲率 $K$ 是内蕴的：它可以仅从第一基本形式计算，无需参考第二基本形式或嵌入。

具体含义是，如果两个曲面是等距的 —— 即具有相同的第一基本形式 —— 它们在对应点处具有相同的高斯曲率。

![惊人定理：K 是内蕴的，仅从 E, F, G 计算](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_4_egregium.png)

这很惊人，因为 $K$ 是*外在定义*的：$K = \det(\text{形状算子}) = (LN - M^2)/(EG - F^2)$。分子 $LN - M^2$ 是从第二基本形式构建的，它编码了表面如何在 $\mathbb{R}^3$ 中弯曲。没有明显的原因表明这个组合可以不用 $\mathrm{II}$ 表达。然而它确实可以。

*证明思路。* 从*Gauss 公式*开始：对 $\mathbf{x}_{ij}$ 再次求导，使用 $\mathbf{x}_{ij} = \sum\Gamma^k_{ij}\mathbf{x}_k + L_{ij}\mathbf{n}$，并应用对称性 $\mathbf{x}_{ijk} = \mathbf{x}_{ikj}$。切向部分给出*Gauss 方程*，经过处理后得到
$$E\cdot K = (\Gamma^2_{12})_u - (\Gamma^2_{11})_v + \Gamma^1_{12}\Gamma^2_{11} - \Gamma^1_{11}\Gamma^2_{12} + \Gamma^2_{12}\Gamma^2_{12} - \Gamma^2_{11}\Gamma^2_{22},$$
或者在一个更简洁的形式之一（这是一个有许多种表述的公式，没有一种特别容易记忆），
$$K = \frac{1}{\sqrt{EG-F^2}}\biggl[\partial_v\biggl(\frac{\sqrt{EG-F^2}}{E}\Gamma^2_{11}\biggr) - \partial_u\biggl(\frac{\sqrt{EG-F^2}}{E}\Gamma^2_{12}\biggr)\biggr].$$
对于正交图（$F = 0$），Brioschi 公式简化为：
$$K = -\frac{1}{2\sqrt{EG}}\biggl[\partial_v\biggl(\frac{E_v}{\sqrt{EG}}\biggr) + \partial_u\biggl(\frac{G_u}{\sqrt{EG}}\biggr)\biggr].$$
对于等温图（$E = G = \lambda$, $F = 0$）：
$$K = -\frac{1}{2\lambda}\Delta\log\lambda,$$
其中 $\Delta = \partial_u^2 + \partial_v^2$ 是欧几里得 Laplacian。*这个*是每个工作几何学家记住的公式，因为它简短而美丽。

公式的具体形式不如结论重要：$K$ 仅依赖于 $E, F, G$ 及其导数。因此仅依赖于 $\mathrm{I}$。因此是内蕴的。$\square$

球面和平面有不同的高斯曲率（$1/r^2$ 对 $0$），因此它们不是局部等距的。即使是很小区域的球面也没有忠实的平面地图。制图师必须妥协：保持角度（Mercator，保角但不等距）、保持面积（Gall-Peters，等面积但不保角），或某种混合。他们不能保留一切，因为 $K$ 必须匹配。

曲面只能以保持 $K$ 的方式弯曲（等距变形）。惊人定理对可能的变形施加了硬性约束。

![等距变换保持高斯曲率](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_5_isometry.png)

## 示例：通过内蕴公式计算球面上的 $K$

球坐标：$E = \sin^2\varphi$, $G = 1$, $F = 0$。使用 Brioschi 的正交图公式：
$$K = -\frac{1}{2\sqrt{EG}}\biggl[\partial_v\biggl(\frac{E_v}{\sqrt{EG}}\biggr) + \partial_u\biggl(\frac{G_u}{\sqrt{EG}}\biggr)\biggr].$$
这里 $u = \theta$, $v = \varphi$，所以 $E_v = E_\varphi = 2\sin\varphi\cos\varphi = \sin 2\varphi$, $G_u = 0$。然后
$$K = -\frac{1}{2\sin\varphi}\cdot\partial_\varphi\biggl(\frac{\sin 2\varphi}{\sin\varphi}\biggr) = -\frac{1}{2\sin\varphi}\cdot\partial_\varphi(2\cos\varphi) = -\frac{1}{2\sin\varphi}\cdot(-2\sin\varphi) = 1.$$

单位球上常数 $K = 1$。我们之前通过外在形状算子计算过这一点；现在看到它也可以从内蕴公式得出。练习的重点不是答案 —— 已经知道答案 —— 而是没有使用任何外在数据。

### 示例：螺线上 $K$ 的计算

螺线 $\mathbf{x}(u, v) = (v\cos u, v\sin u, c u)$ 有 $\mathrm{I} = \mathrm{diag}(v^2 + c^2, 1)$。使用 Brioschi：
$E = v^2 + c^2$, $G = 1$, $\sqrt{EG} = \sqrt{v^2+c^2}$, $E_v = 2v$。
$$K = -\frac{1}{2\sqrt{v^2+c^2}}\partial_v\biggl(\frac{2v}{\sqrt{v^2+c^2}}\biggr).$$
计算内部导数：$\partial_v(2v/\sqrt{v^2+c^2}) = 2/\sqrt{v^2+c^2} - 2v^2/(v^2+c^2)^{3/2} = 2c^2/(v^2+c^2)^{3/2}$。因此
$$K = -\frac{1}{2\sqrt{v^2+c^2}}\cdot\frac{2c^2}{(v^2+c^2)^{3/2}} = -\frac{c^2}{(v^2+c^2)^2}.$$

负值。随着 $|v|\to\infty$ 衰减。螺线处处双曲，曲率集中在中心轴附近。

### 示例：悬链面与螺线等距

悬链面是 $\rho(v) = c\cosh(v/c)$, $z(v) = v$ 的旋转曲面。它的第一基本形式（经过计算）是 $\mathrm{I} = \cosh^2(v/c)\,\mathrm{diag}(c^2, 1)$ 在某些参数化下，经过变量变换后与螺线的度量匹配。

因此 $K_{\text{悬链面}} = K_{\text{螺线}} = -c^2/(v^2+c^2)^2$ 在对应点处。两种在 $\mathbb{R}^3$ 中看起来完全不同的曲面 —— 一个是螺旋形状，另一个是两个环之间的肥皂膜 —— 在内蕴上是相同的。弯曲将一个变成另一个。

这是惊人定理最引人注目的例子：相同的 $\mathrm{I}$，因此相同的 $K$，因此曲面在内蕴上是“相同的”，尽管它们的外在几何（和视觉外观）完全不同。

## 平行移动

一旦有了 Christoffel 符号，我们就可以描述如何沿着曲线“移动”一个切向量，使其保持“平行”。

给定曲面 $S$ 上的一条曲线 $\gamma(t)$ 和一个沿着 $\gamma$ 的向量场 $\mathbf{V}(t)\in T_{\gamma(t)}S$，$\mathbf{V}$ 沿着 $\gamma$ *平行移动* 如果
$$\frac{D\mathbf{V}}{dt} := \mathbf{V}'(t)^{\text{切向}} = 0,$$
即 $\mathbf{V}$ 在 $\mathbb{R}^3$ 中的导数没有切向分量（只有法向分量）。

在坐标 $\mathbf{V} = V^1\mathbf{x}_1 + V^2\mathbf{x}_2$ 下：
$$\frac{D V^k}{dt} = \dot{V^k} + \sum_{i,j}\Gamma^k_{ij}\dot{u_i}V^j = 0.$$

给定曲线的 $V^1, V^2$ 的线性 ODE。存在唯一性给出了每条曲线的平行移动映射 $P_\gamma: T_{\gamma(0)}S\to T_{\gamma(1)}S$。它是一个线性同构，保持度量（在平行移动过程中范数和角度保持不变 —— 这是 Christoffel 符号公式的后果）。

测地线是沿着其*切向量*平行移动的曲线。因此 $\dot\gamma(t) = $ （切向），且 $D\dot\gamma/dt = 0$，这展开为测地线方程。

如果 $\gamma$ 是闭合回路，$P_\gamma: T_pS\to T_pS$ 不必是恒等映射。它与恒等映射的差异（$T_pS$ 中的旋转）是回路的*全纯性*。在平坦表面上，全纯性是平凡的。在弯曲表面上，全纯性是非平凡的，事实上：小回路的全纯性等于包围区域内 $K$ 的积分。这是曲率与平行移动路径独立性失效之间联系的最简单版本。接下来在第 5 章（Gauss-Bonnet）和第 11 章（流形上的曲率）中再次看到它。

## 内蕴与外在，精确区分

让我再仔细画一次这条线。

**内蕴量**（仅从 $\mathrm{I}$ 计算）：
- 长度、角度、面积（第 2 章）。
- Christoffel 符号。
- 高斯曲率 $K$（惊人定理）。
- 测地线。
- 平行移动。
- 全纯性。

**外在量**（需要 $\mathrm{II}$ 或嵌入）：
- Gauss 映射 $N: S\to S^2$。
- 形状算子 $S_p$。
- 主曲率 $k_1, k_2$。
- 平均曲率 $H$。
- 某个方向的法曲率 $\kappa_n$。
- 渐近方向和脐点方向。

惊人定理是 $K$ 从右边列转移到左边的确切意义。$H$ 仍然在右边。

![内蕴与外在几何分开](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/differential-geometry/04-intrinsic-geometry/dg_v2_04_7_intrinsic_extrinsic.png)

仅内蕴量是在没有嵌入的情况下抽象到流形中仍然存在的量。可以在不将其放入 $\mathbb{R}^n$ 的情况下讨论 Riemann 流形上的 Christoffel 符号和高斯曲率。这是使广义相对论成为可能的哲学转变：时空在内蕴上是弯曲的，没有嵌入到平坦的高维空间中。

## 常曲率曲面

一个自然的问题：哪些曲面具有常高斯曲率？

**三个模型曲面**（相似性下）：
- $K = 1$：单位球。
- $K = 0$：平面（或圆柱，或任何平坦曲面）。
- $K = -1$：双曲平面（局部实现为伪球面；根据 Hilbert 定理，在 $\mathbb{R}^3$ 中作为完整嵌入曲面不存在）。

Minding 在 1839 年证明了一个重要的结论：具有相同常高斯曲率的两个曲面局部等距。

这是一个美妙的定理。它说在局部齐次情况下，常曲率是唯一的内蕴不变量。单位球的两块局部等距。伪球面的两块局部等距。平面的两块局部等距。但球面不等距于平面不等距于伪球面。

双曲平面（常 $K = -1$，单连通，完备）是非欧几何的中心对象。它是平行公设失效的模型空间：通过不在直线上的点，有无穷多条直线（测地线）不与给定直线相交。这是 Bolyai 和 Lobachevsky 在 1820 年代的伟大发现，早于 Gauss 发表的工作，但在未发表的通信中被 Gauss 预见到。

## 总结

已经从外在几何过渡到内蕴几何。

- **Christoffel 符号** $\Gamma^k_{ij}$ 仅从 $\mathrm{I}$ 计算，编码基框架如何转动。
- **测地线方程** $\ddot u_k + \sum_{ij}\Gamma^k_{ij}\dot u_i\dot u_j = 0$ 描述了内蕴几何中的“直线”。解由初始点和方向唯一确定。
- **平行移动** 沿着曲线移动向量，保持“切向性”；测地线是其切向量平行移动的曲线。
- **惊人定理** 说 $K$ 是内蕴的，即由 $\mathrm{I}$ 决定。相同的度量，相同的 $K$，无论嵌入如何。
- **制图是不可能的**：球面和平面有不同的 $K$，因此不存在忠实的平面地图。
- **螺线和悬链面是等距的**，戏剧性地说明了惊人定理。
- **常曲率曲面** 是三个模型空间：球面、平面、双曲平面。

下一章（Gauss-Bonnet）带来全局视角：在闭合曲面上积分 $K$ 得到拓扑不变量（$2\pi$ 乘以 Euler 示性数）。这是从局部微分几何到全局拓扑的桥梁，是数学中最美丽的定理之一。

## 附录：更多测地线示例

### 回转曲面上的测地线

对于具有图 $\mathbf{x}(u, v) = (\rho(v)\cos u, \rho(v)\sin u, z(v))$ 的回转曲面，在轮廓的弧长参数化下（因此 $(\rho')^2 + (z')^2 = 1$），度量是 $\mathrm{I} = \mathrm{diag}(\rho^2, 1)$。Christoffel 符号（经过计算）：
$$\Gamma^u_{uv} = \rho'/\rho,\qquad \Gamma^v_{uu} = -\rho\rho'.$$

测地线方程：
$$\ddot u + 2(\rho'/\rho)\dot u\dot v = 0,\qquad \ddot v - \rho\rho'\dot u^2 = 0.$$

第一个方程可以写成 $d/dt(\rho^2\dot u) = 0$，因此 $\rho^2\dot u = L$ 是守恒量。这是*Clairaut 关系*：在回转曲面上的测地线上，量 $\rho\sin\psi = L/\sqrt{\rho^2(\dot u^2 + (\dot v/\rho)^2/\rho^{-2})\cdot\text{stuff}}$ —— 让我干净地陈述：

如果 $\gamma$ 是回转曲面上的测地线，$\psi$ 是 $\gamma'$ 与平行方向之间的夹角，则 $\rho\sin\psi$ 沿 $\gamma$ 保持常数。

这是一个经典的守恒律（本质上是角动量：度量的旋转对称性）。它允许手算锥面、抛物面、双曲面等的测地线。

例如，在锥面 $\rho(v) = v\sin\alpha$, $z(v) = v\cos\alpha$（半角为 $\alpha$ 的锥面）上，度量是 $\mathrm{I} = \mathrm{diag}(v^2\sin^2\alpha, 1)$。测地线：沿着生成线切割锥面并展开；你得到一个平坦圆盘的扇形，测地线是直线。拉回到锥面：展开后的直线对应于绕锥面缠绕的曲线，与生成线成恒定角度（由 Clairaut）。

### 伪球面上的测地线

伪球面具有常 $K = -1$，其测地线是非欧几何中的曲线。可以通过 Klein 模型或 Poincaré 圆盘模型来描述 —— 但在伪球面本身的图中，它们是类似曳物线的曲线。显式计算它们是通过伪球面度量解测地线 ODE 的练习。

有趣的事实：伪球面实现了（一部分）*上半平面模型*的双曲几何，在度量 $ds^2 = (du^2 + dv^2)/v^2$ 对于 $v > 0$。此模型中的测地线是垂直线和以 $u$ 轴为中心的半圆。通过共同点的两条测地线可以有任意小的角度而不相交，这是违反 Euclid 平行公设的表现。

## 附录：通过 $\Delta\log\lambda$ 具体计算 $K$

对于立体投影的单位球的等温坐标：$\mathbf{x}(u, v)$ 有度量 $\mathrm{I} = \lambda(u, v)\,I_2$ 其中
$$\lambda(u, v) = \frac{4}{(1 + u^2 + v^2)^2}.$$

那么 $\log\lambda = \log 4 - 2\log(1+u^2+v^2)$。Laplacian：
$$\Delta\log\lambda = -2\Delta\log(1+u^2+v^2).$$

计算 $\Delta\log(1+u^2+v^2)$。令 $r^2 = u^2+v^2$。则 $\log(1+r^2)$，使用极坐标 Laplacian $\Delta f(r) = f'' + f'/r$：
$$f(r) = \log(1+r^2),\quad f'(r) = 2r/(1+r^2),\quad f''(r) = (2(1+r^2) - 2r\cdot 2r)/(1+r^2)^2 = (2 - 2r^2)/(1+r^2)^2.$$
$$\Delta f = f'' + f'/r = (2-2r^2)/(1+r^2)^2 + 2/(1+r^2) = \bigl[(2-2r^2) + 2(1+r^2)\bigr]/(1+r^2)^2 = 4/(1+r^2)^2.$$

因此 $\Delta\log\lambda = -8/(1+r^2)^2 = -2\lambda$。由等温图公式：
$$K = -\frac{1}{2\lambda}\Delta\log\lambda = -\frac{1}{2\lambda}\cdot(-2\lambda) = 1.$$

球面上常数单位高斯曲率。已经通过三种不同的方法计算了这一点：从形状算子、从 Brioschi 公式和从等温图公式。所有方法都给出相同的结果，正如惊人定理所预测的那样。
