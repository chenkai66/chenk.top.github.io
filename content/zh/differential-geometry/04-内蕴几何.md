经典微分几何中最核心的意外发现在这里：Gauss 曲率——我们用法向量和形算子定义的量——实际上只依赖第一基本形式。住在曲面上的蚂蚁，配有尺子和量角器但不知道 $\mathbb{R}^3$ 的存在，仍然可以探测 Gauss 曲率。这就是 Gauss 的绝妙定理（Theorema Egregium），它从根本上改变了我们对几何的理解。

## Christoffel 符号

在陈述定理之前，需要 Christoffel 符号——描述坐标基向量沿曲面变化方式的簿记工具。

二阶导数 $\mathbf{x}_{uu}$、$\mathbf{x}_{uv}$、$\mathbf{x}_{vv}$ 分解为切向和法向分量：

$$\mathbf{x}_{uu} = \Gamma^1_{11}\mathbf{x}_u + \Gamma^2_{11}\mathbf{x}_v + e\mathbf{N}$$
$$\mathbf{x}_{uv} = \Gamma^1_{12}\mathbf{x}_u + \Gamma^2_{12}\mathbf{x}_v + f\mathbf{N}$$
$$\mathbf{x}_{vv} = \Gamma^1_{22}\mathbf{x}_u + \Gamma^2_{22}\mathbf{x}_v + g\mathbf{N}$$

法向分量给出第二基本形式系数 $e,f,g$。切向分量就是 **Christoffel 符号** $\Gamma^k_{ij}$，完全由 $E$、$F$、$G$ 及其偏导数计算。正交参数化（$F = 0$）时：

$$\Gamma^1_{11} = \frac{E_u}{2E}, \quad \Gamma^1_{12} = \frac{E_v}{2E}, \quad \Gamma^1_{22} = \frac{-G_u}{2E}$$
$$\Gamma^2_{11} = \frac{-E_v}{2G}, \quad \Gamma^2_{12} = \frac{G_u}{2G}, \quad \Gamma^2_{22} = \frac{G_v}{2G}$$

Christoffel 符号不是张量——它们依赖坐标选择，可以在任何一个点通过选择适当坐标使之为零。但它们是构建内蕴几何量的基础砖块。

## Gauss 绝妙定理

**定理（Gauss, 1827）。** 曲面的 Gauss 曲率 $K$ 只依赖第一基本形式系数 $E$、$F$、$G$ 及其一阶和二阶偏导数。

正交参数化（$F = 0$）时的显式公式：

$$K = -\frac{1}{2\sqrt{EG}}\left[\frac{\partial}{\partial u}\left(\frac{G_u}{\sqrt{EG}}\right) + \frac{\partial}{\partial v}\left(\frac{E_v}{\sqrt{EG}}\right)\right].$$

*证明思路。* **Gauss 方程**把 Riemann 曲率张量（由 Christoffel 符号及其导数构造）与 Gauss 曲率联系：$R_{1212} = K(EG - F^2)$。$\Gamma^k_{ij}$ 只由 $E$、$F$、$G$ 决定，所以 $R_{1212}$ 也只由它们决定，因此 $K$ 亦然。

Gauss 称这个结果为"egregium"（杰出的/绝妙的），因为它完全出乎他的预期。$K = \kappa_1\kappa_2$ 的定义涉及法向量和形算子——依赖于曲面如何坐落在 $\mathbb{R}^3$ 中的对象。但最终结果只取决于内蕴度量。

**推论 1。** 等距曲面在对应点有相同的 Gauss 曲率。不拉伸曲面就不能改变 $K$。

**推论 2。** $K$ 是弯曲不变量。弯曲曲面（不拉伸——如卷纸成圆柱）时，每一点的 Gauss 曲率保持不变。

## 例：为什么地图必然变形

球面处处 $K = 1/R^2 > 0$，平面处处 $K = 0$。由绝妙定理，两者之间不存在等距——连局部都不行。因此球面的任何平面地图必定扭曲距离、角度或面积（或三者全部）。

Mercator 投影保角但极地面积膨胀。Lambert 等面积投影保面积但扭曲角度。没有投影能同时保两者——因为平坦度量不可能再现球面的正 Gauss 曲率。

## 例：圆柱面是平坦的

圆柱面 $K = 0$，和平面一样。验证了我们早先的观察：圆柱面与平面局部等距（卷/展纸）。但球面 $K = 1/R^2 > 0$ 不可能与平面等距——你无法把橘子皮展平而不撕裂或拉伸。

## 测地线

**测地线**是曲面上"尽可能直"的曲线。严格定义：曲线 $\gamma(t)$ 的加速度没有切向分量：

$$\nabla_{\gamma'}\gamma' = 0$$

其中 $\nabla$ 是协变导数——普通导数在 $\mathbb{R}^3$ 中的切向投影。等价地，**测地曲率** $\kappa_g = 0$。

直觉：在曲面上行走，始终正视前方（相对曲面不左转不右转），走出的路径就是测地线。测地线在 $\mathbb{R}^3$ 中可能弯曲，但在曲面内它是笔直的。

## 测地线方程

坐标中，$\gamma(t) = \mathbf{x}(u(t),v(t))$ 的测地线方程为：

$$u'' + \Gamma^1_{11}(u')^2 + 2\Gamma^1_{12}u'v' + \Gamma^1_{22}(v')^2 = 0$$
$$v'' + \Gamma^2_{11}(u')^2 + 2\Gamma^2_{12}u'v' + \Gamma^2_{22}(v')^2 = 0$$

二阶非线性 ODE 系统。由存在唯一性定理，给定初始点 $p$ 和初始方向 $\mathbf{v}\in T_pS$，测地线唯一确定——对应 Euclid 几何中"一点加一方向确定一条直线"。

## 例：球面上的测地线

球面上的测地线是大圆——过球心的平面与球面的交线。两点间最短路径沿大圆弧走。

**为什么是大圆？** 大圆的曲率向量指向球心方向，完全垂直于曲面——没有切向分量，$\kappa_g = 0$。

赤道是测地线。每条经线是测地线。但非赤道的纬线不是：它有非零测地曲率 $\kappa_g = \cot\theta_0/R$——它朝最近的极点弯曲。

飞机航线走大圆弧而不走等纬线。伦敦飞东京经过北极圈——大圆路线——比沿纬线飞节省大量距离。

## 例：圆柱面上的测地线

圆柱面 $\mathbf{x}(u,v) = (\cos u,\sin u,v)$，Christoffel 符号全为零（$E = G = 1$，$F = 0$，全部常数）。测地线方程退化为 $u'' = 0$，$v'' = 0$，解为 $u = at+b$，$v = ct+d$——参数平面中的直线。

对应到圆柱面上三族曲线：
- 螺旋线：$a$、$c$ 都不为零，以恒定角度螺旋上升。
- 竖直母线：$a = 0$，平行于轴线的直线。
- 水平圆：$c = 0$，横截面。

展开圆柱面，所有测地线变成直线——与平面等距的预期一致。螺旋线在 $\mathbb{R}^3$ 中看起来弯曲，但在圆柱面内是笔直的。

## 例：环面上的测地线

环面上情况更复杂。外赤道（$u = 0$）和内赤道（$u = \pi$）是测地线。经线（常 $v$）也是。但大多数环面测地线既不封闭也不周期——它们遍历地缠绕环面，永远不精确重复。

## 测地线与最短路径

**定理。** 曲面上两点间的最短曲线（若存在）是测地线。

*证明思路。* 对能量泛函 $E[\gamma] = \frac{1}{2}\int|\gamma'|^2\,dt$ 做变分法，Euler-Lagrange 方程给出测地线方程。（用能量而非长度避免了重参数化不变性的麻烦。）

反向只局部成立：测地线只是局部最短。球面上短于半圆的大圆弧是最短路径，但互补的长弧也是测地线——只是不是最短的。超过**共轭点**（对径点）后，测地线不再最短。

## Gauss-Bonnet 的预告

测地曲率与拓扑有深刻联系。对测地三角形 $T$（三边是测地线，$\kappa_g = 0$），**角度超出量**等于 Gauss 曲率积分：

$$(\alpha_1+\alpha_2+\alpha_3) - \pi = \iint_T K\,dA.$$

球面上（$K > 0$）：三角形内角和大于 $\pi$，三角形"鼓起来"。双曲面上（$K < 0$）：内角和小于 $\pi$，三角形"凹进去"。平坦面上（$K = 0$）：内角和恰好 $\pi$——Euclid 几何。

这是 Gauss-Bonnet 定理的局部版本，第 6 章将完整展开。这是第一个暗示：曲率——一个局部的微分量——携带着拓扑信息。
