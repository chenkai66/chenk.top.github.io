多项式是代数的实验室。几乎每一个环论概念——理想、商、因子分解、不可约性——都是通过多项式例子首先被理解的，然后才被抽象化。这不是巧合：多项式环足够丰富，可以展示所有有趣的现象，同时结构又足够明确，允许显式计算。

在这篇文章中，我们研究环 $R[x]$ 上的多项式。我们发展了除法算法，建立了不可约性标准（Eisenstein 准则、模 $p$ 还原、有理根测试），定义了唯一因子分解整环，并证明了 Gauss 引理——这是从 $\mathbb{Z}$ 到 $\mathbb{Q}$ 因子分解的桥梁。最终，将清楚地了解何时及为何唯一因子分解成立，以及当它失败时发生了什么。




## 为什么多项式在代数中处于核心地位

心理图像：一个多项式 $f(x) \in R[x]$ 是系数在 $R$ 中的“形式表达式”，但它也是一个函数（当在点上求值时）、理想的生成元，以及代数关系的见证。每个视角揭示了同一对象的不同方面。

多项式值得专门研究的三个原因：

1. 多项式生成所有代数扩张：如果 $\alpha$ 是某个有理系数多项式的根，则 $\mathbb{Q}(\alpha) \cong \mathbb{Q}[x]/(p(x))$，其中 $p$ 是 $\alpha$ 的最小多项式。多项式因子分解控制域扩张。

2. 多项式是一个通用的测试场：$\mathbb{Z}[x]$ 是 UFD 但不是 PID，因此可以研究这两个概念之间的差距。根据 Hilbert 基定理，Noether 环上的多项式环也是 Noether 环。

3. 多项式连接代数和几何：多项式的零点形成代数簇——这是代数几何的起点。环论性质（理想、商）对应于几何性质（子簇、交集）。

## $R[x]$：定义、次数和除法算法

想象一下：加法是逐系数进行的；乘法则是系数序列的“卷积”。次数在乘法幺半群中表现得像一个加性赋值。

定义。设 $R$ 是一个交换幺环。*多项式环* $R[x]$ 由形式表达式组成

$$f(x) = a_n x^n + \cdots + a_1 x + a_0$$

其中 $a_i \in R$。运算包括逐点加法和卷积乘法。

卷积乘法：$(f \cdot g)_k = \sum_{i + j = k} f_i g_j$。例如 $(2x + 3)(x^2 + x + 1) = 2x^3 + 2x^2 + 2x + 3x^2 + 3x + 3 = 2x^3 + 5x^2 + 5x + 3$。这种卷积结构使得多项式乘法成为一个结合的、分配的运算，类似于形式幂级数的乘法。

*次数* $\deg f$ 是最大的 $n$ 使得 $a_n \neq 0$。*首项系数* 是 $a_n$；如果 $a_n = 1$，则 $f$ 是*首一*的。

次数和乘法。如果 $R$ 是整环，那么 $\deg(fg) = \deg f + \deg g$。（对于有零因子的环不成立：例如在 $(\mathbb{Z}/4)[x]$ 中 $(2x)(2x) = 0$。）

推论。$R$ 是整环 $\Rightarrow$ $R[x]$ 是整环。

多变量情况。$R[x_1, \ldots, x_n] := R[x_1, \ldots, x_{n-1}][x_n]$，递归定义。这在代数几何中非常重要。

$R[x_1, \ldots, x_n]$ 中的一个单项式是 $x_1^{e_1} \cdots x_n^{e_n}$，其中 $e_i \in \mathbb{Z}_{\geq 0}$。单项式的集合与 $\mathbb{Z}_{\geq 0}^n$ 双射，而 $R[x_1, \ldots, x_n]$ 是这个集合上的自由 $R$-模。因此，元素是单项式的有限 $R$-线性组合。单项式的总次数是 $e_1 + \cdots + e_n$。多元多项式是 Gröbner 基（理想的算法基础）和结式（消去方程组中的变量）等构造的家园。

求值同态。对于 $\alpha \in S \supseteq R$，映射 $\text{ev}_\alpha: R[x] \to S, f \mapsto f(\alpha)$ 是一个环同态。这将形式多项式与函数联系起来。两个多项式可能定义相同的函数但作为多项式不相等：在 $\mathbb{F}_2[x]$ 中，$x$ 和 $x^2$ 都定义了函数 $0 \mapsto 0, 1 \mapsto 1$，但 $x \neq x^2$。

$\text{ev}_\alpha$ 的核是消失于 $\alpha$ 的多项式的理想。如果 $S$ 是整环且 $\alpha$ 是 $R[x]$ 中某个非零多项式的根，则核由 *最小多项式* 生成（即以 $\alpha$ 为根的最小次数的首一多项式）。例如，$\sqrt 2$ 在 $\mathbb{Q}$ 上的最小多项式是 $x^2 - 2$，所以 $\ker(\text{ev}_{\sqrt 2}) = (x^2 - 2)$ 在 $\mathbb{Q}[x]$ 中。商 $\mathbb{Q}[x]/(x^2 - 2) \cong \mathbb{Q}(\sqrt 2)$ 由第一同构定理得出。这是下一篇文章关于域扩张的代数基础。

### 除法算法

核心结论是：设 $F$ 是一个域，$f, g \in F[x]$，$g \neq 0$。存在唯一的 $q, r \in F[x]$ 使得

$$f = qg + r, \quad \deg r < \deg g.$$

证明（存在性）。对 $\deg f$ 进行归纳。如果 $\deg f < \deg g$，取 $q = 0, r = f$。否则：设 $f$ 和 $g$ 的最高次项分别为 $a_n x^n$ 和 $b_m x^m$ 且 $n \geq m$。设 $f_1 = f - (a_n/b_m) x^{n-m} g$。那么 $\deg f_1 < n$，应用归纳假设。（唯一性：如果有两个分解，减去并利用次数。）$\square$

除法算法需要除以最高次项系数，这就是为什么 $F$ 必须是域的原因。

![Polynomial long division step by step](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_1_division.png)

为什么这很重要。除法算法使 $F[x]$ 成为 Euclidean 整环（Euclidean 函数：次数）。这是算术基本定理的多项式类比，并是所有多项式 UFD 理论的技术基础。

算法结果：Euclidean 算法可以在最大次数线性时间内计算两个多项式的 GCD（假设算术运算是常数大小的）。这是计算机代数系统中多项式 GCD 计算的基础，并支持许多进一步的算法：部分分式分解、Sturm 序列用于实根计数、Berlekamp 在有限域上的因子分解算法。

推论。$F[x]$ 是 Euclidean 整环 $\Rightarrow$ PID $\Rightarrow$ UFD。$F[x]$ 中的每个理想都是主理想，由唯一的首一多项式生成。

因子定理。$\alpha \in F$ 是 $f$ 的根当且仅当 $(x - \alpha) \mid f$。（通过将 $f$ 除以 $x - \alpha$ 并在 $\alpha$ 处评估余项。）

推论。一个次数为 $n$ 的多项式在域上最多有 $n$ 个根。在 $\mathbb{Z}/8$ 上不成立：$x^2 - 1$ 有根 $1, 3, 5, 7$。

例题 1。在 $\mathbb{Q}[x]$ 中，将 $f = x^4 + 2x^3 - x + 3$ 除以 $g = x^2 + x - 1$。

步骤 1：$x^4/x^2 = x^2$。$f - x^2 g = x^3 + x^2 - x + 3$。

步骤 2：$x^3/x^2 = x$。余项：$3$。

结果：$q = x^2 + x, r = 3$。验证：$(x^2 + x)(x^2 + x - 1) + 3 = x^4 + 2x^3 - x + 3$。$\checkmark$

例题 1b。在 $\mathbb{Q}[x]$ 中，将 $f = x^3 + 1$ 除以 $g = x + 1$。

步骤 1：$x^3/x = x^2$，$f - x^2 g = -x^2 + 1$。

步骤 2：$-x^2/x = -x$，余项 $x + 1$。

步骤 3：$(x+1)/x = 1$ 等待，这给出一个非多项式。重新做一次。

步骤 2：$-x^2 - (-x)(x+1) = -x^2 + x^2 + x = x$。目前为止的余项：$x + 1$。

步骤 3：$x/x = 1$，$(x + 1) - 1 \cdot (x + 1) = 0$。所以 $q = x^2 - x + 1, r = 0$，确认 $x^3 + 1 = (x + 1)(x^2 - x + 1)$。

数值例子：通过 Euclidean 算法求 GCD。在 $\mathbb{Q}[x]$ 中，找到 $\gcd(x^4 - 1, x^3 - 1)$。除法：$x^4 - 1 = x \cdot (x^3 - 1) + (x - 1)$。然后 $\gcd(x^3 - 1, x - 1)$：$x^3 - 1 = (x^2 + x + 1)(x - 1) + 0$。所以 $\gcd = x - 1$。确认 $x^4 - 1 = (x-1)(x+1)(x^2+1)$ 和 $x^3 - 1 = (x-1)(x^2+x+1)$ 共享因子 $x-1$。

例题：Bezout 在 $\mathbb{Q}[x]$ 中。从上面的 Euclidean 算法中，$x - 1 = (x^4 - 1) - x \cdot (x^3 - 1)$。所以 $1 \cdot (x^4 - 1) + (-x)(x^3 - 1) = x - 1$，表示 GCD 为线性组合。这是多项式环中的 Bezout 恒等式，是部分分式分解和许多微积分及组合技巧背后的引擎。

数值例子：$\mathbb{Q}[x]$ 的理想和可除性。$(f) \subseteq (g)$ 当且仅当 $g \mid f$。所以 $\mathbb{Q}[x]$ 的理想格是多项式的可除性格的对偶。给定两个多项式，它们的 GCD 生成 $(f) + (g)$，它们的 LCM 生成 $(f) \cap (g)$。这是整数关系 $\gcd(m, n)\mathbb{Z} = m\mathbb{Z} + n\mathbb{Z}$ 和 $\text{lcm}(m, n)\mathbb{Z} = m\mathbb{Z} \cap n\mathbb{Z}$ 的多项式类比。

## 不可约性判别准则

想象一下，不可约多项式就像是素数的多项式版本。一个多项式是否不可约很大程度上取决于基环：$x^2 + 1$ 在 $\mathbb{R}$ 上是不可约的，在 $\mathbb{C}$ 上可以分解为 $(x+i)(x-i)$，在 $\mathbb{F}_5$ 上可以分解为 $(x+2)(x+3)$。

定义是这样的：如果 $f \in R[x]$ 的次数大于零，并且 $f = gh$ 在 $R[x]$ 中成立时，$g$ 或 $h$ 必须是单位元，那么 $f$ 就是 $R$ 上的不可约多项式。

在一个域 $F$ 上，$F[x]$ 的单位元是非零常数，因此不可约性意味着：$f$ 不能分解成两个次数严格小于 $f$ 的多项式的乘积。

依赖于基环并不是麻烦事，而是关键所在。“在哪些域上 $f$ 可以分解？”这个问题是 Galois 理论的起点。多项式在不同素数模下的分解模式通过 Chebotarev 密度定理与它的 Galois 群联系起来。

### 有理根测试

定理是这样的：如果 $f = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$ 有一个有理根 $p/q$（最简形式），那么 $p \mid a_0$ 且 $q \mid a_n$。

证明方法是：将 $f(p/q) = 0$ 两边乘以 $q^n$，利用 $p, q$ 互质。

举个例子：$f(x) = x^3 - 3x + 1$ 是否在 $\mathbb{Q}$ 上不可约？可能的有理根是 $\pm 1$。计算 $f(1) = -1$，$f(-1) = 3$。没有有理根。因为 $f$ 的次数是 3，若可约则必有一个一次因子，从而有一个有理根。但不存在这样的有理根。所以 $f$ 是不可约的。

数值例子：$f(x) = 2x^3 + x^2 - 7x - 6$。可能的有理根是 $\pm 1, \pm 2, \pm 3, \pm 6, \pm 1/2, \pm 3/2$。检查发现 $f(2) = 16 + 4 - 14 - 6 = 0$。所以 $(x - 2)$ 是一个因子。进行除法：$f = (x - 2)(2x^2 + 5x + 3) = (x-2)(2x+3)(x+1)$。三个一次因子在 $\mathbb{Z}$ 上。

另一个数值例子：$f = 6x^3 - 7x^2 + 1$。可能的有理根是 $\pm 1, \pm 1/2, \pm 1/3, \pm 1/6$。计算 $f(1) = 0$。所以 $(x - 1)$ 是一个因子。进行除法：$f = (x - 1)(6x^2 - x - 1)$。二次项可以分解为 $(3x + 1)(2x - 1)$。所以 $f = (x-1)(3x+1)(2x-1)$。

为什么这很重要？这是最便宜、最快的测试方法。对于 $\mathbb{Q}[x]$ 中的三次多项式，有理根测试足以确定其可约性（因为可约性要求存在一次因子）。对于更高次的多项式，它排除了一次因子的可能性，但仍有可能存在更高次的不可约因子。

还有一个重要教训：有理根受到整系数的限制。具有整系数的多项式生活在一个离散的世界中，其有理根必须位于由系数决定的一个有限列表中。这种有限性使得测试变得可行。

### Eisenstein 判别法

定理是这样的：设 $f = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$（$n \geq 1$）。如果一个素数 $p$ 满足：
1. $p \nmid a_n$，
2. $p \mid a_i$ 对所有 $i < n$ 成立，
3. $p^2 \nmid a_0$，

那么 $f$ 在 $\mathbb{Q}$ 上是不可约的。

证明方法是：如果 $f = gh$ 在 $\mathbb{Z}[x]$ 中成立（Gauss 引理），则对 $p$ 取模：$\bar f = \bar a_n x^n$。根据 $\mathbb{F}_p[x]$ 中的唯一分解性，$\bar g$ 和 $\bar h$ 必须是 $x$ 的幂次。因此 $p \mid g(0), h(0)$，从而 $p^2 \mid a_0 = g(0)h(0)$，与条件 (3) 矛盾。证毕。

![Eisenstein 判别法应用于一个具体的多项式](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_4_eisenstein.png)

为什么这很重要？Eisenstein 判别法提供了一个快速的充分（但不必要）测试。当它适用时，不可约性立即得到；当它不适用时，可以尝试代换如 $x \to x + 1$ 或者使用模 $p$ 还原。

Eisenstein 判别法的伪装形式：多项式 $f(x)$ 不可约当且仅当 $f(x + a)$ 不可约，其中 $a$ 是任意常数。所以如果 Eisenstein 判别法在 $f(x)$ 上失败，可以尝试 $f(x + 1), f(x - 1), f(x + 2), \ldots$。有时适当的平移会揭示出隐藏的 Eisenstein 结构。

例如：分圆多项式 $\Phi_p(x) = x^{p-1} + \cdots + 1$ 在 $\mathbb{Q}$ 上是不可约的。代换 $x \to x + 1$：

$$\Phi_p(x + 1) = \frac{(x+1)^p - 1}{x} = x^{p-1} + \binom{p}{1} x^{p-2} + \cdots + \binom{p}{p-1}.$$

Eisenstein 判别法适用于素数 $p$：$\binom{p}{k}$ 被 $p$ 整除，对于 $1 \leq k \leq p-1$，最高次项系数为 1，常数项为 $p$（不是 $p^2$）。

### 模 $p$ 还原

定理是这样的：设 $f \in \mathbb{Z}[x]$ 是首一多项式，次数为 $n$。如果 $\bar f \in \mathbb{F}_p[x]$ 在某个素数 $p$ 下是不可约的，那么 $f$ 在 $\mathbb{Q}$ 上也是不可约的。

证明方法是：如果 $f = gh$ 在 $\mathbb{Z}[x]$ 中成立，且 $\deg g, \deg h \geq 1$（并且首一，因为 $f$ 是首一的），那么 $\bar f = \bar g \bar h$ 且次数相同。这与 $\bar f$ 的不可约性矛盾。

逆命题不成立：$x^4 + 1$ 在 $\mathbb{Q}$ 上是不可约的，但在模每个素数都是可约的。

为什么这很重要？模 $p$ 还原是数论中最强大的技术之一：它将无限的、困难的问题（在 $\mathbb{Z}$ 上的分解）转化为有限的、机械的问题（在 $\mathbb{F}_p$ 上的分解）。当它适用时，可以快速证明不可约性。这个策略也是现代算术几何的原型：通过模每个素数 $p$ 来研究全局问题，并组装局部数据。

不可约性的策略如下：
1. 有理根测试（完全处理次数 $\leq 3$ 的情况）。
2. Eisenstein 判别法（尝试小素数；如果需要，代换 $x \to x + a$）。
3. 模小素数还原。
4. 直接比较系数。

对于高次多项式，有时这些方法都不够。在这种情况下，多项式的结构可能会给出线索：分圆多项式、形如 $f(x) g(x) + 1$ 的多项式，或与线性递推相关的多项式。专门的技术（如 Newton 多边形、$p$-adic 分析）可以处理更难的情况。计算机代数系统如 Pari/GP、SageMath 和 Magma 可以算法地分解大次数多项式在许多环上的因子；对于“这个多项式是否不可约”的研究问题，这些工具是不可或缺的。

举个例子：$f = x^4 + x + 1$ 是否在 $\mathbb{Q}$ 上不可约？

模 2：$\bar f = x^4 + x + 1$ 在 $\mathbb{F}_2$ 中没有根（$\bar f(0) = 1, \bar f(1) = 1$）。如果它可约，必须是两个二次多项式的乘积。$\mathbb{F}_2[x]$ 中唯一的不可约二次多项式是 $x^2 + x + 1$。因此需要 $\bar f = (x^2 + x + 1)^2 = x^4 + x^2 + 1 \neq x^4 + x + 1$。矛盾。所以 $\bar f$ 在 $\mathbb{F}_2$ 上不可约，从而 $f$ 在 $\mathbb{Q}$ 上也不可约。

## 唯一分解整环

心理图像：UFD 是一个整环，其中每个元素都有“素因子分解”，唯一性只考虑相伴和顺序。UFD 的算术本质上与 $\mathbb{Z}$ 一样良好。

整环 $R$ 是 UFD，如果每个非零非单位元素都可以分解成不可约元素，唯一性只考虑相伴和顺序。

举个例子。$\mathbb{Z}$, $F[x]$ 对于 $F$ 是一个域, $\mathbb{Z}[i]$, $\mathbb{Z}[x]$。

对于多项式环：$\mathbb{Q}[x], \mathbb{R}[x], \mathbb{C}[x], \mathbb{F}_p[x]$ 都是 UFD（并且是 PID，因为基是域）。$\mathbb{Z}[x]$ 是 UFD 但不是 PID。$\mathbb{Q}[x, y]$ 是 UFD 但不是 PID。一旦超过一个变量，“好”环的层次就会无限细分。

可以证明，每个 PID 都是 UFD。

证明分为两部分：存在性（每个非零非单位元素都可以分解）使用升链条件（没有无穷的真因子链）。唯一性使用 PID 中不可约元素是素元素的事实（$p \mid ab \Rightarrow p \mid a$ 或 $p \mid b$）。

假设 $a$ 没有分解。则 $a = a_1 b_1$ 且 $a_1, b_1$ 都是非单位，且（不失一般性）$a_1$ 没有分解。迭代得到 $a, a_1, a_2, \ldots$ 且 $(a) \subsetneq (a_1) \subsetneq (a_2) \subsetneq \cdots$，这是一个升链。PID 是 Noetherian，矛盾。

*唯一性的草图。* 在 PID 中，给定一个不可约元素 $p$，理想 $(p)$ 是极大理想（任何更大的主理想 $(a)$ 包含 $p$ 有 $p = ax$，因此由不可约性 $a$ 或 $x$ 是单位，给出 $(a) = (p)$ 或 $(a) = R$）。极大理想 $\Rightarrow$ 素理想。因此在 PID 中，不可约 $\Rightarrow$ 素，这给出了通过标准论证的唯一因子分解：如果 $p_1 \cdots p_k = q_1 \cdots q_m$，则 $p_1 \mid q_1 \cdots q_m$，因此由素性 $p_1 \mid q_j$ 对于某个 $j$，即 $p_1$ 与 $q_j$ 相伴。消去并归纳。

![UFD 中的唯一因子分解为有限链](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_5_ufd_chain.png)

UFD 结构使“初等数论”推广到其他环：gcd、lcm、素因子分解都适用。UFD 的失败（如在 $\mathbb{Z}[\sqrt{-5}]$ 中）表明更深的算术复杂性，由 Dedekind 的理想理论解决。

实际后果：在 UFD $R$ 中，有理函数域 $\text{Frac}(R)$ 有干净的部分分式分解。$\text{Frac}(R)$ 中的每个元素都可以写成形式为 $u/p^k$ 的项之和，其中 $p$ 是素元素。这是微积分中所有部分分式技巧的代数根源，推广到任意 UFD。

### 失败：$\mathbb{Z}[\sqrt{-5}]$

$6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})$。

所有四个因子都是不可约的。

*对于 $2$:* 如果 $2 = \alpha\beta$，则 $N(\alpha)N(\beta) = N(2) = 4$。如果都不是单位，则 $N(\alpha) = N(\beta) = 2$。但 $a^2 + 5b^2 = 2$ 没有整数解。因此 $2$ 是不可约的。类似地，其他因子也是如此。

四个因子两两不相伴（不同的范数或范数不相关）。因此 $6$ 有两个本质不同的不可约因子分解。UFD 失败。

为什么？在 $\mathbb{Z}[\sqrt{-5}]$ 中，不可约 $\not\Rightarrow$ 素。$2$ 是不可约的，但 $2 \mid (1+\sqrt{-5})(1-\sqrt{-5}) = 6$ 而 $2 \nmid (1 \pm \sqrt{-5})$ 在 $\mathbb{Z}[\sqrt{-5}]$ 中。不可约与素之间的差距正是 UFD 失败的地方。

类群衡量一个数环距离 PID 有多远。类数为 1 意味着 PID；对于 $\mathbb{Z}[\sqrt{-5}]$ 类数为 2。

Dedekind 的洞察：尽管元素可能有多个不可约因子分解，*理想* 在任何“Dedekind 整环”（包含 $\mathbb{Z}[\sqrt{-5}]$ 和大多数代数整数环）中唯一地分解为素理想的乘积。对于 $\mathbb{Z}[\sqrt{-5}]$：

$$(2) = \mathfrak{p}^2, \quad (3) = \mathfrak{q}_1 \mathfrak{q}_2, \quad (1 + \sqrt{-5}) = \mathfrak{p}\mathfrak{q}_1, \quad (1 - \sqrt{-5}) = \mathfrak{p}\mathfrak{q}_2,$$

其中 $\mathfrak{p} = (2, 1 + \sqrt{-5}), \mathfrak{q}_1 = (3, 1 + \sqrt{-5}), \mathfrak{q}_2 = (3, 1 - \sqrt{-5})$。因此 $(6) = \mathfrak{p}^2 \mathfrak{q}_1 \mathfrak{q}_2$，无论从哪个元素级因子分解开始，理想的因子分解都是相同的。歧义在于如何将素理想分组为元素，而不是素结构本身。




## 从 $\mathbb{Z}$ 到 $\mathbb{Z}[x]$ 再到 $\mathbb{Q}[x]$：Gauss 引理

定义 $f \in \mathbb{Z}[x]$ 的内容为 $c(f) = \gcd$ 系数。$f$ 是*本原的*如果 $c(f) = 1$。

对于 UFD $R$ 上的多项式，内容定义为系数的 gcd（在单位意义下是良定义的）。内容扩展为一个乘法函数：$c(fg) = c(f) c(g)$（这是 Gauss 引理的内容版本）。每个非零 $f \in R[x]$ 唯一地（在单位意义下）分解为 $f = c(f) \cdot f^*$，其中 $f^*$ 是本原的。

Gauss 引理指出，本原多项式的乘积是本原的。

*证明。* 如果 $fg$ 的所有系数都能被素数 $p$ 整除，模 $p$：$\bar f \bar g = 0$ 在 $\mathbb{F}_p[x]$ 中。整环 $\Rightarrow$ $\bar f = 0$ 或 $\bar g = 0$，与本原性矛盾。$\square$

进一步可得：$f \in \mathbb{Z}[x]$ 本原，$f = gh$ 在 $\mathbb{Q}[x]$ 中 $\Rightarrow$ 存在 $g^*, h^* \in \mathbb{Z}[x]$ 使得 $f = g^* h^*$，次数相同。

可以证明：$\mathbb{Z}[x]$ 是 UFD。更一般地，$R$ 是 UFD $\Rightarrow$ $R[x]$ 是 UFD。通过归纳 $R[x_1, \ldots, x_n]$ 是 UFD。

*证明 $\mathbb{Z}[x]$ 的草图。* 使用 $\mathbb{Z}$ 是 UFD 和 $\mathbb{Q}[x]$ 是 UFD。对于非零 $f \in \mathbb{Z}[x]$，写 $f = c(f) \cdot f^*$ 其中 $f^*$ 是本原的。通过 $\mathbb{Z}$-UFD 分解 $c(f)$，通过 $\mathbb{Q}[x]$-UFD 分解 $f^*$。根据 Gauss 的推论，$f^*$ 的因子可以在 $\mathbb{Z}[x]$ 中选取且是本原的。唯一性：结合 $\mathbb{Z}$-唯一性、$\mathbb{Q}[x]$-唯一性和 Gauss 引理。

同样的论证适用于任何 UFD $R$ 代替 $\mathbb{Z}$，使用 $\text{Frac}(R)$ 代替 $\mathbb{Q}$。

Gauss 引理是整系数和有理系数因子分解之间的桥梁。没有它，回答“这个 $\mathbb{Z}[x]$-多项式是否不可约？”的问题会更难使用 $\mathbb{Q}[x]$ 工具。

第二个后果：它告诉我们“因子清除分母。”如果 $f \in \mathbb{Z}[x]$ 是首一的并在 $\mathbb{Q}$ 上分解为首一因子，那么这些因子自动在 $\mathbb{Z}[x]$ 中。这就是为什么有理根测试产生整数根而不是仅仅有理根。

### 层次回顾

| 层次 | 示例 | 为什么这一层 |
|-------|---------|----------------|
| 域 | $\mathbb{Q}, \mathbb{F}_p$ | 每个非零元素可逆 |
| Euclidean，非域 | $\mathbb{Z}, F[x]$ | 除法算法存在 |
| PID，非 Euclidean | $\mathbb{Z}[(1+\sqrt{-19})/2]$ | 每个理想为主理想，无 Euclidean 函数 |
| UFD，非 PID | $\mathbb{Z}[x]$ | 唯一分解成立，$(2, x)$ 不为主理想 |
| 整环，非 UFD | $\mathbb{Z}[\sqrt{-5}]$ | $6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$ |

## 实际中的因式分解

**例4.** 在 $\mathbb{Q}$ 上分解 $f = 2x^4 + 3x^3 - 4x^2 - 3x + 2$。

有理根测试：候选根为 $\pm 1, \pm 2, \pm 1/2$。计算得 $f(1) = 0$，$f(-2) = 0$。除以 $(x-1)(x+2) = x^2 + x - 2$：

$$f = (x^2 + x - 2)(2x^2 + x - 1) = (x-1)(x+2)(2x-1)(x+1)$$

验证一下：展开 $(x-1)(x+2)(2x-1)(x+1) = (x^2+x-2)(2x^2+x-1)$。乘起来：$2x^4 + x^3 - x^2 + 2x^3 + x^2 - x - 4x^2 - 2x + 2 = 2x^4 + 3x^3 - 4x^2 - 3x + 2$。正确。

**例5.** $f = x^4 - 10x^2 + 1$ 在 $\mathbb{Q}$ 上是否不可约？

有理根测试：$\pm 1$ 都不行。

尝试二次因子：$(x^2 + ax + b)(x^2 + cx + d)$。比较系数得 $a + c = 0$，所以 $c = -a$。$bd = 1$。$a(d - b) = 0$，因此 $a = 0$ 或 $d = b$。

如果 $a = 0$：$b + d = -10, bd = 1$。方程 $t^2 + 10t + 1$ 的根是 $-5 \pm 2\sqrt 6$，无理数。

如果 $d = b$：$b^2 = 1$，所以 $b = \pm 1$。$a^2 = 12$ 或 $a^2 = 8$，都是无理数。

因此 $f$ 不可约。（这是 $\sqrt 2 + \sqrt 3$ 的最小多项式。）

**例5b.** 验证 $\sqrt 2 + \sqrt 3$ 是 $x^4 - 10x^2 + 1$ 的根。设 $\alpha = \sqrt 2 + \sqrt 3$。$\alpha^2 = 2 + 2\sqrt 6 + 3 = 5 + 2\sqrt 6$。$\alpha^2 - 5 = 2\sqrt 6$。$(\alpha^2 - 5)^2 = 24$。所以 $\alpha^4 - 10\alpha^2 + 25 = 24$，即 $\alpha^4 - 10\alpha^2 + 1 = 0$。正确。

因为 $\mathbb{Q}(\sqrt 2, \sqrt 3)$ 在 $\mathbb{Q}$ 上的次数是 $4$（四个基元素 $1, \sqrt 2, \sqrt 3, \sqrt 6$ 线性无关），$\alpha$ 的最小多项式的次数整除 $4$。由于 $\alpha \notin \mathbb{Q}$ 和 $\alpha^2 \notin \mathbb{Q}$，最小多项式的次数大于 $2$。结合 $f(\alpha) = 0$ 和 $\deg f = 4$，$f$ 就是最小多项式。

**例6.** $f = x^5 + 6x^4 + 9x^3 + 12x + 3$。用 Eisenstein 判别法，取 $p = 3$：最高次项系数 $1$（$3 \nmid 1$）；$6, 9, 0, 12, 3$ 都能被 $3$ 整除；常数项 $3$ 不能被 $9$ 整除。Eisenstein 判别法适用，所以 $f$ 不可约。

**例：当 Eisenstein 直接失效时。** $f = x^4 + 1$。没有素数 $p$ 能直接应用 Eisenstein 判别法。但代换 $x \to x + 1$：$(x+1)^4 + 1 = x^4 + 4x^3 + 6x^2 + 4x + 2$。用 Eisenstein 判别法，取 $p = 2$：最高次项系数 $1$，其他项 $4, 6, 4, 2$ 都能被 $2$ 整除，常数项 $2$ 不能被 $4$ 整除。所以 $(x+1)^4 + 1$ 不可约，从而 $x^4 + 1$ 也不可约。（代换保持不可约性，因为 $x \to x + 1$ 是 $\mathbb{Q}[x]$ 的环自同构。）

**例：$\Phi_5(x) = x^4 + x^3 + x^2 + x + 1$。** 用代换后的 Eisenstein 判别法：$\Phi_5(x+1) = ((x+1)^5 - 1)/x$。直接展开：$(x+1)^5 - 1 = x^5 + 5x^4 + 10x^3 + 10x^2 + 5x$。除以 $x$：$x^4 + 5x^3 + 10x^2 + 10x + 5$。用 Eisenstein 判别法，取 $p = 5$：所有条件都满足。所以 $\Phi_5$ 不可约。这是关于分圆多项式的普遍事实的一个特例，$p = 5$。

**例7.** 在 $\mathbb{Q}$ 上分解 $x^6 - 1$。根是六次单位根。分解为：

$$x^6 - 1 = (x^3 - 1)(x^3 + 1) = (x - 1)(x^2 + x + 1)(x + 1)(x^2 - x + 1)$$

四个不可约因子。每个分圆多项式 $\Phi_d$ 对于 $d \mid 6$ 贡献一次：$\Phi_1 = x - 1, \Phi_2 = x + 1, \Phi_3 = x^2 + x + 1, \Phi_6 = x^2 - x + 1$。

一般公式：$x^n - 1 = \prod_{d \mid n} \Phi_d(x)$。对于 $n = 12$：$x^{12} - 1 = \Phi_1 \Phi_2 \Phi_3 \Phi_4 \Phi_6 \Phi_{12}$。次数总和：$1 + 1 + 2 + 2 + 2 + 4 = 12$。正确。

这种分解反映了 $n$ 次单位根按其阶 $d \mid n$ 划分的几何事实。每个 $\Phi_d$ 收集了原始的 $d$ 次单位根。

![三次单位根在复平面上的分布](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_7_root_complex.png)

**例8.** $x^3 - 2$ 在 $\mathbb{C}$ 上的根：$\sqrt[3]{2}, \sqrt[3]{2}\omega, \sqrt[3]{2}\omega^2$，其中 $\omega = e^{2\pi i/3}$。这三个点在复平面上形成一个等边三角形，位于半径为 $\sqrt[3]{2}$ 的圆上。在 $\mathbb{Q}$ 上，$x^3 - 2$ 不可约（用 Eisenstein 判别法，取 $p = 2$）。分裂域是 $\mathbb{Q}(\sqrt[3]{2}, \omega)$，是 $\mathbb{Q}$ 的六次扩张，Galois 群是 $S_3$。

**例9.** 在 $\mathbb{Q}$ 和 $\mathbb{R}$ 上分解 $x^4 - 4$。在 $\mathbb{Q}$ 上：$x^4 - 4 = (x^2 - 2)(x^2 + 2)$。两个因子在 $\mathbb{Q}$ 上都不可约（判别式 $\pm 8$）。在 $\mathbb{R}$ 上：$x^2 - 2 = (x - \sqrt 2)(x + \sqrt 2)$，但 $x^2 + 2$ 仍然不可约（没有实根）。在 $\mathbb{C}$ 上：完全分解为线性因子 $(x - \sqrt 2)(x + \sqrt 2)(x - i\sqrt 2)(x + i\sqrt 2)$。随着域的扩大，分解越来越细。

**例10.** 在 $\mathbb{Z}$ 上分解 $x^4 + 4$。尝试 Sophie Germain 方法：$x^4 + 4 = x^4 + 4x^2 + 4 - 4x^2 = (x^2 + 2)^2 - (2x)^2 = (x^2 + 2x + 2)(x^2 - 2x + 2)$。两个二次因子。每个在 $\mathbb{Z}$ 上都不可约（判别式 $4 - 8 = -4$，没有实根）。所以 $x^4 + 4$ 在 $\mathbb{Z}$ 上可约，尽管 $x^4 + 1$ 不可约。这种分解不明显；这是不可约性问题中的经典陷阱之一。

**例11.** 在 $\mathbb{Z}[x, y]$ 中分解 $f(x, y) = x^2 - y^2$。利用平方差公式：$x^2 - y^2 = (x - y)(x + y)$。两个因子都不可约（一次多项式，内容为 $1$）。所以 $f$ 有两个不可约因子。相比之下，$x^2 + y^2$ 在 $\mathbb{Z}[x, y]$ 中不可约（因为它在 $\mathbb{R}[x, y]$ 中不可约，所以在 $\mathbb{Q}[x, y]$ 中也不可约，再由 Gauss 引理，在 $\mathbb{Z}[x, y]$ 中也不可约）。

**例12.** 在 $\mathbb{Q}$ 上分解 $f(x, y, z) = x^2 + y^2 + z^2 - xy - yz - xz$。这是一个对称二次型。尝试 $f(x, x, x) = 3x^2 - 3x^2 = 0$，所以 $f$ 在对角线上消失。这提示 $(x - y)$ 可能是 $f$ 的因子（将 $f$ 视为 $x$ 的多项式）。计算：$f$ 作为 $x$ 的多项式是 $x^2 - (y + z) x + (y^2 + z^2 - yz)$。判别式：$(y + z)^2 - 4(y^2 + z^2 - yz) = y^2 + 2yz + z^2 - 4y^2 - 4z^2 + 4yz = -3y^2 + 6yz - 3z^2 = -3(y - z)^2$。所以根是 $x = ((y + z) \pm (y - z)\sqrt{-3})/2$。不在 $\mathbb{Q}$ 中。所以 $f$ 在 $\mathbb{Q}$ 上不可约。在 $\mathbb{Q}(\sqrt{-3})$ 上，它分解为两个 $x$ 的线性因子（系数在 $\mathbb{Q}(\sqrt{-3})[y, z]$ 中）。

**例13.** $f(x) = x^p - x \in \mathbb{F}_p[x]$，其中 $p$ 是素数。在 $\mathbb{F}_p$ 上，$\mathbb{F}_p$ 的每个元素都是根（由 Fermat 小定理：$a^p = a$ 对于 $a \in \mathbb{F}_p$ 成立）。所以 $f(x) = \prod_{a \in \mathbb{F}_p}(x - a)$，是一个 $p$ 个线性因子的乘积。多项式 $x^p - x$ 有 $\mathbb{F}_p$ 中的所有元素作为根，这很不寻常：一个 $p$ 次多项式在 $p$ 元素的域中有 $p$ 个根。

**例14.** 在 $\mathbb{F}_2$ 上分解 $x^4 + x^3 + x^2 + x + 1$。这是 $\Phi_5(x)$，第五分圆多项式。在 $\mathbb{F}_2$ 上，看 $2$ 在模 $5$ 下的阶：$2, 4, 3, 1$（$2$ 的幂模 $5$），所以阶是 $4$。根据有限域上的分圆多项式理论，$\Phi_5(x)$ 在 $\mathbb{F}_2$ 上不可约当且仅当 $\text{ord}_5(2) = 4 = \deg \Phi_5$。确实如此。所以 $\Phi_5(x)$ 在 $\mathbb{F}_2$ 上不可约。直接验证：它在 $\mathbb{F}_2$ 中没有根（$f(0) = 1, f(1) = 5 = 1$）；如果它分解为两个二次多项式，唯一的不可约二次多项式是 $x^2 + x + 1$，而 $(x^2 + x + 1)^2 = x^4 + x^2 + 1$，不等于 $\Phi_5$。所以不可约。

**例15.** 在 $\mathbb{F}_3$ 上分解 $x^9 - x$。在 $\mathbb{F}_9$ 上，$x^9 - x$ 有 $\mathbb{F}_9$ 中所有 $9$ 个元素作为根，所以 $x^9 - x = \prod_{\alpha \in \mathbb{F}_9}(x - \alpha)$ 在 $\mathbb{F}_9$ 上成立。在 $\mathbb{F}_3$ 上，按 Frobenius 映射 $\alpha \mapsto \alpha^3$ 的轨道分组。不动点是 $\mathbb{F}_3$（三个元素，贡献三个线性因子）。剩下的 $6$ 个元素分成 $3$ 个大小为 $2$ 的轨道（因为它们的最小多项式次数为 $2$）。所以分解为 $3$ 个线性因子 $\times$ $3$ 个不可约二次因子。总次数：$3 \cdot 1 + 3 \cdot 2 = 9$。正确。

## 接下来的内容

我们已经介绍了初等环论的两大支柱：一般环、理想和商（第五篇文章），以及多项式环和因子分解（这篇文章）。这个层次结构

$$\text{域} \subset \text{欧几里得整环} \subset \text{主理想整环} \subset \text{唯一因子分解整环} \subset \text{整环}$$

已经完全建立，包含关系严格，且因子分解保证逐步增加。

下一篇文章将探讨*域论和扩张*：如何从较小的域构建更大的域？答案涉及多项式环的商 $F[x]/(f(x))$，这引出了代数扩张、分裂域和Galois理论。

值得记住的总结：多项式环把代数变成了一种计算游戏。除法算法是引擎，因子分解是目标，不可约性测试是工具箱。当一切顺利时，我们得到唯一因子分解整环；当不顺利时，我们就进入了更深的代数数论领域。

*预览一下这将导向何处*。$x^2 + 1 \in \mathbb{R}[x]$ 是不可约的，所以 $\mathbb{R}[x]/(x^2 + 1) \cong \mathbb{C}$ 是一个域。$x^2 + 1 \in \mathbb{Q}[x]$ 也是不可约的，所以 $\mathbb{Q}[x]/(x^2 + 1) \cong \mathbb{Q}(i)$ 是 $\mathbb{Q}$ 的一个二次扩域。在这两种情况下，哪些多项式是不可约的决定了哪些域扩张存在。Galois理论研究这些扩张的对称群，多项式因子分解与群论之间的桥梁是经典代数中最深刻的概念之一。这就是我们要去的方向。

*阅读推荐*。Dummit和Foote的第9-10章涵盖了多项式环和域扩张。Hungerford的《代数》内容更密集但非常优秀。对于计算角度，Cox-Little-O'Shea的《理想、簇和算法》介绍了Gröbner基和多元多项式环的算法方面。关于不可约性测试，Lang的《代数》第4章是一个经典之作，包含许多实例。

*最后一点反思*。多项式因子分解是这样一个主题，每本教科书都列出相同的定理，但实际操作需要亲自动手熟悉。要内化不可约性测试，不是靠记忆而是通过在一百个例子上应用它们。做了五十个之后，你开始识别结构：分圆形状、Eisenstein候选者经过替换后的情况、模 $2$ 可以分解但模 $3$ 不行的多项式。模式识别才是真正的技能，而定理只是它的脚手架。

第二个反思：多项式环理论是现代代数锻造的工坊。Galois在20岁时就草拟了多项式Galois理论；Dedekind发明了理想来处理数环中的因子分解；Hilbert的基定理开启了交换代数；Noether围绕商环和模重新组织了所有这些内容，这是现代学科的语言。你在经典代数中遇到的每一个名字背后，都有一个多项式环作为核心故事的一部分。

最后一条具体建议：当遇到多项式不可约性问题时，按顺序尝试有理根测试、几个小素数的模约简、几个素数上的Eisenstein判别法（必要时进行替换），以及直接系数比较。如果仍然卡住，寻找特殊结构：分圆因子、二项式的乘积、形式为 $f(x^k)$ 的多项式。大多数教科书中的问题可以通过这些方法解决。
