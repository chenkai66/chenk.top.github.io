多项式是代数的实验室。几乎每一个环论概念——理想、商、因子分解、不可约性——都是通过多项式例子首先被理解的，然后才被抽象化。这不是巧合：多项式环足够丰富，可以展示所有有趣的现象，同时结构又足够明确，允许显式计算。

在这篇文章中，我们研究环 $R[x]$ 上的多项式。我们发展了除法算法，建立了不可约性标准（Eisenstein 准则、模 $p$ 还原、有理根测试），定义了唯一因子分解整环，并证明了 Gauss 引理——这是从 $\mathbb{Z}$ 到 $\mathbb{Q}$ 因子分解的桥梁。最终，将清楚地了解何时及为何唯一因子分解成立，以及当它失败时发生了什么。

## 为什么多项式在代数中处于核心地位

心理图像：一个多项式 $f(x) \in R[x]$ 是系数在 $R$ 中的“形式表达式”，但它也是一个函数（当在点上求值时）、理想的生成元，以及代数关系的见证。每个视角揭示了同一对象的不同方面。

多项式值得专门研究的三个原因：

1. 多项式生成所有代数扩张：如果 $\alpha$ 是某个有理系数多项式的根，则 $\mathbb{Q}(\alpha) \cong \mathbb{Q}[x]/(p(x))$，其中 $p$ 是 $\alpha$ 的最小多项式。多项式因子分解控制域扩张。

2. 多项式是一个通用的测试场：$\mathbb{Z}[x]$ 是 UFD 但不是 PID，因此可以研究这两个概念之间的差距。根据 Hilbert 基定理，Noether 环上的多项式环也是 Noether 环。

3. 多项式连接代数和几何：多项式的零点形成代数簇——这是代数几何的起点。环论性质（理想、商）对应于几何性质（子簇、交集）。

## $R[x]$：定义、次数和除法算法

心理图像：加法是逐系数进行的；乘法是系数序列的“卷积”。次数像乘法幺半群中的加性赋值。

设 $R$ 是一个交换幺环。*多项式环* $R[x]$ 由形式表达式

$$f(x) = a_n x^n + \cdots + a_1 x + a_0$$

组成，其中 $a_i \in R$。运算：逐系数加法和卷积乘法。

卷积乘法：$(f \cdot g)_k = \sum_{i + j = k} f_i g_j$。所以 $(2x + 3)(x^2 + x + 1) = 2x^3 + 2x^2 + 2x + 3x^2 + 3x + 3 = 2x^3 + 5x^2 + 5x + 3$。卷积结构使多项式乘法成为一种结合且分配的运算，类似于形式幂级数的乘法。

*次数* $\deg f$ 是最大的 $n$ 使得 $a_n \neq 0$。*首项系数* 是 $a_n$；如果 $a_n = 1$，则 $f$ 是*首一的*。

如果 $R$ 是整环，则 $\deg(fg) = \deg f + \deg g$。（对于有零因子的环失效：$(2x)(2x) = 0$ 在 $(\mathbb{Z}/4)[x]$ 中。）

进一步可得：$R$ 是整环 $\Rightarrow$ $R[x]$ 是整环。

$R[x_1, \ldots, x_n] := R[x_1, \ldots, x_{n-1}][x_n]$，递归定义。在代数几何中非常重要。

$R[x_1, \ldots, x_n]$ 中的一个单项式是 $x_1^{e_1} \cdots x_n^{e_n}$，其中 $e_i \in \mathbb{Z}_{\geq 0}$。单项式的集合与 $\mathbb{Z}_{\geq 0}^n$ 一一对应，而 $R[x_1, \ldots, x_n]$ 是该集合上的自由 $R$-模。因此，一个元素是有限个单项式的 $R$-线性组合。单项式的总次数是 $e_1 + \cdots + e_n$。多变量多项式是 Gröbner 基（理想的算法基础）和结式（消去系统中的变量）等构造的家园。

对于 $\alpha \in S \supseteq R$，映射 $\text{ev}_\alpha: R[x] \to S, f \mapsto f(\alpha)$ 是一个环同态。这将形式多项式与函数联系起来。两个多项式可能给出相同的函数，但作为多项式不相等：在 $\mathbb{F}_2[x]$ 中，$x$ 和 $x^2$ 都定义了函数 $0 \mapsto 0, 1 \mapsto 1$，但 $x \neq x^2$。

$\text{ev}_\alpha$ 的核是消失在 $\alpha$ 处的多项式的理想。如果 $S$ 是整环且 $\alpha$ 是 $R[x]$ 中某个非零多项式的根，则核由 $\alpha$ 的*最小多项式*生成（即以 $\alpha$ 为根的次数最小的首一多项式）。例如，$\sqrt 2$ 在 $\mathbb{Q}$ 上的最小多项式是 $x^2 - 2$，因此 $\ker(\text{ev}_{\sqrt 2}) = (x^2 - 2)$ 在 $\mathbb{Q}[x]$ 中。商 $\mathbb{Q}[x]/(x^2 - 2) \cong \mathbb{Q}(\sqrt 2)$ 由第一同构定理得出。这是下一篇文章关于域扩张的代数基础。

### 除法算法

可以证明：设 $F$ 是一个域，$f, g \in F[x]$，$g \neq 0$。存在唯一的 $q, r \in F[x]$ 使得

$$f = qg + r, \quad \deg r < \deg g.$$

*证明（存在性）。* 对 $\deg f$ 归纳。如果 $\deg f < \deg g$，取 $q = 0, r = f$。否则：设首项分别为 $a_n x^n$ 对于 $f$ 和 $b_m x^m$ 对于 $g$ 且 $n \geq m$。设 $f_1 = f - (a_n/b_m) x^{n-m} g$。则 $\deg f_1 < n$，应用归纳假设。（唯一性：如果有两种分解，相减并使用次数。）$\square$

除法算法需要除以首项系数，这就是为什么 $F$ 必须是域的原因。

![多项式长除法步骤](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_1_division.png)

这很关键，因为除法算法使 $F[x]$ 成为 Euclidean 整环（Euclidean 函数：次数）。这是算术基本定理的多项式类比，是所有多项式 UFD 理论的技术基础。

算法后果：Euclidean 算法可以在最大次数的线性时间内计算两个多项式的 GCD（假设算术操作是常数大小的）。这是计算机代数系统中多项式 GCD 计算的基础，并支持许多进一步的算法：部分分式分解、Sturm 链用于实根计数、Berlekamp 在有限域上的因式分解算法。

进一步可得：$F[x]$ 是 Euclidean 整环 $\Rightarrow$ PID $\Rightarrow$ UFD。$F[x]$ 中的每个理想都是主理想，由唯一的首一多项式生成。

$\alpha \in F$ 是 $f$ 的根 $\iff (x - \alpha) \mid f$。（通过将 $f$ 除以 $x - \alpha$ 并在 $\alpha$ 处求余数。）

进一步可得：次数为 $n$ 的多项式在域上有至多 $n$ 个根。在 $\mathbb{Z}/8$ 上失效：$x^2 - 1$ 有根 $1, 3, 5, 7$。

在 $\mathbb{Q}[x]$ 中将 $f = x^4 + 2x^3 - x + 3$ 除以 $g = x^2 + x - 1$。

步骤 1: $x^4/x^2 = x^2$。$f - x^2 g = x^3 + x^2 - x + 3$。

步骤 2: $x^3/x^2 = x$。余数：$3$。

结果：$q = x^2 + x, r = 3$。验证：$(x^2 + x)(x^2 + x - 1) + 3 = x^4 + 2x^3 - x + 3$。$\checkmark$

在 $\mathbb{Q}[x]$ 中将 $f = x^3 + 1$ 除以 $g = x + 1$。

步骤 1: $x^3/x = x^2$，$f - x^2 g = -x^2 + 1$。

步骤 2: $-x^2/x = -x$，余数 $x + 1$。

步骤 3: $(x+1)/x = 1$ 等待，这会得到一个非多项式。让我重做。步骤 2: $-x^2 - (-x)(x+1) = -x^2 + x^2 + x = x$。目前为止的余数：$x + 1$。步骤 3: $x/x = 1$，$(x + 1) - 1 \cdot (x + 1) = 0$。所以 $q = x^2 - x + 1, r = 0$，确认 $x^3 + 1 = (x + 1)(x^2 - x + 1)$。

在 $\mathbb{Q}[x]$ 中，找到 $\gcd(x^4 - 1, x^3 - 1)$。除法：$x^4 - 1 = x \cdot (x^3 - 1) + (x - 1)$。然后 $\gcd(x^3 - 1, x - 1)$：$x^3 - 1 = (x^2 + x + 1)(x - 1) + 0$。所以 $\gcd = x - 1$。确认 $x^4 - 1 = (x-1)(x+1)(x^2+1)$ 和 $x^3 - 1 = (x-1)(x^2+x+1)$ 共享因子 $x-1$。

从上面的 Euclidean 算法，$x - 1 = (x^4 - 1) - x \cdot (x^3 - 1)$。所以 $1 \cdot (x^4 - 1) + (-x)(x^3 - 1) = x - 1$，将 gcd 表达为线性组合。这是多项式环中的 Bezout 恒等式，是部分分式分解和许多微积分及组合技巧背后的引擎。

$(f) \subseteq (g)$ 当且仅当 $g \mid f$。因此，$\mathbb{Q}[x]$ 的理想格是多项式的可除性格的对偶。给定两个多项式，它们的 gcd 生成 $(f) + (g)$ 而它们的 lcm 生成 $(f) \cap (g)$。这是整数关系 $\gcd(m, n)\mathbb{Z} = m\mathbb{Z} + n\mathbb{Z}$ 和 $\text{lcm}(m, n)\mathbb{Z} = m\mathbb{Z} \cap n\mathbb{Z}$ 的多项式类比。

## 不可约性准则

心理图像：不可约多项式是素数的多项式类比。一个多项式是否不可约取决于基环：$x^2 + 1$ 在 $\mathbb{R}$ 上不可约，在 $\mathbb{C}$ 上分解为 $(x+i)(x-i)$，在 $\mathbb{F}_5$ 上分解为 $(x+2)(x+3)$。

$f \in R[x]$ 是正次数的 *不可约* 多项式，如果 $f = gh$ 在 $R[x]$ 中意味着 $g$ 或 $h$ 是单位。

在一个域 $F$ 上，$F[x]$ 的单位是非零常数，因此不可约性意味着：$f$ 不能分解为严格较小次数的多项式的乘积。

依赖于基环并不是麻烦：这是整个要点。“$f$ 在哪些域上可分解？”这个问题是 Galois 理论的种子。多项式在不同素数模下的分解模式通过 Chebotarev 密度定理与其 Galois 群相关联。

### 有理根测试

可以证明，如果 $f = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$ 有一个有理根 $p/q$ 且是最简分数，则 $p \mid a_0$ 且 $q \mid a_n$。

*证明。* 将 $f(p/q) = 0$ 乘以 $q^n$，利用互质的 $p, q$。

$f(x) = x^3 - 3x + 1$ 在 $\mathbb{Q}$ 上是否不可约？可能的有理根：$\pm 1$。$f(1) = -1$，$f(-1) = 3$。没有有理根。由于 $\deg f = 3$，可约性意味着有一个一次因子，从而有一个有理根。不存在这样的根。因此 $f$ 是不可约的。

数值示例：$f(x) = 2x^3 + x^2 - 7x - 6$：可能的有理根：$\pm 1, \pm 2, \pm 3, \pm 6, \pm 1/2, \pm 3/2$。检查：$f(2) = 16 + 4 - 14 - 6 = 0$。因此 $(x - 2)$ 是一个因子。除法：$f = (x - 2)(2x^2 + 5x + 3) = (x-2)(2x+3)(x+1)$。在 $\mathbb{Z}$ 上有三个一次因子。

数值示例：$f = 6x^3 - 7x^2 + 1$ 的有理根：可能的有理根：$\pm 1, \pm 1/2, \pm 1/3, \pm 1/6$。$f(1) = 0$。因此 $(x - 1)$ 是一个因子。除法：$f = (x - 1)(6x^2 - x - 1)$。二次多项式分解为 $(3x + 1)(2x - 1)$。因此 $f = (x-1)(3x+1)(2x-1)$。

这是最便宜、最快的测试。对于 $\mathbb{Q}[x]$ 中的三次多项式，有理根测试单独确定可约性（因为可约性迫使有一个一次因子）。对于更高次的多项式，它排除了一次因子，但留下了高次不可约因子的可能性。

它还有一个元教训：根的有理性受系数整性的约束。具有整数系数的多项式生活在离散的世界中，其有理根必须位于由系数决定的有限列表上。这种有限性使得测试可计算。

### Eisenstein 准则

可以证明，$f = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$ ($n \geq 1$)。如果素数 $p$ 满足：
1. $p \nmid a_n$，
2. $p \mid a_i$ 对于 $i < n$，
3. $p^2 \nmid a_0$，

则 $f$ 在 $\mathbb{Q}$ 上不可约。

*证明。* 如果 $f = gh$ 在 $\mathbb{Z}[x]$ 中（Gauss 引理见下文），模 $p$：$\bar f = \bar a_n x^n$。由 $\mathbb{F}_p[x]$ 中的唯一因子分解，$\bar g$ 和 $\bar h$ 是 $x$ 的幂。因此 $p \mid g(0), h(0)$，故 $p^2 \mid a_0 = g(0)h(0)$，与 (3) 矛盾。$\square$

![Eisenstein 准则应用于一个具体多项式](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_4_eisenstein.png)

Eisenstein 提供了一个快速的充分（但非必要）测试。当它适用时，不可约性立即成立；当它不适用时，尝试替换如 $x \to x + 1$ 或使用模 $p$ 还原。

Eisenstein 通过平移隐藏：多项式 $f(x)$ 不可约当且仅当 $f(x + a)$ 对任何常数 $a$ 不可约。因此，如果 Eisenstein 在 $f(x)$ 上失败，尝试 $f(x + 1), f(x - 1), f(x + 2), \ldots$。有时正确的平移会揭示隐藏的 Eisenstein 结构。

$\Phi_p(x) = x^{p-1} + \cdots + 1$ 在 $\mathbb{Q}$ 上不可约。替换 $x \to x + 1$：

$$\Phi_p(x + 1) = \frac{(x+1)^p - 1}{x} = x^{p-1} + \binom{p}{1} x^{p-2} + \cdots + \binom{p}{p-1}.$$

Eisenstein 适用于素数 $p$：$\binom{p}{k}$ 对于 $1 \leq k \leq p-1$ 可被 $p$ 整除，首项系数为 $1$，常数项为 $p$（不是 $p^2$）。

### 模 $p$ 还原

可以证明，$f \in \mathbb{Z}[x]$ 是首一的，次数为 $n$。如果 $\bar f \in \mathbb{F}_p[x]$ 对某个素数 $p$ 不可约，则 $f$ 在 $\mathbb{Q}$ 上不可约。

*证明。* 如果 $f = gh$ 在 $\mathbb{Z}[x]$ 中且 $\deg g, \deg h \geq 1$（并且是首一的，因为 $f$ 是首一的），则 $\bar f = \bar g \bar h$ 且次数相同。与 $\bar f$ 的不可约性矛盾。

逆命题不成立：$x^4 + 1$ 在 $\mathbb{Q}$ 上不可约但在每个素数模下都可约。

模 $p$ 还原是数论中最强大的技术之一：它将无限且困难的问题（在 $\mathbb{Z}$ 上的因子分解）转化为有限且机械的问题（在 $\mathbb{F}_p$ 上的因子分解）。当它适用时，它提供了不可约性的快速证明。这一策略也是现代算术几何的原型：通过模每个素数 $p$ 并组装局部数据来研究全局问题。


1. 有理根测试（完全处理次数 $\leq 3$ 的情况）。
2. Eisenstein（尝试小素数；如果需要，替换 $x \to x + a$）。
3. 模小素数还原。
4. 直接比较系数。

对于更高次的多项式，有时这些都不够。在这种情况下，多项式的结构可能会提供线索：分圆多项式、形如 $f(x) g(x) + 1$ 的多项式，或与线性递推相关的多项式。专门技术（Newton 多边形、$p$-adic 分析）处理更难的情况。计算机代数系统如 Pari/GP、SageMath 和 Magma 可以算法地在许多环上分解大次数多项式；对于研究中的“这个多项式是否不可约”问题，这些工具是不可或缺的。

$f = x^4 + x + 1$ 在 $\mathbb{Q}$ 上是否不可约？

模 2：$\bar f = x^4 + x + 1$ 在 $\mathbb{F}_2$ 中没有根（$\bar f(0) = 1, \bar f(1) = 1$）。如果它可分解，必须是两个二次多项式的乘积。$\mathbb{F}_2[x]$ 中唯一的不可约二次多项式是 $x^2 + x + 1$。因此需要 $\bar f = (x^2 + x + 1)^2 = x^4 + x^2 + 1 \neq x^4 + x + 1$。矛盾。因此 $\bar f$ 在 $\mathbb{F}_2$ 上不可约，故 $f$ 在 $\mathbb{Q}$ 上不可约。

## 唯一分解整环

心理图像：UFD 是一个整环，其中每个元素都有“素因子分解”，唯一性只考虑相伴和顺序。UFD 的算术本质上与 $\mathbb{Z}$ 一样良好。

整环 $R$ 是 UFD，如果每个非零非单位元素都可以分解成不可约元素，唯一性只考虑相伴和顺序。

举个例子。$\mathbb{Z}$, $F[x]$ 对于 $F$ 是一个域, $\mathbb{Z}[i]$, $\mathbb{Z}[x]$。

对于多项式环：$\mathbb{Q}[x], \mathbb{R}[x], \mathbb{C}[x], \mathbb{F}_p[x]$ 都是 UFD（并且是 PID，因为基是域）。$\mathbb{Z}[x]$ 是 UFD 但不是 PID。$\mathbb{Q}[x, y]$ 是 UFD 但不是 PID。一旦超过一个变量，“好”环的层次就会无限细分。

可以证明，每个 PID 都是 UFD。

证明分为两部分：存在性（每个非零非单位元素都可以分解）使用升链条件（没有无穷的真因子链）。唯一性使用 PID 中不可约元素是素元素的事实（$p \mid ab \Rightarrow p \mid a$ 或 $p \mid b$）。

假设 $a$ 没有分解。则 $a = a_1 b_1$ 且 $a_1, b_1$ 都是非单位，且（不失一般性）$a_1$ 没有分解。迭代得到 $a, a_1, a_2, \ldots$ 且 $(a) \subsetneq (a_1) \subsetneq (a_2) \subsetneq \cdots$，这是一个升链。PID 是 Noetherian，矛盾。

*唯一性的草图。* 在 PID 中，给定一个不可约元素 $p$，理想 $(p)$ 是极大理想（任何更大的主理想 $(a)$ 包含 $p$ 有 $p = ax$，因此由不可约性 $a$ 或 $x$ 是单位，给出 $(a) = (p)$ 或 $(a) = R$）。极大理想 $\Rightarrow$ 素理想。因此在 PID 中，不可约 $\Rightarrow$ 素，这给出了通过标准论证的唯一因子分解：如果 $p_1 \cdots p_k = q_1 \cdots q_m$，则 $p_1 \mid q_1 \cdots q_m$，因此由素性 $p_1 \mid q_j$ 对于某个 $j$，即 $p_1$ 与 $q_j$ 相伴。消去并归纳。

![UFD 中的唯一因子分解为有限链](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/06-polynomial-rings/aa_v2_06_5_ufd_chain.png)

UFD 结构使“初等数论”推广到其他环：gcd、lcm、素因子分解都适用。UFD 的失败（如在 $\mathbb{Z}[\sqrt{-5}]$ 中）表明更深的算术复杂性，由 Dedekind 的理想理论解决。

实际后果：在 UFD $R$ 中，有理函数域 $\text{Frac}(R)$ 有干净的部分分式分解。$\text{Frac}(R)$ 中的每个元素都可以写成形式为 $u/p^k$ 的项之和，其中 $p$ 是素元素。这是微积分中所有部分分式技巧的代数根源，推广到任意 UFD。

### 失败：$\mathbb{Z}[\sqrt{-5}]$

$6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})$。

所有四个因子都是不可约的。

*对于 $2$:* 如果 $2 = \alpha\beta$，则 $N(\alpha)N(\beta) = N(2) = 4$。如果都不是单位，则 $N(\alpha) = N(\beta) = 2$。但 $a^2 + 5b^2 = 2$ 没有整数解。因此 $2$ 是不可约的。类似地，其他因子也是如此。

四个因子两两不相伴（不同的范数或范数不相关）。因此 $6$ 有两个本质不同的不可约因子分解。UFD 失败。

为什么？在 $\mathbb{Z}[\sqrt{-5}]$ 中，不可约 $\not\Rightarrow$ 素。$2$ 是不可约的，但 $2 \mid (1+\sqrt{-5})(1-\sqrt{-5}) = 6$ 而 $2 \nmid (1 \pm \sqrt{-5})$ 在 $\mathbb{Z}[\sqrt{-5}]$ 中。不可约与素之间的差距正是 UFD 失败的地方。

类群衡量一个数环距离 PID 有多远。类数为 1 意味着 PID；对于 $\mathbb{Z}[\sqrt{-5}]$ 类数为 2。

Dedekind 的洞察：尽管元素可能有多个不可约因子分解，*理想* 在任何“Dedekind 整环”（包含 $\mathbb{Z}[\sqrt{-5}]$ 和大多数代数整数环）中唯一地分解为素理想的乘积。对于 $\mathbb{Z}[\sqrt{-5}]$：

$$(2) = \mathfrak{p}^2, \quad (3) = \mathfrak{q}_1 \mathfrak{q}_2, \quad (1 + \sqrt{-5}) = \mathfrak{p}\mathfrak{q}_1, \quad (1 - \sqrt{-5}) = \mathfrak{p}\mathfrak{q}_2,$$

其中 $\mathfrak{p} = (2, 1 + \sqrt{-5}), \mathfrak{q}_1 = (3, 1 + \sqrt{-5}), \mathfrak{q}_2 = (3, 1 - \sqrt{-5})$。因此 $(6) = \mathfrak{p}^2 \mathfrak{q}_1 \mathfrak{q}_2$，无论从哪个元素级因子分解开始，理想的因子分解都是相同的。歧义在于如何将素理想分组为元素，而不是素结构本身。

## 从 $\mathbb{Z}$ 到 $\mathbb{Z}[x]$ 再到 $\mathbb{Q}[x]$：Gauss 引理

定义 $f \in \mathbb{Z}[x]$ 的内容为 $c(f) = \gcd$ 系数。$f$ 是*本原的*如果 $c(f) = 1$。

对于 UFD $R$ 上的多项式，内容定义为系数的 gcd（在单位意义下是良定义的）。内容扩展为一个乘法函数：$c(fg) = c(f) c(g)$（这是 Gauss 引理的内容版本）。每个非零 $f \in R[x]$ 唯一地（在单位意义下）分解为 $f = c(f) \cdot f^*$，其中 $f^*$ 是本原的。

Gauss 引理指出，本原多项式的乘积是本原的。

*证明。* 如果 $fg$ 的所有系数都能被素数 $p$ 整除，模 $p$：$\bar f \bar g = 0$ 在 $\mathbb{F}_p[x]$ 中。整环 $\Rightarrow$ $\bar f = 0$ 或 $\bar g = 0$，与本原性矛盾。$\square$

进一步可得：$f \in \mathbb{Z}[x]$ 本原，$f = gh$ 在 $\mathbb{Q}[x]$ 中 $\Rightarrow$ 存在 $g^*, h^* \in \mathbb{Z}[x]$ 使得 $f = g^* h^*$，次数相同。

可以证明：$\mathbb{Z}[x]$ 是 UFD。更一般地，$R$ 是 UFD $\Rightarrow$ $R[x]$ 是 UFD。通过归纳 $R[x_1, \ldots, x_n]$ 是 UFD。

*证明 $\mathbb{Z}[x]$ 的草图。* 使用 $\mathbb{Z}$ 是 UFD 和 $\mathbb{Q}[x]$ 是 UFD。对于非零 $f \in \mathbb{Z}[x]$，写 $f = c(f) \cdot f^*$ 其中 $f^*$ 是本原的。通过 $\mathbb{Z}$-UFD 分解 $c(f)$，通过 $\mathbb{Q}[x]$-UFD 分解 $f^*$。根据 Gauss 的推论，$f^*$ 的因子可以在 $\mathbb{Z}[x]$ 中选取且是本原的。唯一性：结合 $\mathbb{Z}$-唯一性、$\mathbb{Q}[x]$-唯一性和 Gauss 引理。

同样的论证适用于任何 UFD $R$ 代替 $\mathbb{Z}$，使用 $\text{Frac}(R)$ 代替 $\mathbb{Q}$。

Gauss 引理是整系数和有理系数因子分解之间的桥梁。没有它，回答“这个 $\mathbb{Z}[x]$-多项式是否不可约？”的问题会更难使用 $\mathbb{Q}[x]$ 工具。

第二个后果：它告诉我们“因子清除分母。”如果 $f \in \mathbb{Z}[x]$ 是首一的并在 $\mathbb{Q}$ 上分解为首一因子，那么这些因子自动在 $\mathbb{Z}[x]$ 中。这就是为什么有理根测试产生整数根而不是仅仅有理根。
