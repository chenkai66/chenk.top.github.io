每个学代数的人都会遇到这样的困境：手里的数系不够用了。古希腊人发现 $\sqrt{2}$ 是无理数，也就是说 $x^2 - 2 = 0$ 在有理数域 $\mathbb{Q}$ 中无解。怎么办？不是放弃方程，而是把域扩大。域扩张正是这种"扩大"的严格数学语言，也是 Galois 理论的全部基础设施。

本文从零开始搭建域扩张的理论框架：扩张次数与向量空间结构、单扩张与极小多项式、塔式法则及其应用、分裂域与代数闭包、可分性与完美域。掌握这些工具后，下一篇文章就可以直接进入 Galois 理论的核心定理。

---

## 动机：解多项式需要更大的域

考虑多项式 $f(x) = x^2 + 1$。在实数域 $\mathbb{R}$ 上它没有根（因为 $x^2 \geq 0$），但到了复数域 $\mathbb{C} = \mathbb{R}(i)$，它就分解为 $(x-i)(x+i)$。这种现象在代数中无处不在：

- $x^2 - 2$ 在 $\mathbb{Q}$ 中无根，到了 $\mathbb{Q}(\sqrt{2})$ 就有根 $\pm\sqrt{2}$。
- $x^3 - 2$ 在 $\mathbb{Q}$ 中无根；实根 $\sqrt[3]{2}$ 属于 $\mathbb{Q}(\sqrt[3]{2})$；要得到全部三个根还需进一步扩张到 $\mathbb{Q}(\sqrt[3]{2}, \omega)$（$\omega = e^{2\pi i/3}$ 是三次本原单位根）。
- $x^4 + 1$ 在 $\mathbb{Q}$ 上不可约（尽管它在 $\mathbb{R}$ 上可约为两个二次因子），要分裂需要八次单位根域 $\mathbb{Q}(\zeta_8)$。

模式很明确：给定一个域 $K$ 上不可完全分解的多项式，我们构造一个更大的域 $L \supseteq K$，使其在 $L$ 中完全分解。域扩张理论回答三个根本问题：这样的大域如何构造？它比基域"大多少"？最小的这样的域何时存在、是否唯一？

从历史角度看，这条线索贯穿了从文艺复兴到十九世纪的代数学发展。二次方程的求根公式人人会用，Cardano 和 Tartaglia 在十六世纪找到了三次方程的公式，Ferrari 随后解决了四次方程。五次方程却抵抗了所有尝试。理解"为什么"需要全新的视角——不是对根本身下功夫，而是研究根所在域扩张的对称性。域扩张理论因此不仅仅是技术工具，它是代数学最深层结构性结果的表达语言。

## 域扩张与扩张次数

**定义。** 若 $K \subseteq L$ 都是域（等价地，存在域的单射同态 $K \hookrightarrow L$），则称 $L/K$ 为一个**域扩张**，$K$ 为**基域**，$L$ 为**扩域**。记号 $L/K$ 不是商——它只是约定俗成地表示 $L$ 扩张了 $K$。

由于 $L$ 是包含 $K$ 的域，它天然地成为 $K$ 上的向量空间：向量加法就是 $L$ 中的加法，$K$-标量乘法就是 $L$ 中的乘法。扩张的**次数**定义为

$$[L : K] = \dim_K L,$$

即 $L$ 作为 $K$-向量空间的维数。若 $[L:K]$ 有限，称 $L/K$ 为**有限扩张**；否则为**无限扩张**。

**例 1（$\mathbb{C}/\mathbb{R}$，次数 2）。** 任何复数都可以写成 $a + bi$（$a, b \in \mathbb{R}$），而 $\{1, i\}$ 是 $\mathbb{R}$-线性无关的（$a + bi = 0$ 迫使 $a = b = 0$）。因此 $[\mathbb{C}:\mathbb{R}] = 2$。

**例 2（$\mathbb{Q}(\sqrt{2})/\mathbb{Q}$，次数 2）。** $\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$ 是一个域（乘法利用 $(\sqrt{2})^2 = 2$；逆元 $(a+b\sqrt{2})^{-1} = (a-b\sqrt{2})/(a^2-2b^2)$，分母非零因为 $\sqrt{2}$ 是无理数）。基为 $\{1, \sqrt{2}\}$，故 $[\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2$。

**例 3（$\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$，次数 3）。** 令 $\alpha = \sqrt[3]{2}$。我们断言 $[\mathbb{Q}(\alpha):\mathbb{Q}] = 3$，基为 $\{1, \alpha, \alpha^2\}$。线性无关性的验证：若 $a + b\alpha + c\alpha^2 = 0$（$a,b,c \in \mathbb{Q}$），且某系数非零，则 $\alpha$ 满足一个次数不超过 2 的有理多项式。但 $\alpha$ 的极小多项式 $x^3 - 2$ 在 $\mathbb{Q}$ 上不可约（有理根检验：$\pm 1, \pm 2$ 都不是根）。不可约三次多项式的根不可能满足二次关系，矛盾。

**例 4（$\mathbb{R}/\mathbb{Q}$，无限扩张）。** $\mathbb{R}$ 不可数，而 $\mathbb{Q}$ 上任何有限维向量空间都是可数的（可数集的有限并仍然可数）。因此 $[\mathbb{R}:\mathbb{Q}] = \infty$。更具体地，不同素数的平方根 $\sqrt{2}, \sqrt{3}, \sqrt{5}, \sqrt{7}, \ldots$ 在 $\mathbb{Q}$ 上线性无关，这已经给出了无穷维。

**代数元与超越元。** $L$ 中的元素 $\alpha$ 称为 $K$ 上的**代数元**，若存在非零多项式 $f(x) \in K[x]$ 使得 $f(\alpha) = 0$。否则 $\alpha$ 是 $K$ 上的**超越元**。

**命题。** 有限扩张一定是代数扩张。

*证明。* 设 $[L:K] = n$。对任意 $\alpha \in L$，$n+1$ 个元素 $1, \alpha, \alpha^2, \ldots, \alpha^n$ 在 $n$ 维空间中必线性相关：存在不全为零的 $a_0, \ldots, a_n \in K$，使得 $a_0 + a_1\alpha + \cdots + a_n\alpha^n = 0$。这就是多项式关系，故 $\alpha$ 是代数元。$\blacksquare$

反之不成立：所有代数数组成的域 $\overline{\mathbb{Q}}/\mathbb{Q}$ 是代数扩张，但次数无穷。

## 单扩张与极小多项式

**定义。** 给定域扩张 $L/K$ 和元素 $\alpha \in L$，**单扩张** $K(\alpha)$ 是 $L$ 中包含 $K$ 和 $\alpha$ 的最小子域。具体地：

$$K(\alpha) = \left\{ \frac{f(\alpha)}{g(\alpha)} : f, g \in K[x], \ g(\alpha) \neq 0 \right\}.$$

### 代数的情形

设 $\alpha$ 是 $K$ 上的代数元。在所有以 $\alpha$ 为根的 $K[x]$ 中的非零多项式里，取次数最小的首一多项式。

**定义。** $\alpha$ 在 $K$ 上的**极小多项式** $m_\alpha(x)$（或 $\min_K(\alpha)$）是 $K[x]$ 中以 $\alpha$ 为根、次数最小的首一多项式。

**定理（极小多项式的性质）。** 设 $\alpha$ 在 $K$ 上的极小多项式为 $m(x)$，$\deg m = n$。则：

1. $m(x)$ 在 $K$ 上不可约。
2. 若 $f(x) \in K[x]$ 满足 $f(\alpha) = 0$，则 $m(x) \mid f(x)$。
3. $K(\alpha) \cong K[x]/(m(x))$（作为 $K$-代数同构）。
4. $[K(\alpha):K] = n$，且 $\{1, \alpha, \ldots, \alpha^{n-1}\}$ 是 $K(\alpha)$ 在 $K$ 上的一组基。

*证明。*

(1) 设 $m(x) = g(x)h(x)$，$\deg g, \deg h < \deg m$。则 $0 = m(\alpha) = g(\alpha)h(\alpha)$。由于 $L$ 是域（从而是整环），$g(\alpha) = 0$ 或 $h(\alpha) = 0$。无论哪种情况，都找到了一个次数比 $m$ 小的以 $\alpha$ 为根的多项式（除以首项系数使其首一），与极小性矛盾。

(2) 做带余除法 $f(x) = q(x)m(x) + r(x)$，$\deg r < \deg m$。代入 $\alpha$：$0 = f(\alpha) = q(\alpha) \cdot 0 + r(\alpha) = r(\alpha)$。若 $r \neq 0$，则得到一个次数更小的以 $\alpha$ 为根的多项式——矛盾。故 $r = 0$。

(3) 赋值同态 $\operatorname{ev}_\alpha : K[x] \to L$，$f(x) \mapsto f(\alpha)$ 的像是 $K[\alpha]$，核是 $(m(x))$。因 $K[x]$ 是 PID 且 $m$ 不可约，$(m(x))$ 是极大理想，故 $K[x]/(m(x)) \cong K[\alpha]$ 是域。而 $K[\alpha]$ 是 $L$ 中包含 $K$ 和 $\alpha$ 的最小子域（任何这样的子域都包含所有 $\alpha$ 的多项式表达式），因此 $K[\alpha] = K(\alpha)$。

(4) 在 $K[x]/(m(x))$ 中，每个陪集唯一地由一个次数 $< n$ 的多项式代表。基为 $\{\overline{1}, \overline{x}, \ldots, \overline{x^{n-1}}\}$，对应到同构下就是 $\{1, \alpha, \ldots, \alpha^{n-1}\}$。$\blacksquare$

**关键推论。** 在代数情形下，$K(\alpha) = K[\alpha]$。也就是说，$\alpha$ 的有理表达式都可以化为多项式表达式。原因是逆元自动存在：若 $f(\alpha) \neq 0$，则 $\gcd(f, m) = 1$（因为 $m$ 不可约且 $m \nmid f$），由 Bezout 等式可找到 $s, t \in K[x]$ 使 $s(x)f(x) + t(x)m(x) = 1$，代入 $\alpha$ 得 $f(\alpha)^{-1} = s(\alpha)$。

### 计算实例：$\mathbb{Q}(\sqrt[3]{2})$ 中的运算

令 $\alpha = \sqrt[3]{2}$，极小多项式 $m(x) = x^3 - 2$。$[\mathbb{Q}(\alpha):\mathbb{Q}] = 3$，基 $\{1, \alpha, \alpha^2\}$。

**乘法。** $(1+\alpha)(2 - \alpha + \alpha^2) = 2 - \alpha + \alpha^2 + 2\alpha - \alpha^2 + \alpha^3 = 2 + \alpha + 2 = 4 + \alpha$，用了 $\alpha^3 = 2$。

**求逆。** 求 $(1+\alpha)^{-1}$：对 $1+x$ 和 $x^3-2$ 做辗转相除。

$x^3 - 2 = (x^2 - x + 1)(x + 1) + (-3)$，

所以 $-3 = (x^3-2) - (x^2-x+1)(x+1)$，即 $1 = -\frac{1}{3}(x^3-2) + \frac{1}{3}(x^2-x+1)(x+1)$。

代入 $\alpha$：$(1+\alpha)^{-1} = \frac{1}{3}(\alpha^2 - \alpha + 1)$。

验证：$\frac{1}{3}(\alpha^2 - \alpha + 1)(1 + \alpha) = \frac{1}{3}(\alpha^2 - \alpha + 1 + \alpha^3 - \alpha^2 + \alpha) = \frac{1}{3}(1 + 2) = 1$。正确。

### 超越的情形

若 $\alpha$ 是 $K$ 上的超越元，则赋值同态 $\operatorname{ev}_\alpha : K[x] \to K[\alpha]$ 是单射（没有非零多项式在 $\alpha$ 处为零），所以 $K[\alpha] \cong K[x]$ 是多项式环，不是域。此时 $K(\alpha) \cong K(x)$（有理函数域），$[K(\alpha):K] = \infty$。

**例。** $\pi$ 是 $\mathbb{Q}$ 上的超越数（Lindemann, 1882），因此 $\mathbb{Q}(\pi) \cong \mathbb{Q}(x)$，$[\mathbb{Q}(\pi):\mathbb{Q}] = \infty$。

## 塔式法则及其推论

塔式法则是扩张次数在域链中的乘法性，它是计算扩张次数最常用的工具。

**定理（塔式法则）。** 若 $K \subseteq M \subseteq L$ 且 $[M:K], [L:M]$ 都有限，则 $[L:K]$ 有限且

$$[L:K] = [L:M] \cdot [M:K].$$

*证明。* 设 $\{e_1, \ldots, e_m\}$ 是 $M/K$ 的基，$\{f_1, \ldots, f_n\}$ 是 $L/M$ 的基。我们证明 $\{e_i f_j\}_{1\leq i\leq m,\, 1\leq j\leq n}$ 是 $L/K$ 的基，从而 $[L:K] = mn$。

*张成性。* 任取 $\ell \in L$。由 $\{f_j\}$ 张成 $L/M$：$\ell = \sum_j \mu_j f_j$，$\mu_j \in M$。再由 $\{e_i\}$ 张成 $M/K$：$\mu_j = \sum_i a_{ij} e_i$，$a_{ij} \in K$。代入：$\ell = \sum_{i,j} a_{ij}(e_i f_j)$。

*线性无关。* 设 $\sum_{i,j} a_{ij}(e_i f_j) = 0$，$a_{ij} \in K$。整理为 $\sum_j (\sum_i a_{ij}e_i) f_j = 0$。每个括号 $\sum_i a_{ij}e_i \in M$，而 $\{f_j\}$ 在 $M$ 上线性无关，故每个括号为 0。再由 $\{e_i\}$ 在 $K$ 上线性无关，得 $a_{ij} = 0$。$\blacksquare$

### 应用 1：次数整除性

若 $K \subseteq M \subseteq L$ 且 $[L:K]$ 有限，则 $[M:K]$ 整除 $[L:K]$。特别地，若 $\alpha \in L$，则 $\deg \min_K(\alpha)$ 整除 $[L:K]$。

### 应用 2：$\sqrt{2}+\sqrt{3}$ 生成 $\mathbb{Q}(\sqrt{2},\sqrt{3})$

**计算扩张次数。** $[\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2$。接下来证明 $\sqrt{3} \notin \mathbb{Q}(\sqrt{2})$：若 $\sqrt{3} = a + b\sqrt{2}$（$a,b \in \mathbb{Q}$），平方得 $3 = a^2 + 2b^2 + 2ab\sqrt{2}$。由 $\sqrt{2}$ 的无理性，$2ab = 0$。若 $a=0$，则 $2b^2=3$，即 $b^2=3/2$，$b$ 无理；若 $b=0$，则 $a^2=3$，$a$ 无理。矛盾。因此 $x^2-3$ 在 $\mathbb{Q}(\sqrt{2})$ 上不可约，$[\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}(\sqrt{2})] = 2$。

塔式法则：$[\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}] = 2 \times 2 = 4$。基为 $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$。

**证明 $\mathbb{Q}(\sqrt{2}+\sqrt{3}) = \mathbb{Q}(\sqrt{2},\sqrt{3})$。** 令 $\alpha = \sqrt{2}+\sqrt{3}$。

- $\alpha^2 = 5 + 2\sqrt{6}$，故 $\sqrt{6} = (\alpha^2-5)/2 \in \mathbb{Q}(\alpha)$。
- $\alpha\sqrt{6} = \sqrt{12}+\sqrt{18} = 2\sqrt{3}+3\sqrt{2}$。
- $\alpha\sqrt{6} - 2\alpha = 3\sqrt{2}+2\sqrt{3}-2\sqrt{2}-2\sqrt{3} = \sqrt{2}$。

故 $\sqrt{2} = \alpha(\sqrt{6}-2) \in \mathbb{Q}(\alpha)$，$\sqrt{3} = \alpha - \sqrt{2} \in \mathbb{Q}(\alpha)$。于是 $\mathbb{Q}(\alpha) \supseteq \mathbb{Q}(\sqrt{2},\sqrt{3})$，反向包含显然，等号成立。

$\alpha$ 在 $\mathbb{Q}$ 上的极小多项式：$\alpha^2 - 5 = 2\sqrt{6}$，$(\alpha^2-5)^2 = 24$，即 $\alpha^4 - 10\alpha^2 + 1 = 0$。所以 $\min_\mathbb{Q}(\sqrt{2}+\sqrt{3}) = x^4 - 10x^2 + 1$。

### 应用 3：经典不可能性

尺规作图可构造的实数 $\alpha$ 必满足 $[\mathbb{Q}(\alpha):\mathbb{Q}]$ 是 2 的幂。

- **倍立方**需构造 $\sqrt[3]{2}$，但 $[\mathbb{Q}(\sqrt[3]{2}):\mathbb{Q}] = 3$，不是 2 的幂。不可能。
- **三等分 $60°$ 角**需构造 $\cos 20°$，其极小多项式 $8x^3-6x-1$ 在 $\mathbb{Q}$ 上不可约，故 $[\mathbb{Q}(\cos 20°):\mathbb{Q}] = 3$。不可能。
- **化圆为方**需构造 $\sqrt{\pi}$，而 $\pi$ 是超越数，$[\mathbb{Q}(\sqrt{\pi}):\mathbb{Q}] = \infty$。不可能。

## 分裂域与代数闭包

到目前为止，我们都是逐个添加根。但许多问题——尤其是 Galois 理论——需要一次添加一个多项式的所有根。

**定义。** 设 $f(x) \in K[x]$ 次数为 $n$。$f$ 在 $K$ 上的**分裂域**是一个扩张 $L/K$，使得：

1. $f$ 在 $L$ 中完全分解：$f(x) = c(x-\alpha_1)\cdots(x-\alpha_n)$，$\alpha_i \in L$。
2. $L = K(\alpha_1, \ldots, \alpha_n)$——即 $L$ 由根生成。

条件 (2) 保证了极小性：$L$ 是使 $f$ 完全分裂的最小域。

**定理（分裂域的存在性）。** 每个非常数多项式 $f(x) \in K[x]$ 都有分裂域。

*证明。* 对 $\deg f$ 归纳。基础步骤 $\deg f = 1$ 平凡。归纳步骤：设 $p(x)$ 是 $f$ 的不可约因子，$K_1 = K[x]/(p(x))$ 中 $f$ 有根 $\alpha_1$，$f(x) = (x-\alpha_1)g(x)$，$\deg g = \deg f - 1$。由归纳假设 $g$ 在 $K_1$ 上有分裂域 $L$，$L$ 就是 $f$ 在 $K$ 上的分裂域。$\blacksquare$

**定理（分裂域的唯一性）。** $f(x) \in K[x]$ 的任意两个分裂域作为 $K$-扩张同构。

唯一性的证明对 $[L_1:K]$ 归纳，逐步把部分同构延拓到更多根上。这个定理让我们可以说"$f$ **的**分裂域"（在同构意义下）。

### 计算实例：$x^4 - 2$ 在 $\mathbb{Q}$ 上的分裂域

$x^4 - 2$ 的根是 $\sqrt[4]{2}, \, i\sqrt[4]{2}, \, -\sqrt[4]{2}, \, -i\sqrt[4]{2}$。分裂域为

$$L = \mathbb{Q}(\sqrt[4]{2}, \, i).$$

用塔式法则：$[\mathbb{Q}(\sqrt[4]{2}):\mathbb{Q}] = 4$（Eisenstein 判别法，$p=2$），$[L:\mathbb{Q}(\sqrt[4]{2})] = 2$（$i \notin \mathbb{Q}(\sqrt[4]{2}) \subset \mathbb{R}$），故

$$[L:\mathbb{Q}] = 4 \times 2 = 8.$$

注意 $[L:\mathbb{Q}] = 8$ 而 $\deg f = 4$。一般地，分裂域的次数可以远大于多项式的次数。

### 正规扩张

分裂域自然地引出了正规性的概念。

**定义。** 代数扩张 $L/K$ 称为**正规**的，如果 $K[x]$ 中每个在 $L$ 里有一个根的不可约多项式都在 $L$ 中完全分裂。

**定理。** 有限扩张 $L/K$ 正规当且仅当 $L$ 是 $K$ 上某个多项式的分裂域。

**例。** $\mathbb{Q}(\sqrt{2})/\mathbb{Q}$ 是正规的（$x^2-2$ 的分裂域）。但 $\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}$ **不**正规：$x^3-2$ 在 $\mathbb{Q}(\sqrt[3]{2}) \subset \mathbb{R}$ 中只有一个根 $\sqrt[3]{2}$，另两个根 $\sqrt[3]{2}\omega, \sqrt[3]{2}\omega^2$ 是非实复数，不在 $\mathbb{Q}(\sqrt[3]{2})$ 中。这是关键区别：添加不可约多项式的一个根不一定得到正规扩张。

### 代数闭包

**定义。** 域 $K$ 是**代数闭**的，若 $K[x]$ 的每个非常数多项式都有根。$K$ 的**代数闭包** $\overline{K}$ 是既代数闭又是 $K$ 的代数扩张的域。

每个域都有代数闭包，且在同构意义下唯一（证明需要 Zorn 引理）。最熟悉的例子是 $\overline{\mathbb{Q}} \subset \mathbb{C}$，即所有代数数组成的域。$\mathbb{C}$ 本身则是 $\overline{\mathbb{R}}$（代数基本定理）。

## 可分性与完美域

在特征 0 中，域扩张的理论非常干净。但在正特征中会出现微妙的病理现象：不可约多项式可能有重根。可分性的概念正是用来识别和排除这种病理的。

**定义。** 多项式 $f(x) \in K[x]$ 称为**可分**的，如果它在分裂域中没有重根。代数元 $\alpha$ 称为可分的，如果 $\min_K(\alpha)$ 是可分多项式。代数扩张 $L/K$ 称为可分的，如果 $L$ 中每个元素都是 $K$ 上的可分元。

**命题（导数判别法）。** $f(x)$ 有重根当且仅当 $\gcd(f, f') \neq 1$（$f'$ 是形式导数）。

*证明。* 若 $\alpha$ 是重根，写 $f(x) = (x-\alpha)^2 g(x)$。则 $f'(\alpha) = 0$，故 $(x-\alpha) \mid \gcd(f, f')$。反之，若 $\alpha$ 是单根，写 $f(x) = (x-\alpha)h(x)$，$h(\alpha) \neq 0$，则 $f'(\alpha) = h(\alpha) \neq 0$，故 $\alpha$ 不是 $f'$ 的根。$\blacksquare$

**推论。** 不可约多项式 $p(x)$ 不可分当且仅当 $p'(x) = 0$。

在特征 0 中，$p'(x) = 0$ 意味着 $p$ 是常数——矛盾。因此**特征 0 的不可约多项式一定可分**。这就是为什么在只涉及 $\mathbb{Q}$ 或 $\mathbb{R}$ 的课程中很少讨论可分性。

在特征 $p > 0$ 中，$p'(x) = 0$ 意味着 $p(x)$ 是 $x^p$ 的多项式：$p(x) = a_0 + a_1 x^p + a_2 x^{2p} + \cdots$。

**定义。** 域 $K$ 称为**完美**的，如果 $K$ 上每个不可约多项式都可分。

- 特征 0 的域自动完美。
- 有限域 $\mathbb{F}_q$ 是完美的（Frobenius $a \mapsto a^p$ 在有限集上单射即满射）。
- $\mathbb{F}_p(t)$（$\mathbb{F}_p$ 上的有理函数域）**不**完美：$x^p - t$ 不可约（对 $\mathbb{F}_p[t]$ 中的素元 $t$ 用 Eisenstein），但在分裂域中 $x^p - t = (x - t^{1/p})^p$，只有一个根且重数为 $p$。

**可分性为什么对 Galois 理论重要。** 有限扩张 $L/K$ 是 **Galois 扩张**当且仅当它既正规又可分。对 Galois 扩张，$|\operatorname{Gal}(L/K)| = [L:K]$——Galois 群的阶等于扩张次数。若失去可分性，$|\operatorname{Aut}_K(L)| < [L:K]$，Galois 对应就会失败。好消息是，对完美域（包括 $\mathbb{Q}$ 和所有有限域）的扩张，可分性自动满足，只需检查正规性。
