## 域：乘法也有逆的环

**域** (Field) 是交换含幺环，且每个非零元素都有乘法逆元。等价地说，域恰好有两个理想：$(0)$ 和自身。能除以任何非零元素，就没有空间容纳有趣的理想结构——域是环论意义下最"简单"的对象。

$$
F \text{ 是域} \iff \forall\, a \neq 0,\; \exists\, a^{-1} : aa^{-1} = 1
$$

常见的域：$\mathbb{Q}$, $\mathbb{R}$, $\mathbb{C}$, 有限域 $\mathbb{F}_p = \mathbb{Z}/p\mathbb{Z}$。

$\mathbb{Z}$ 不是域（2 没有乘法逆）。域是环论的"原子"——不能再做非平凡商。每个整环都能嵌入它的分式域（如 $\mathbb{Z} \hookrightarrow \mathbb{Q}$）。

## 域扩张

**域扩张** $K/F$：$F$ 是 $K$ 的子域。此时 $K$ 天然是 $F$ 上的向量空间。维数 $[K:F] = \dim_F K$ 叫**扩张次数**。

$$
[K:F] = \dim_F K
$$

**塔公式：** $F \subseteq L \subseteq K$ 时 $[K:F] = [K:L] \cdot [L:F]$。

*证明。* $L/F$ 的基 $\{v_1, \ldots, v_m\}$，$K/L$ 的基 $\{w_1, \ldots, w_n\}$，则 $\{v_iw_j\}$ 是 $K/F$ 的基（$mn$ 个元素）。$\square$

塔公式的威力：若 $[K:F]$ 是素数，则 $F$ 和 $K$ 之间没有中间域。

![域扩张塔图](https://blog-pic-ck.oss-cn-beijing.aliyuncs.com/posts/zh/abstract-algebra/04-%E5%9F%9F%E4%B8%8E%E6%89%A9%E5%BC%A0/fig_concept.png)

### 例 1：$\mathbb{Q}(\sqrt{2})$

$\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$，2 次扩张，基 $\{1, \sqrt{2}\}$。

验证是域：$a + b\sqrt{2} \neq 0$ 时逆元为 $\frac{a - b\sqrt{2}}{a^2 - 2b^2}$。分母非零因为 $\sqrt{2}$ 无理。

### 例 2：4 次扩张

$\mathbb{Q}(\sqrt{2}, \sqrt{3})/\mathbb{Q}$ 次数为 4，基 $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$。用塔公式：$[\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2$，$[\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}(\sqrt{2})] = 2$（$\sqrt{3} \notin \mathbb{Q}(\sqrt{2})$ 需要验证——如果 $\sqrt{3} = a + b\sqrt{2}$，平方得 $3 = a^2 + 2b^2 + 2ab\sqrt{2}$，迫使 $ab = 0$，然后 $a^2 = 3$ 或 $2b^2 = 3$，在 $\mathbb{Q}$ 中都无解）。

## 代数元与超越元

$\alpha \in K$ 是 $F$ 上的**代数元**，如果它满足某个非零 $F$ 系数多项式。否则叫**超越元**。

$\pi$ 和 $e$ 是 $\mathbb{Q}$ 上的超越元。$\sqrt{2}$ 是代数元，满足 $x^2 - 2 = 0$。

代数元 $\alpha$ 有唯一的首一不可约**极小多项式** $m_\alpha(x) \in F[x]$。

**定理。** 若 $\alpha$ 的极小多项式次数为 $n$，则：

$$
F(\alpha) \cong F[x]/(m_\alpha(x)), \quad [F(\alpha):F] = n
$$

基为 $\{1, \alpha, \alpha^2, \ldots, \alpha^{n-1}\}$。

*证明。* 代入映射 $\text{ev}_\alpha: F[x] \to K$ 的核是 $(m_\alpha)$（不可约 = 极大理想），所以 $F[x]/(m_\alpha)$ 是域，同构于 $F(\alpha)$。$\square$

这个定理是生产域扩张的引擎：取一个不可约多项式，做商，就添加了它的根。

### 例 3：$\mathbb{Q}(\sqrt[3]{2})$

极小多项式 $x^3 - 2$（Eisenstein, $p = 2$），次数 3，基 $\{1, \sqrt[3]{2}, \sqrt[3]{4}\}$。乘法规则：$\beta = \sqrt[3]{2}$，$\beta^3 = 2$ 用来化简高次项。

## 分裂域

多项式 $f \in F[x]$ 在 $K$ 上**分裂**：$f$ 在 $K[x]$ 中完全分解为一次因式。$f$ 的**分裂域**是使 $f$ 分裂的最小扩域。

$$
\text{Split}_F(f) = F(\alpha_1, \ldots, \alpha_n), \quad f(x) = c\prod(x - \alpha_i)
$$

**定理。** 分裂域存在且同构意义下唯一。

### 例 4：$x^3 - 2$ 的分裂域

3 个根：$\sqrt[3]{2}$，$\omega\sqrt[3]{2}$，$\omega^2\sqrt[3]{2}$（$\omega = e^{2\pi i/3}$，本原 3 次单位根）。分裂域 $\mathbb{Q}(\sqrt[3]{2}, \omega)$，次数 6。

计算：$[\mathbb{Q}(\sqrt[3]{2}):\mathbb{Q}] = 3$，$\omega$ 满足 $x^2 + x + 1$（$\mathbb{Q}$ 上不可约），$\gcd(2,3) = 1$ 保证总次数 $= 3 \times 2 = 6$。这个"6"正是 Galois 群 $S_3$ 的阶。

## 有限域

**定理。** 每个素数幂 $q = p^n$ 恰好对应一个（同构意义下）$q$ 元域 $\mathbb{F}_q$。不存在其他大小的有限域。

构造：$\mathbb{F}_{p^n}$ 是 $x^{p^n} - x$ 在 $\mathbb{F}_p$ 上的分裂域。或取 $\mathbb{F}_p[x]$ 中任一 $n$ 次不可约多项式 $f$，做商 $\mathbb{F}_p[x]/(f)$。

### 例 5：$\mathbb{F}_4$

$\mathbb{F}_2[x]/(x^2 + x + 1)$，设 $\alpha$ 是 $x$ 的像。$\mathbb{F}_4 = \{0, 1, \alpha, \alpha+1\}$，$\alpha^2 = \alpha + 1$。

乘法群 $\mathbb{F}_4^*$ 是 3 阶循环群：$\alpha^3 = \alpha(\alpha+1) = \alpha^2 + \alpha = 1$。每个有限域的乘法群都是循环的——这对编码和密码学至关重要。

### 例 6：$\mathbb{F}_8$

$\mathbb{F}_2[x]/(x^3 + x + 1)$。$\alpha^3 = \alpha + 1$，$\mathbb{F}_8$ 有 8 个元素，乘法群 $\mathbb{F}_8^*$ 是 7 阶循环群。$\alpha$ 是它的生成元。

## 代数闭包

域 $F$ **代数闭**：$F[x]$ 中每个非常数多项式都有根。$F$ 的**代数闭包** $\overline{F}$ 是包含 $F$ 的最小代数闭域。

$\mathbb{C}$ 是 $\mathbb{R}$ 的代数闭包（代数基本定理）。$\overline{\mathbb{Q}}$——代数数——是 $\mathbb{C}$ 的可数子域。代数闭包的存在需要 Zorn 引理。

## 可分性

多项式**可分** (Separable)：在分裂域中无重根。代数扩张 $K/F$ 可分，如果每个元素的极小多项式都可分。

特征 0 中（$\mathbb{Q}$, $\mathbb{R}$）每个不可约多项式自动可分——不需要担心。特征 $p$ 中可能出问题：$x^p - t \in \mathbb{F}_p(t)[x]$ 不可约但只有一个根（重数 $p$）。

可分性是 Galois 理论基本定理的前提。Galois 扩张 = 正规 + 可分。
