主板：硬件大舞台

¶主板的本质

定义：主板（Motherboard）是计算机的"骨架"和"神经网络"，连接并协调所有硬件组件。

生活化比喻：

  • 主板 = 城市的交通网络（道路、桥梁）
  • CPU = 市政府（决策中心）
  • 内存 = 快递站（临时中转）
  • 硬盘 = 仓库（永久存储）
  • 显卡 = 电视台（图形处理）

所有部件通过主板上的"道路"（总线）交换数据。

¶PCIe 插槽：扩展之王

PCIe = Peripheral Component Interconnect Express

¶PCIe 代数与带宽

代数 单通道带宽 x1 x4 x16 发布年份
PCIe 2.0 500 MB/s 500 MB/s 2 GB/s 8 GB/s 2007
PCIe 3.0 1 GB/s 1 GB/s 4 GB/s 16 GB/s 2010
PCIe 4.0 2 GB/s 2 GB/s 8 GB/s 32 GB/s 2017
PCIe 5.0 4 GB/s 4 GB/s 16 GB/s 64 GB/s 2022

x1/x4/x16 的含义：

  • x1 = 1 条数据通道（最窄）
  • x4 = 4 条数据通道
  • x16 = 16 条数据通道（最宽）

生活化比喻：

  • x1 = 单车道小路
  • x4 = 4 车道公路
  • x16 = 16 车道高速公路

¶常见设备使用的 PCIe 通道数

设备 通道数 带宽需求 插槽类型
显卡（RTX 4090） x16 32 GB/s PCIe x16 物理插槽
NVMe SSD x4 8 GB/s M.2 或 PCIe x4
高端网卡（10GbE） x4 1.25 GB/s PCIe x4/x8
声卡 x1 100 MB/s PCIe x1
USB 扩展卡 x1 500 MB/s PCIe x1

¶集成显卡 vs 独立显卡

¶集成显卡（iGPU）

定义：集成在 CPU 内部的显卡。

代表：

  • Intel UHD Graphics 770（i5-13400）
  • AMD Radeon Graphics（Ryzen 5 7600）

优点：

  • ✅ 不占预算（CPU 自带）
  • ✅ 功耗低（15-25W）
  • ✅ 办公够用

缺点：

  • ❌ 性能弱（无法玩 3A 大作）
  • ❌ 占用系统内存（共享内存，如 512MB-2GB）

适合场景：

  • 办公（Word/Excel/PPT）
  • 看视频（1080p/4K 流畅）
  • 轻度游戏（LOL 低画质）

¶独立显卡（dGPU）

定义：独立的图形处理芯片 + 显存。

代表：

  • NVIDIA RTX 系列（RTX 4060/4070/4080/4090）
  • AMD RX 系列（RX 7600/7700/7800/7900）

优点：

  • ✅ 性能强（3A 大作高画质）
  • ✅ 独立显存（不占系统内存）
  • ✅ 专业功能（光追、DLSS、CUDA）

缺点：

  • ❌ 价格贵（¥2000-¥15000）
  • ❌ 功耗高（150-450W）
  • ❌ 体积大（占用 2-3 个插槽位）

主板供电（VRM）深度解析

¶VRM 是什么？

VRM = Voltage Regulator Module（电压调节模块）

作用：将 12V 电源转换为 CPU 需要的 1.0-1.4V

生活化比喻：

  • VRM = 变压器（把 220V 市电转成 5V 手机充电器）
  • 相数越多 = 变压器越多（电力越稳定）

¶VRM 相数对照表

CPU 功耗（TDP） 推荐 VRM 相数 主板定位 价格
< 65W (i5-13400) 6+2 相 入门 B760 ¥800
125W (i5-13600K) 10+2 相 中端 B760 ¥1000
150W (i7-13700K) 14+2 相 高端 Z790 ¥1500
250W+ (i9-13900K 超频) 18+2 相以上 旗舰 Z790 ¥2000+

危险示例：

❌ i9-13900K 超频（300W）+ 入门 B760（8 相）

结果：VRM 过热 → 降频 → 甚至烧毁主板！

正确搭配：

✅ i9-13900K + 旗舰 Z790（18 相）

¶HDMI vs DP 详细对比

特性 HDMI 2.0 HDMI 2.1 DP 1.4 DP 2.0
最大分辨率 4K@60Hz 8K@60Hz / 4K@120Hz 8K@60Hz 16K@60Hz
最大刷新率 1080p@144Hz 4K@144Hz 1440p@240Hz 4K@240Hz

音频回传（ARC） ✅ 支持 ✅ eARC ❌ 不支持 ❌ 不支持
应用场景 电视、投影仪 次世代游戏主机 游戏显示器 未来标准

¶选购建议

场景 1：连接 4K 电视

  • 推荐：HDMI 2.1 ✅
  • 理由：支持 ARC 音频回传，电视音响可用

场景 2：连接游戏显示器（1440p 165Hz）

  • 推荐：DP 1.4 ✅
  • 理由：高刷新率支持更好

场景 3：连接办公显示器（1080p 60Hz）

  • 推荐：HDMI 2.0 或 DP 1.2 ✅
  • 理由：都够用，选便宜的线

场景 4：多显示器（3 屏）

  • 推荐：1×DP + 2×HDMI
  • 理由：充分利用显卡所有接口
